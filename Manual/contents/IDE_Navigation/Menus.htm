Menu

Górną część strony GameMaker IDE zajmuje Pasek Menu. Znajduje się tu wiele rozwijanych menu, których można używać do nawigacji IDE , wykonywania zadań lub otwierania dodatkowych okien z narzędziami pomocnymi w tworzeniu gier. Wymieniamy tu domyślne menu, które są dostępne, ale niektóre edytory asset dodają dodatkowe menu w zależności od kontekstu. Te dodatkowe są wyjaśnione na stronach odpowiednich dla edytora, który je stworzył.

Dostępne są następujące menu domyślne:

  • Menu Plik
  • Menu Edycja
  • Menu budowania
  • Menu Windows
  • Menu Narzędzia
  • Menu Marketplace
  • Menu Układy
  • Menu Pomoc
