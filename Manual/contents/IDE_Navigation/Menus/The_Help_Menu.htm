Menu Pomoc

To menu związane jest z uzyskaniem pomocy podczas korzystania z GameMakera. Zawiera ono następujące pozycje menu:

  • Open Manual - Otwiera podręcznik pomocy. Zauważ, że kliknięcie przycisku pomocy na pasku menu również otworzy dokument, podobnie jak naciśnięcie przycisku .
  • Baza Wiedzy - To przeniesie Cię do Bazy Wiedzy GameMakera, gdzie znajdziesz artykuły na temat konfiguracji platformy, znanych problemów, używania GameMakera i wielu innych rzeczy.
  • Zgłoś błąd - Jeśli znajdziesz jakieś błędy lub problemy podczas korzystania z GameMakera możesz je zgłosić tutaj.
  • Open Log In Explorer - Otwiera katalog w systemie, w którym można znaleźć plik ui.log, który jest zwykle potrzebny do zgłaszania błędów związanych z IDE.
  • GameMaker.io - Otwiera stronę internetową programu GameMaker.
  • GameMaker Community - To otworzy forum GameMaker Community w Twojej przeglądarce. Zauważ, że jest to forum prowadzone przez społeczność , więc jeśli masz zgłoszenie błędu lub potrzebujesz pomocy technicznej powinieneś najpierw udać się na Oficjalną Stronę Wsparcia (patrz wyżej).
  • Release Notes - To otworzy Release Notes dla GameMakera w Twojej przeglądarce. W przypadku konieczności cofnięcia aktualizacji, możesz uzyskać instalatory poprzednich wersji klikając na link z numerem wersji na tej stronie.
  • Aktualne Runtime Release Notes - Ponieważ Runtime może być aktualizowany niezależnie od IDE, posiada własny zestaw Runtime Release Notes. Wybranie tej opcji spowoduje otwarcie ich w przeglądarce, ale należy pamiętać, że najnowszy runtime może nie być tym, którego używasz. Aby dowiedzieć się, na jakim runtime się znajdujesz, lub aby uzyskać informacje o tym, jak dodać, usunąć lub aktywować runtimes, zobacz sekcję Preferencje Runtime.
  • Wymagane SDK - Ze względu na wieloplatformową naturę GameMakera, wymagane jest wiele pakietów SDK innych firm. Kliknięcie tego przeniesie Cię do artykułu pomocy technicznej, który wyszczególnia co jest potrzebne dla każdej platformy.
  • Otwórz projekt w eksploratorze - spowoduje to otwarcie lokalizacji zapisu aktualnie załadowanego projektu w eksploratorze plików systemu operacyjnego.
  • Refresh System Fonts -Odświeży to system fonts, który GameMaker jest w stanie zobaczyć, i jest przydatne do wykrywania nowych fonts, które zostały zainstalowane, gdy IDE był otwarty.
  • Licences - Otwiera to okno zawierającelicencje wszystkich bibliotek firm trzecich używanych w GameMakerze.
  • About - Otwiera to okno "About GameMaker", w którym możesz uzyskać informacje o wersji produktu (niezbędne do zgłaszania wszelkich błędów lub problemów) oraz zobaczyć napisy.
