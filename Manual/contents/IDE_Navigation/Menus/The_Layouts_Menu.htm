Menu Układy

To menu jest związane z zapisywaniem i ładowaniem różnych układów strony IDE. Stronę IDE można skonfigurować w dowolny sposób, przy czym niektóre elementy (takie jak Przeglądarka Asset, lub Output Windows) są rozmieszczone inaczej niż domyślnie. Dzięki temu możesz dostosować GameMakera do swojego sposobu pracy, a dzięki opcjom w tym menu możesz zapisać układy bieżącego projektu jako plik *.xml, który następnie można wczytać do dowolnego projektu, aby ustawić takie same układy. W tym menu dostępne są następujące opcje:

  • Wczytaj układ - spowoduje to otwarcie eksploratora plików, w którym można wybrać do wczytania zapisany wcześniej plik układu *.xml.
  • Zapisz układ - spowoduje to otwarcie eksploratora plików, w którym można nadać nazwę i zapisać bieżący układ strony IDE jako plik *.xml.
  • Resetuj układ - powoduje przywrócenie domyślnych ustawień bieżącego układu strony IDE.
