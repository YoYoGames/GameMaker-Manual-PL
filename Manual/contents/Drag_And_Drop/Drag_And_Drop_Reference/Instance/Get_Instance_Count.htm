Uzyskaj liczbę instancji

Za pomocą tej czynności można znaleźć liczbę instancji dowolnego adresu object aktualnie aktywnych w ramach witryny room. Można wybrać dowolny adres object z listy zasobów lub użyć specjalnego słowa kluczowego all aby uzyskać liczbę wszystkich aktywnych instancji. Musisz także podać zmienną docelową, do której zwrócisz wartość, aby można ją było wykorzystać w dalszych działaniach. Jeśli na stronie flag zmienną docelową będzie "Temp", to zostanie utworzona nowa zmienna tymczasowa (lokalną) zmienną, która przechowa dla Ciebie zwróconą wartość, jak pokazano na poniższym przykładzie.

Składnia działania:

Argumenty:

Argument Opis
Object Nazwa strony object, której obecność należy sprawdzić.
Target Zmienna, której ma dotyczyć zwracana wartość.

Przykład:

Powyższy kod bloku akcji tworzy zmienną lokalną (tymczasową), a następnie przypisuje jej liczbę instancji object "obj_Squirrel". Wartość ta jest następnie sprawdzana i instancja jest niszczona, jeśli liczba instancji jest większa niż 100.
