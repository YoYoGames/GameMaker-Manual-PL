Ustaw Sprite

Ten blok akcji ustawia stronę sprite_index instancji na inną stronę sprite z przeglądarki Asset. Wybierasz stronę sprite, na którą ma zostać zmieniona, a następnie podajesz obrazek (klatkę animacji), który ma zostać wyświetlony. Zwróć uwagę, że jeśli nie chcesz, by sprite animował się po ustawieniu ramki, musisz użyć akcji Set Animation Speed i ustawić ją na 0 lub jeśli chcesz, by sprite kontynuowała animację, ustaw wartość tak, by używała wbudowanej zmiennej image_index zamiast.

Zmiana adresu sprite spowoduje także zmianę maski kolizji instancji, chyba że w Edytorze obiektów podano także osobny adres mask_index. Zwróć uwagę, że aby zmiany w tej akcji będą widoczne, instancja powinna albo nie mieć zdarzenia rysowania (a więc GameMaker domyślnie narysuje sprite), albo być rysowana za pomocą akcji Draw Self.

Składnia działania:

Argumenty:

Argument Opis
Sprite Nowy adres sprite, który ma zostać użyty (-1, jeśli chcesz usunąć adres sprite).
Frame Klatka animacji, która ma być wyświetlana początkowo (jeśli nie są używane żadne dodatkowe klatki, należy użyć wartości 0).

Przykład:

Powyższy kod bloku akcji ustawia nową stronę sprite oraz szereg innych właściwości dotyczących sposobu wyświetlania tej strony sprite.
