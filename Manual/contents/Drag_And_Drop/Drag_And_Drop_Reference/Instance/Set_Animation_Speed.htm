Ustaw szybkość animacji

Ten blok akcji ustawia image_speed (szybkość animacji) dla strony sprite przypisanej do instancji. Idealną liczbę klatek na sekundę ustawiasz w Sprite Editorze, a następnie możesz użyć tej wartości aby ją zmodyfikować. Domyślna prędkość animacji jest ustawiona na 1, co oznacza, że gra będzie próbowała utrzymać liczbę klatek na sekundę, którą zdefiniowałeś dla sprite, ale możesz użyć wartości większych niż 1, aby animować szybciej (klatki mogą być pomijane), lub wartości dziesiętnych, aby animować wolniej (klatki będą wyświetlane w różnych krokach), a nawet wartości ujemnych, aby odwrócić szybkość animacji.

Składnia działania:

Argumenty:

Argument Opis
Speed Nowa wartość prędkości modyfikatora.

Przykład:

Powyższy kod bloku akcji ustawia nową stronę sprite oraz szereg innych właściwości dotyczące sposobu wyświetlania tej strony sprite, w tym ustawienie jej animacji z połową zdefiniowanej prędkości.
