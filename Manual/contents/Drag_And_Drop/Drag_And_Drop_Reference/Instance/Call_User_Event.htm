Wywołanie zdarzenia użytkownika

Za pomocą tej akcji można wywołać dowolne z 12 dostępnych Innych - zdarzeń użytkownika. Zdarzenie użytkownika to takie, do którego można dodać kod scripts lub kod akcji, a następnie wywołać je w dowolnym momencie za pomocą tej akcji. akcji, dzięki czemu są to zasadniczo niestandardowe zdarzenia object.

Składnia działania:

Argumenty:

Argument Opis
Event Numer zdarzenia użytkownika, z którym ma zostać nawiązane połączenie (0-11).

Przykład:

Powyższy kod bloku akcji sprawdza zmienną, a następnie wywołuje zdarzenie użytkownika [0] lub zdarzenie użytkownika [1], w zależności od tego, czy wynik sprawdzenia będzie miał wartość true czy false.
