Change Object Instance

Za pomocą tej akcji możesz zmienić jedną instancję w inną. Musisz podać adres object, na który ma zostać zamieniona instancja, a akcja zmieni instancję, wykonując zdarzenia Destroy Event i Clean Up Event dla początkowej instancji oraz Create Event dla nowej instancji. początkowej instancji oraz zdarzenie Create dla nowej instancji. Zauważ, że jeśli nie określisz kierunku, prędkości itd... to nowa instancja "odziedziczy" te wartości. Na przykład, jeśli początkowa instancja ma image_angle (obrót) 180, porusza się z prędkością 2px na tick gry i ma image_speed (prędkość animacji) 0.5, to instancja, w którą się zmieni, zachowa te wartości, chyba że zostaną one jawnie ustawione w zdarzeniu Create event nowej instancji.

Warto zauważyć, że zmiana instancji oznacza, że nie możesz wykonywać żadnych dalszych działań z tą instancją aż do następnego tiku gry, w szczególności próbować uzyskać dostęp do zmiennych itp. Zasadniczo Zasadniczo zmieniasz instancję, ale nie jest ona faktycznie dostępna do końca bieżącego kroku, więc bezpośredni dostęp do zmiennych w niej zawartych (na przykład wywołanie inst_changed.x) nie zadziała.
