If Key Down (Rollback)

Ta akcja sprawdza, czy dane wejście jest trzymane dla bieżącego gracza.

Pozwala na dołączenie po prawej stronie akcji, które uruchamiają się dopiero po znalezieniu danego wejścia.

Action Syntax:

Argumenty:

Argument Opis
Input Nazwa wejścia do sprawdzenia
Not Włącz, aby sprawdzić, czy wejście nie jest trzymane

Przykład:

Powyższe akcje przesuwają gracza w lewo lub w prawo w zależności od wejścia.

Wstecz: Rollback Akcje wieloosobowe
Następny: If Key Pressed (Rollback)
