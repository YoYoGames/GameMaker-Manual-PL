Cofanie akcji dla wielu graczy

Akcje Rollback służą do tworzenia gier wieloosobowych z wykorzystaniem systemu Rollback w GameMakerze.

Dla wprowadzenia zobacz: Rollback Multiplayer

Aby uzyskać listę zmiennych używanych w systemie Rollback, zobacz: Rollback Variables

Następujące akcje są przewidziane dla Rollback multiplayer:

  • Tworzenie gry
  • Dołącz do gry
  • Rozpocznij grę
  • Opuścić grę
  • Określenie wejścia
  • Określanie opóźnienia ramki wejściowej
  • Zdefiniuj wejście pozorne
  • Określenie Gracz
  • Wyświetlanie zdarzeń
  • Pobierz informacje
  • Get Input
  • If Key Down (Rollback)
  • W przypadku naciśnięcia klawisza (Rollback)
  • If Key Up (Rollback)
  • Jeśli zadziałało zdarzenie Rollback
  • Synchronizacja na klatce
  • Użyj losowego wejścia
  • Użyj ręcznego startu
  • Wyślij wiadomość
