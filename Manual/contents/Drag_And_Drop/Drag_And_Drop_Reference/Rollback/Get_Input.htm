Get Input (Rollback)

Ta akcja zwraca adres struct zawierający wartości wejściowe dla gracza. Przyjmuje opcjonalny argument określający ID gracza, dla którego zostaną zwrócone dane wejściowe.

Zaleca się używanie If Key Down (Rollback), If Key Pressed (Roll back) i If Key Up ( Rollback) do sprawdzania wejść w multiplayerze.

Kiedy jest używany w instancji gracza, która została utworzona przez Define Player (Rollback), nie musisz określać ID gracza, więc możesz po prostu podać player_id.

Kiedy używasz ich w innych instancjach lub instancjach graczy, które zostały utworzone ręcznie, musisz określić ID gracza.

Informacje o tym, jakie dane wejściowe są zwracane i jak można je zmienić, można znaleźć w części Definiowanie danych wejściowych.

Action Syntax:

Argumenty:

Argument Opis
Player ID ID gracza, który ma otrzymać dane wejściowe
Target Zmienna, w której ma być zapisany zwrócony struct
