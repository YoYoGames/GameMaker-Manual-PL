If Key Up (Rollback)

Ta akcja sprawdza, czy dane wejście zostało właśnie zwolnione dla bieżącego gracza.

Pozwala na dołączenie po prawej stronie akcji, które uruchamiają się dopiero po znalezieniu danego wejścia.

Action Syntax:

Argumenty:

Argument Opis
Input Nazwa wejścia do sprawdzenia
Not Włącz to, aby sprawdzić, czy wejście nie zostało zwolnione

Wstecz: Rollback Akcje wieloosobowe
Dalej: Jeśli zdarzenie Rollback zadziałało
