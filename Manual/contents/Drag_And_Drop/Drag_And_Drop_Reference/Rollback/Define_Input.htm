Określenie wejścia (Rollback)

Funkcja służy do definiowania własnych wejść dla wszystkich graczy. Przyjmuje parę Wejście/Wartość, gdzie Wejście jest nazwą wejścia, a Wartość jest przypisaną mu stałą wejściową.

Aby dowiedzieć się, jak przypisać wejścia niestandardowe, zobacz sekcję Sterowanie niestandardowe.

Jeśli jest używana, funkcja ta musi być uruchomiona przed akcją join/create.

Action Syntax:

Argumenty:

Argument Opis
Input Nazwa wejścia do zdefiniowania
Value Stała wejściowa do przypisania (patrz Sterowanie własne)

Przykład:

Powyższa akcja definiuje wejścia fire, left, right, up i down dla wszystkich graczy.
