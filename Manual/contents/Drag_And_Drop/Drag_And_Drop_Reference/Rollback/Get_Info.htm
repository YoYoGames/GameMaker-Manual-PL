Pobierz informacje o cofaniu

Funkcja zwraca adres struct zawierający informacje o graczu. Powinna być używana tylko podczas ręcznego zarządzania instancjami graczy. Jeśli używasz Define Player, to każda instancja gracza automatycznie otrzyma te informacje jako zmienne.

Informacje o zawartości zwróconego struct, patrz: rollback_get_info

Action Syntax:

Argumenty:

Argument Opis
Player ID ID gracza, dla którego chcesz uzyskać informacje
Target Zmienna, w której ma być zapisany zwrócony struct

Wstecz: Rollback Akcje wieloosobowe
Dalej: Get Input
