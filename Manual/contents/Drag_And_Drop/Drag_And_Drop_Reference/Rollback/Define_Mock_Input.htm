Określenie wejścia pozorowanego (Rollback)

Tej akcji można użyć podczas testu synchron izacji, aby przypisać tymczasowe "wyśmiewane" wejście do określonego odtwarzacza. Zastępuje to elementy sterujące zdefiniowane dla gracza oraz wszelkie losowe dane wejściowe z testu synchronizacji.

Action Syntax:

Argumenty:

Argument Opis
Player ID Numer gracza, który otrzyma ten zestaw wejściowy.
Input Nazwa wejścia do zdefiniowania
Value Stała wejściowa do przypisania (patrz Sterowanie własne)

Przykład:

Powyższa akcja definiuje wejścia kpiące fire, left, right, up i down dla gracza 1.
