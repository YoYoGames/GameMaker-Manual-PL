Rozpocznij grę "Rollback

Ta akcja wymusza rozpoczęcie bieżącej gry wieloosobowej, zanim wszyscy gracze do niej dołączą. Musisz najpierw stworzyć grę wieloosobową używając opcji Create Rollback Game.

System normalnie czeka, aż wszyscy gracze dołączą, zanim automatycznie rozpocznie grę. Ta akcja może być użyta do rozpoczęcia gry wcześniej, zanim wszyscy gracze dołączą.

Możesz zadzwonić do Use Manual Start, aby Rollback czekał na to połączenie nawet po dołączeniu wszystkich graczy.

Uruchomienie tej akcji spowoduje uruchomienie zdarzenia Rollback Start.

Action Syntax:

Wstecz: Rollback Akcje wieloosobowe
Następny: Leave Rollback Game
