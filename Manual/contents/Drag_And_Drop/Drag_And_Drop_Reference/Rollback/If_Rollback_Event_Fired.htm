Jeśli zadziałało zdarzenie Rollback

Ta akcja jest używana w zdarzeniu Rollback Event, aby sprawdzić, jaki rodzaj zdarzenia został odpalony.

Pozwala na dołączenie akcji po prawej stronie, które uruchamiają się tylko wtedy, gdy dane zdarzenie zostało odpalone.

Action Syntax:

Argumenty:

Argument Opis
Event Zdarzenie Rollback do sprawdzenia
Not Włącz to, aby sprawdzić, czy dane zdarzenie nie zostało odpalone (co oznacza, że warunek będzie prawdziwy dla wszystkich innych odpalonych zdarzeń)

Wstecz: Rollback Akcje wieloosobowe
Dalej: Sync On Frame
