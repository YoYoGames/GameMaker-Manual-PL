Użyj ręcznego startu (Rollback)

Domyślnie gra wieloosobowa rozpoczyna się automatycznie po dołączeniu wszystkich graczy.

Wywołanie tej akcji wyłącza to zachowanie automatycznego uruchamiania, więc gra czeka, aż uruchomisz ją ręcznie.

Jeśli zostanie użyta, ta akcja musi zostać uruchomiona przed akcją join/create.

Action Syntax:

Przykład:

Powyższy blok umożliwia ręczne uruchomienie, a następnie dołączenie lub utworzenie nowej gry wieloosobowej.
