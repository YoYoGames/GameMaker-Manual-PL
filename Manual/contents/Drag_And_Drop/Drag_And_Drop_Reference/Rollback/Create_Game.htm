Utwórz grę typu Rollback

Ta akcja hostuje nową grę wieloosobową. Pobiera liczbę graczy, którzy muszą być połączeni, aby gra się rozpoczęła, i opcjonalnie pozwala wyłączyć Test Sync.

W przypadku wywołania z wyłączoną funkcją Sync Test, system czeka na dołączenie określonej liczby graczy przed rozpoczęciem gry. Upewnij się, że nie tworzysz ani nie modyfikujesz żadnych zarządzanych instancji do czasu rozpoczęcia gry.

WSKAZÓWKA Możesz opcjonalnie wymusić rozpoczęcie gry przed dołączeniem wszystkich graczy, uruchamiając Start Game.

Obecnie w jednej grze można mieć tylko do 4 graczy.

OSTRZEŻENIE Nie zaleca się ustawiania domyślnego regionu w tej czynności, chyba że pozwalasz graczowi wybrać go poprzez menu.

Action Syntax:

Argumenty:

Argument Opis
Count of players Liczba graczy, którzy muszą być podłączeni, aby gra się rozpoczęła. Jeśli używany jest Define Player, jest to liczba instancji, które są automatycznie tworzone.
Sync test Określa, czy test synchronizacji powinien być włączony, i jest domyślnie włączony. Gdy jest wyłączony, włączona jest funkcjonalność online.
Region Określa region, w którym zostanie utworzona wirtualna strona room. Aby uzyskać informacje na temat uzyskiwania regionów, zobacz rollback_create_game().

Wstecz: Rollback Akcje wieloosobowe
Następny: Dołącz do gry
