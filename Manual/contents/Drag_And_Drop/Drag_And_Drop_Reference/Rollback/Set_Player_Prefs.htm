Ustawianie preferencji gracza (Rollback)

Kiedy grasz w grę używając Sync Test, wszyscy "zdalni" gracze otrzymują losowe wartości dla swoich zdefiniowanych wejść, jako podstawowy sposób testowania. Możesz wyłączyć lub ponownie włączyć to zachowanie za pomocą tej akcji.

Action Syntax:

Argumenty:

Argument Opis
Enabled Włączanie lub wyłączanie losowego wprowadzania danych dla zdalnych graczy podczas testowania

Wstecz: Rollback Akcje wieloosobowe
Następny: Użyj ręcznego startu
