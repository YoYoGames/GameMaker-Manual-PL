Określenie gracza (Rollback)

Ta akcja mówi systemowi Rollback, że object powinien być używany dla graczy. Jest to opcjonalne, a kiedy jest użyte, system automatycznie stworzy instancje dla graczy, którzy się łączą i zniszczy instancje dla graczy, którzy się rozłączają.

Możesz określić warstwę, na której będą tworzone instancje graczy. Jeśli warstwa nie zostanie określona, gracze zostaną stworzeni na warstwie "Instancje", która musi istnieć w room.

Aby uzyskać informacje na temat tworzenia gry z tą funkcją lub bez niej, zobacz Definiowanie obiektu gracza.

Jeśli jest używana, funkcja ta musi być uruchomiona przed akcją join/start.

Każda instancja utworzona za pomocą tej funkcji otrzymuje zmienne instancji wymienione na tej stronie.

Action Syntax:

Argumenty:

Argument Opis
Object object do wykorzystania dla instancji graczy
Layer OPCJONALNE Nazwa warstwy, na której będą tworzone instancje graczy. "Instancje" domyślnie (gdy nie podano wartości).
