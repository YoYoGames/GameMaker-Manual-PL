Pobierz Buffer Rozmiar

Tej akcji można użyć do pobrania rozmiaru strony buffer w bajtach. Należy podać unikatową wartość identyfikatora buffer zwróconą przez akcję Utwórz bufor, a następnie podać zmienną docelową, do której zostanie zwrócona wartość rozmiaru. Zmienna docelowa może zostać oznaczona jako tymczasowa zmienna lokalna, w którym to przypadku zostanie utworzona i usunięta z pamięci po zakończeniu bieżącej akcji script lub zdarzenia.

Składnia działania:

Argumenty:

Argument Opis
Buffer Indeks buffer (przechowywany w zmiennej)
Target Zmienna docelowa, która ma przechowywać zwróconą wartość

Przykład:

Powyższy kod bloku akcji pobiera rozmiar strony buffer, a następnie wykorzystuje tę wartość do ustawienia pozycji odczytu/zapisu na koniec aktualnie przechowywanych danych.
