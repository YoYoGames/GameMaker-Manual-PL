Pobierz Buffer Pozycja

Tej akcji można użyć do pobrania bieżącej pozycji odczytu/zapisu na stronie buffer. Podczas tworzenia strony buffer pozycja odczytu/zapisu jest ustawiona na 0 bajtów, a w miarę dodawania danych do strony buffer pozycja ta będzie się kolejno przesuwać, co oznacza, że można użyć tej akcji, aby uzyskać liczbę bajtów w buffer, która jest ustawiona w dowolnym momencie. Użytkownik podaje unikatową wartość identyfikatora buffer zwróconą przez akcję Create Buffer, a następnie podaje zmienną docelową, która ma zwrócić wartość pozycji. Zmienna docelowa może zostać oznaczona jako tymczasowa zmienna lokalna, w którym to przypadku zostanie utworzona i usunięta z pamięci po zakończeniu bieżącej akcji script lub zdarzenia.

Składnia działania:

Argumenty:

Argument Opis
Buffer Indeks buffer (przechowywany w zmiennej)
Position Pozycja odczytu/zapisu (w bajtach) w buforze

Przykład:

Powyższy kod bloku akcji pobiera pozycję odczytu/zapisu dla podanego bufora i jeśli nie jest ona równa 0 (tzn. na początku strony buffer), to jest ustawiana na początek strony buffer.

Powrót: Akcje buforowe
Następny: Jeśli Buffer istnieje
