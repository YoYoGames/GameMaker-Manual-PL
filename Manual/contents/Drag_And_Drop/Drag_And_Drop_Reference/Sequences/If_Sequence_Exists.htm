Jeśli istnieje strona Sequence

Tej akcji można użyć do sprawdzenia, czy podany element sequence istnieje w grze room. Podajesz identyfikator elementu, zwrócony przez akcję Create Sequence, i jeśli element sequence istnieje w grze room, to akcja zwróci wynik true, w przeciwnym razie zwróci wynik false.

Jeśli flag argument "Not", to akcja sprawdzi, czy nie istnieje taki element sequence i jeśli nie zostanie znaleziony, zwróci true.

Te akcje będą teraz uruchamiane, jeśli blok "if" zakończy się wynikiem true, podczas gdy wszelkie akcje wykonane w innym miejscu zostaną wykonane po bloku "if".
