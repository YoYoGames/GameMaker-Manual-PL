Zapisywanie do pliku Ini

Za pomocą tej akcji można zapisać do pliku Ini wartość numeryczną lub adres string. Wcześniej należy otworzyć plik Ini do odczytu/zapisu za pomocą akcji Otwórz plik Ini, a następnie za pomocą tej akcji podać typ danych do zapisania (liczba rzeczywista lub adres string ujęty w ""), a także nagłówek sekcji, klucz w sekcji oraz wartość klucza. Strukturę pliku Ini można sobie wyobrazić na poniższym rysunku:

Jak widać, pliki Ini są dzielone na sekcje - przy czym każda sekcja ma swoją nazwę nagłówkową - oraz Następnie każda sekcja jest dzielona na różne pary klucz-wartość (podobnie jak struktura danych Mapy). Zauważ, że pliki Ini można utworzyć samodzielnie za pomocą dowolnego programu do przetwarzania tekstu (np. Notatnika na Windows), a następnie umieścić je jako plik dołączony (Included File ), z którego można czytać i do którego można pisać w grze.

WAŻNE! Próba wczytania danych z pliku Ini, który nie został wcześniej otwarty za pomocą akcji Otwórz plik Ini, spowoduje poważny błąd i awarię gry.

WAŻNE! Powoduje to jedynie zapisanie danych do pamięci i aby ostatecznie zapisać dane do pliku na dysku, należy wywołać polecenie Close Ini File, w przeciwnym razie dane nie zostaną zapisane.

Składnia działania:

Argumenty:

Argument Opis
Type Typ danych do zapisania, albo string (ujęty w cudzysłów ""), albo real number
Section Sekcja pliku Ini, do której należy zapisywać (jako string)
Key Klucz w sekcji, do którego należy dokonać zapisu (jako string)
Value Wartość do wypisania, albo string albo liczba rzeczywista, zależnie od argumentu "type" (powyżej)

Przykład:

Powyższy kod bloku akcji otworzy plik ini, następnie zapisze dane do trzech różnych kluczy w w tym samym nagłówku sekcji (jeden string i dwa numery rzeczywiste), a następnie ponownie zamknie plik ini (zapisując dane na dysku).
