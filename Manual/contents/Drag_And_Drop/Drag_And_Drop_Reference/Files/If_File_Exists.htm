Jeśli plik istnieje

Za pomocą tej akcji można sprawdzić, czy plik o podanej nazwie istnieje. Wybierasz plik do sprawdzenia (jako żądło i łącznie z extension), a funkcja zwróci true, jeśli plik istnieje, oraz false, jeśli nie. Zwróć uwagę, że funkcja zwróci true, jeśli plik istnieje w pakiecie z grą jako plik dołączony (Included File ) lub jeśli plik został zapisany podczas działania gry (więcej informacji na ten temat znajduje się w rozdziale System plików ).

Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku:

Składnia działania:

Argumenty:

Argument Opis
Type Typ danych do odczytania, albo string (ujęty w cudzysłów ""), albo real number

Przykład:

Powyższy kod bloku akcji utworzy zmienną przechowującą dane buffer, a następnie sprawdzi, czy czy plik zapisu buffer został już wcześniej utworzony. Jeśli plik istnieje, zostanie wczytany do utworzonej zmiennej buffer, ale jeśli go nie ma, zostanie utworzony, zapisany, a następnie zapisany i gotowy do wczytania następnym razem.
