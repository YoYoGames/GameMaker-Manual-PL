Ustaw oś czasu wystąpienia

Ta akcja przypisuje do instancji stronę timeline z przeglądarki Asset. Podajesz adres timeline do przypisania, a następnie możesz wybrać, czy timeline loop ma się pojawić, gdy osiągnie ostatni moment (następna klatka po uruchomieniu ostatniego momentu będzie loop z powrotem do do pierwszego momentu timeline ) oraz czy uruchomić timeline od razu, czy nie. Jeśli nie uruchomisz timeline od razu, możesz użyć akcji Set Timeline State, aby uruchomić ją później w innym wydarzeniu lub script.

Składnia działania:

Argumenty:

Argument Opis
Timeline Zasób timeline, który ma być przypisany do instancji
Loop Flaga, aby strona timeline loop była wyświetlana po ostatniej chwili (domyślnie flaga)
Start Timeline Znacznik, aby strona timeline była uruchamiana natychmiast lub nie (domyślnie oznaczona)

Przykład:

Powyższy kod bloku akcji przypisze timeline do instancji wywołującej akcji oraz ustawi moment początkowy na 10, a następnie prędkość timeline na 2.
