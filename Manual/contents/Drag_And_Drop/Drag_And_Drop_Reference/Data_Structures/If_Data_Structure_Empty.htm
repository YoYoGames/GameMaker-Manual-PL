Jeśli struktura danych jest pusta

Za pomocą tej akcji można sprawdzić, czy podana struktura danych jest pusta, czy nie. Najpierw należy wybrać typ struktury danych do sprawdzenia (lista, mapa lub stos - bez siatek, ponieważ nie mogą one być puste), a następnie podać zmienną która przechowuje indeks struktury danych (zwrócony przez akcje Create List, Create Map i Create Stack). Akcja zwróci true jeśli jeśli struktura danych jest pusta, a false jeśli nie jest. Zauważ, że możesz flag "not" w akcji, aby sprawdzić, czy struktura danych zawiera dane, sprawdzając, czy nie jest pusta.

Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku:

Te akcje będą teraz uruchamiane, jeśli "if" zakończy się wynikiem true, podczas gdy wszelkie zostaną wykonane po bloku "if".

Składnia działania:

Argumenty:

Argument Opis
Structure Typ struktury danych do sprawdzenia (Lista, Mapa lub Stos)
Variable Zmienna przechowująca indeks struktury danych

Przykład:

Powyższy kod bloku akcji sprawdza, czy struktura danych, do której odwołuje się w zmiennej globalnej nie jest pusta, a jeśli tak, to struktura danych jest czyszczona.

Powrót: Działania związane ze strukturami danych
Następny: Tworzenie listy
