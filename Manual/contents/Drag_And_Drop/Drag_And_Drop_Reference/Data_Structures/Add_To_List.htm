Dodaj do listy

Tej akcji można użyć do dodania nowej wartości (dowolnego typu danych) do listy, która zostanie dodana na końcu. Użytkownik podaje zmienną przechowującą indeks listy (zwróconą przez akcję Utwórz listę) oraz wartość do przechowania.

Składnia działania:

Argumenty:

Argument Opis
List Indeks (przechowywany w zmiennej) listy, którą należy dodać do
Value Wartość, która ma zostać dodana do listy

Przykład:

Powyższy kod bloku akcji tworzy zmienną globalną, a następnie tworzy nową strukturę danych listy, przypisując jej wartość indeksu do zmiennej globalnej. Zakres jest następnie zmieniany tak, aby wszystkie instancje object "obj_Enemy_Parent" dodały do listy swoje unikalne wartości ID instancji.
