Uzyskaj indeks elementu listy

Za pomocą tej akcji można odzyskać pozycję indeksu elementu na podanej liście. W tym celu należy podać zmienną przechowującą indeks listy (zwróconą przez akcję Utwórz listę) oraz wartość, dla której chcemy znaleźć pozycję indeksu w ramach listy, a także zmienną docelową do przechowywania zwróconej pozycji indeksu dla elementu (która może być oznaczona jako tymczasowa zmienna lokalna, używana do końca działania strony script lub zdarzenia). Należy pamiętać, że jeśli szukana wartość nie istnieje w obrębie listy, to nie będzie zwracana pozycja indeksu listy, a więc wartością zwracaną będzie -1.

Składnia działania:

Argumenty:

Argument Opis
List Indeks (zapisany w zmiennej) listy, z której ma zostać usunięty
Value Wartość, dla której ma być sprawdzana lista
Target Zmienna docelowa do przechowywania zwróconego indeksu

Przykład:

Powyższy kod bloku akcji sprawdza, czy na pozycji instancji nie występuje kolizja a jeśli taka zostanie znaleziona, to unikalna wartość ID instancji jest przechowywana w zmiennej tymczasowej, a następnie sprawdzane jest, czy istnieje ona w strukturze danych listy. Jeśli istnieje, to element jest usuwany z listy, a instancja niszczona.

Powrót: Działania związane ze strukturami danych
Następny: Wstawianie do listy
