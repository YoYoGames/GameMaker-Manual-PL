Wyczyść strukturę danych

Ta czynność spowoduje wyczyszczenie wszystkich wartości z podanej struktury danych. Musisz najpierw wybrać typ struktury danych, którą chcesz wyczyścić, a następnie podać indeks struktury danych (indeks jest wartością zwracaną i przechowywaną w zmiennej podczas tworzenia strukturę danych). Wyczyszczona struktura danych jest uważana za "pustą" (tzn. nie zawiera żadnych wpisów), z wyjątkiem siatek. W przypadku siatek działanie to spowoduje po prostu wyczyszczenie wszystkich komórek siatki do wartości 0.

UWAGA: Nie zwalnia to struktury danych z pamięci, a jedynie oczyszcza ją z wszelkich informacji. Aby zwolnić pamięć przypisaną do struktury danych, musisz użyć akcji Uwolnij strukturę danych.

Składnia działania:

Argumenty:

Argument Opis
Structure Typ czyszczonej struktury danych (Lista, Mapa, Siatka lub Stos).
Variable Zmienna przechowująca indeks struktury danych, którą należy wyczyścić

Przykład:

Powyższy kod bloku akcji sprawdza, czy struktura danych, do której odwołuje się w zmiennej globalnej nie jest pusta, a jeśli tak, to struktura danych jest czyszczona.

Powrót: Działania związane ze strukturami danych
Następny: Dowolna struktura danych
