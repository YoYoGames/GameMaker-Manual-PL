Jeśli struktura danych istnieje

Za pomocą tego działania można sprawdzić, czy zmienna indeksuje strukturę danych, czy nie. Najpierw należy wybrać typ struktury danych do sprawdzenia (Lista, Mapa, Siatka lub Stos), a następnie podać zmienną, która przechowuje indeks struktury danych indeks (zwracany przez akcje Utwórz listę, Utwórz mapę, Utwórz siatkę i Utwórz stos). Akcja zwróci true, jeśli struktura danych istnieje, i false, jeśli nie. Zauważ, że możesz flag "not" w akcji, aby sprawdzić, czy zmienna nie posiada indeksu struktury danych.

Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku:

Te akcje będą teraz uruchamiane, jeśli "if" zakończy się wynikiem true, podczas gdy wszelkie zostaną wykonane po bloku "if".

Składnia działania:

Argumenty:

Argument Opis
Structure Typ struktury danych do sprawdzenia (List, Map, Grid lub Stack)
Variable Zmienna przechowująca indeks struktury danych

Przykład:

Powyższy kod bloku akcji sprawdza, czy podana struktura danych istnieje, a jeśli tak, to jest zwalniana z pamięci.

Powrót: Działania związane ze strukturami danych
Następny: Jeśli struktura danych jest pusta
