Utwórz listę

Ta akcja utworzy nową strukturę danych listy i zwróci wartość indeksu, aby można było później uzyskać dostęp do listy za pomocą innych akcji związanych ze strukturą danych. Indeks listy zostanie zwrócony do podanej zmiennej docelowej, która może być Można ją utworzyć wcześniej za pomocą polecenia Przypisz zmienną lub Zadeklaruj zmienną albo flag. Można też zaznaczyć pole wyboru "Temp", aby nadać nazwę i utworzyć tymczasową zmienną lokalną która będzie przechowywać wartość do końca działania strony script lub zdarzenia. Nowo utworzona struktura danych listy jest uważana za "pustą", tzn. nie zawiera żadnych pozycji listy. Zauważ, że można tworzyć dodatkowe listy DS, klikając ikonę plusa obok akcji i wybierając inną zmienną, która będzie przechowywać identyfikator listy.

WAŻNE! Tworzenie dowolnej struktury danych zużywa pamięć na platformie docelowej, dlatego wszystkie struktury danych powinny być usuwane, gdy nie są już potrzebne, za pomocą akcji Uwolnij strukturę danych. w przeciwnym razie dojdzie do wycieku pamięci, który może pogorszyć wydajność gry lub nawet spowodować jej awarię. Jest to szczególnie istotne w przypadku używania tymczasowych zmiennych lokalnych do przechowywania indeksów struktur danych, ponieważ zmienne te są usuwane po zakończeniu kodu lub zdarzenia, ale nie oznacza to, że struktura danych również zostaje usunięta! Struktura danych nadal będzie istnieć, tylko nie będzie można się do niej odwołać, więc albo użyj zmiennej instancji i zwolnij strukturę w późniejszym czasie, albo zwolnij strukturę przed końcem przed końcem zdarzenia lub funkcji, jeśli jej indeks jest przechowywany w zmiennej tymczasowej.

Składnia działania:

Argumenty:

Argument Opis
Target Zmienna docelowa, w której ma być zapisany indeks listy

Przykład:

Powyższy kod bloku akcji tworzy zmienną globalną, a następnie tworzy nową strukturę danych Lista przypisując jej wartość indeksu do zmiennej globalnej. Zakres jest następnie zmieniany tak, aby wszystkie instancje object "obj_Enemy_Parent" dodawały do listy swoje unikalne wartości ID instancji.
