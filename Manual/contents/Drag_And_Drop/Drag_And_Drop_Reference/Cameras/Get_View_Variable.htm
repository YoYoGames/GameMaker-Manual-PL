Pobierz zmienną widoku

Ta akcja umożliwia uzyskanie wartości pewnych wbudowanych zmiennych związanych z danym portem widoku. Wybierasz zmienną portu widoku, którą chcesz pobrać, następnie podajesz port widoku, do którego ma trafić zmienna (od 0 do 7) i zmienną docelową, która ma przechowywać zwróconą wartość (może być ona oznaczona jako tymczasowa zmienna lokalna). wartość (może być oznaczona jako tymczasowa zmienna lokalna). Port widoku to obszar ekranu, który będzie używany do rysowania danego widoku z kamery (ustawionego w edytorze room ), a różne zmienne, które można uzyskać w ten sposób przedstawiono w poniższej tabeli:

Zmienna Opis
Camera Zwracana wartość będzie unikatową wartością identyfikacyjną kamery, która ma być przypisana do portu (więcej informacji można znaleźć tutaj ).
Visibility Spowoduje to zwrócenie wartości true lub false, gdzie true oznacza, że port widoku jest widoczny (tzn. jest rysowany na ekranie), a false oznacza, że nic nie jest rysowane.
View Port X Coordinate Jest to pozycja X portu widoku w oknie gry.
View Port Y Coordinate Jest to pozycja Y portu widoku w oknie gry.
View Port Width Jest to szerokość (w pikselach) okna podglądu.
View Port Height Jest to wysokość (w pikselach) portu podglądu.
View Port Surface ID Spowoduje to zwrócenie unikatowej wartości identyfikatora powierzchni przypisanej do portu widoku lub -1, jeśli nie przypisano żadnej powierzchni.

Składnia działania:

Argumenty:

Argument Opis
Variable Wbudowana zmienna widoku, aby uzyskać
View Port widoku, który ma być docelowy (od 0 do 7)
Target Zmienna docelowa, która ma przechowywać zwracaną wartość (może być oznaczona jako tymczasowa zmienna lokalna)

Przykład:

Powyższy kod bloku akcji pobiera bieżący identyfikator kamery przypisany do widoku port [0], a następnie sprawdza, czy jest on taki sam jak identyfikator zapisany w zmiennej globalnej. Jeśli nie, to kamera jest ustawiana na nowy identyfikator.
