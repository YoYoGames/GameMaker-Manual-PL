Powrót

Ta akcja jest używana podczas tworzenia funkcji bloku akcji i służy do zwracania wartości do instancji wywołującej script. Wartością zwracaną może być wszystko, co może przechowywać zmienna - np. jak string, wartość liczbowa, wskaźnik lub identyfikator zasobu - lub może to być sama zmienna. Należy pamiętać, że gdy wywołamy akcję return w funkcji, funkcja zostanie zakończona - zwracając podaną wartość - i nie zostaną wywołane żadne inne akcje po wykonaniu return, chociaż jeśli w zdarzeniu po wywołaniu funkcji znajdują się akcje, zostaną one wykonane.

Składnia działania:

Argumenty:

Argument Opis
Return Wartość zwracana.

Przykład:

Powyższa funkcja bloku akcji ustawia zmienną tymczasową na słowo kluczowe noone, a następnie sprawdza zmienną wejściową, jaką wartość ma w danej chwili. Jeśli zmienna jest większa niż 100, to tworzona jest instancja, a jej unikalny identyfikator jest zapisywany w zmiennej tymczasowej, która następnie jest zwracana z funkcji.
