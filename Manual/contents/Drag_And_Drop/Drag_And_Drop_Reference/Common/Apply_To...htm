Aplikuj do...

Tej akcji można użyć do zmiany zakresu akcji dla wszystkich kolejnych akcji, które są do niej dołączone. Podobnie jak w przypadku if jeśli umieścisz kolejne akcje kolejne akcje na prawo od akcji Zastosuj do..., będą one uruchamiane z zakresu object lub instancji. Zakres wybiera się z menu podręcznego, które pojawia się po kliknięciu małej strzałki w prawym górnym rogu akcji. różnych zestawów objects - lub nawet jednej instancji, jeśli masz identyfikator instancji - aby uruchamiać wiele akcji przed kontynuowaniem kodu bieżącej object.

Należy pamiętać, że aby dodać akcje do bloku "Dotyczy", należy je upuścić z boku akcji, jak pokazano na poniższej ilustracji:

Działania te będą teraz uruchamiane ze strony object lub instancji, do której odnosi się blok "Stosuje się do", podczas gdy wszelkie akcje zrzucone gdzie indziej będą wykonywane po bloku "Stosuje się do".
