Ustaw położenie wzdłuż ścieżki

Ta akcja ustawia pozycję instancji wzdłuż strony path. Wartość jest normalizowana w zakresie 0 - 1, więc jeśli ustawisz ją na przykład na 0,5, instancja zostanie przesunięta dokładnie na środek strony path. Zauważ, że aby ta akcja miała jakikolwiek efekt, musisz wcześniej wywołać akcję Start Following Path, aby przypisać instancję do strony path.

Składnia działania:

Argumenty:

Argument Opis
Position Pozycja (od 0 do 1) wzdłuż strony path, na której ma być umieszczona instancja

Przykład:

Powyższy kod bloku akcji uruchamia instancję na stronie path, a następnie generuje losową wartość dziesiętną z przedziału od 0 do 1. Wartość ta jest przechowywana w tymczasowej zmiennej lokalnej i używana do ustawiania pozycji instancji wzdłuż path.
