Uzyskaj wynik

Możesz użyć tej akcji, aby pobrać wartość zmiennej instancji utworzonej dla "wyniku". Podajesz zmienną docelową do przechowywania zwróconej wartości "score" (która może być oznaczona jako tymczasowa zmienna lokalna), ale zauważ, że jeśli zmienna nie została wcześniej ustawiona (za pomocą polecenia Set Score), to zostanie ona utworzona, a akcja zwróci 0 (w przeciwieństwie do zwykłych zmiennych instancji, które spowodują błąd, jeśli spróbujesz uzyskać do nich dostęp bez uprzedniego ustawienia ich).

Argumenty:

Argument Opis
Target Zmienna docelowa do przechowywania zwróconej wartości "score" (może być oznaczona jako tymczasowa zmienna lokalna)

Przykład:

Powyższy kod bloku akcji pobierze bieżącą wartość wyniku i zapisze ją w tymczasowej zmiennej lokalnej zmiennej lokalnej. Ta zmienna jest następnie konwertowana na string i przechowywana w innej tymczasowej zmiennej lokalnej, zanim zostanie narysowana na ekranie.
