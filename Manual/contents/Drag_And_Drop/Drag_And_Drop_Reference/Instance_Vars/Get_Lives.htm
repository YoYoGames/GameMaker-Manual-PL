Zdobądź życie

Możesz użyć tej akcji, aby pobrać wartość zmiennej instancji utworzonej dla "lives". Podajesz zmienną docelową do przechowywania zwróconej wartości "lives" (która może być oznaczona jako tymczasowa zmienna lokalna), ale zauważ, że jeśli zmienna nie została wcześniej ustawiona (za pomocą Set Lives), to zostanie ona utworzona, a akcja zwróci 0 (w odróżnieniu od zwykłych zmiennych instancji, które spowodują błąd, jeśli spróbujesz uzyskać do nich dostęp bez uprzedniego ustawienia ich).

Składnia działania:

Argumenty:

Argument Opis
Target Zmienna docelowa do przechowywania zwróconej wartości "lives" (może być oznaczona jako tymczasowa zmienna lokalna)

Przykład:

Powyższy kod bloku akcji pobierze wartość dla "lives" i zapisze ją w tymczasowej zmiennej lokalnej. Zmienna ta zostanie następnie użyta do określenia, ile stosów sprites powinno zostać narysowanych.
