Ustaw alfa cząstek

Ta akcja ustawi alfę zarówno w momencie tworzenia cząstki, jak i przez cały czas jej istnienia. Najpierw określasz identyfikator typu cząstki (powinien on być taki sam jak wartość zwrócona podczas tworzenia cząstki za pomocą akcji Utwórz typ cząstki), a następnie możesz ustawić wartości alfa, które mają być używane.

Można wybrać trzy wartości alfa, a cząstka będzie się między nimi mieszać przez cały czas życia cząstki. Na przykład, jeśli cząstka ma wartości alfa 0,5, 1 i 0, a jej czas życia wynosi 60 klatek gry, to w momencie utworzenia (klatka 0) będzie miała alfę 0,5, potem będzie zanikać, aż osiągnie pełną alfę 1 (klatka 30), a następnie stanie się niewidoczna z alfą 0 (klatka 60).

Składnia działania:

Argumenty:

Argument Opis
Type Unikalna wartość identyfikacyjna typu cząstki, dla której ma zostać ustawiony kolor
Start Początkowa alfa, której należy użyć
Middle Pomiędzy alfami można użyć
End Ostateczna alfa, której należy użyć

Przykład:

Powyższy kod bloku akcji utworzy nowy typ cząstki i przypisze jej unikatową wartość ID do zmiennej globalnej. Następnie ustawi wszystkie właściwości typu cząstki.

Powrót: Akcje cząstek
Następny: Ustaw czas życia cząstek
