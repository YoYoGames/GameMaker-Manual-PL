Ustaw sprajty cząstek

Tej akcji można użyć do ustawienia strony sprite, której cząstka może używać podczas rysowania. Należy podać unikatową wartość identyfikacyjną typu, dla którego ma zostać ustawiona strona sprite (zwróconą podczas tworzenia typu za pomocą akcji Utwórz typ cząstki) oraz zasób sprite, który ma zostać użyty. Jeśli strona sprite ma pod-obrazy, cząstka będzie animować przez ramki z prędkością ustawioną w edytorze obrazów dla strony sprite. Należy także pamiętać, że jeśli ustawisz wartość koloru cząstki (za pomocą akcji Ustaw kolor cząstki), kolor ten zostanie zmieszany z używaną stroną sprite, podobnie jak ustawienie mieszania obrazu na instancji spowoduje zmieszanie koloru z instancją sprite (użyj koloru białego, aby nie mieszać).

UWAGA: Jeśli użyjesz tej akcji, nie powinieneś używać akcji Ustaw kształt cząstki, ponieważ możesz mieć tylko stronę sprite lub kształt, ale nie obie.

Składnia działania:

Argumenty:

Argument Opis
Type Wartość identyfikacyjna typu, który ma być ustawiony pod adresem sprite.
Sprite Adres sprite, który należy przypisać do typu cząstki.

Przykład:

Powyższy kod bloku akcji utworzy nowy typ cząstki i przypisze jej unikatową wartość ID do zmiennej globalnej. Następnie ustawi wszystkie właściwości typu cząstki.

Powrót: Akcje cząstek
Następnie: Ustaw kształt cząstki
