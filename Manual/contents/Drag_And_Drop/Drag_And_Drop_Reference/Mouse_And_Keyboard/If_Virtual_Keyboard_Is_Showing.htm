Jeśli wyświetlana jest klawiatura wirtualna

Ta akcja służy do sprawdzania, czy jest wyświetlana wirtualna klawiatura systemu operacyjnego. Akcja przyjmie wartość true, jeśli klawiatura jest otwarta lub otwiera się, lub false, jeśli klawiatura wirtualna jest zamknięta lub zamyka się. Jeśli włączysz modyfikator not, działanie to zostanie odwrócone, a zwróconą wartość true / false będzie można zapisać w zmiennej docelowej, którą można oznaczyć jako temp (lokalną).

Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku:

Te akcje będą teraz uruchamiane, jeśli w bloku "if" zakończy się wynikiem true, podczas gdy wszelkie działania porzucone w innym miejscu zostaną wykonane po bloku "if".
