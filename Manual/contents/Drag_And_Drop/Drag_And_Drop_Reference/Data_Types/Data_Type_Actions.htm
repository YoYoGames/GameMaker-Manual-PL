Typy danych Biblioteka akcji

GameMaker ma wiele różnych typów danych, takich jak liczby całkowite, strings, wskaźniki itd. a niektóre z nich można konwertować na inne. W tej części podręcznika przedstawiono różne działania, jakie można wykonać, aby poradzić sobie z różnymi typami danych i je przekonwertować.

UWAGA: więcej informacji na temat różnych typów danych można znaleźć tutaj.

Biblioteka Typy ma następujące funkcje:

Decimal To Integer
String To Number
Number To String

Powrót: Indeks
Następny: Działania losowe
