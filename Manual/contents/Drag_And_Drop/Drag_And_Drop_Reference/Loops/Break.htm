Przerwa

Tej akcji można użyć tylko w ramach akcji loop (np. Repeat lub While), a jej wywołanie spowoduje przerwanie działania loop w tym momencie. Zazwyczaj akcja loop będzie wykonywana określoną liczbę razy lub do momentu ale można użyć akcji Break, aby przedwcześnie zakończyć działanie loop w dowolnym momencie (działania w obrębie loop po akcji Break nie zostaną uruchomione).
