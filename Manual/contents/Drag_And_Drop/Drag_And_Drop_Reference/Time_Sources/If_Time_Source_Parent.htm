Jeśli [Źródło czasu] Rodzic

Ta czynność umożliwia sterowanie innymi czynnościami w oparciu o rodzica danego Źródła czasu.

Sprawdza, czy źródło czasu podane w polu "Rodzic" jest (lub nie jest) rodzicem źródła czasu podanego w polu "Źródło czasu".

Składnia działania:

Argumenty:

Argument Opis
Time Source Strona Identyfikator źródła czasu aby sprawdzić rodzica
Not Włącz tę opcję, aby odwrócić warunek
Parent Adres Identyfikator źródła czasu do porównania z rodzicem poprzednio podanego Źródła Czasu

Powrót: Działania związane ze źródłem czasu
Następny: Jeśli Źródło czasu Stan
