Przekonfiguruj źródło czasu

Akcja ta pozwala na zmianę ustawień dla istniejącego Źródła czasu, określonego w polu "Źródło czasu".

Dostarczone tu ustawienia są takie same jak w przypadku Create Time Source, z wyjątkiem rodzica.

Spowoduje to zresetowanie i dezaktywację Źródła Czasu, więc musisz uruchomić je ponownie.

Action Syntax:

Argumenty:

Argument Opis
Time Source Strona Identyfikator źródła czasu rekonfigurować
Period Długość okresu źródła czasu, jak długo trwa jego wygaśnięcie
Units Jednostki, w których wyrażony jest okres (sekundy lub klatki)
Callback Adres method do wywołania po wygaśnięciu źródła czasu
Arguments OPCJONALNE Tablica zawierająca argumenty, które mają być przekazane do funkcji callback.
Repetitions OPCJONALNE Ile razy Źródło Czasu powinno się uruchomić w sumie, lub -1 dla nieokreślonego powtarzania.
Expiry Type OPCJONALNE Typ wygaśnięcia dla źródła czasu

Powrót: Czynności związane ze źródłem czasu.
Następny: Resetuj źródło czasu
