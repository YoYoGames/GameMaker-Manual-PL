Pobierz Pozostały czas

Ta czynność pozwala uzyskać czas pozostały do wygaśnięcia podanego źródła czasu. Wartość ta jest wyrażona w jednostkach użytych dla danego źródła czasu.

Argumenty:

Argument Opis
Time Source Strona Identyfikator źródła czasu aby uzyskać pozostały czas dla
Target Zmienna do przechowywania wartości zwracanej
Temp Czy zmienna ma być utworzona jako zmienna tymczasowa?

Powrót: Działania związane ze źródłem czasu
