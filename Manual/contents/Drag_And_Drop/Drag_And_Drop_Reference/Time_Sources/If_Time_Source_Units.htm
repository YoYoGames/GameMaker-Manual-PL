Jeśli [Źródło czasu] Jednostki

Ta czynność umożliwia sterowanie innymi czynnościami w oparciu o typ jednostki danego Źródła czasu.

Sprawdza, czy jednostka Stała jednostki źródła czasu podana w polu "Jednostki" jest (lub nie jest) typem jednostki źródła czasu podanym w polu "Źródło czasu".

Składnia działania:

Argumenty:

Argument Opis
Time Source Strona Identyfikator źródła czasu aby sprawdzić stan
Not Włącz tę opcję, aby odwrócić warunek
Units Strona Źródło czasu Jednostka stała do sprawdzenia z

Powrót: Działania związane ze źródłem czasu
Następny: Od prędkości BPM do sekund
