Jeśli [Źródło czasu] Państwo

Ta akcja umożliwia sterowanie innymi akcjami na podstawie stanu danego Źródła czasu.

Sprawdza, czy stan Stała stanu źródła czasu podany w polu "Stan" jest (lub nie jest) stanem źródła czasu podanym w polu "Źródło czasu".

Składnia działania:

Argumenty:

Argument Opis
Time Source Strona Identyfikator źródła czasu aby sprawdzić stan
Not Włącz tę opcję, aby odwrócić warunek
State Strona Stała stanu źródła czasu do sprawdzenia z

Przykład:


Ten blok kodu powinien być uruchamiany w zdarzeniu Naciśnięcie klawisza.

Sprawdza, czy stan źródła czasu jest "aktywny", i w takim przypadku wstrzymuje działanie tego źródła czasu.

Jeśli jednak stan jest "wstrzymany", źródło czasu jest uruchamiane ponownie.
