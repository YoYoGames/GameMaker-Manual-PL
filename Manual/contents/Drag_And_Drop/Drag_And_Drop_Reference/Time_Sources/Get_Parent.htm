Pobierz Rodzic

Ta czynność umożliwia uzyskanie nadrzędnego źródła czasu dla podanego źródła czasu.

Zwracana wartość może być identyfikatorem niestandardowego źródła czasu lub jednego z wbudowanych źródeł czasu, w zależności od tego, jak zostało utworzone dane źródło czasu.

Składnia działania:

Argumenty:

Argument Opis
Time Source Strona Identyfikator źródła czasu aby uzyskać dane rodzica
Target Zmienna do przechowywania nadrzędnego źródła czasu
Temp Czy zmienna ma być utworzona jako zmienna tymczasowa?

Powrót: Działania związane ze źródłem czasu
Następny: Zdobądź okres
