Serwer Micro Web

Podstawową funkcją tego niewielkiego narzędzia jest łączenie się z niektórymi platformami docelowymi dla programu GameMaker (np. platformą HTML5) i serwowanie na nich plików. W trybie offline służy ono także do obsługi plików do podręcznika GameMakera.

Po uruchomieniu gry na urządzeniu docelowym wymagającym Micro Web Server zostanie otwarte nowe okno, w którym gra zostanie wysłana do wybranego urządzenia:

UWAGA Przy pierwszym uruchomieniu gry na obiekcie, który wymaga strony Micro Web Server, może pojawić się alert zapory. W takim przypadku należy zezwolić na komunikację przez zaporę sieciową server.

Po uruchomieniu witryny Micro Web Server możesz uzyskać dostęp do swojej gry z dowolnego urządzenia lokalnego (a nawet zdalnie, jeśli przekierowanie portów zostało prawidłowo skonfigurowane). Wystarczy uzyskać dostęp do lokalnego adresu IP komputera przez port widoczny w tym oknie, aby móc grać w grę z innego urządzenia, np. telefonu komórkowego lub innego komputera.

Na przykład, załóżmy, że lokalny adres IP komputera to 192.168.0.110(przeczytaj, jak go znaleźć), a port wyświetlany w oknie Micro Web Server to 51264. Po uruchomieniu gry na komputerze użytkownik łączy się z adresem 192.168. 0.110: 51264 za pomocą przeglądarki w telefonie komórkowym i gra w swoją grę.

Na stronie Server wyświetlana jest pewna ilość informacji debugowania związanych z obsługiwanymi plikami, a na dole znajduje się przycisk umożliwiający ich wyczyszczenie. Obok przycisku "Wyczyść" znajduje się adres path do głównego pliku JS projektu, a pod nim opcja Szybkość sieci. Domyślnie jest ona ustawiona na "Bez limitu", ale jej kliknięcie spowoduje otwarcie listy różnych opcji sieciowych:

Zmiana tego ustawienia spowoduje również zmianę opcji Rozmiar pakietu po prawej stronie. Możesz również edytować to ustawienie ręcznie - w takim przypadku opcja Prędkość sieci zostanie ustawiona na "Niestandardowa". Ustawienie jej na wartość inną niż maksymalna spowoduje zmniejszenie szybkości transferu sieciowego i umożliwi testowanie gier w środowisku o niskiej przepustowości.

Widoczna jest również lista Dozwolonych adresów IP wraz z numerem używanego portu , ale nie można ich edytować w witrynie Micro Web Server. Zamiast tego należy je ustawić w ustawieniach Menedżera urządzeń dla danego urządzenia docelowego lub w Preferencjach HTML5, jeśli używasz urządzenia "Domyślnego".
