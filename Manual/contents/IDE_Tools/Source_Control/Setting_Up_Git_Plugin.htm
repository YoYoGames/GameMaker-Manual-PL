Konfigurowanie wtyczki Git

Zanim będziemy mogli korzystać z wtyczki Git, musimy skonfigurować plik wykonywalny Git do użycia, przechodząc do sekcji Plugins - Source Control (Git) w Preferencjach:

Plik wykonywalny Git path powinien zostać automatycznie wypełniony, jeśli został znaleziony przez GameMaker, jednak jeśli pole jest puste, można kliknąć przycisk i ręcznie wybrać plik git.exe znajdujący się w folderze cmd instalacji Git, który zwykle znajduje się w C:\Program Files\Git\.

Pozostałe opcje dostępne na stronie Git można znaleźć na stronie Preferencje wtyczek.

Zanim będziesz mógł używać Source Control w swoim projekcie, musisz wejść w Opcje gry i włączyć Kontrolę źródła:
