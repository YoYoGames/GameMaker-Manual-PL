Preferencje systemu Windows

W preferencjach witryny Windows dostępne są następujące opcje:

  • Lokalizacja Visual Studio: Ta opcja służy do wskazania programowi GameMaker instalacji Microsoft Visual Studio na komputerze, na którym tworzony jest program.
  • Domyślny typ opakowania: Podczas tworzenia ostatecznego pakietu wykonywalnego Windows można wybrać między utworzeniem instalatora (przy użyciu NSIS) lub skompresowanego pliku ZIP który będzie zawierał grę i wszelkie dodatkowe pliki. Domyślnie podczas kompilacji zostanie wyświetlone okno dialogowe wyboru między tymi dwiema opcjami, ale można ustawić, aby zawsze używana była jedna lub druga. Należy pamiętać, że gdy opcja ta jest ustawiona na Pokaż okno dialogowe, w oknie dialogowym, które pojawi się podczas kompilacji, włączenie opcji "Zapamiętaj opcję pakowania" spowoduje ustawienie preferencji na wybraną przez Ciebie opcję.
