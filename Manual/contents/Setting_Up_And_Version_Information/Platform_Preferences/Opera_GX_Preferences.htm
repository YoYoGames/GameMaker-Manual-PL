Preferencje Opery GX

W preferencjach Opera GX dostępne są następujące opcje:

  • Lokalizacja Emscripten SDK: To jest lokalizacja Emscripten SDK, wymaganego dla kompilacji YYC wykonanych dla Opery GX. Informacje na temat konfigurowania SDK można znaleźć w rozdziale Konfiguracja dla Oper y GX.
  • Domyślny port internetowy Server : Umożliwia zmianę domyślnego portu dla strony internetowej Opera GX server. Domyślną wartością jest 51264.
  • Domyślny adres internetowy Server: W tym miejscu można zmienić domyślny adres internetowy server dla strony internetowej Opera GX server. Domyślna wartość to 127.0.0.1.
  • Domyślne dozwolone adresy IP witryny Server: W tym miejscu można dodać dowolne dozwolone adresy IP dla strony server. Będą one ustawione jako domyślne dla wszystkich urządzeń dodanych za pomocą Target Managera.
