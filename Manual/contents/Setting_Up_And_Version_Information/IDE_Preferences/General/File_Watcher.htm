Preferencje obserwatora plików

File Watcher (Obserwator plików ) to nazwa części witryny IDE, która obserwuje pliki związane z projektem pod kątem wszelkich zmian wprowadzonych przez zewnętrzne oprogramowanie (np. edycja obrazu w zewnętrznym edytorze) lub zmian wewnętrznych, takich jak edycja skryptu. asset. Zasadniczo nie ma potrzeby edytowania tych ustawień, ale w niektórych systemach - np. w tych, w których dyskiem zapisującym pliki projektu jest dysk SSD - funkcja obserwowania plików może informować o zmianach podczas wykonywania takich czynności, jak zwykłe zapisywanie projektu projektu lub przy każdej edycji projektu asset, w którym to przypadku może być konieczna zmiana tych ustawień.

Dostępne są następujące ustawienia:

  • Włącz obserwatora plików: W tym miejscu można włączyć lub wyłączyć system File Watcher. Po wyłączeniu tej funkcji witryna IDE nie będzie wychwytywać żadnych plików zmienionych przez źródła zewnętrzne, dlatego należy zachować ostrożność przy wyłączaniu tę funkcję. Funkcja File Watcher jest domyślnie włączona.
  • Ustaw tolerancję obserwatora plików w MS po zapisie W IDE: Dostosowuje czas między zapisaniem pliku a sprawdzeniem zmian przez obserwatora plików. Wartość tę można dostosować, jeśli podczas zapisywania projektu W takim przypadku należy zwiększyć opóźnienie, aż komunikat przestanie się pojawiać. Domyślną wartością jest 2000 ms, ale może ona wynosić od 1 ms do 10000 ms.
