Preferencje dotyczące zasobów

Preferencje zawierają opcje określające sposób rysowania pewnych zasobów w projektach:

  • Początkowy domyślny kolor rysowania dla nowych projektów: Jeśli użyjesz strony font lub narysujesz kształt za pomocą funkcji GameMaker lub akcji GML Visual bez podania koloru, użyty zostanie kolor ustawiony w tym miejscu. Domyślnym kolorem jest biały ( ARGB: $FFFFFFFF).
  • Początkowy domyślny kolor okna dla nowych projektów: Ta opcja służy do ustawiania koloru tła strony HTML dla celu HTML5, a także do ustawiania koloru wyświetlanego poza stroną rooms na wszystkich celach (w tym HTML5). Domyślnym kolorem jest czarny (ARGB: $000000FF).
