Preferencje edytora splajnu

Preferencje edytora splajnów są używane służą przede wszystkim do określania wyglądu gładkiej strony paths w Edytorze ścieżek, choć mają też wpływ na niektóre inne aspekty IDE (np. edytor kształtów fizyki). Physics Shape Editor). Dostępne są następujące opcje:

  • Wstaw path precyzja punktu (px): Ustawia wartość precyzji (w pikselach) dla wstawiania punktów na stronie path, tzn. mysz musi znajdować się w tym zakresie, aby wstawić punkt, a nie dodać nowy. Domyślnie jest to 20px.
  • Szerokość punktu zakotwiczenia: Ustaw szerokość punktów zakotwiczenia w pikselach. Domyślnie jest to 8px;
  • Wysokość punktu zakotwiczenia: Określa wysokość punktów zakotwiczenia w pikselach. Domyślnie jest to 8px;
  • Kolor ścieżki: Kolor, który będzie używany do wyświetlania strony path podczas jej edycji. Domyślnie jest to ( ARGB) #FFFFFFFF.
  • Kolor podświetlonego segmentu path : Kolor, który będzie używany do podświetlania wybranych segmentów path. Domyślnie (ARGB) #FFFFFFFF.
  • Nieważny kolor path : Kolor używany do wyróżnienia strony path, która nie jest ważna. Domyślnie jest to (ARGB) #FF0000FF.
  • Kolor punktów zakotwiczenia: kolor używany do rysowania punktów zakotwiczenia na stronie path. Domyślnie jest to (ARGB) #FFC0C0C0.
  • Kolor nieprawidłowych punktów zakotwiczenia: Kolor, który ma być używany do rysowania punktów zakotwiczenia path. Domyślnie jest to (ARGB) #FF800000.
  • Wybrany kolor punktu zakotwiczenia: Kolor, który zostanie użyty do narysowania wyróżnionych (wybranych) punktów zakotwiczenia path. Domyślnie jest to (ARGB) #FF00A4FF.
  • Wybrany nieważny kolor punktu zakotwiczenia: Kolor, który będzie używany do rysowania punktów zakotwiczenia, które są wyróżnione, ale nieważne. Domyślnie jest to (ARGB) #FFA400FF.
  • Kolor punktu zakotwiczenia po najechaniu myszą: Kolor, który będzie używany do rysowania aktywnych punktów zakotwiczenia. Domyślnie jest to (ARGB) #FFFFFFFF.
  • Kolor nieważnego punktu zakotwiczenia po najechaniu myszą: kolor używany do rysowania punktów zakotwiczenia, które są aktywne, ale nieważne. Domyślnie jest to (ARGB) #FF0000FF.
  • Kolor nakładki maski: ustawia kolor nakładki dla masek, jeśli są one włączone (patrz wyżej). Domyślnie jest to (ARGB) #FF404040.
