Preferencje wtyczek

Preferencje wtyczek służą do kontrolowania wszelkich dodatkowych wtyczek, które zostały dodane do GameMakera IDE. Domyślnie jedna sekcja jest poświęcona wtyczce Source Control, która służy do integracji projektów GameMakera z Git.

Kontrola źródeł (Git)

W tej sekcji omówiono preferencje, które są wymagane, aby wtyczka Git SCM działała w programie GameMaker.

  • Ścieżka do pliku wykonywalnego Git: Jest to lokalizacja pliku wykonywalnego Git, którego GameMaker używa do wykonywania wszystkich poleceń Source Control. Jak pokazano na powyższym obrazku, będzie on znajdował się w folderze cmd w instalacji Git.

Poniżej można skonfigurować narzędzie Scalanie i narzędzie Różnica dla strony Source Control z tymi ustawieniami(opcjonalnie):

  • Merge (Tool location): W tym miejscu można określić pełny plik path do narzędzia do scalania Git.
  • Merge (Opcje narzędzia): W tym polu można dodać dowolne dodatkowe polecenia, które będą uruchamiane przy każdym użyciu narzędzia Scalanie.
  • Różnica (położenie narzędzia): W tym miejscu określany jest pełny plik path do narzędzia Git diff.
  • Różnica (opcje narzędzia): W tym polu można dodać dowolne dodatkowe polecenia, które mają być uruchamiane przy każdym użyciu narzędzia diff.
