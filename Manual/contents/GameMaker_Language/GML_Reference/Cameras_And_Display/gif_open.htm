gif_open

Za pomocą tej funkcji można utworzyć pusty obrazek w formacie GIF, do którego można dodać dane. Po wywołaniu funkcji należy określić szerokość i wysokość GIF-a (w pikselach, nie ma górnego ograniczenia rozmiaru, z wyjątkiem dostępnej pamięci), a funkcja zwróci unikalną wartość ID używaną do identyfikacji prezentu w kolejnych funkcjach lub zwróci -1, jeśli nie można zainicjować GIF-a (na przykład, jeśli szerokość/wysokość jest zbyt duża dla dostępnej pamięci). Można również podać opcjonalny argument określający "kolor przezroczysty" dla GIF-a. Jest to wartość RGB (bez składowej alfa), która spowoduje wyczyszczenie gifa do tego koloru, zanim cokolwiek zostanie do niego dodane. Jeśli nie podasz argumentu kolor przezroczysty, zostanie użyty domyślny kolor czarny.

Należy pamiętać, że podczas korzystania z tej funkcji należy wywołać polecenie gif_save() aby zakończyć tworzenie GIF-a przed otwarciem innego pliku do zapisu (dlatego każdemu gif_open() musi towarzyszyć gif_save()).

Składnia:

gif_open(width, height, [clear_colour]);

Argument Typ Opis
width Real The width of the gif to create
height Real The height of the gif to create
[clear_colour] Kolor OPTIONAL Kolor (RGB), na który ma zostać wyczyszczony gif

Zwroty:

GIF ID

Przykład:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

Powyższy kod utworzy plik obrazu GIF zawierający 30 klatek pobranych z powierzchni aplikacji, a następnie zapisze go.
