window_get_cursor

Za pomocą tej funkcji można uzyskać informacje o bieżącym kursorze używanym w oknie gry, a wartością zwracaną jest jedna z poniższych stałych (aby ustawić bieżący kursor, można użyć funkcji window_set_cursor() która również przyjmuje te stałe):

Kursor Stała
Stała Kursor
cr_none
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

Składnia:

window_get_cursor();

Zwroty:

Cursor Constant

Przykład:

if mouse_check_button_pressed(mb_left)
{
    if window_get_cursor() != cr_drag window_set_cursor(cr_drag);
}

Powyższy kod zmieni kursor okna na standardowy kursor przeciągania okna, jeśli został naciśnięty lewy przycisk myszy i nie został on wcześniej zmieniony.
