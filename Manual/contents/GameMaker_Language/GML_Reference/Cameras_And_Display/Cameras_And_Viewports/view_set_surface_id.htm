view_set_surface_id

Za pomocą tej zmiennej można ustawić zawartość portu widoku, która ma być rysowana na powierzchni. Podczas pracy z powierzchniami często wymagane jest przechwycenie całego widocznego obszaru ekranu na powierzchnię, dlatego za pomocą tej funkcji można przypisać ją do portu widoku. Oznacza to, że wszystko, co jest wyświetlane w wybranym porcie widoku, zostanie narysowane na przypisanej powierzchni, a zawartość tego portu widoku nie będzie już wyświetlana, co oznacza, że trzeba będzie albo:

  • Włącz nowy widok i rysuj powierzchnię tylko w tym widoku (używając view_current aby sprawdzić, który widok jest rysowany)
  • Narysuj powierzchnię w zdarzeniu Draw GUI lub Post Draw instancji, ponieważ te zdarzenia są niezależne od widoków.

Podczas korzystania z tej funkcji należy podać indeks portu widoku (od 0 do 7) oraz indeks powierzchni (albo indeks application_surface lub unikatową wartość indeksu zwróconą przez funkcję surface_create()) lub, jeśli powierzchnia została już wcześniej przypisana i chcesz ją usunąć, wartość -1. Więcej przykładów ustawiania portu widoku na powierzchnię można znaleźć w zmiennej view_surface_id.

Składnia:

view_set_surface_id(view_port, surf)

Argument Typ Opis
view_port Real The view port to target (0 - 7)
surf Identyfikacja powierzchni The surface to apply to the view

Zwroty:

Real

Przykład:

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

Powyższy kod sprawdzi, czy do portu widoku[0] została przypisana powierzchnia, a jeśli nie, to zostanie ona przypisana.
