camera_get_end_script

Za pomocą tej funkcji można uzyskać identyfikator funkcji skryptu przypisanej jako funkcja końcowa dla podanej kamery. Jeśli nie przypisano żadnej funkcji script, funkcja zwraca -1.

Składnia:

camera_get_end_script(camera_id)

ArgumentTyp Opis
camera_id The unique camera ID value returned when you created the camera

Zwroty:

Script Function or -1 if no function is assigned

Przykład:

var scr = camera_get_end_script(camera_view[0]);
if scr != cutScene
{
    camera_set_update_script(camera_view[0], cutScene);
}

Powyższy kod sprawdza funkcję script przypisaną jako begin script dla kamery przypisanej do portu widoku[0] i jeśli nie jest to "cutScene", jest ona ustawiana na tę funkcję.
