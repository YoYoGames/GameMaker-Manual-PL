view_get_camera

Za pomocą tej funkcji można uzyskać unikatową wartość identyfikatora kamery przypisanej do danego portu podglądu (od 0 do 7). Jeśli żadna kamera nie jest przypisana, funkcja zwraca -1.

Składnia:

view_get_camera(view_port)

ArgumentTyp Opis
view_port The view port to target (0 - 7)

Zwroty:

Camera ID

Przykład:

var cam = view_get_camera(0);
var cw = camera_get_view_width(cam);
var ch = camera_get_view_height(cam);
camera_set_view_pos(cam, mouse_x - (cw / 2), mouse_y - (ch / 2));

Powyższy kod pobierze identyfikator kamery przypisanej do portu widokowego[0], a następnie zmieni jej położenie.

Powrót: Kamery i porty podglądu
Następny: view_get_visible
