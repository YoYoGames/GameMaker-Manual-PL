view_set_yport

Ta funkcja służy do ustawiania pozycji y podanego portu widoku. Podajesz indeks portu widoku (od 0 do 7) oraz nową pozycję, na której ma się on znaleźć.

Składnia:

view_set_yport(view_port, y)

ArgumentTyp Opis
view_port The view port to target (0 - 7)
y The new y position

Zwroty:

N/A

Przykład:

if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)
{
    view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));
}

Powyższy kod sprawdzi pozycję y portu widoku[0] i jeśli nie znajduje się on w wymaganym miejscu, zostanie ustawiony w tej pozycji.

Powrót: Kamery i porty podglądu
