camera_get_view_y

Za pomocą tej funkcji można pobrać pozycję y widoku dla danej kamery. Należy pamiętać, że funkcja ta jest ważna tylko dla kamer utworzonych przy użyciu camera_create_view() lub dodanych w room edytora.

Składnia:

camera_get_view_y(camera_id)

ArgumentTyp Opis
camera_id The unique camera ID value returned when you created the camera

Zwroty:

Przykład:

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

Powyższy kod pobiera położenie kamery przypisanej do portu widoku[0], a następnie rysuje tekst względem tego położenia.

Powrót: Kamery i porty podglądu
Następny: camera_get_view_width
