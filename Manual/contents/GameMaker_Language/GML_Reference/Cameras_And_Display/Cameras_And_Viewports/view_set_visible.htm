view_set_visible

Za pomocą tej funkcji można ustawić widoczność danego portu widoku. Funkcja pobiera indeks portu widoku (od 0 do 7) oraz wartość logiczną true, jeśli jest on widoczny, lub false, jeśli nie jest.

Składnia:

view_set_visible(view_port, visible)

ArgumentTyp Opis
view_port The view port to target (0 - 7)
visible Visibility toggle (true jest widoczny, a false jest niewidoczny)

Zwroty:

N/A

Przykład:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

Powyższy kod sprawdzi, czy port widoku[0] jest widoczny, a jeśli nie jest, zostanie ustawiony na visible.
