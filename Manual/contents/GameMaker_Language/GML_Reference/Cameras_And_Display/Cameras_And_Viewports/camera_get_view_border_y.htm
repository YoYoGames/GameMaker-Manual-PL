camera_get_view_border_y

Za pomocą tej funkcji można pobrać wartość granicy dla obiektu/instancji następującej po podanej kamerze wzdłuż osi y (granica pionowa). Wartość zwracana jest w pikselach.

Składnia:

camera_get_view_border_y(camera_id)

ArgumentTyp Opis
camera_id The unique camera ID value returned when you created the camera

Zwroty:

Przykład:

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
    camera_set_view_border(view_camera[0], 200, 200);
}

Powyższy kod pobiera wartości xborder i yborder kamery przypisanej do portu widoku[0], a następnie sprawdza, czy odpowiadają one podanej wartości. Jeśli nie, to kamera widoku jest ustawiana na podaną wartość.
