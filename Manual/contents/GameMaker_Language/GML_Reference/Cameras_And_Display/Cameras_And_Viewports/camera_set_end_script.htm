camera_set_end_script

Ta funkcja może być użyta do ustawienia funkcji skryptu, która będzie wywoływana na końcu każdej klatki gry, w której kamera jest przypisana do widocznego i aktywnego portu widoku, po tym jak wszystko dla tej kamery zostanie wyrenderowane. Podajesz unikalną wartość ID kamery (zwracaną przez różne camera_create() oraz identyfikator funkcji, która ma zostać wywołana. Kolejność wywoływania funkcji dołączonych do kamer oraz faktycznego renderowania widoku z kamery jest następująca:

  • Kamery dla wszystkich widocznych i aktywnych portów widokowych mają funkcję aktualizacji o nazwie
  • Następnie dla każdego widoku:
    • Zostanie wywołana funkcja rozpoczęcia dla kamery dla danego widoku
    • Dla danego widoku wykonywane są zdarzenia rysowania
    • Zostanie wywołana funkcja zakończenia dla kamery
  • Przejdź do następnego aktywnego widocznego widoku i powtórz czynność...

Składnia:

camera_set_end_script(camera_id, script)

Argument Typ Opis
camera_id Identyfikator kamery The unique camera ID value returned when you created the camera.
script Funkcja skryptu The script to run each game frame

Zwroty:

N/A

Przykład:

camera_set_end_script(view_camera[0], endCamera);

Powyższy kod ustawia koniec script dla kamery przypisanej do portu podglądu [0].
