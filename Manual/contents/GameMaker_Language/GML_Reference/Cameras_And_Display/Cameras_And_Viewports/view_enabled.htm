view_enabled

Ta zmienna kontroluje, czy porty widoku widoczne w witrynie room są włączone, czy nie. Jeśli masz porty widoku ustawione jako widoczne, a następnie wyłączysz tę opcję, cała strona room będzie rysowana na ekranie w skali odpowiadającej rozmiarowi okna, a nie poszczególne kamery będą rysowane przez porty widoku.

Składnia:

view_enabled;

Zwroty:

(true: enabled, false: disabled)

Przykład:

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

Powyższy kod sprawdza, czy widoki są włączone dla room, a jeśli nie są, upewnia się, że port widoku[0] jest widoczny, a następnie włącza widoki dla room.

Powrót: Kamery i porty podglądu
Następny: view_visible
