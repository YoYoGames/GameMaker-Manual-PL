view_get_visible

Ta funkcja służy do sprawdzania widoczności podanego portu widoku. Funkcja zwróci true, jeśli jest on widoczny, i false, jeśli nie jest.

Składnia:

view_get_visible(view_port)

ArgumentTyp Opis
view_port The view port to target (0 - 7)

Zwroty:

Boolean

Przykład:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

Powyższy kod sprawdzi, czy port widoku[0] jest widoczny, a jeśli nie jest, zostanie ustawiony na visible.

Powrót: Kamery i porty podglądu
Następny: view_get_xport
