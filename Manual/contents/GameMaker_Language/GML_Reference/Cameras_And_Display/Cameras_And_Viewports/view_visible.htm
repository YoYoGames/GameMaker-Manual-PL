widok_widoczny

Ta zmienna może być użyta do sprawdzenia, czy dany port widoku jest aktualnie widoczny, czy nie. Można również ustawić tę zmienną, aby efektywnie włączać lub wyłączać widok, ustawiając jej wartość na true (widoczny) lub false (niewidoczny). Należy pamiętać, że nawet jeśli port widoku jest ustawiony jako widoczny, to jeśli porty widoku nie są włączone (za pomocą wbudowanej zmiennej view_enabled lub włączenie ich w edytorze Room ), to nie będą one rysowane na ekranie.

Składnia:

view_visible[0 ... 7];

Zwroty:

(true: enabled, false: disabled)

Przykład:

if !view_visible[0]
{
    view_visible[0] = true;
}

Powyższy kod sprawdza, czy widok[0] jest widoczny, a jeśli nie jest, ustawia go jako widoczny.
