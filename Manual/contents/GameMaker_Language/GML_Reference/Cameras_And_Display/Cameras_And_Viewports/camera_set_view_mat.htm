camera_set_view_mat

Ta funkcja ustawia matrycę widoku dla danej kamery. Użytkownik podaje unikatową wartość ID kamery (zwracaną przez inne camera_create() oraz matrycę widoku, która ma zostać zastosowana. Więcej informacji na temat tworzenia macierzy widoku można znaleźć w rozdziale Funkcje macierzy, a konkretnie matrix_build_lookat().

Należy pamiętać, że jeśli kamera jest automatycznie śledzona przez object - tzn. została utworzona za pomocą camera_create_view() z indeksem object / identyfikatorem instancji, który nie jest równy -1, lub ustawiasz kamerę zdefiniowaną w edytorze Room, lub ustawiasz kamerę domyślną - wówczas macierz ta zostanie nadpisana w następnej klatce gry.

Składnia:

camera_set_view_mat(camera_id, matrix)

ArgumentTyp Opis
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Zwroty:

Przykład:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Powyższy kod tworzy nową kamerę i przypisuje ją do portu widoku[0]. Kamera ta ma następnie ustawione macierze widoku i rzutowania.
