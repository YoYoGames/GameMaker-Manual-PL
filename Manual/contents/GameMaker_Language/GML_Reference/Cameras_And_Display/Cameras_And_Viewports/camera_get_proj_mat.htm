camera_get_proj_mat

Ta funkcja służy do pobierania macierzy projekcji. Funkcja zwraca tablicę macierzy, którą można następnie wykorzystać w innych funkcjach macierzowych lub ustawić macierz projekcji innego aparatu (za pomocą funkcji camera_set_proj_mat()).

Składnia:

camera_get_proj_mat(camera_id)

ArgumentTyp Opis
camera_id The unique camera ID value returned when you created the camera

Zwroty:

Matrix Array

Przykład:

mat = camera_get_proj_mat(camera_view[0]);

Powyższy kod pobiera macierz rzutowania dla kamery przypisanej do portu widoku[0].

Powrót: Kamery i porty podglądu
Następny: camera_get_update_script
