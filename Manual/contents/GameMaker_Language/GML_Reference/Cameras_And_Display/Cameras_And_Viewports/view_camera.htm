widok_kamery

Ta tablica przechowuje unikalny identyfikator kamery przypisany do danego portu widoku. Można ją ustawić na nową kamerę lub odczytać, aby uzyskać bieżącą kamerę, zwracając -1, jeśli nie przypisano żadnej kamery. W witrynie room może być aktywnych maksymalnie 8 portów widoku (wartości tablicy od 0 do 7), a do każdego z nich można przypisać kamerę, po prostu ustawiając tę zmienną na wartość identyfikatora kamery (zwracaną przez funkcje camera_create() lub camera_create_view()). Jeśli ustawiono kamerę w porcie widoku za pomocą edytora Room, zmienna ta będzie zawierać identyfikator kamery dla tego portu.

Składnia:

view_camera[0 ... 7];

Zwroty:

Camera ID or -1 (if no camera is assigned)

Przykład:

if view_camera[0] == -1
{
    view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

Powyższy kod sprawdza, czy do portu widoku [0] jest dołączona kamera, a jeśli nie, to jest ona tworzona i ustawiana na tym porcie.
