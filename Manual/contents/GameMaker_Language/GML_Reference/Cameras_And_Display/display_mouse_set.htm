display_mouse_set

Za pomocą tej funkcji można zmienić lub ustawić położenie myszy na ekranie gry, co może być przydatne na przykład w grach FPS. Funkcja będzie działać tylko wtedy, gdy gra jest w centrum uwagi, a użycie adresu ALT + Tab spowoduje odblokowanie myszy.

Składnia:

display_mouse_set(x, y);

ArgumentTyp Opis
x The x coordinate in the display.
y The y coordinate in the display.

Zwroty:

Przykład:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

Powyższy kod spowoduje wyśrodkowanie myszy na ekranie gry.
