window_set_max_width

Za pomocą tej funkcji można ustawić maksymalną szerokość okna gry. Jeśli włączysz opcję zmiany rozmiaru okna w opcjach gry dla platformy docelowej, gracz będzie mógł zmienić rozmiar okna gry na dowolny, jednak za pomocą tej funkcji możesz ograniczyć maksymalną szerokość do określonego przez siebie rozmiaru. Jeśli chcesz powrócić do domyślnego zachowania (tzn. braku minimum), użyj wartości -1.

UWAGA: Ta funkcja jest dostępna tylko na platformach Windows, macOS i Linux.

Składnia:

window_set_max_width(width);

ArgumentTyp Opis
width The maximum width in pixels for the game window.

Zwroty:

Przykład:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

Powyższy kod ustawi minimalną i maksymalną szerokość oraz wysokość okna gry.
