window_device DEPRECATED

Ta funkcja zwróci bieżący wskaźnik urządzenia d3d, który można następnie (na przykład) przekazać do biblioteki DLL lub Dylib na stronach Windows i macOS.

UWAGA Ta funkcja została wycofana z GameMakera na rzecz os_get_info().

Składnia:

window_device();

Zwroty:

Pointer

Przykład:

gfx_pointer = window_device();

Powyższy kod zapisze wskaźnik urządzenia d3d w zmiennej.

Powrót: Okno gry
Następny: window_view_mouse_get_x
