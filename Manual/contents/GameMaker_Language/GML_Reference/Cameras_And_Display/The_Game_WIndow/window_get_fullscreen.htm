window_get_fullscreen

Ta funkcja zwraca, czy okno gry jest w trybie pełnoekranowym (true), czy nie (false).

Składnia:

window_get_fullscreen();

Zwroty:

Przykład:

if window_get_fullscreen()
{
    draw_text(32, 32, "Fullscreen is ON");
}
else
{
    draw_text(32, 32, "Fullscreen is OFF");
}

Powyższy kod sprawdzi, czy okno jest w trybie pełnoekranowym i w zależności od zwróconej wartości narysuje komunikat tekstowy.
