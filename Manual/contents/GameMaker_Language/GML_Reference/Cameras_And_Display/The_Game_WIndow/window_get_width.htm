window_get_width

Za pomocą tej funkcji można uzyskać bieżącą szerokość (w pikselach) okna gry.

Składnia:

window_get_width();

Zwroty:

Przykład:

ww = window_get_width();

Powyższy kod zapisze bieżącą szerokość okna w zmiennej "ww".

Powrót: Okno gry
Następny: window_get_height
