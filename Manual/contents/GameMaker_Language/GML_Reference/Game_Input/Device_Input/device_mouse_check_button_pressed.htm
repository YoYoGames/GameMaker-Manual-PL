device_mouse_check_button_pressed

Ta funkcja zwraca true lub false w zależności od tego, czy określone przez użytkownika urządzenie zostało "dotknięte" (kliknięte), czy nie. Argument urządzenie odnosi się do numeru dotknięcia, który może wynosić od 0 do n, a maksymalna liczba dotknięć, które można wykryć, zależy w dużym stopniu od używanego urządzenia i systemu operacyjnego, na którym działa (większość urządzeń obsługuje co najmniej 4 jednoczesne dotknięcia). Ta funkcja jest wywoływana tylko raz przez faktyczne naciśnięcie przycisku, a do sprawdzenia przycisków myszy można użyć stałych wymienionych na tej stronie. Należy pamiętać, że strona mb_right będzie wykrywana tylko w przypadku wykrycia podwójnego dotknięcia i przytrzymania go przy drugim dotknięciu (to zachowanie można wyłączyć za pomocą funkcji device_mouse_dbclick_enable()).

Składnia:

device_mouse_check_button_pressed(device, button);

ArgumentTyp Opis
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Zwroty:

Przykład:

if device_mouse_check_button_pressed(0, mb_left)
{
    press = true;
}

Powyższy kod sprawdza, czy urządzenie zostało naciśnięte, i ustawia zmienną na wartość true, jeśli tak się stało.
