gamepad_set_axis_deadzone

Za pomocą tej funkcji można ustawić "martwą strefę" dla osi joysticka. Określamy gniazdo urządzenia, które chcemy ustawić, a następnie ustawiamy wartość z zakresu od 0 do 1. Jeśli wartość wejściowa jest mniejsza od podanej wartości, oś joysticka jest traktowana jako 0. Należy pamiętać, że jest to ustawienie globalne, które będzie miało wpływ na wszystkie osie wszystkich joysticków podłączonych do określonego gniazda urządzenia.

Użycie tej funkcji nie oznacza, że wartość osi rozpocznie się od wartości strefy martwej, ponieważ funkcja gamepad_axis_value() zawsze będzie zwracać znormalizowaną wartość z przedziału od -1 i 0 lub 0 i 1. Na przykład ustawienie strefy martwej na 0,2 będzie oznaczać, że naciśnięcie drążka w prawo zacznie zwracać wartość z przedziału 0 - 1 dopiero po przekroczeniu wartości surowej osi 0,2 - zatem gdy wartość surowa wynosi 0,2, wartość zwracana będzie równa 0, gdy wartość surowa wynosi 0,5, wartość zwracana będzie równa 0,375, a jeśli wartość surowa wynosi 0,9, wartość zwracana będzie równa 0,875.

Składnia:

gamepad_set_axis_deadzone(device, deadzone);

ArgumentTyp Opis
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

Zwroty:

Przykład:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

Powyższy kod ustawi martwą strefę osi joysticka na dowolnym z 4 podłączonych urządzeń na 0,05.
