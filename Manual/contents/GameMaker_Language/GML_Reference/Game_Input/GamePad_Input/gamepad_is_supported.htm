gamepad_jest_obsługiwany

Za pomocą tej funkcji można dowiedzieć się, czy platforma docelowa obsługuje pady do gier (zwraca true), czy nie (zwraca false).

Składnia:

gamepad_is_supported();

Zwroty:

Przykład:

global.GP = gamepad_is_supported();

Powyższy kod sprawdza, czy dany gamepad jest obsługiwany, i zapisuje wartość zwracaną w zmiennej globalnej, aby można ją było sprawdzić w przyszłości.
