gamepad_get_button_threshold

Tej funkcji można użyć do wykrycia bieżącego ustawienia progu przycisków analogowych dla danego urządzenia. Domyślnym progiem dla wszystkich przycisków analogowych jest 0,5, a zakres wynosi od 0 do 1. Próg określa, w którym momencie przycisk jest uznawany za "wciśnięty" w grach, które wymagają, aby działał on jak przycisk cyfrowy.

Składnia:

gamepad_get_button_threshold(device);

ArgumentTyp Opis
device Which gamepad device "slot" to check.

Zwroty:

Przykład:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Powyższy kod sprawdza próg przycisku analogowego gamepada podłączonego do urządzenia "slot" 0 i jeśli nie jest to wartość domyślna 0,5, ustawia ją na tę wartość.
