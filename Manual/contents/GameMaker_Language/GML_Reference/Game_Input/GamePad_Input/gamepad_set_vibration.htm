gamepad_set_vibration

Za pomocą tej funkcji można ustawić wibracje silników gamepada, przy czym dla każdego z silników można wybrać wartość od 0 (brak wibracji) do 1 (pełne wibracje). Należy pamiętać, że ta funkcja nie ma ograniczenia czasowego, dlatego konieczne jest użycie zmiennej lub alarmu, aby switch wyłączyć wibracje (ustawić silniki na 0) po upływie określonego czasu, w przeciwnym razie gamepad będzie wibrować bez końca.

UWAGA: Ta funkcja jest obecnie dostępna tylko dla standardowych modułów docelowych systemu Windows, PS4 i Xbox One.

Składnia:

gamepad_set_vibration(device, left_motor, right_motor);

ArgumentTyp Opis
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

Zwroty:

Przykład:

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

Powyższego kodu można użyć (na przykład) w zdarzeniu kolizji, aby gamepad podłączony do "gniazda" 0 wibrował przez pół sekundy, a po upływie tego czasu ustawiony alarm był używany do switch ponownego wyłączenia go.
