gamepad_remove_mapping

Funkcja ta może być użyta do usunięcia aktualnego mapowania urządzenia z podanego indeksu slotu gamepada. Po wywołaniu tej funkcji, indeks slotu będzie musiał zostać przemapowany przy pomocy funkcji gamepad_test_mapping() jeśli chcemy, aby do poprawnego wykrywania danych wejściowych można było używać stałych gamepada (bezpośrednie dane wejściowe można zawsze uzyskać za pomocą funkcji gamepad_axis/button/hat_count() oraz gamepad_axis/button/hat_value() ).

Składnia:

gamepad_remove_mapping(index);

ArgumentTyp Opis
index Which gamepad index "slot" to remove the mapping from.

Zwroty:

Przykład:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

Powyższy kod usunie mapowanie z podanego slotu indeksu gamepada na podstawie wartości zmiennej.

Powrót: Wejście gamepada
Następny: gamepad_is_supported
