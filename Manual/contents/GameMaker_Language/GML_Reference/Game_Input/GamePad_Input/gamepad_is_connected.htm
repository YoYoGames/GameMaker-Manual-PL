gamepad_jest_podłączony

Ta funkcja zwraca, czy do podanego "gniazda" podłączony jest gamepad (zwraca true), czy nie (zwraca false). Zazwyczaj używasz tej funkcji w połączeniu z funkcją gamepad_get_device_count() aby uzyskać prawidłową liczbę dostępnych gamepadów i/lub "slotów" na gamepady. Należy pamiętać, że może wystąpić niewielkie opóźnienie pomiędzy podłączeniem gamepada przez użytkownika a wykryciem go przez GameMaker (ma to miejsce szczególnie w przypadku kontrolerów podłączonych przez bluetooth).

UWAGA: Na niektórych platformach - zwłaszcza konsolach - funkcja ta może zwracać wartość false natychmiast po podłączeniu/wybraniu gamepada i może wymagać sprawdzenia w alarmie kilka klatek (kroków) później, aby prawidłowo wykryć gamepada.

Składnia:

gamepad_is_connected(numb);

ArgumentTyp Opis
device Which gamepad "slot" to check.

Zwroty:

Przykład:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

Powyższy kod loops przegląda dostępne gamepady (lub gniazda gamepadów), a następnie sprawdza każdy z nich pod kątem podłączonego gamepada. zwrócona wartość jest następnie używana do ustawienia globalnej tablicy na true lub false w celu wykorzystania w przyszłych sprawdzeniach.
