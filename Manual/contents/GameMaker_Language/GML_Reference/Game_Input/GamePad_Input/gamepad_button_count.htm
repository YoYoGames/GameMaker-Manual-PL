gamepad_button_count

Ta funkcja zwraca całkowitą liczbę przycisków dostępnych dla gamepada podłączonego do podanego "gniazda" urządzenia.

Składnia:

gamepad_button_count(device);

ArgumentTyp Opis
device Which gamepad device "slot" to check.

Zwroty:

Przykład:

b_num = gamepad_button_count(0);

Powyższy kod zwróci liczbę przycisków dostępnych na gamepadzie podłączonym do urządzenia "slot" 0 i zapisze tę wartość w zmiennej "b_num".

Powrót: Wejście gamepada
Następny: gamepad_button_value
