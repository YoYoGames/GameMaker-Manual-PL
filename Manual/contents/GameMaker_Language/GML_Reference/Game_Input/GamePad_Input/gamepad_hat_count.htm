gamepad_hat_count

Ta funkcja zwróci całkowitą liczbę kapeluszy dostępnych dla gamepada podłączonego do podanego "slotu" urządzenia. Kapelusze ogólnie odnoszą się do przycisków góra/dół/lewo/prawo. Należy pamiętać, że w przypadku celu Windows kapelusze są dostępne tylko dla kontrolerów DirectInput (czyli od gniazda 4 w górę).

Składnia:

gamepad_hat_count(device);

ArgumentTyp Opis
device Which gamepad device "slot" to check.

Zwroty:

Przykład:

h_num = gamepad_hat_count(4);

Powyższy kod zwróci liczbę kapeluszy dostępnych na gamepadzie podłączonym do urządzenia "slot" 4 i zapisze tę wartość w zmiennej "h_num".
