gamepad_button_check

Ta funkcja zwróci true lub false w zależności od tego, czy dany przycisk gamepada został wykryty jako przytrzymany, czy nie. Jeśli sprawdzany jest przycisk analogowy, wówczas sprawdzanie będzie uwzględniać ustawienie pro gu i zwróci true tylko wtedy, gdy surowa wartość wejściowa przekroczy podany próg (tę surową wartość można uzyskać za pomocą funkcji gamepad_button_value()).

Składnia:

gamepad_button_check(device, button);

Argument Typ Opis
device Real Which gamepad device "slot" to check.
button Przycisk gamepada Stała Which gamepad button constant to check for.

Zwroty:

Boolean

Przykład:

if gamepad_button_check(0, gp_face1)
{
    if canshoot = true
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

Powyższy kod wykryje, czy dolny przycisk górnej tarczy (X na kontrolerze ps3) jest przytrzymywany, a jeśli tak, to wystrzeli instancję "pocisku" i włączy alarm.
