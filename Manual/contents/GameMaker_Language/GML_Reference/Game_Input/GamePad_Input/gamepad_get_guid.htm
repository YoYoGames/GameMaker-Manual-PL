gamepad_get_guid

Za pomocą tej funkcji można pobrać GUID dla gamepada podłączonego do podanego indeksu slotu. Podajemy slot gamepada, który chcemy sprawdzić (od 0 do 11), a funkcja zwróci jeden z poniższych ciągów znaków:

  • "brak", jeśli nie jest dostępny żaden identyfikator GUID lub nie jest podłączony żaden gamepad
  • "indeks urządzenia poza zakresem", jeśli indeks gniazda gamepada jest poza zakresem
  • GUID string, coś w stylu "050000005e040000fd020000ffff3f00", jeśli slot ma przypisany gamepad i można uzyskać dla niego GUID

Funkcja ta jest zwykle używana w połączeniu z gamepad_get_description() do wygenerowania części SDL string wymaganej do remappingu kontrolerów na Windows, Android i macOS.

Uwagi dotyczące platformy
Platforma Notatka
Opera GX Na platformie docelowej Opera GX ta funkcja zawsze będzie zwracać "brak" dla urządzeń bez wejścia bezpośredniego

Składnia:

gamepad_get_guid(index);

ArgumentTyp Opis
index Which gamepad "slot" index to check

Zwroty:

Przykład:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Powyższy kod pobiera identyfikator GUID oraz opis nazwy strings, a następnie łączy je i zapisuje ostateczną wersję string w zmiennej globalnej do przyszłego użytku.
