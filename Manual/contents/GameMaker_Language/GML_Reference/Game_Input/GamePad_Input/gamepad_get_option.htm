gamepad_get_option

Funkcja ta służy do pobierania wartości danej opcji. Podajesz "slot" gamepada do sprawdzenia oraz opcję string, a funkcja zwróci związaną z nią wartość. Dostępne opcje string zależą od platformy, na której jest uruchamiany projekt, zgodnie z poniższą listą:

Opcja Platforma Opis Zwraca
"allow_rotation" tvOS Zwraca informację, czy pilot Siri Remote ma mieć blokadę obrotu (tzn. może być używany tylko w pionie), czy nie. Gdy pilot jest odblokowany, obrócenie go spowoduje zmianę wprowadzanych danych, np. obrót o 90º w lewo sprawi, że "prawa" strona panelu dotykowego powróci do pozycji "w górę" itd. Boolean
"dpad_absolute" tvOS Zwraca, czy panel dotykowy pilota Siri Remote akceptuje wejście bezwzględne, czy względne. W trybie bezwzględnym za środek panelu dotykowego uznaje się pozycję (0,0) - czyli miejsce, w którym spoczywa drążek analogowy lub gdzie znajdowałby się środek dpada - a ruch wokół tej pozycji będzie generował dane wejściowe. Po ustawieniu opcji względnej pozycja (0,0) jest traktowana jako miejsce pierwszego dotknięcia panelu i wszystkie dane wejściowe będą generowane z tej pozycji, a nie ze środka panelu. Boolean

Składnia:

gamepad_get_option(device, option_name);

ArgumentTyp Opis
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

Zwroty:

Boolean, Real, String, or N/A (depends on platform and option)

Przykład:

for(var i = 0; i < 12; ++i;)
{
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
    {
        if gamepad_get_option(i, "allow_rotation") == false
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

Powyższy kod loops przegląda wszystkie gniazda gamepadów i sprawdza, czy na platformie tvOS nie ma pilota "Siri Remote". Jeśli taki pilot zostanie wykryty, ustawia go tak, aby umożliwiał obracanie danych wejściowych.
