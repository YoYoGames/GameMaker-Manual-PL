gamepad_get_device_count

Funkcja ta podaje albo liczbę podłączonych game padów, albo liczbę dostępnych "slotów", do których można podłączyć game pady.

Rzeczywista wartość zwracana zależy od platformy i wewnętrznej konfiguracji tej platformy i jako taka funkcja powinna być używana w połączeniu z funkcją gamepad_is_connected() aby upewnić się co do dokładnej liczby podłączonych klocków w danym momencie.

Składnia:

gamepad_get_device_count();

Zwroty:

Real

Przykład:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i)
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

Powyższy kod loops przez dostępne pady do gier (lub gniazda gamepadów), a następnie sprawdza każdy z nich pod kątem podłączonego gamepada. zwrócona wartość jest następnie używana do ustawienia globalnej tablicy na true lub false do użycia w przyszłych kontrolach.
