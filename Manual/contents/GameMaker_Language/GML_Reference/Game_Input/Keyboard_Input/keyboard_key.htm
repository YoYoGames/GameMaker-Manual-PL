keyboard_key

Za pomocą tej zmiennej można uzyskać kod klawisza, który jest aktualnie wciśnięty. Jeśli w momencie sprawdzania nie zostanie wciśnięty żaden klawisz, zmienna zwróci 0.

Składnia:

keyboard_key;

Zwroty:

Virtual Key Constant

Przykład:

switch (keyboard_key)
{
    case vk_numpad1: gun = weapon[0][0]; break;
    case vk_numpad2: gun = weapon[1][0]; break;
    case vk_numpad3: gun = weapon[2][0]; break;
}

Powyższy kod wykorzystuje wartość zmiennej keyboard_key do ustawienia zmiennej na taką samą wartość jak tablica.

Powrót: Wejście klawiatury
Następny: keyboard_lastkey
