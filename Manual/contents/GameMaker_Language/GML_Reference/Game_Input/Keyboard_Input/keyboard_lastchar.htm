keyboard_lastchar

Ta zmienna przechowuje adres string ostatniego naciśniętego klawisza. Zmienna ta nie jest przeznaczona tylko do odczytu i można ją zmienić, na przykład ustawić na "" (pusta string), jeśli już się nią posługiwano.

Składnia:

keyboard_lastchar;

Zwroty:

Przykład:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

Powyższy kod sprawdza, czy zmienna lastkey jest większa od -1, a jeśli tak, to dodaje do zmiennej "str" wartość ostatniego klawisza jako string.
