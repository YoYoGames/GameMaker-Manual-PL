keyboard_check_released

Za pomocą tej funkcji można sprawdzić, czy klucz został zwolniony, czy nie. W przeciwieństwie do funkcji keyboard_check() funkcja ta będzie uruchamiana tylko raz za każdym razem, gdy klawisz zostanie podniesiony, więc aby została ponownie uruchomiona, klawisz musi zostać najpierw naciśnięty, a następnie ponownie zwolniony. Funkcja przyjmuje wartość kodu klucza zwróconą przez funkcję ord() (tylko wielkie litery od A-Z lub cyfry od 0-9), lub dowolną stałą vk_* wymienioną na głównej stronie Keyboard Input.

Składnia:

keyboard_check_released(key);

ArgumentTyp Opis
key The key to check the released state of.

Zwroty:

Przykład:

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

Powyższy kod sprawdzi, czy klawisz "P" został zwolniony, a jeśli tak, utworzy instancję "obj_Pause".
