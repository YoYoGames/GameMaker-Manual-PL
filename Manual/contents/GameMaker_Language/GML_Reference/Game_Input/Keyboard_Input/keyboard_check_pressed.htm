keyboard_check_pressed

Za pomocą tej funkcji można sprawdzić, czy dany klawisz został naciśnięty, czy nie. W przeciwieństwie do funkcji keyboard_check() funkcja ta będzie uruchamiana tylko raz za każdym razem, gdy klawisz zostanie naciśnięty, więc aby została uruchomiona ponownie, klawisz musi zostać najpierw zwolniony, a następnie ponownie naciśnięty. Funkcja przyjmuje wartość keycode zwróconą przez funkcję ord() (tylko wielkie litery od A-Z lub cyfry od 0-9), lub dowolną stałą vk_* wymienioną na głównej stronie Keyboard Input.

Składnia:

keyboard_check_pressed(key);

Argument Typ Opis
key Stała klucza wirtualnego (vk_*) The key to check the pressed state of.

Zwroty:

Boolean

Przykład:

if keyboard_check_pressed(vk_anykey)
{
    room_goto_next();
}

Powyższy kod powoduje przejście do następnej strony room, jeśli gracz naciśnie dowolny klawisz klawiatury (działa to jak monit "Naciśnij dowolny klawisz, aby kontynuować").

Powrót: Wejście klawiatury
Następny: keyboard_check_released
