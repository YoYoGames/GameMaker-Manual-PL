keyboard_string

Ta zmienna przechowuje adres string zawierający ostatnie (co najwyżej) 1024 znaki wpisane na klawiaturze. Ta strona string będzie zawierać tylko znaki drukowalne, ale poprawnie zareaguje na naciśnięcie klawisza backspace, usuwając ostatni znak. Zmienna ta nie jest przeznaczona tylko do odczytu i można ją zmienić, na przykład ustawić na "" (pusta string), jeśli już się nią zajmowano, a także można używać funkcji String Functions do manipulowania nią. Należy pamiętać, że podczas korzystania z wyświetlanej na ekranie klawiatury wirtualnej tylko ta zmienna będzie aktualizowana na podstawie danych wprowadzanych z klawiatury.

Składnia:

keyboard_string;

Zwroty:

Przykład:

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

Powyższy kod ograniczy długość klawiatury string do 15 znaków, a następnie usunie te, które przekraczają ten limit, kopiując pierwsze piętnaście znaków z powrotem do zmiennej.
