gest_get_tap_count

Ta funkcja służy do sprawdzania, czy liczenie stuknięć jest włączone, czy wyłączone. Funkcja zwróci true, jeśli jest włączona, i false w przeciwnym przypadku. Liczenie stuknięć można włączyć lub wyłączyć za pomocą funkcji gesture_tap_count(). Gdy funkcja jest włączona, każde zdarzenie stuknięcia będzie miało dodatkowy wpis na mapie DS o nazwie "tapcount", który będzie rejestrował liczbę stuknięć. Gdy funkcja jest włączona, oznacza to, że wszystkie zdarzenia stukania będą wyzwalane, np. dwa stuknięcia będą wyzwalały zarówno zdarzenie stuknięcia pojedynczego, jak i podwójnego, przy czym liczba stuknięć dla zdarzenia stuknięcia pojedynczego wynosi 1, a dla zdarzenia stuknięcia podwójnego - 2. Wartość licznika stuknięć zostanie zresetowana do 0 po upływie czasu ustawionego dla wykrycia podwójnego stuknięcia (patrz funkcja gesture_double_tap_time()). Jeśli liczenie stuknięć jest wyłączone, to stuknięcie początkowe nie zostanie zarejestrowane, dopóki nie upłynie czas podwójnego stuknięcia i nie zostanie wykryte drugie stuknięcie. Domyślnie funkcja ta jest włączona.

Składnia:

gesture_get_tap_count();

Zwroty:

Przykład:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

Powyższy kod sprawdza, czy liczenie stuknięć jest włączone, a jeśli nie jest, to jest włączane.
