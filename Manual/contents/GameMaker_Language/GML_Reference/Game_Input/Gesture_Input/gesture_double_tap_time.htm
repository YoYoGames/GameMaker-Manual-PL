gest_double_tap_time

Ta funkcja służy do ustawiania czasu, jaki musi upłynąć między dwoma dotknięciami/kliknięciami, aby wywołać zdarzenie gestu dwukrotnego stuknięcia. Czas jest mierzony w sekundach, a jego wartość domyślna wynosi 0,16.

Składnia:

gesture_double_tap_time(time);

Argument Typ Opis
time Real The time (in seconds) to set for double tap gesture event detection.

Zwroty:

N/A

Przykład:

if gesture_get_double_tap_time() != 0.16
{
    gesture_get_double_tap_time(0.16);
}

Powyższy kod sprawdza, czy czas podwójnego stuknięcia dla gestów jest ustawiony na 0,16 sekundy, a jeśli nie jest, ustawia go na tę wartość.
