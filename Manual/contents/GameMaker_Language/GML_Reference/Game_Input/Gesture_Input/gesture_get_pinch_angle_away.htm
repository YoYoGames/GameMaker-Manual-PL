gest_get_pinch_angle_away

Ta funkcja służy do określania kąta, o jaki musi oddalać się dotyk od innego dotyku, aby potencjalnie można było uruchomić gest uszczypnięcia. Kąt jest mierzony w stopniach, a jego wartość domyślna to 45°.

Składnia:

gesture_get_pinch_angle_away();

Zwroty:

(inches)

Przykład:

if gesture_get_pinch_angle_away() != 45
{
    gesture_pinch_angle_away(45);
}

Powyższy kod sprawdza, czy kąt uszczypnięcia dla gestów jest ustawiony na 45°, a jeśli nie jest, ustawia go na tę wartość
