gest_rotate_angle

Ta funkcja służy do ustawiania kąta, który musi zostać przekroczony przez parę dotknięć, aby zostało wyzwolone gest rozpoczęcia obrotu. Kąt jest mierzony w stopniach, a jego wartość domyślna wynosi 5°.

Składnia:

gesture_rotate_angle();

Zwroty:

(inches)

Przykład:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

Powyższy kod sprawdza, czy kąt obrotu dla gestów jest ustawiony na 5°, a jeśli nie jest, ustawia go na tę wartość.

Powrót: Wprowadzanie za pomocą gestów
Następny: event_data
