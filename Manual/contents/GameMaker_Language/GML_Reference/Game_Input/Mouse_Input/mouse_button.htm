mouse_button

Ta zmienna tylko do odczytu zwraca przycisk myszy, który jest aktualnie wciśnięty (aktualnie, czyli w tym kroku) i może zwrócić dowolną specjalną stałą myszy z wyjątkiem mb_any.

Składnia:

mouse_button;

Zwroty:

Mouse Button Constant (except mb_any)

Przykład:

if mouse_button = mb_left
{
    x -= 1;
}

Przesuwa bieżącą instancję w lewo, jeśli lewy przycisk myszy jest wciśnięty.

Wstecz: Wejście myszy
Następny: mouse_check_button
