mouse_lastbutton

Ta zmienna zwraca ostatni naciśnięty przycisk myszy i może zwrócić dowolną specjalną stałą myszy z wyjątkiem mb_any (można również ustawić tę zmienną na jedną z tych stałych).

Składnia:

mouse_lastbutton;

Zwroty:

Mouse Button Constant (except mb_any)

Przykład:

if mouse_lastbutton = mb_left
{
    x -= 1;
}

Powoduje przesunięcie bieżącej instancji w lewo, jeśli ostatnim naciśniętym przyciskiem był lewy przycisk myszy.

Wstecz: Wejście myszy
Następny: mouse_wheel_up
