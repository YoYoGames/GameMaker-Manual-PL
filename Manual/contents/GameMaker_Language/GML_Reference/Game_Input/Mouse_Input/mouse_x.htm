mysz_x

Ta zmienna tylko do odczytu zwraca bieżącą pozycję myszy w osi x na stronie room.

Składnia:

mouse_x;

Zwroty:

Przykład:

x = median(64, mouse_x, room_width - 64);

Powyższy kod będzie utrzymywał instancję w pozycji x myszy tak długo, jak długo będzie się ona znajdować w granicach 64 pikseli z każdej strony strony strony room.

Wstecz: Wejście myszy
Następny: mouse_y
