mouse_wheel_down

Ta funkcja zwraca true, jeśli kółko myszy jest obracane w dół, a false w przeciwnym przypadku.

Składnia:

mouse_wheel_down();

Zwroty:

Przykład:

if mouse_wheel_down()
{
    y += 10;
}

Powoduje przesunięcie bieżącej instancji w dół ekranu, jeśli kółko myszy zostanie obrócone w górę.

Wstecz: Wejście myszy
Następny: mouse_x
