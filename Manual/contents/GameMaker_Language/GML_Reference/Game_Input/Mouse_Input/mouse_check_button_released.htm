mouse_check_button_released

Ta funkcja zwróci true, jeśli sprawdzany przycisk myszy został zwolniony, lub false, jeśli nie został zwolniony. Funkcja ta zostanie wywołana tylko raz dla każdego przycisku myszy, gdy zostanie on zwolniony, a żeby wywołać ją ponownie, przycisk będzie musiał być ponownie naciśnięty i zwolniony. Przycisk myszy, który ma zostać sprawdzony, należy wybrać z jednej z poniższych stałych:

Składnia:

mouse_check_button_released(numb);

Argument Typ Opis
numb Przycisk myszy Stała Which mouse button constant to check for.

Zwroty:

Boolean

Przykład:

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

Powyższy kod sprawdzi, czy prawy przycisk myszy został zwolniony, a jeśli tak, to ustawi prędkość instancji na dziesiątą część odległości między bieżącą pozycją x/y a pozycją myszy x/y.

Wstecz: Wejście myszy
Następny: mouse_clear
