keyboard_virtual_hide

Tej funkcji można użyć do ukrycia wirtualnej klawiatury na urządzeniu, na którym uruchomiona jest gra. Wywołanie tej funkcji spowoduje wygenerowanie systemowego zdarzenia asynchronicznego, w którym mapa async_load DS zostanie wypełniona następującymi parami klucz/wartość:

  • "event_type" - będzie to "stan klawiatury wirtualnej", gdy zostanie wywołany przez funkcje klawiatury wirtualnej.
  • "screen_height" - wysokość wirtualnej klawiatury (w pikselach). Wartość ta będzie równa 0, jeśli klawiatura jest niewidoczna.
  • "keyboard_status" - aktualny stan klawiatury, zwracany jako jeden z następujących ciągów znaków:
    • "ukrycie"
    • "hidden"
    • "showing"
    • "widoczna"

Składnia

keyboard_virtual_hide();

Zwroty:

Przykład:

if input == true
{
    input = false;
    keyboard_virtual_hide();
}

Powyższy kod ukryje wirtualną klawiaturę systemu operacyjnego, jeśli podana zmienna nie jest ustawiona na false.
