keyboard_virtual_height

Ta funkcja zwraca bieżącą wysokość wirtualnej klawiatury w pikselach, w zależności od rozmiaru wyświetlacza. Jeśli klawiatura nie jest widoczna, zwrócone zostanie 0.

Składnia

keyboard_virtual_height();

Zwroty:

Przykład:

if keyboard_virtual_status() == true
{
    key_h = keyboard_virtual_height();
}

Powyższy kod sprawdzi stan wirtualnej klawiatury systemu operacyjnego i jeśli jest ona widoczna, ustawi zmienną na wysokość klawiatury.
