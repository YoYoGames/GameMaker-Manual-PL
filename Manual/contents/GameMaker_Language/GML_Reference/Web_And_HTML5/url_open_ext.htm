url_open_ext

Spowoduje to otwarcie określonego adresu URL w przeglądarce wybranego urządzenia docelowego lub, jeśli używasz modułu HTML5, w aktualnie otwartej przeglądarce. Parametr "target", który określasz, jest taki sam jak standardowa wartość JavaScript "name", gdy używasz metody open() i powinieneś być świadomy, że wszystkie oprócz "_self" mogą spowodować zablokowanie przez przeglądarkę lub zapytanie użytkownika, czy chce na to pozwolić.

Ważne cele to:

Target Opis
_blank Otwiera linkowany dokument w nowym oknie lub karcie (nie będzie to działać, jeśli wyskakujące okienka są blokowane przez użytkownika, w tym przypadku można użyć funkcji clickable_* zamiast tego).
Siebie Otwiera połączony dokument w tej samej ramce, w której został kliknięty (jest to ustawienie domyślne).
_rodzic Otwiera połączony dokument w ramce nadrzędnej.
Nie. Otwiera połączony dokument w pełnej objętości okna.

Składnia:

url_open_ext(url, target);

Argument Typ Opis
url String The URL (website address) to link to.
target String This is the target area to open the URL in.

Zwroty:

N/A

Przykład:

url_open_ext("http://gamemaker.io", "_blank");

Otworzy to stronę główną GameMakera w nowym oknie.
