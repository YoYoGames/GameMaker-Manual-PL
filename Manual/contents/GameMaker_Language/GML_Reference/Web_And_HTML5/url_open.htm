url_open

Spowoduje to otwarcie określonego adresu URL w przeglądarce wybranego urządzenia docelowego lub, jeśli korzystasz z modułu HTML5, w aktualnie otwartej przeglądarce.

Składnia:

url_open(url);

Argument Typ Opis
url String The URL (website address) to link to.

Zwroty:

N/A

Przykład:

url_open("http://gamemaker.io");

Otworzy to stronę główną GameMakera w bieżącym oknie.
