url_open_full

Spowoduje to otwarcie określonego adresu URL w przeglądarce wybranego urządzenia docelowego lub, jeśli używasz modułu HTML5, w aktualnie otwartej przeglądarce. Parametr "target", który określasz jest taki sam jak standardowa wartość JavaScript "name", gdy używasz metody open() (pamiętaj, że wszystkie oprócz "_self" mogą spowodować zablokowanie przez przeglądarkę lub zapytanie użytkownika, czy chce na to pozwolić), a "options" jest taki sam jak parametr JavaScript "specs" do kontrolowania, jakie właściwości powinno wyświetlać nowe okno / zakładka (nie wszystkie przeglądarki mogą obsługiwać wszystkie funkcje).

Ważne cele to:

Target Opis
_blank Otwiera linkowany dokument w nowym oknie lub karcie (nie będzie to działać, jeśli wyskakujące okienka są blokowane przez użytkownika, w tym przypadku można użyć funkcji clickable_* zamiast tego).
Siebie Otwiera połączony dokument w tej samej ramce, w której został kliknięty (jest to ustawienie domyślne).
_rodzic Otwiera połączony dokument w ramce nadrzędnej.
Nie. Otwiera połączony dokument w pełnej objętości okna.

Ważne opcje to:

Parametry Opis
'height=[px]' Wysokość okna, przy czym minimalna wartość to 100.
'width=[px]' Szerokość okna, przy czym minimalna wartość to 100.
'left=[px]' Lewa pozycja okna.
'top=[px]' Górna pozycja okna (tylko IE).
'location=[boolean]' Czy wyświetlać pole adresu (domyślnie 1).
'menubar=[boolean]' Czy wyświetlać pasek menu, czy nie (domyślnie 1).
'resizable=[boolean]' Czy okno można zmienić rozmiar, czy nie (domyślnie 1).
'scrollbars=[boolean]' Czy wyświetlać paski przewijania (domyślnie 1).
'status=[boolean]' Czy dodać pasek stanu, czy nie (domyślnie 1).
'titlebar=[boolean]' Czy wyświetlać pasek tytułowy czy nie. Jest to ignorowane, chyba że aplikacja wywołująca jest aplikacją HTML lub zaufanym oknem dialogowym (domyślnie 1);
'toolbar=[boolean]' Czy wyświetlać pasek narzędzi przeglądarki (domyślnie tak).

Składnia:

url_open_full(url, target, options);

Argument Typ Opis
url String The URL (website address) to link to.
target String This is the target area to open the URL in (see description).
options String Standard browser options (see description).

Zwroty:

N/A

Przykład:

url_open_full("http://gamemaker.io", "_blank", "resizable=0, height=200, scrollbars=0");

Spowoduje to otwarcie strony głównej GameMakera w nowym oknie, które nie może być zmienione, ma wysokość 200 pikseli i nie ma pasków przewijania.
