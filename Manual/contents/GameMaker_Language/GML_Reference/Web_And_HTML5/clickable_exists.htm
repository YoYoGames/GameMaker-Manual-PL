clickable_exists

Ta funkcja zwraca, czy klikalna ikona DOM została utworzona z podanym indeksem, czy nie. Należy pamiętać, że wartość używana do sprawdzania musi być wcześniej zainicjalizowana, w przeciwnym razie pojawi się błąd powodujący zamknięcie GameMakera.

Składnia:

clickable_exists(index);

Argument Typ Opis
index Identyfikator klikalny The index of the clickable to check.

Zwroty:

Boolean

Przykład:

if !clickable_exists(home_but)
{
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

Powyższy kod sprawdza, czy zmienna "home_but" już istnieje, a jeśli nie, tworzy klikalną ikonę DOM na pozycji (32, 32) strony, na której uruchomiona jest gra. Ikona używa adresu sprite, do którego odwołuje się strona texture jako "spr_MS_Home". Po kliknięciu ikona otworzy nowe okno o podanym adresie URL i zdefiniowanych dla niego właściwościach.
