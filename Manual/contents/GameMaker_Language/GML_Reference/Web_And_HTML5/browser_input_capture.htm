browser_input_capture

Za pomocą tej funkcji można określić, czy okno przeglądarki ma przechwytywać wszystkie dane wejściowe (ustawiamy ją na false), czy też gra ma przechwytywać dane wejściowe (ustawiamy ją na true). Należy pamiętać, że ta funkcja jest przeznaczona wyłącznie do użytku z modułem HTML5.

Składnia:

browser_input_capture(enable);

Argument Typ Opis
enable Boolean Enable or disable game input capture

Zwroty:

N/A

Przykład:

browser_input_capture(true);

Powyższy kod ustawia grę tak, aby przechwytywała wszystkie dane wejściowe przeglądarki na platformie docelowej HTML5.

Powrót: Sieć i HTML5
Następny: webgl_enabled
