mp_linear_step_object

Za pomocą tej funkcji każesz instancji wykonać "krok" w kierunku określonego punktu, określonego przez wartości xgoal i ygoal. Wielkość kroku (czyli o ile pikseli instancja powinna się przesunąć w każdym kroku) jest określana przez wielkość kroku, a jeśli instancja znajduje się już w danej pozycji, to zatrzyma się i nie będzie się dalej poruszać, w przeciwieństwie do innych, prostszych funkcji, takich jak move_towards_point(). Rozmiar kroku to także odległość "przed", o jaką object będzie sprawdzać każdy krok pod kątem kolizji, a Ty możesz wybrać, czy instancja zatrzyma się po kolizji z object lub wybraną przez Ciebie instancją.

Składnia:

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

Argument Typ Opis
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Identyfikatorzasobu lub instancjiobiektu The object that is to block the path. Can be an object index, an instance id or the special keyword, all

Zwroty:

Boolean

Przykład:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Powyższy kod sprawi, że obiekt object będzie się poruszał w kierunku myszy z prędkością 5 pikseli na krok, sprawdzając tylko, czy nie koliduje z obiektem object "obj_Wall". Po osiągnięciu pozycji myszy utworzy on obiekt object "obj_Explosion" i zniszczy się.
