mp_grid_get_cell

Za pomocą tej funkcji możesz sprawdzić dowolną daną komórkę mp_grid, aby zobaczyć, czy została ona oznaczona jako zajęta, czy nie, podając indeks (wcześniej utworzonej) mp_grid i współrzędne x i y komórki do sprawdzenia.

Jeśli została ona zajęta lub sprawdzana pozycja jest poza granicami siatki, funkcja zwróci -1, w przeciwnym razie zwróci 0.

Składnia:

mp_grid_get_cell(id, x , y);

Argument Typ Opis
id Identyfikator siatki MP Index of the mp_grid that is to be used
x1 Real The x coordinate of the grid to check.
y1 Real The y coordinate of the grid to check.

Zwroty:

Real

Przykład:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

Powyższy kod sprawdzi komórkę mp_grid odpowiadającą pozycji myszy i jeśli jest zajęta ustawi zmienną image_blend na kolor czerwony, a jeśli nie jest zajęta ustawi ją na kolor zielony.
