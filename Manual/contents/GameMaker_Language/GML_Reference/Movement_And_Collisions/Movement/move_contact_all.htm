move_contact_all

Ta funkcja przesunie instancję wykonującą kod o zadaną liczbę pikseli w określonym kierunku, aż napotka ona inną instancję z poprawną maską. Możesz użyć -1 lub 0 dla maxdist (domyślnie 1000px), tzn. GameMaker będzie przesuwał instancję o 1000 pikseli w górę, dopóki nie znajdzie się ona poza zasięgiem kolizji.

Składnia:

move_contact_all(dir, maxdist);

ArgumentTyp Opis
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Zwroty:

Przykład:

if !place_meeting(x, y + 1, all)
{
    move_contact_all(270, -1);
}

Powyższy kod sprawdzi, czy pod instancją nie ma kolizji, a jeśli nie ma, to przesunie ją w dół, aż do wystąpienia kolizji lub do pokonania 1000 pikseli.
