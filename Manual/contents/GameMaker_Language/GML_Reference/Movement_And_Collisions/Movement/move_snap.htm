move_snap

Ta funkcja służy do "przyciągania" instancji do siatki o określonym rozmiarze. Zostanie ona przytrzaśnięta do najbliższej odpowiadającej jej pozycji na "niewidzialnej" siatce, którą określają wartości hsnap i vsnap.

Składnia:

move_snap(hsnap, vsnap);

ArgumentTyp Opis
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Zwroty:

Przykład:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

Powyższy kod sprawdza wszystkie instancje "obj_Pieces", czy są one przypięte do siatki 32x32 pikseli, a jeśli nie, to przypina je do najbliższej pozycji w tej siatce.
