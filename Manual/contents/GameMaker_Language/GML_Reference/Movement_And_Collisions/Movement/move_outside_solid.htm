przesuń_na_zewnątrz_solid

Za pomocą tej funkcji możesz powiedzieć instancji, by wyszła z kolizji z dowolną instancją oznaczoną jako solid w dowolnym kierunku i o dowolną liczbę pikseli co krok, przy czym wartość -1 lub 0 dla maxdist to domyślnie 1000px, czyli GameMaker będzie przesuwał instancję o 1000 pikseli w górę, dopóki nie wyjdzie ona z kolizji.

Składnia:

move_outside_solid(dir, maxdist);

ArgumentTyp Opis
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Zwroty:

Przykład:

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

Powyższy kod będzie wchodził do zdarzenia kolizji i sprawdzał, czy "druga" instancja w kolizji jest oznaczona jako "stała". Jeśli tak, to przesunie instancję poza obszar kolizji.
