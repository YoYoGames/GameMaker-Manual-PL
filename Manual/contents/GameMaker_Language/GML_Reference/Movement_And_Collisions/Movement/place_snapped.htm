miejsce_przechwycone

Za pomocą tej funkcji można sprawdzić, czy początek instancji (jej pozycja x i y) jest wyrównany do siatki o podanych przez użytkownika wartościach hsnap i vsnap.

Składnia:

place_snapped(hsnap, vsnap);

ArgumentTyp Opis
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

Zwroty:

Przykład:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

Powyższy kod sprawdza wszystkie instancje "obj_Pieces", czy są one przypięte do siatki 32x32 pikseli, a jeśli nie, to przypina je do najbliższej pozycji w tej siatce.
