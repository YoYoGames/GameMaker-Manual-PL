collision_rectangle_list

Ta funkcja jest taka sama jak funkcja collision_rectangle() z tą różnicą, że zamiast wykrywać tylko jedną instancję kolizji na raz, będzie ona wykrywać wiele instancji. Podajesz pozycje x/y lewego górnego i prawego dolnego rogu obszaru, który ma być sprawdzany, oraz object, a także możesz określić, czy sprawdzanie ma być dokładne (wtedy wszystkie sprawdzane instancje będą musiały mieć dokładne maski kolizji) oraz czy sprawdzanie ma obejmować instancję wywołującą, czy nie.

Podajesz również listę DS, więc wartości id wszystkich instancji, które kolidują z instancją wywołującą, zostaną dodane na koniec podanej listy. Możesz wyczyścić listę przed wywołaniem tej funkcji, tak by zawierała ona tylko wyniki tego wywołania. Możesz także uporządkować instancje według odległości od środka prostokątnego obszaru do ich początków. Funkcja zwraca liczbę znalezionych instancji lub 0, jeśli nie znaleziono żadnej.

Zwróć uwagę, że zamiast indeksu object możesz podać słowo kluczowe instance all, aby sprawdzić wszystkie instancje w bieżącym room, co może obejmować instancję wykonującą kod (zależnie od argumentu notme ).

Składnia:

collision_rectangle_list(x1, y1, x2, y2, obj, prec, notme, list, ordered);

Argument Typ Opis
x1 Real The x coordinate of the left side of the rectangle to check.
y1 Real The y coordinate of the top side of the rectangle to check.
x2 Real The x coordinate of the right side of the rectangle to check.
y2 Real The y coordinate of the bottom side of the rectangle to check.
obj Identyfikatorzasobu lub instancjiobiektu The object to check for instance collisions.
prec Boolean Whether the check is based on precise collisions (trueco jest wolniejsze) lub ogólnie jego ramki (false, szybciej).
notme Boolean Whether the calling instance, if relevant, should be excluded (true) lub nie (false).
list Identyfikator listy DS The DS list to use to store the IDs of colliding instances.
ordered Boolean Whether the list should be ordered by distance (true) lub nie (false).

Zwroty:

Real (The number of instances found to be in collision)

Przykład:

var _list = ds_list_create();
var _num = collision_rectangle_list(x - 100, y - 100, x + 100, y + 100, obj_Enemy, false, true, _list, false);
if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

Powyższy kod sprawdzi prostokątny obszar 100 pikseli wokół pozycji instancji wywołującej pod kątem kolizji z instancjami "obj_Enemy". Jeśli wystąpią kolizje, to utworzona wcześniej lista jest zapętlana i każda instancja, która brała udział w kolizji, jest niszczona.
