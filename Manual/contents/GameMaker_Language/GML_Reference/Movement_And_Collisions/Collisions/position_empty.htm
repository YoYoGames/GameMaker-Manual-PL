position_empty

Ta funkcja sprawdza, czy dana pozycja wchodzi w kolizję z dowolną instancją z ważną maską kolizji w danej pozycji.

Składnia:

position_empty(x, y);

ArgumentTyp Opis
x The x position to check.
y The y position to check.

Zwroty:

Przykład:

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if position_empty(xx, yy)
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

Funkcja ta sprawdzi, czy w losowym miejscu na stronie room nie doszło do kolizji, a jeśli nie, utworzy instancję obj_Bomb.
