collision_rectangle

Collision_rectangle używa pierwszych czterech argumentów (x1,y1,x2,y2) do zdefiniowania obszaru w obrębie bieżącej strony room, a następnie sprawdza, czy jakakolwiek strona object określona przez argument "obj" jest w kolizji z tym obszarem. Kolizja ta może być sprawdzana jako dokładna lub nie, a także można zdecydować się na sprawdzenie instancji wykonującej sam kod lub nie. Weźmy pod uwagę poniższy obrazek:

Tutaj instancja w środku używa prostokąta kolizji do sprawdzania, czy piłka objects. Niebieskie kulki nie mają dokładnego prostokąta ograniczającego i jak widać, nawet jeśli sprite nie dotyka prostokąta, to i tak dojdzie do kolizji (nawet jeśli ustawisz opcję "precise" w funkcji na true), ponieważ prostokąt ograniczający sprite zachodzi na prostokąt kolizji. Z drugiej strony, zielona kula zostanie uwzględniona w kolizji tylko wtedy, gdy rzeczywista sprite będzie zachodzić na prostokąt. Pamiętaj, że aby kolizje precyzyjne zostały uwzględnione, zarówno na stronie object sprite, jak i w funkcji kolizji musi być zaznaczona opcja precise. Należy również zauważyć, że wartością zwracaną przez funkcję może być identyfikator dowolnej z instancji uznanych za kolizyjne.

Zauważ, że zamiast indeksu object można podać instancję id aby sprawdzić konkretną instancję, lub słowa kluczowe instancji all, lub other (w zależności od zdarzenia i bieżącego zakresu kodu).

Składnia:

collision_rectangle(x1, y1, x2, y2, obj, prec, notme);

Argument Typ Opis
x1 Real The x coordinate of the left side of the rectangle to check.
y1 Real The y coordinate of the top side of the rectangle to check.
x2 Real The x coordinate of the right side of the rectangle to check.
y2 Real The y coordinate of the bottom side of the rectangle to check.
obj Identyfikatorzasobu lub instancjiobiektu The object to check for instance collisions.
prec Boolean Whether the check is based on precise collisions (trueco jest wolniejsze) lub ogólnie jego ramki (false, szybciej).
notme Boolean Whether the calling instance, if relevant, should be excluded (true) lub nie (false).

Zwroty:

Instance ID or noone

Przykład:

var inst;
inst = collision_rectangle(50, 50, 200, 100, obj_Ball, false, true);
if inst != noone
{
    with (inst) instance_destroy();
}

Ten krótki kod używa collision_rectangle do sprawdzenia obszaru w room od 50x, 50y (lewy górny prostokąt) do 200x, 200y (prawy dolny prostokąt) dla instancji object o nazwie "obj_ball". Wartość zwracaną zapisuje w zmiennej tymczasowej, która następnie jest sprawdzana, czy wartość ta jest identyfikatorem instancji, czy też słowem kluczowym nikt. Jeśli nie jest to nikt, to używa zapisanego identyfikatora instancji do zniszczenia strony object.
