lista_punktów_kolizji

Ta funkcja jest taka sama jak funkcja collision_point() z tą różnicą, że zamiast wykrywać tylko jedną instancję w kolizji na raz, będzie ona wykrywać wiele instancji. Podajesz pozycję x/y punktu, który ma być sprawdzany, i możesz ustawić, czy sprawdzanie ma być dokładne (wtedy wszystkie sprawdzane instancje będą musiały mieć dokładne maski kolizji) oraz czy sprawdzanie ma obejmować instancję wywołującą, czy nie.

Podajesz również listę DS, więc wartości id wszystkich instancji, które kolidują z instancją wywołującą, zostaną dodane na koniec podanej listy. Możesz wyczyścić listę przed wywołaniem tej funkcji, tak by zawierała ona tylko wyniki tego wywołania. Masz także możliwość uporządkowania instancji według ich odległości od podanego punktu do ich początków. Funkcja zwraca liczbę znalezionych instancji lub 0, jeśli nie znaleziono żadnej.

Zwróć uwagę, że zamiast indeksu object możesz podać słowo kluczowe instance all, aby sprawdzić wszystkie instancje w bieżącym room, co może obejmować instancję wykonującą kod (zależnie od argumentu notme ).

Składnia:

collision_point_list(x, y, obj, prec, notme, list, ordered);

Argument Typ Opis
x Real The x coordinate of the point to check.
y Real The y coordinate of the point to check.
obj Identyfikatorzasobu lub instancjiobiektu The object to check for instance collisions.
prec Boolean Whether the check is based on precise collisions (trueco jest wolniejsze) lub ogólnie jego ramki (false, szybciej).
notme Boolean Whether the calling instance, if relevant, should be excluded (true) lub nie (false).
list Identyfikator listy DS The DS list to use to store the IDs of colliding instances.
ordered Boolean Whether the list should be ordered by distance (true) lub nie (false).

Zwroty:

Real (The number of instances found to be in collision)

Przykład:

var _list = ds_list_create();
var _num = collision_point_list(x, y, obj_Enemy, false, true, _list, false);
if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

Powyższy kod sprawdzi pozycję instancji wywołującej pod kątem kolizji z instancjami "obj_Enemy". Jeśli są jakieś kolizje, to utworzona lista jest zapętlana i każda instancja, która była w kolizji, jest niszczona.
