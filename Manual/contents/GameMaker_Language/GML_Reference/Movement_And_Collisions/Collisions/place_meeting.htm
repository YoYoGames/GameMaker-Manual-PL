miejsce_spotkania

Za pomocą tej funkcji możesz sprawdzić, czy pozycja nie koliduje z inną instancją lub wszystkimi instancjami obiektu object, używając maski kolizji instancji, która uruchamia kod sprawdzający. Używając tej funkcji, prosisz program GameMaker o przesunięcie instancji do nowej pozycji, sprawdzenie kolizji, cofnięcie się i poinformowanie Cię, czy kolizja została znaleziona, czy nie. Działa to w przypadku precyzyjnych kolizji, ale tylko wtedy, gdy zarówno instancja, jak i sprawdzana strona object mają zaznaczone maski precyzyjnych kolizji, w przeciwnym razie stosowane są tylko kolizje typu bounding box. Ta funkcja akceptuje także specjalne słowa kluczowe all i other. Zwróć uwagę, że jeśli chcesz uzyskać unikalny identyfikator instancji object, z którą się zderzasz, powinieneś użyć instance_place().

Należy pamiętać, że podane współrzędne x/y zostaną spolszczone do najbliższej liczby całkowitej przed wykonaniem sprawdzenia.

Składnia:

place_meeting(x, y, obj);

Argument Typ Opis
x Real The x position to check.
y Real The y position to check.
obj Identyfikatorzasobu lub instancjiobiektu The instance or object to check for.

Zwroty:

Boolean

Przykład:

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

Powyższy kod sprawdza, czy po lewej stronie instancji nie występuje kolizja, i przesuwa instancję, jeśli jej nie ma.

Powrót: Kolizje
Następny: position_empty
