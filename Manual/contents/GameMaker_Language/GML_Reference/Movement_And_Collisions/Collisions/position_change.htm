pozycja_zmiana

Funkcja ta sprawdza pozycję pod kątem kolizji z dowolnymi instancjami w podanym punkcie, a jeśli taka kolizja występuje, zmienia wszystkie instancje w kolizji na instancje wybranego obiektu object. Możesz ustawić argument "perf" na true, co zmusi GameMakera do wykonania zdarzeń Destroy i Clean Up znalezionej instancji oraz zdarzenia Create nowej instancji, lub false, by nie wykonywać żadnego z tych zdarzeń. Należy pamiętać, że jeśli nie wykonasz zdarzeń Destroy, Clean Up i Create, każda utworzona instancja, która używa zmiennej normalnie zdefiniowanej w zdarzeniu create, spowoduje awarię gry, ponieważ zmienna ta nie będzie już istnieć.

Składnia:

position_change(x, y, obj, perf);

Argument Typ Opis
x Real The x coordinate of where to change colliding instances.
y Real The y coordinate of where to change colliding instances.
obj Obiekt Aktywa The new object the calling object will change into.
perf Boolean Whether to perform that new object's Create event (true) or not (false).

Zwroty:

N/A

Przykład:

position_change(200, 200, obj_Bird, true);

To zmieni wszystkie instancje zderzające się w punkcie (200,200) w instancję "obj_Bird", wykonując dla każdej z nich zdarzenie "obj_Bird "Create.

Powrót: Kolizje
Następny: position_destroy
