Facebook

Facebook jest jednym z najważniejszych portali społecznościowych w Internecie, a wiele gier i twórców oprogramowania ma obecnie swoje własne strony, na których umieszcza gry i multimedia. GameMaker posiada stronę extension, która umożliwia połączenie z Facebookiem i wykonywanie pewnych czynności i zadań, na które pozwala API tego serwisu. Przed rozpoczęciem korzystania z extension zaleca się odwiedzenie strony dla programistów Facebooka, ponieważ dokładne zrozumienie sposobu działania Facebook Graph oraz dostępnych połączeń i uprawnień jest niezbędne, aby w pełni wykorzystać te funkcje.

Informacje o tym, jak skonfigurować i używać Facebooka extension na wszystkich dostępnych platformach, można znaleźć pod następującym linkiem:

  • Android, iOS i HTML5: Integracja z Facebookiem

Poniżej wymieniono przestarzałe funkcje, które działały tylko w starszym interfejsie API Facebooka. W związku z tym nie można ich już używać w projektach. Zostały one wymienione tutaj na wypadek, gdybyś zaimportował stary projekt zawierający te funkcje. Aby korzystać z aktualnych funkcji Facebooka, musisz aktywować API Facebooka w odpowiednich opcjach gry dla platformy docelowej, co spowoduje pobranie i zainstalowanie extension. Do extension dołączony jest "minipodręcznik" w formacie *.pdf, w którym wyjaśniono, jak z niego korzystać, a także przedstawiono wszystkie definicje funkcji i przykłady. Wszystkie odniesienia do starych funkcji facebook_*() będą musiały zostać zastąpione (i zaktualizowane), aby używać odpowiedników extension fb_*().

Wymienione poniżej funkcje służą do integracji gry z serwisem Facebook:

  • facebook_init
  • facebook_login
  • facebook_request_publish_permissions
  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
