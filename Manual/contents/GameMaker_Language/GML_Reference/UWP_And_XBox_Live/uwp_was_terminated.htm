uwp_was_terminated

Za pomocą tej funkcji można dowiedzieć się, czy gra została wcześniej wyłączona przez system (tzn. czy nie jest to pierwsze uruchomienie gry). Jeśli tak było, funkcja zwróci true, w przeciwnym razie zwróci false. Przykładem użycia może być sprawdzenie, czy gra została wcześniej zakończona przez system, a następnie (jeśli zwróci true) wczytanie poprzedniego etapu gry, na którym znajdował się gracz. W przypadku kierowania na stronę XBox One, wymagane zachowanie gry w przypadku wykrycia tego faktu będzie różne w różnych wersjach dokumentów certyfikacyjnych, dlatego warto sprawdzić najnowszą wersję, aby dowiedzieć się, co powinna zrobić gra.

Składnia:

uwp_was_terminated();

Zwroty:

Przykład:

if xboxlivewas_terminated()
{
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
}

Powyższy kod sprawdza, czy aplikacja nie została już wcześniej zakończona przez system, a jeśli tak, to ładuje stronę buffer i ustawia zmienną globalną.
