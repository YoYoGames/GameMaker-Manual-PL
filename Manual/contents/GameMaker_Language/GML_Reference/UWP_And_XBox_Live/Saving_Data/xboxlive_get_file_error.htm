xboxlive_get_file_error

Ta funkcja zwróci stałą (pokazaną poniżej), która pomoże wyjaśnić, dlaczego funkcja pliku zawiodła, tak abyś mógł podjąć odpowiednie działania w kodzie gry. Gdy jedna z funkcji pliku ulegnie awarii, możesz wywołać tę funkcję, aby dowiedzieć się, na czym dokładnie polegał błąd.

UWAGA: W przypadku korzystania z funkcji asynchronicznego zapisu/wczytywania, mapa async_load mapa będzie miała dodatkowy klucz "error", który będzie zawierał jedną ze stałych (pokazanych poniżej).

Składnia:

xboxlive_get_file_error();

Zwroty:

Xbox File Error Constant

Stała błędu pliku Xbox
Stała Opis
xboxlive_fileerror_outoflocalstorage Jeśli taka sytuacja wystąpi podczas zapisywania gry, oznacza to, że cała dostępna przestrzeń na dysku została wykorzystana, a użytkownik odrzucił ofertę zwolnienia miejsca (przedstawioną przez system operacyjny). Gra powinna potwierdzić użytkownikowi, że zamierza kontynuować grę bez zapisywania. Jeśli użytkownik nadal chce mieć możliwość zapisywania, gra powinna ponowić próbę zapisania gry.
xboxlive_fileerror_quotaexceeded W takim przypadku gra powinna powiadomić użytkownika, że nie może on zapisać więcej danych, dopóki nie zwolni trochę miejsca, a następnie przedstawić mu sposób, w jaki może to zrobić.
xboxlive_fileerror_noerror Funkcja zapisywania i wczytywania plików przebiegała bez problemów.

Przykład:

buffer_save(buff, "SaveGame.bin");
if (xboxlive_get_file_error() != xboxlive_fileerror_noerror)
{
    show_debug_message("Save failed. Error code = " + string(xboxlive_get_file_error()));
}

Powyższy kod sprawdza, czy wystąpił błąd podczas zapisywania danych, a jeśli tak, to kod błędu jest wypisywany na konsolę.
