xboxlive_stats_set_stat_real

Ta funkcja służy do ustawiania wartości statystyki dla podanego identyfikatora użytkownika. Podaje się identyfikator użytkownika zwrócony przez (na przykład) funkcję xboxlive_get_user(), następnie stat string do ustawienia(jeśli stat string jeszcze nie istnieje, to zostanie utworzony nowy stat i ustawiony na podaną wartość) oraz wartość (rzeczywistą), na którą ma zostać ustawiony stat. Należy pamiętać, że nazwa statystyki musi być wyłącznie alfanumeryczna, bez symboli i spacji.

Podczas ustawiania wartości statystyk wszystkie poprzednie wartości zostaną zastąpione, dlatego do Ciebie należy ustalenie, czy wartość statystyk powinna zostać zaktualizowana, czy nie (np. sprawdzenie, czy wysoki wynik jest rzeczywiście najwyższy) poprzez porównanie aktualnej wartości statystyk z nową wartością przed jej ustawieniem.

Funkcja zwróci -1, jeśli wystąpił błąd lub identyfikator użytkownika jest nieprawidłowy, lub dowolną inną wartość, jeśli funkcja została pomyślnie wywołana.

Składnia:

xboxlive_stats_set_stat_real(user_id, stat, value);

ArgumentTyp Opis
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (a real)

Zwroty:

Przykład:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Powyższy kod sprawdza zmienną i jeśli jest to true, to sprawdza wartość statystyki "PercentDone". Jeśli wartość ta jest mniejsza niż 100, generowana jest wartość i statystyka ustawiana jest na tę wartość.
