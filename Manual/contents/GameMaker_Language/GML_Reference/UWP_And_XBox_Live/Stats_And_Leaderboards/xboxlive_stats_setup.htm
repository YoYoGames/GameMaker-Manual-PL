xboxlive_stats_setup

Funkcja ta musi zostać wywołana przed użyciem jakichkolwiek innych funkcji statystyk Xbox Live, a jej zadaniem jest po prostu zainicjowanie wymaganych bibliotek w systemie. Argument "user_id" to nieprzetworzony identyfikator użytkownika zwrócony przez funkcję xboxlive_get_user()natomiast argumenty "service_config" i "title_id" to unikalne identyfikatory Twojej gry w Centrum Rozwojowym Xbox Live.

Ta funkcja jest przeznaczona wyłącznie do stosowania w statystykach opartych na zdarzeniach (2013).

Składnia:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Argument Typ Opis
user_id Identyfikator użytkownika konsoli Xbox The ID pointer of the user to check.
service_config_id String This is the config_id string for the game.
title_id String The unique ID for your game on the Xbox Dev Center.

Zwroty:

N/A

Przykład:

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Powyższy kod inicjalizuje system statystyk dla podanego identyfikatora użytkownika.
