xboxlive_achievement_load_leaderboard

Ta funkcja wysyła zapytanie do server o informacje na temat podanego leaderboard. Tablica musi być utworzona wcześniej w pulpicie deweloperskim Twojej gry, a podany identyfikator jest tym, który musisz podać jako pierwszy argument funkcji. Wywoła ona wywołanie zwrotne Social Asynchronous Event, które zawiera async_load zawierającą mapę wypełnioną odpowiednimi parami klucz/wartość. Klucz id na stronie DS Map jest używany do identyfikacji właściwego wywołania zwrotnego (dla danego zdarzenia asynchronicznego może istnieć więcej niż jedna funkcja wyzwalająca) i będzie sparowany ze stałą xboxlive_achievement_leaderboard_info oraz kilkoma innymi parami klucz-wartość dla każdego gracza. Dokładna zawartość mapy jest przedstawiona poniżej:

  • "id" - w przypadku tej funkcji powinno być xboxlive_achievement_leaderboard_info
  • "leaderboardid" - Unikalny identyfikator tablicy liderów zdefiniowany na pulpicie nawigacyjnym dostawcy.
  • "numentries" - liczba wejść do tabeli liderów, które otrzymałeś.
  • "PlayerN" - Nazwa gracza, gdzie "N" jest liczbą całkowitą odpowiadającą jego pozycji na liście graczy w tabeli rankingowej.
  • "PlayeridN" - unikalny identyfikator użytkownika gracza, "N".
  • "RankN" -Ranga gracza "N" w tabeli rankingowej.
  • "ScoreN" - wynik gracza "N".

Można określić dokładną część tabeli liderów, która ma zostać wyświetlona, podając minimalną i maksymalną wartość indeksu, a także zażądać filtrowania wyników, aby wyświetlić tylko wyniki znajomych lub wyniki wszystkich graczy, używając następujących stałych filtra:

Filtr osiągnięć Xbox Live Stała
Stała Opis
xboxlive_achievement_filter_friends_only  Pobierz tylko wyniki znajomych graczy.
xboxlive_achievement_filter_all_players  Uzyskaj wszystkie wyniki dla wszystkich graczy.

Składnia:

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

Argument Typ Opis
ident String The unique ID of the leaderboard as shown on the developer dashboard.
minindex Real The starting index value to get the leaderboard data from.
maxindex Real The maximum index value to get the leaderboard data from.
filter Filtr osiągnięć Xbox Live Stała Whether to filter the results to those that are on your friends list or to all players (see the Description above)

Zwroty:

N/A

Przykład rozszerzony:

Poniższy kod zostałby prawdopodobnie wywołany po zalogowaniu się gracza na swoje konto w grze, aby wyświetlić listę wszystkich znajomych, którzy znaleźli się na liście liderów w danej grze:

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

Spowoduje to wysłanie zapytania o informacje na temat danego leaderboard i wygenerowanie asynchronicznego wywołania zwrotnego ze specjalną mapą async_load DS zawierającą następujące dane:

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_leaderboard_info
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++;)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

Powyższy kod sprawdza zwróconą stronę DS Map w Społecznym zdarzeniu asynchronicznym i jeśli jej "id" pasuje do sprawdzanej stałej, sprawdza, czy wysyłana tablica liderów pasuje do oryginalnego wywołania funkcji, a następnie wykonuje pętlę po mapie i zapisuje wszystkie różne wartości w kilku tablicach.
