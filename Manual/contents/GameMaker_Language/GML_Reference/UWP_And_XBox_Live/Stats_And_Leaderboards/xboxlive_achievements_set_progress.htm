xboxlive_achievements_set_progress

Ta funkcja służy do uaktualniania postępów w zdobywaniu osiągnięć. Podajesz identyfikator użytkownika zwrócony np. przez funkcję xboxlive_get_user(), a następnie łańcuch osiągnięcia (jest to identyfikator osiągnięcia przypisany w XDP/Windows Dev Center podczas jego tworzenia). Na koniec ustawiasz wartość postępu na wartość z przedziału od 0 do 100. Zauważ, że system osiągnięć odrzuci aktualizacje, które są niższe niż aktualna wartość postępu wartość postępu.

Składnia:

xboxlive_achievements_set_progress(user_id, achievement, progress);

ArgumentTyp Opis
user_id The user ID of the user to set the achievement for
achievement The achievement to set (a string)
progress The progress value to set (from 0 to 100)

Przykład rozszerzony:

Poniżej znajduje się rozszerzony przykład użycia tej funkcji. Na początek należy ją wywołać w jakimś zdarzeniu, takim jak Rozpoczęcie lub Utworzenie pokoju:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Powyższy kod zostanie wywołany w celu uzyskania listy wszystkich pozycji w globalnej tabeli liderów w grze i wygeneruje wywołanie zwrotne zdarzenia asynchronicznego społecznościowego, którym zajmiemy się w następujący sposób:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Powyższy kod tworzy wartość procentową na podstawie pewnych zmiennych globalnych, a następnie używa jej do ustawienia postępu osiągnięcia dla osiągnięcia "Game_Completed".
