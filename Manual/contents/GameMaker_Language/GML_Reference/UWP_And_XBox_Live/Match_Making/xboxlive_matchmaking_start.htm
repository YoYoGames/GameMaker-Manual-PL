xboxlive_matchmaking_start

Ta funkcja musi zostać wywołana przed rozpoczęciem sesji dla wielu graczy dla danego użytkownika. Pobiera ona wskaźnik ID użytkownika, który można pobrać dla konkretnego modułu gry, wywołując funkcję xboxlive_user_for_pad() i inicjalizuje API gry wieloosobowej.

Składnia:

xboxlive_matchmaking_start(user_id);

ArgumentTyp Opis
user_id The user ID pointer to use

Zwroty:

Przykład:

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

Powyższy kod pobiera identyfikator użytkownika dla podanego indeksu klocka, a następnie inicjalizuje możliwości gry wieloosobowej.
