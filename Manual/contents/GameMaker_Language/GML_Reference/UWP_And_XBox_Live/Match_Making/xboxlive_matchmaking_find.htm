xboxlive_matchmaking_find

Ta funkcja wyszuka sesję dla wielu graczy przy użyciu szablonu sesji, lejka dopasowującego i szablonu Secure Device Association, które należy wcześniej utworzyć w pulpicie nawigacyjnym XDP. Identyfikator użytkownika, który można pobrać dla konkretnego padu do gry, można uzyskać, wywołując funkcję xboxlive_user_for_pad() funkcji. Funkcja zwróci unikalną wartość Request ID, która może być użyta do zidentyfikowania właściwego zdarzenia Social Asynchronous Event dla tej funkcji. Zdarzenie to zostanie wywołane po utworzeniu sesji i będzie zawierało DS Map w zmiennej async_load zawierającą następujące pary klucz/wartość:

  • "requestid" - zawiera identyfikator wniosku zwrócony przez funkcję wywołującą
  • "status" - zawiera adres string "session_created", informujący o rodzaju zdarzenia, które zostało wywołane.
  • "num_results" - zawiera 1 lub więcej, jeśli znaleziono sesję, w przeciwnym razie zawiera 0.
  • "sessionid<index> " - zawiera identyfikator sesji lub -1 w przypadku niepowodzenia(<index/gt; jest wartością od 0 do "num_results" - 1).
  • "sessionOwner<index> " - zawiera identyfikator gospodarza sesji, jeśli sesja została znaleziona(<index> jest wartością z zakresu od 0 do "num_results" - 1).
  • "error" - zawiera 0 w przypadku powodzenia, lub -1 w przypadku niepowodzenia

Składnia:

xboxlive_matchmaking_find(user_id, template, hopper, sdatemplate, [matchattributes]);

Argument Typ Opis
user_id Identyfikator użytkownika konsoli Xbox The user ID pointer to use
template String The name of the session template
hopper String The name of the matchmaking hopper
sdatemplate String The name of the secure device association template
matchattributes Real OPTIONAL Atrybuty meczu

Zwroty:

Async Request ID

Przykład:

var userID = xboxone_user_for_pad(global.PadIndex);
result = xboxone_matchmaking_find(userID, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Powyższe polecenie pobierze identyfikator użytkownika na podanym gamepadzie, a następnie spróbuje znaleźć dla niego sesję kojarzenia.
