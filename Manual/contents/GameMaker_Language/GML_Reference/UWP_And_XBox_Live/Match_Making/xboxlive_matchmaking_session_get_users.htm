xboxlive_matchmaking_session_get_users

Ta funkcja utworzy i wypełni mapę DS szczegółami dotyczącymi użytkowników w określonej sesji lub -1, jeśli wystąpił błąd. Wskaźnik do identyfikatora sesji można uzyskać z mapy async_load mapy DS, która jest zwracana w asynchronicznym zdarzeniu społecznościowym po utworzeniu lub znalezieniu sesji (zob. xboxlive_matchmaking_create() więcej szczegółów). Zwrócona mapa DS będzie zawierała następujące pary klucz-wartość:

  • "num_results" - zawiera liczbę użytkowników w sesji
  • "userId<index>" - zawiera identyfikator użytkownika (<index> jest wartością z zakresu od 0 do "num_results" - 1)
  • "userIsOwner<index>" - zawiera 1, jeśli użytkownik jest hostem, w przeciwnym razie 0 (<index> jest wartością od 0 do "num_results" - 1)

Składnia:

xboxlive_matchmaking_session_get_users(session_id);

ArgumentTyp Opis
session_id The session ID pointer to use

Zwroty:

Async Request ID

Przykład:

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
{
    for (var i = 0; i < session_map[? "num_results"]; i++)
    {
        if (session_map[? "userIsOwner" + string(i)] == 1)
        {
            global.OwnerID = session_map[? "userId" + string(i)];
        }
    }
}

Powyższe działanie spowoduje pobranie danych wszystkich użytkowników uczestniczących w sesji kojarzenia i ustawienie zmiennej globalnej na identyfikator właściciela sesji.
