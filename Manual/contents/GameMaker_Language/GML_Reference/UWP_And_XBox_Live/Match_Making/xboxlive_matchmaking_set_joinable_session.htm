xboxlive_matchmaking_set_joinable_session

Ta funkcja ustawia (utworzoną wcześniej) sesję jako dostępną dla innych użytkowników, którzy mogą się do niej przyłączyć za pośrednictwem interfejsu użytkownika systemu. Użytkownik ma tylko jedną sesję, do której może się przyłączyć na raz, a kiedy opuści sesję (lub sesja się zakończy), zostanie ona usunięta, jednak można ją usunąć ręcznie, przekazując -1 jako argument sesji.

Składnia:

xboxlive_matchmaking_set_joinable_session(local_user, session_that_is_joinable);

ArgumentTyp Opis
local_user The local user ID pointer.
session_that_is_joinable The session ID to make joinable, or -1.

Zwroty:

Przykład:

if (global.session_ID != -1)
{
    xboxlive_matchmaking_set_joinable_session(xboxlive_user_for_pad(0), global.session_ID);
}

Powyższy kod sprawdza, czy istnieje poprawna wartość identyfikatora sesji, a jeśli takowa zostanie wykryta, ustawia sesję jako możliwą do dołączenia.
