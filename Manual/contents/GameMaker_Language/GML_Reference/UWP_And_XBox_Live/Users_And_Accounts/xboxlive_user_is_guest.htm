xboxlive_user_is_guest

Ta funkcja zwróci true, jeśli podany wskaźnik ID użytkownika jest użytkownikiem-gościem, i false, jeśli nie jest. Wskaźnik ID użytkownika można uzyskać za pomocą funkcji xboxlive_get_user().

Składnia:

xboxlive_user_is_guest(user_id);

ArgumentTyp Opis
user_id The ID pointer of the user to check.

Zwroty:

Przykład:

if xboxlive_user_is_guest(user_id[0])
{
    global.user_name[0] = "GUEST";
}
else
{
    global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]);
}

Powyższe polecenie przechowuje wskaźnik identyfikatora użytkownika aktywującego w zmiennej globalnej.
