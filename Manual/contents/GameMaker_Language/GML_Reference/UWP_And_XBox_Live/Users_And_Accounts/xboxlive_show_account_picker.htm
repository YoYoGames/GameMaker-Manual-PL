xboxlive_show_account_picker

Funkcja otworzy ekran wyboru konta XBox, aby użytkownik mógł się zalogować, jeśli nie jest jeszcze zalogowany (jeśli jest zalogowany, funkcja nic nie zrobi). Pamiętaj, że ta funkcja ma zastosowanie tylko w przypadku korzystania z wersji UWP na PC, a nie na konsoli Xbox, ponieważ aby uruchomić grę na konsoli Xbox, użytkownik musi być już zalogowany.

Składnia:

xboxlive_show_account_picker();

Zwroty:

Przykład:

if !xboxlive_user_is_signed_in()
{
    if !xboxlive_user_is_signing_in()
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

Powyższy kod sprawdza, czy użytkownik jest zalogowany do usługi XBox Live. Jeśli jest zalogowany, kod pobiera gamertag użytkownika i zapisuje go w zmiennej globalnej do wykorzystania w przyszłości. Jeśli użytkownik nie jest zalogowany, kod sprawdza, czy jest w trakcie logowania - w takim przypadku nic więcej się nie dzieje, a jeśli nie jest, otwiera okno wyboru konta, zachęcając do zalogowania się.
