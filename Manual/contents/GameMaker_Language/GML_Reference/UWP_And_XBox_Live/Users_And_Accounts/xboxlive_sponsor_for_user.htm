xboxlive_sponsor_for_user

Ta funkcja umożliwia uzyskanie identyfikatora sponsora podanego użytkownika.

Składnia:

xboxlive_sponsor_for_user(user_id);

Argument Typ Opis
user_id Identyfikator użytkownika konsoli Xbox The ID pointer of the user to check.

Zwroty:

Xbox Sponsor ID

Przykład:

var _uid = xboxlive_user_for_pad(0);
sponsor = xboxlive_sponsor_for_user(_uid);

Powyższy kod pobiera wskaźnik identyfikatora użytkownika przypisanego do gniazda gamepada [0], a następnie używa tego identyfikatora do pobrania identyfikatora sponsora użytkownika.
