xboxlive_get_activating_user

Za pomocą tej funkcji można pobrać wskaźnik ID użytkownika, który uruchomił grę z konsoli.

UWAGA: Sposób użycia tego wskaźnika zależy od tego, co chcesz zrobić w grze, ale nie powinieneś polegać na tym, że będzie to identyfikator aktualnie grającego użytkownika, ponieważ użytkownik aktywujący może się wylogować w trakcie gry.

Składnia:

xboxlive_get_activating_user();

Zwroty:

Xbox User ID

Przykład:

global.activUser = xboxlive_get_activating_user();

Powyższe polecenie przechowuje wskaźnik identyfikatora użytkownika aktywującego w zmiennej globalnej
