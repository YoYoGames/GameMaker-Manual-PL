xboxlive_get_user_count

Za pomocą tej funkcji można określić całkowitą liczbę użytkowników zalogowanych do systemu XBox. Zwrócona zostanie liczba całkowita z zakresu 0 - N, gdzie N to liczba zalogowanych użytkowników -1.

Składnia:

xboxlive_get_user_count();

Zwroty:

Przykład:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

Powyższe polecenie loops przegląda wszystkich zalogowanych użytkowników i zapisuje ich unikalne identyfikatory w tablicy.
