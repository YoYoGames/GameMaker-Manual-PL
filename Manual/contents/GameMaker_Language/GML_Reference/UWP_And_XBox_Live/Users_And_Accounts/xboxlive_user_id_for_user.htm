xboxlive_user_id_for_user

Ta funkcja przekształca wskaźnik identyfikatora użytkownika na adres string.

Składnia:

xboxlive_user_id_for_user(user_id);

ArgumentTyp Opis
user_id The ID pointer of the user to check.

Zwroty:

Przykład:

var _uid = xboxlive_user_for_pad(0);
uid_string = xboxlive_user_id_for_user(_uid);

Powyższy kod pobiera wskaźnik identyfikatora użytkownika przypisanego do gniazda gamepada [0], a następnie zapisuje ten identyfikator użytkownika jako string do zmiennej instancji.
