uwp_was_closed_by_user

Za pomocą tej funkcji można dowiedzieć się, czy gra została wcześniej zamknięta przez użytkownika podczas gry (tzn. czy została zakończona poprawnie, a nie przez system). Jeśli tak, to funkcja zwróci true, w przeciwnym razie zwróci false.

Składnia:

uwp_was_closed_by_user();

Zwroty:

Przykład:

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

Powyższy kod sprawdza, czy aplikacja została wcześniej zamknięta przez użytkownika, a jeśli tak, to ustawia zmienną globalną.
