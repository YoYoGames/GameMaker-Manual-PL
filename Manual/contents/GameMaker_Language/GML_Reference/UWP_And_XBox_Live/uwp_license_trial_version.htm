uwp_license_trial_version

Za pomocą tej funkcji można sprawdzić, czy gra jest objęta licencją próbną, czy nie. Jeśli tak, funkcja zwróci true, a w przeciwnym razie false.

Składnia:

uwp_license_trial_version();

Zwroty:

Przykład:

if uwp_license_trial_version()
{
    global.LevelCap = 10;
}

Powyższy kod sprawdza, czy aplikacja jest objęta licencją próbną, a jeśli tak, to ustawia zmienną globalną.

Powrót: UWP i XBox Live
Następny: uwp_license_trial_user
