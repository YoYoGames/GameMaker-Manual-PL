rozwiązywanie sieci

Ta funkcja zwraca adres IP podanego adresu URL.

Składnia:

network_resolve(url);

ArgumentTyp Opis
url The URL to get the IP of (a string).

Zwroty:

Przykład:

game_ip = network_resolve("www.macsweeneygames.com");

Powyższy kod zwróci adres IP podanego adresu URL i zapisze go w zmiennej "game_ip".

Powrót: Praca w sieci
Następny: network_set_config
