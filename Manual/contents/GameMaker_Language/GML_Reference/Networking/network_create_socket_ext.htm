network_create_socket_ext

Funkcja służy do utworzenia nowego gniazda klienta dla twojej gry, aby komunikować się przez sieć. Musisz określić typ gniazda (patrz lista stałych poniżej) i podać port do wykorzystania, a funkcja zwróci unikalne id, które powinno być używane we wszystkich dalszych wywołaniach funkcji dla tego gniazda, lub wartość mniejszą niż 0, jeśli połączenie nie powiedzie się.

Składnia:

network_create_socket_ext(protocol, port);

Argument Typ Opis
protocol Gniazdo Typ stały The network protocol to use
port Real The port to use

Zwroty:

Network Socket ID

Przykład:

client = network_create_socket_ext(network_socket_udp, 6510);

Powyższy kod utworzy nowe gniazdo UDP na porcie 6510.

Powrót: Networking
Następny: network_connect
