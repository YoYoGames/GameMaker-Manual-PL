network_create_socket

Funkcja służy do utworzenia nowego gniazda klienckiego dla Twojej gry w celu komunikacji przez sieć. Musisz zdefiniować typ gniazda (patrz lista stałych poniżej), a funkcja zwróci unikalne id dla tego gniazda, które powinno być używane we wszystkich dalszych wywołaniach funkcji dla tego gniazda, lub wartość mniejszą niż 0, jeśli połączenie nie powiedzie się.

WSKAZÓWKA Możesz użyć IP "127.0.0.1", aby połączyć się z powrotem do tego samego urządzenia, na którym uruchomiona jest gra.

Składnia:

network_create_socket(type);

Argument Typ Opis
type Gniazdo Typ stały The type of socket connection to create (see the constants listed above).

Zwroty:

Network Socket ID

Przykład:

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

Powyższy kod sprawdzi czy gra jest uruchomiona w przeglądarce czy nie i utworzy nowe gniazdo TCP lub Web przed próbą połączenia się przez nie z podanym adresem IP na podanym porcie.
