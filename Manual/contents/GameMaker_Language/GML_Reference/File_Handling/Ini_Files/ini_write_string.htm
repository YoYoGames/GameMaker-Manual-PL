ini_write_string

Za pomocą tej funkcji można zapisać plik string (tekst) do pliku danych ini. Pliki ini są dzielone na sekcje, a następnie każda sekcja jest dzielona na pary klucz-wartość. Typowy plik ini wygląda więc następująco:

Warto zauważyć, że używanie cudzysłowów i znaków escape podczas pisania ini strings może spowodować problemy podczas późniejszego odczytywania danych, ponieważ mogą one zostać przedwcześnie obcięte. Problemem może być na przykład zapisanie następującej treści:

ini_write_string("test2", "section", "hello \"Fritz\"");

Składnia:

ini_write_string(section, key, value);

Argument Typ Opis
section String The section of the .ini to write to.
key String The key within the relevant section of the .ini to write to.
value String The string to write to the relevant destination.

Zwroty:

N/A

Przykład:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

Powyższy kod otwiera plik ini do odczytu i zapisu, a następnie zapisuje adres string przechowywany w zmiennej globalnej "Name" w sekcji "Save" z kluczem "Player", po czym ponownie zamyka plik.
