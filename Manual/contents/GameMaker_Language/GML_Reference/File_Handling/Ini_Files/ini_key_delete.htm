ini_key_delete

Za pomocą tej funkcji można usunąć wybrany klucz (i odpowiadającą mu wartość) z pliku ini.

Składnia:

ini_key_delete(section, key);

ArgumentTyp Opis
section The section to delete a key from.
key The key to delete.

Zwroty:

Przykład:

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

Ten przykład otworzy plik "savedata.ini" i wpisze wartość do "save1" > "Score". Następnie usunie klucz "Score" i zamknie plik .ini.
