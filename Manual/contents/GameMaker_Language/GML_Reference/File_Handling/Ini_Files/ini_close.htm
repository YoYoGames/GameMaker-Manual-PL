ini_close

Funkcję tę należy wywołać w momencie, gdy zakończy się odczytywanie lub zapisywanie w dowolnym otwartym pliku ini. Jeżeli nie użyjesz tej funkcji po użyciu którejkolwiek z funkcji zapisu do pliku ini, to nic nie zostanie zapisane na dysku, ponieważ informacje o pliku są przechowywane w pamięci do momentu wywołania tej funkcji, która wymusza zapis. Jeśli spróbujesz otworzyć plik ini, nie zamknąwszy wcześniej innego (lub tego samego), również otrzymasz błąd.

Funkcja zwróci również adres string z zakodowanym w nim plikiem ini. Ten plik string można zapisać w pliku server i/lub użyć go ponownie wraz z funkcją ini_open_from_string() aby ponownie utworzyć plik ini.

Składnia:

ini_close();

Zwroty:

String

Przykład:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Spowoduje to otwarcie pliku "savedata.ini" i odczytanie wartości z sekcji "save1" z kluczem "score", a następnie zamknięcie pliku ini. Jeśli w sekcji "save1" nie ma żadnej wartości o nazwie "score" lub nie ma pliku "savedata.ini", zostanie zwrócone 0 i utworzony nowy plik ini.
