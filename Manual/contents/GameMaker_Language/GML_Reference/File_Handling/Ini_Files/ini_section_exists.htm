ini_section_exists

Ta funkcja sprawdza, czy dana sekcja istnieje w aktualnie otwartym pliku ini i zwraca true, jeśli tak, lub false w przeciwnym wypadku. Nie jest to konieczne sprawdzanie, aby uniknąć błędów, ponieważ w przypadku braku sekcji odczyt z nieistniejącej sekcji zwróci wartość domyślną, jednak może być przydatne do sprawdzenia, czy w pliku ini zapisano określone dane.

Składnia:

ini_section_exists(section);

ArgumentTyp Opis
section The section in the open .ini file to check for.

Zwroty:

Przykład:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

To ustawi zmienną global.savegame na "1", jeśli "save1" NIE zostanie znaleziony. Przy użyciu for loop można to wykorzystać jako podstawowy sposób sprawdzania, ile jest save'ów w danej grze (np. tworzenie systemu dla wielu jednoczesnych save'ów w jednej grze).
