ini_open_from_string

Ta funkcja utworzy tymczasowy plik ini na podstawie adresu string i otworzy go do odczytu/zapisu. Adres string powinien być poprawnie sformatowany jako plik ini (tzn. z sekcjami, kluczami i wartościami podziału wierszy), w przeciwnym razie plik ini nie zostanie utworzony poprawnie. Należy pamiętać, że ten plik ini jest tymczasowy i zostanie usunięty z pamięci w momencie jej zamknięcia, co spowoduje utratę wszelkich informacji, które były w nim przechowywane, jednak funkcja ini_close() zwraca jednak adres string pełnego pliku ini, który można zapisać na stronie server lub na dysku.

Składnia:

ini_open_from_string(string);

ArgumentTyp Opis
string The string containing all the ini information.

Zwroty:

N/A

Przykład:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

Powyższy kod tworzy tymczasowe ini, w którym przechowywane są informacje z string "str", a następnie odczytuje dane z tego ini przed jego ponownym zamknięciem.
