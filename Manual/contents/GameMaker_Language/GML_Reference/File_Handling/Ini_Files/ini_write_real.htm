ini_write_real

Za pomocą tej funkcji można zapisać wartość (numeryczną) do pliku danych ini. Pliki ini są dzielone na sekcje, a następnie każda sekcja jest dzielona na pary klucz-wartość. Typowy plik ini wygląda więc następująco:

Składnia:

ini_write_real(section, key, value);

ArgumentTyp Opis
section The section of the .ini to write to.
key The key within the relevant section of the .ini to write to.
value The real value to write to the relevant destination.

Zwroty:

Przykład:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

To ustawi score na 1000, następnie otworzy plik "savedata.ini" i wpisze tę wartość do "save1" > "score". Następnie ustawi score2 na rzeczywistą wartość w "save1" > "score", czyli 1000, i zamknie plik .ini ponownie. Gdyby nie znaleziono tej wartości, wynik2 zostałby ustawiony na 0.
