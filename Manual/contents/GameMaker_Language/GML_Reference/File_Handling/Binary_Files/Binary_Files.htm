Pliki binarne

W rzadkich sytuacjach może zajść potrzeba odczytania danych z plików binarnych. Do tego celu można użyć tych funkcji, choć generalnie zalecamy, aby do tworzenia, zapisywania i wczytywania danych binarnych używać dedykowanych funkcji buforowych, a nie tych funkcji.

UWAGA: Te funkcje nie działają, gdy modułem docelowym jest HTML5.

Służą do tego następujące procedury niskopoziomowe:

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
