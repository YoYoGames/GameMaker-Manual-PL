katalog_tworzony

Ta funkcja tworzy w obszarze zapisu katalog o podanej nazwie.

UWAGA: Ta funkcja nie będzie działać w języku HTML5, ponieważ nie można tworzyć ani usuwać katalogów w lokalnej pamięci masowej przeglądarki.

OSTRZEŻENIE! Ta funkcja może nie działać tak, jak tego oczekujesz, ponieważ GameMaker jest sandboxed! Więcej informacji na ten temat znajdziesz w rozdziale System plików.

Składnia:

directory_create(dname)

ArgumentTyp Opis
dname The name of the directory to create.

Zwroty:

Przykład:

if !directory_exists("Games")
{
    directory_create("Games");
}

Sprawdzi, czy podany katalog istnieje w lokalnym folderze danych, a jeśli nie, utworzy go dla Ciebie.
