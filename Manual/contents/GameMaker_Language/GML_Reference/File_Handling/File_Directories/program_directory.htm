program_katalog

Spowoduje to wyświetlenie katalogu, w którym znajduje się plik wykonywalny gry. Jednak nie zawsze jest to użyteczne, zwłaszcza że niektóre urządzenia uruchamiają exe z pliku *.zip, więc to polecenie zwróci to samo, niezależnie od tego, skąd gra jest uruchamiana.

OSTRZEŻENIE! Ta funkcja może nie działać zgodnie z oczekiwaniami użytkownika, ponieważ GameMaker jest sandboxed! Więcej informacji na ten temat znajdziesz w rozdziale System plików.

UWAGA: Jeśli wyłączysz piaskownicę, nadal nie będziesz mógł zapisywać w tym katalogu, aby zapobiec ewentualnym problemom z usuwaniem wymaganych plików gry.

Składnia:

program_directory

Zwroty:

Przykład:

dir = program_directory;

Spowoduje to zapisanie w zmiennej katalogu, w którym znajduje się plik wykonywalny.

Powrót: Katalogi plików
Następny: directory_exists
