katalog_zniszczenie

Ta funkcja usuwa katalog o podanej nazwie z obszaru zapisu.

UWAGA: Ta funkcja nie będzie działać w języku HTML5, ponieważ nie można tworzyć ani usuwać katalogów w lokalnej pamięci masowej przeglądarki.

OSTRZEŻENIE! Ta funkcja może nie działać tak, jak tego oczekujesz, ponieważ GameMaker jest sandboxed! Więcej informacji na ten temat znajdziesz w rozdziale System plików.

Składnia:

directory_destroy(dname)

ArgumentTyp Opis
dname The name of the directory to remove.

Zwroty:

Przykład:

if directory_exists("DLC")
{
    directory_destroy("DLC");
}

Spowoduje to sprawdzenie, czy podany katalog istnieje w lokalnym folderze danych, a jeśli tak, to zostanie usunięty.
