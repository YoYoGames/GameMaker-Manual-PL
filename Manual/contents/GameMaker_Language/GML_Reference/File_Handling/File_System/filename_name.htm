nazwa_pliku

Użycie tej funkcji powoduje zwrócenie części nazwy wskazanego pliku z adresem extension, ale bez adresu path.

Składnia:

filename_name(fname);

ArgumentTyp Opis
fname The file to use.

Zwroty:

String

Przykład:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

Powyższy kod pobiera nazwę (w postaci string) pierwszego pliku typu "doc" znalezionego w podanym katalogu.

Powrót: System plików
Następny: filename_path
