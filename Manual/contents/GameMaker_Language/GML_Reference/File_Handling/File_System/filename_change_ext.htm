nazwa_pliku_zmiana_ext

Ta funkcja zwraca wskazaną nazwę pliku z adresem extension (zawierającym kropkę) zmienionym na nowy adres extension. Używając pustego adresu string jako nowego adresu extension, można całkowicie usunąć część extension.

Składnia:

filename_change_ext(fname, newext);

ArgumentTyp Opis
fname The file to use.
newext The new extension to use.

Zwroty:

Przykład:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

Powyższy kod pobiera nazwę pliku (jako string) z usuniętą częścią extension.

Powrót: System plików
Następny: get_open_filename
