plik_usuń

Ta funkcja usuwa z systemu podany plik. Należy zauważyć, że ta funkcja usunie tylko te pliki, które GameMaker jest w stanie utworzyć i przetworzyć: pliki ini, pliki tekstowe i binarne, a także te pliki, które służą do przechowywania zasobów stworzonych w grze, takich jak sprites czy powierzchnie. Nie usunie natomiast żadnego innego pliku. Funkcja zwróci również true, jeśli plik został pomyślnie usunięty, lub false w innych okolicznościach.

OSTRZEŻENIE! Ta funkcja może nie działać zgodnie z oczekiwaniami użytkownika, ponieważ GameMaker jest sandboxed! Więcej informacji na ten temat znajduje się w rozdziale dotyczącym limitów systemu plików.

Składnia:

file_delete(fname);

ArgumentTyp Opis
fname The name of the file to delete.

Zwroty:

Przykład:

if file_exists("level.txt")
{
    file_delete("level.txt");
}

Spowoduje to sprawdzenie, czy dany plik istnieje, a jeśli tak, to zostanie usunięty.
