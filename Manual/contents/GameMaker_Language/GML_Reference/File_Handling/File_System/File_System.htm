System plików

W programie GameMaker istnieje wiele funkcji, które umożliwiają manipulowanie plikami wszystkich typów na różne sposoby:

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

Do zmiany nazw plików można użyć następujących funkcji:

UWAGA Funkcje te nie działają na rzeczywistych plikach! Zajmują się one jedynie stroną strings związaną z różnymi elementami path i powinny być używane w połączeniu z innymi funkcjami plików.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Istnieją również specjalne funkcje do uzyskiwania nazw plików, które mogą być używane do ładowania i zapisywania plików z filtrami typu pliku. Funkcje te powodują otwarcie okna eksploratora, a tym samym utratę ostrości gry. Oznacza to, że dane wejściowe, od których gra jest zależna, mogą nie zostać poprawnie zarejestrowane. Dlatego zalecamy, aby (jeśli używasz tych funkcji w zdarzeniu klawiatury) używać zdarzenia podniesienia klawisza, a nie zdarzenia naciśnięcia lub opuszczenia.

UWAGA Te funkcje są dostępne tylko w systemach Windows i macOS.

OSTRZEŻENIE Te funkcje mogą nie działać tak, jak tego oczekujesz, ponieważ GameMaker jest sandboxed! Więcej informacji na ten temat można znaleźć w rozdziale System plików.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
