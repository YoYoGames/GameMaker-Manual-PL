załadować_csv

Funkcja wczytuje plik w formacie CSV i konwertuje go na siatkę DS, zwracając unikalną wartość ID dla utworzonej siatki.

Twój plik CSV powinien być zgodny ze specyfikacją CSV. Jeśli twoja składnia CSV nie jest poprawna, otrzymasz nieoczekiwane wyniki podczas ładowania pliku.

Składnia:

load_csv(filename)

Argument Typ Opis
filename String The name of the file to open (as a string)

Zwroty:

DS Grid ID

Przykład:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
    for (var j = 0; j < hh; j++;)
    {
        draw_text(xx, yy, string(file_grid[# i, j]));
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

Powyższy kod otworzy podany plik CSV i zapisze zwróconą siatkę DS w zmiennej "plik_siatki". Siatka ta jest następnie parsowana w kilku for lopach, a jej zawartość rysowana na ekranie.
