sha1_file

W kryptografii SHA-1 jest kryptograficzną funkcją skrótu zaprojektowaną przez Agencję Bezpieczeństwa Narodowego Stanów Zjednoczonych i jest stosowana w kilku powszechnie używanych aplikacjach, takich jak popularny Git, gdzie służy do sprawdzania zmian w plikach, oraz w protokołach TLS i SSL, PGP, SSH, S/MIME i IPsec. Ta funkcja pobiera plik wejściowy i zwraca 160-bitowy skrót wiadomości w formacie ASCII, unikalny dla tego pliku, który może być użyty do weryfikacji integralności w dowolnym późniejszym czasie.

UWAGA: SHA-1 nie jest całkowicie bezpieczny i może zostać złamany. Więcej informacji można znaleźć na tej stronie.

Składnia:

sha1_file(filename)

Argument Typ Opis
filename String The file to generate the sha1 hash for.

Zwroty:

String

Przykład:

hash = sha1_file(working_directory + "game_data.ini")

Powyższy kod wygeneruje hash sha1 dla podanego pliku i zapisze zwróconą wartość w zmiennej "hash".
