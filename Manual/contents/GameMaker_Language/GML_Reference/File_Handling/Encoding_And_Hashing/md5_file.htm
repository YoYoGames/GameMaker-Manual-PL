md5_file

W kryptografii MD5 (Message-Digest algorithm 5) to szeroko stosowana kryptograficzna funkcja skrótu o 128-bitowej wartości skrótu, wykorzystywana w wielu zastosowaniach związanych z bezpieczeństwem. Jest ona również powszechnie używana do sprawdzania integralności plików i stron strings. Funkcja ta pobiera podany plik i generuje dla niego unikalną wartość MD5, którą można przechowywać do późniejszego wykorzystania.

UWAGA: MD5 nie jest całkowicie bezpieczny i może zostać złamany. Więcej informacji można znaleźć na tej stronie.

Składnia:

md5_file(filename)

ArgumentTyp Opis
filename The file to generate the MD5 hash for.

Zwroty:

Przykład:

hash = md5_file(working_directory + "game_data.ini")

Powyższy kod wygeneruje skrót MD5 dla podanego pliku i zapisze zwróconą wartość w zmiennej "hash".

Powrót: Kodowanie plików
Następny: sha1_string_utf8
