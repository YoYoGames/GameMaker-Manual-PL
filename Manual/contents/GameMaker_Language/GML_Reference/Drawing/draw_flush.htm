draw_flush

Za pomocą tej funkcji można przepłukać cały potok rysowania. Jest to funkcja przeznaczona wyłącznie do debugowania i generalnie nie powinna być używana, chyba że zostanie wskazana przez pracownika pomocy technicznej YoYo Games. ponieważ jej bezmyślne używanie może spowodować poważne problemy z wydajnością gry.

Składnia:

draw_flush();

Zwroty:

Przykład:

draw_flush();

Powyższy kod przepłukuje rurociąg rysujący.
