część_emitenta_zniszcz

Ta funkcja usunie określony emiter z danego systemu i wyczyści go z pamięci (spowoduje to również zatrzymanie produkcji cząsteczek przez dany emiter, ale NIE usunie ich z room). Funkcja ta powinna być wywoływana zawsze wtedy, gdy dany emiter nie jest już potrzebny w systemie, aby zapobiec wyciekom pamięci i błędom.

Składnia:

part_emitter_destroy( ps, ind );

Argument Typ Opis
ps Identyfikator systemu cząstek stałych The particle system to destroy the emitter from.
ind Identyfikator emitera cząstek stałych The index of the emitter to destroy.

Zwroty:

N/A

Przykład:

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit1);
}

Powyższy kod sprawdzi, czy emiter cząstek indeksowany w zmiennej "p_emit" istnieje w danym systemie cząstek, a jeśli tak, to zostanie zniszczony.

Powrót: Emitery cząstek
