część_emitująca_tworzy

Funkcja służy do tworzenia nowego emitora cząstek w podanym systemie cząstek. Funkcja zwróci wartość indeksu nowego emitera, który musi być przechowywany w zmiennej i używany we wszystkich dalszych funkcjach odwołujących się do emitera, a sam emiter musi być zniszczony, gdy nie jest już używany, aby zapobiec wyciekom pamięci (można to osiągnąć za pomocą specjalnych funkcji emitter destroy lub przez zniszczenie całego systemu cząstek, do którego należy emiter).

UWAGA ID emiterów zwracane przez tę funkcję są używane ponownie, np. jeśli najpierw stworzysz trzy emitery cząstek, będą one miały przypisane ID 0, 1 i 2. Jeśli następnie zniszczysz drugi emiter (który otrzymał ID 1) używając part_emitter_destroy a następnie stworzysz kolejny emiter, to ten emiter ponownie wykorzysta ID 1. Ponieważ emiter o ID 2 jest w użyciu, następny emiter otrzyma ID 3 itd.

Składnia:

part_emitter_create(ps);

Argument Typ Opis
ps Identyfikator systemu cząstek stałych The particle system to create the emitter in.

Zwroty:

Particle Emitter ID

Przykład:

p_emit1 = part_emitter_create(Sname);

Spowoduje to utworzenie nowego emitera cząstek i zapisanie jego indeksu w zmiennej "p_emit".

Powrót: Emitery cząstek stałych
Następny: part_emitter_clear
