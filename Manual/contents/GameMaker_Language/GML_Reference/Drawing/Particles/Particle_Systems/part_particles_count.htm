particles_count

Za pomocą tej funkcji można sprawdzić, czy w systemie cząsteczek są aktualnie utworzone jakieś cząsteczki w witrynie room, a także zwrócić ich liczbę.

Składnia:

part_particles_count(ind);

Argument Typ Opis
ind Identyfikator systemu cząstek stałych The index of the particle system.

Zwroty:

Real

Przykład:

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

Powyższy kod sprawdzi liczbę cząstek w lokalnym systemie cząstek indeksowanym w zmiennej "Sname" i jeśli nie ma żadnych, zniszczy system, a następnie instancję.

Powrót: Układy cząstek stałych
Następny: part_system_automatic_update
