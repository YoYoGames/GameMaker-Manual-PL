część_systemu_aktualizacji

Ta funkcja powoduje przesunięcie danego systemu cząsteczek o jeden krok w czasie gry i może być używana do symulowania zachowania systemu cząsteczek, gdy automatyczna aktualizacja jest wyłączona, lub do przesunięcia systemu cząsteczek do określonego punktu w jednym kroku. Na przykład, jeśli masz system cząsteczek dla śniegu i chcesz, aby wyglądało na to, że śnieg pada od razu po wejściu gracza na stronę room, możesz przesunąć cząsteczki śniegu o kilka sekund w zdarzeniu tworzenia instancji, tak aby przy pierwszym narysowaniu cząsteczki były na całym ekranie (pokazuje to poniższy przykład kodu). Możesz także użyć tej funkcji, gdy automatyczne uaktualnianie systemu jest wyłączone przez funkcję part_system_automatic_update() do przesuwania układu.

Składnia:

part_system_update(ind);

Argument Typ Opis
ind Identyfikator systemu cząstek stałych The index of the particle system to update.

Zwroty:

N/A

Przykład:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

Powyższy kod spowoduje przesunięcie o 3 sekundy systemu cząstek indeksowanego w zmiennej globalnej "RainSys".
