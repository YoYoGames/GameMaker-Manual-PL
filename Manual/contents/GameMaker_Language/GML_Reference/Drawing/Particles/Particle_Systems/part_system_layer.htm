warstwa_częściowa_systemu

Ta funkcja służy do umieszczania na stronie switch układu cząsteczek z bieżącej warstwy na nową. Podajesz wartość identyfikatora układu cząsteczek (zwracaną przez funkcję part_system_create_layer()), a następnie unikalny identyfikator warstwy (zwracany przez funkcję layer_create() lub nazwę warstwy, która ma być użyta jako string - na przykład "instance_layer" - jak zdefiniowano w edytorze room ), a układ zostanie przeniesiony do nowej warstwy.

Składnia:

part_system_layer(ps, layer);

Argument Typ Opis
ps Identyfikator systemu cząstek stałych The particle system ID
layer Ciąg lub identyfikator warstwy The new layer ID

Zwroty:

N/A

Przykład:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

Powyższy kod sprawdzi, na jakiej warstwie znajduje się układ cząsteczek, a jeśli nie znajduje się na podanej warstwie, zostanie zmieniony.

Powrót: Układy cząstek stałych
