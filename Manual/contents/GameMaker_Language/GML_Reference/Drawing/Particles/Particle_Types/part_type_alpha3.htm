typ_części_alfa3

Za pomocą tej funkcji można ustawić gradient trzech wartości alfa (przezroczystości) dla każdej utworzonej cząstki danego typu. Pierwsza alfa jest tą, z którą wszystkie cząstki będą zaczynać, druga alfa jest tą, która będzie mieszana w połowie życia cząstki, a trzecia alfa jest tą, z którą cząstka będzie kończyć życie. W czasie życia cząstki nastąpi płynna gradientowa zmiana alf z jednej do drugiej.

Składnia:

part_type_alpha3(ind, alpha1, alpha2, alpha3);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
alpha1 Real The starting alpha of the particle.
alpha2 Real The halfway point alpha of the particle.
alpha3 Real The ending alpha of the particle.

Zwroty:

N/A

Przykład:

part_type_alpha3( part_Health, 0.5, 1, 0);

Powyższy kod sprawi, że wszystkie utworzone cząstki typu indeksowanego w zmiennej "part_Health" będą miały przez cały czas życia mieszankę alfa o wartościach od 0,5 do 1 do 0.
