typ_części_kolor1

Ta funkcja służy do ustawiania typu cząstki jako pojedynczego koloru przez cały czas życia każdej pojedynczej cząstki.

Składnia:

part_type_colour1(ind, colour1);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
colour1 Kolor The single colour to make the particle type.

Zwroty:

N/A

Przykład:

part_type_colour1(global.Snow_Part, c_white);

Powyższy kod spowoduje, że wszystkie cząstki utworzone dla typu cząstki indeksowanego w zmiennej globalnej "Snow_Part" będą tylko białe.
