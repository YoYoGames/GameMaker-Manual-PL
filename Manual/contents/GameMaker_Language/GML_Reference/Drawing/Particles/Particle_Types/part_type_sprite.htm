part_type_sprite

Tej funkcji można użyć do ustawienia typu cząstki, aby korzystała z niestandardowej strony sprite z gry assets. Możesz wybrać stronę sprite, a następnie powiedzieć programowi GameMaker, aby użył strony sprite i animował ją lub nie, a jeśli wybierzesz animację, możesz także wybrać, aby podobrazy zmieniały się w takim odstępie czasu, że animacja będzie się zaczynać i kończyć na początku i końcu czasu życia każdej utworzonej cząstki (jeśli więc czas życia jest ustawiony na minimum 15 kroków i maksimum 30 kroków, to niezależnie od wartości czasu życia animacja będzie dopasowana do niego, przy czym będzie działać szybciej przy krótszych czasach życia i wolniej przy dłuższych). Można także wybrać losowy obrazek, dzięki czemu jeśli strona sprite nie jest animowana, do utworzenia strony sprite zostanie wybrany losowo inny obrazek, a jeśli cząstka jest animowana, animacja rozpocznie się w losowym punkcie.

Składnia:

part_type_sprite(ind, sprite, animate, stretch, random);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
sprite Aktywa Sprite The index of the sprite to set the particle type to.
animate Boolean Whether to follow the sprite's animation normally (true) or not (false).
stretch Boolean Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random Boolean Whether to choose a random sub-image (true) or not (false).

Zwroty:

N/A

Przykład:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Powyższy kod ustawia typ cząstki zaindeksowany w zmiennej globalnej "p1" na kształt cząstki sprite zaindeksowanej w "spr_Coins", a następnie ustawia różne inne właściwości cząstki.
