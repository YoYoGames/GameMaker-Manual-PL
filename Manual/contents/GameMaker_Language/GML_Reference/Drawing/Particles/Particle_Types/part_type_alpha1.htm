typ_części_alfa1

Ta funkcja służy do ustawiania typu cząstki tak, aby miała jedną wartość alfa (przezroczystość) przez cały czas życia każdej pojedynczej cząstki. Wartość ta może wynosić od 0 (przezroczysta) do 1 (nieprzezroczysta).

Składnia:

part_type_alpha1(ind, alpha1);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
alpha1 Real The alpha of the particle.

Zwroty:

N/A

Przykład:

part_type_alpha1(global.Snow_Part, 0.5);

Powyższy kod spowoduje, że wszystkie cząstki utworzone dla typu cząstki indeksowanego w zmiennej globalnej "Snow_Part" będą miały wartość alfa równą 0,5 (półprzezroczyste).
