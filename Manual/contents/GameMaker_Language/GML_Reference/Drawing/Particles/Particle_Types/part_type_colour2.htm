typ_części_kolor2

Za pomocą tej funkcji można ustawić gradient dwóch kolorów dla każdej utworzonej cząstki danego typu. Pierwszy kolor to kolor, od którego wszystkie cząstki będą zaczynać, a drugi to kolor, na którym cząstka będzie kończyć, przy czym w czasie życia cząstki nastąpi płynna zmiana koloru gradientu z jednego koloru na drugi.

Składnia:

part_type_colour2(ind, colour1, colour2);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
colour1 Kolor The colour the particle will start at.
colour2 Kolor The colour the particle will end at.

Zwroty:

N/A

Przykład:

part_type_colour2( part_Health, c_red, c_white);

Powyższy kod sprawi, że wszystkie utworzone cząstki typu indeksowanego w zmiennej "part_Health" będą miały kolor od czerwonego do białego przez cały czas ich życia.
