typ_części_grawitacja

Ta funkcja ustawia siłę grawitacji, która ma oddziaływać na każdą tworzoną cząstkę danego typu. Wartość siły grawitacji jest dodawana do prędkości cząstki co krok i zazwyczaj jest to mała wartość, np. 0,5, natomiast kierunek jest kierunkiem "przyciągania" grawitacyjnego i jest zgodny ze standardowymi kierunkami GameMakera: 0° - w prawo, 90° - w górę, 180° - w lewo i 270° - w dół.

Składnia:

part_type_gravity(ind, grav_amount, grav_direction);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
grav_amount Real Strength of the gravity.
grav_direction Real The direction of the gravity.

Zwroty:

N/A

Przykład:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Powyższy kod ustawi różne wartości cząstek, w tym grawitację, która doda 0,2 do prędkości w każdym kroku o kierunku 270º, co spowoduje, że cząstka będzie ciągnięta "w dół", w kierunku dolnej części ekranu.
