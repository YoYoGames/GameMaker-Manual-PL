typ_części_kolor_rgb

Za pomocą tej funkcji można ustawić mieszankę kolorów czerwonego, zielonego i niebieskiego dla wszystkich tworzonych cząstek danego typu. Podajesz minimalną i maksymalną wartość dla każdego z trzech składników, a utworzone cząstki będą miały losowy kolor oparty na podanym zakresie parametrów. Wszystkie wartości muszą zawierać się w przedziale od 0 do 255.

Składnia:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
rmin Real The minimum the final colour's red component can be.
rmax Real The maximum the final colour's red component can be.
gmin Real The minimum the final colour's green component can be.
gmax Real The maximum the final colour's green component can be.
bmin Real The minimum the final colour's blue component can be.
bmax Real The maximum the final colour's blue component can be.

Zwroty:

N/A

Przykład:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

Powyższy kod powoduje, że każda cząstka emitowana przez typ cząstki indeksowany w zmiennej globalnej "Blood_Part" ma tylko różne odcienie czerwieni.
