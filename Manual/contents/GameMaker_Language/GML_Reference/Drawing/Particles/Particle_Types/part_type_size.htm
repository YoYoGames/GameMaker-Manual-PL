part_type_size

Funkcja ta służy do określania rozmiaru cząstki podczas jej tworzenia, a także może być używana do zwiększania lub zmniejszania rozmiaru cząstki w czasie jej życia. Wartości minimalnego i maksymalnego rozmiaru domyślnie wynoszą 1 (co oznacza, że rozmiar jest taki sam jak rozmiar cząstki bazowej po ustawieniu skali). part_type_scale() Jeśli ustawimy je na tę samą wartość, początkowy rozmiar wszystkich cząstek będzie taki sam, ale jeśli ustawimy różne wartości, każda cząstka zacznie od losowego rozmiaru z tego zakresu, na przykład jeśli ustawimy minimalną wartość 0,5, a maksymalną 2, wszystkie cząstki zostaną utworzone z rozmiarem od połowy do dwukrotności skali bazowej.

Można także ustawić wartość przyrostu, która spowoduje, że cząstka sprite będzie rosła (jeśli jest liczbą dodatnią) lub malała (jeśli jest liczbą ujemną) w czasie życia cząstki, przy czym podana wartość będzie dodawana do rozmiaru cząstki w każdym kroku. Wartość ta może wynosić co najmniej (+/-)0,01.

Na koniec można ustawić współczynnik "wiggle". Jest to wartość, która będzie losowo dodawana lub odejmowana do rozmiaru na każdym etapie życia cząstek. Oczywiście większe wartości są bardziej wyraźne niż mniejsze, a wartość ta może być nawet ujemna, przy czym maksymalny zakres wynosi od -20 do 20.

Składnia:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to change.
size_min Real The minimum size the particle can start at.
size_max Real The maximum size the particle can start at.
size_incr Real How much the particle should increase or decrease per step.
size_wiggle Real How much should randomly be added or subtracted from the particles size per step.

Zwroty:

N/A

Przykład:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

Powyższy kod ustawi różne wartości cząstek, w tym rozmiar, który zostanie podwojony podczas tworzenia każdej cząstki, a następnie będzie zwiększał rozmiar o 0,01 w każdym kroku, z możliwością zmiany o 0,1.
