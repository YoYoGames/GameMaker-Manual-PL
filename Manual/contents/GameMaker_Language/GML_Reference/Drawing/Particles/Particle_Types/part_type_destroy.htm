part_type_destroy

Za pomocą tej funkcji można usunąć z gry cząstki określonego typu. Gdy użyjesz tej funkcji, wszystkie cząstki danego typu znikną ze strony room, a sama cząstka zostanie usunięta z pamięci, więc funkcji tej należy używać tylko wtedy, gdy cząstka nie jest już potrzebna.

Składnia:

part_type_destroy(ind);

Argument Typ Opis
ind Typ cząstki ID The index of the particle type to destroy.

Zwroty:

N/A

Przykład:

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

Powyższy kod sprawdza system cząsteczek, czy w witrynie room są aktualnie widoczne jakieś cząsteczki, a jeśli nie, to cząsteczki, system i instancja są niszczone.
