tekstura_get_uvs

Ta funkcja zwraca tablicę 1D z 4 elementami reprezentującymi współrzędne UV dla podanego wskaźnika texture, wypełniając tablicę następującymi wartościami:

  • [0] = lewa
  • [1] = góra
  • [2] = prawo
  • [3] = dół
  • [4] = ilość pikseli, którą kompilator asset odciął od lewej strony sprites (tylkosprite assets )
  • [5] = ilość pikseli, którą kompilator asset odciął od górnej strony sprites (tylkosprite assets )
  • [6] = znormalizowany procent danych pikseli z oryginalnej szerokości sprites, które zostały zapisane na stronie texture (tylkosprite assets )
  • [7] = znormalizowany procent danych pikseli z oryginalnej wysokości sprites, które zostały zapisane na stronie texture (tylkosprite assets )

Tę wartość można następnie wykorzystać w innych funkcjach rysowania, w szczególności w ogólnych funkcjach 3D i niektórych prymitywnych funkcjach 2D, a także w funkcjach Shader. Jeśli potrzebujesz UVS dla sprite, możesz użyć sprite_get_uvs(), a dla font, font_get_uvs().

Składnia:

texture_get_uvs(texid)

Argument Typ Opis
texid Tekstura The texture pointer to get the UVS for

Zwroty:

Array (4 - 8 elements)

Przykład:

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

Powyższy kod najpierw pobiera tablicę texture dla powierzchni przechowywanej w surf_back, a następnie pobiera współrzędne UV dla tej tablicy texture. Następnie ze zwróconej tablicy pobiera lewe, górne, prawe i dolne współrzędne UV i zapisuje je w zmiennych lokalnych.
