texturegroup_get_tilesets

Za pomocą tej funkcji można pobrać indeks tilesetu każdego z zasobów tilesets przypisanych do stron texture w ramach danej grupy texture. Należy podać identyfikator grupy texture string (zdefiniowanej w edytorze grup texture ), a funkcja zwróci tablicę 1D, w której każdy wpis zawiera indeks tilesetu dla danego zasobu tilesetowego. Jeżeli funkcja się nie powiedzie, tzn. zostanie podana nieprawidłowa grupa lub grupa nie ma przypisanego adresu texture, to tablica będzie pusta (0 długości).

Składnia:

texturegroup_get_tilesets(tex_id);

Argument Typ Opis
tex String The name of the texture group to check (a string)

Zwroty:

Array

Przykład:

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i]));
}

Powyższy kod pobierze indeksy tilesetów dla grupy texture "MainMenu", a następnie wyświetli te identyfikatory w oknie wyjściowym konsoli.
