gpu_set_cullmode

Za pomocą tej funkcji można ustawić tryb kadrowania tylnej powierzchni. Wielokąt (prymityw) ma przednią i tylną stronę. Za przednią stronę uważa się tę, w której wierzchołki są zdefiniowane w kolejności przeciwnej do ruchu wskazówek zegara. Zazwyczaj rysowane są obie strony, ale jeśli tworzymy kształt zamknięty (np. ostrosłup), to jest to marnowanie mocy obliczeniowej, ponieważ tylna strona trójkąta nigdy nie będzie widoczna. Właśnie w takich przypadkach można skorzystać ze strony switch. Pozwala to zaoszczędzić około połowy czasu rysowania, choć oznacza konieczność zdefiniowania wielokątów we właściwy sposób, aby uniknąć problemów.

Użyta wartość trybu kadrowania musi być jedną z poniższych stałych (wartość domyślna to cull_noculling):

Stała Opis
cull_noculling Nie będzie przeprowadzany ubój
cull_clockwise Wszystkie trójkąty zgodne z ruchem wskazówek zegara zostaną usunięte.
cull_counterclockwise Wszystkie trójkąty w kierunku przeciwnym do ruchu wskazówek zegara zostaną usunięte.

Składnia:

gpu_set_cullmode(cullmode);

Argument Typ Opis
cullmode Tryb uśmiercania Ciągły The cull mode to use (see list above)

Zwroty:

N/A

Przykład:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Powyższy kod pobiera bieżący tryb kadrowania i jeśli nie jest to cull_clockwise, ustawia go na tę stałą.
