gpu_set_texrepeat_ext

Za pomocą tej funkcji można określić, czy pojedynczy "slot" samplera ma powtarzać podany adres texture podczas używania Shaderów w GameMakerze. Ustawienie jej na true spowoduje powtarzanie texture, jeśli współrzędne uv będą poza zakresem 0-1, podczas gdy ustawienie false będzie oznaczało brak powtarzania. Prawdopodobnym zastosowaniem tych funkcji jest powtarzanie strony texture w 3D, ale aby to działało i nie ciągnęło obrazów z reszty strony texture, używana strona sprite musi być oznaczona jako "Oddzielna strona Texture " w edytorze Sprite.

UWAGA To ustawienie zostanie zastąpione wartością ustawioną podczas wywoływania funkcji gpu_set_texrepeat().

Składnia:

gpu_set_texrepeat_ext(sampler_id, enable);

ArgumentTyp Opis
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Zwroty:

Przykład:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Powyższy kod sprawdza, czy filtrowanie texture jest włączone lub wyłączone dla określonego identyfikatora próbnika (przechowywanego w zmiennej lokalnej) i odpowiednio je przełącza.
