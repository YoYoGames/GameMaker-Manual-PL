gpu_pop_state

Ta funkcja usuwa ze stosu poprzedni stan układu GPU i stosuje go. Zobacz gpu_push_state() aby uzyskać więcej informacji.

Składnia:

gpu_pop_state();

Zwroty:

Przykład:

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

Powyższy kod "zapisze" aktualny stan układu GPU na stosie, a następnie zmieni pewne ustawienia układu GPU i narysuje grupę instancji, po czym zresetuje stan układu GPU do poprzedniego.
