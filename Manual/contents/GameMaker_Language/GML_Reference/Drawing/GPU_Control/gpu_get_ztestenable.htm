gpu_get_ztestenable

Za pomocą tej funkcji można sprawdzić, czy testowanie metodą z-testów jest włączone (funkcja zwraca wartość true), czy nie (funkcja zwraca wartość false). Domyślną wartością jest to, że testowanie z-testów jest wyłączone, więc funkcja zwróci false.

Składnia:

gpu_get_ztestenable();

Zwroty:

Przykład:

if gpu_get_ztestenable() == false
{
    gpu_set_ztestenable(true);
}

Powyższy kod sprawdza, czy testowanie z-testów jest włączone, a jeśli jest wyłączone, to jest ponownie włączane.
