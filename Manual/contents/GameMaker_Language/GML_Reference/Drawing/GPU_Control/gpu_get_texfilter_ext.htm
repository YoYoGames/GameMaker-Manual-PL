gpu_get_texfilter_ext

Za pomocą tej funkcji można sprawdzić, czy interpolacja texture (interpolacja liniowa) jest włączona (zwraca true) czy nie (zwraca false) dla danego shader samplera texture.

UWAGA: W przypadku celu HTML5 ta funkcja będzie działać tylko z włączoną obsługą WebGL.

Składnia:

gpu_get_texfilter_ext(sampler_id);

Argument Typ Opis
sampler_id Uchwyt do próbnika cieni The sampler id from the shader.

Zwroty:

Boolean

Przykład:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Powyższy kod sprawdza, czy filtrowanie texture jest włączone lub wyłączone dla określonego identyfikatora próbnika (przechowywanego w zmiennej lokalnej) i odpowiednio je przełącza.
