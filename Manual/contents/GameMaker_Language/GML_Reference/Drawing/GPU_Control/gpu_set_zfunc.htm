gpu_set_zfunc

Za pomocą tej funkcji można ustawić tryb porównywania testowania bufora z (zob. gpu_set_ztestenable() więcej informacji). Dostępne wartości to dowolne z poniższych stałych (wartość domyślna to cmpfunc_lessequal):

Stała Opis
cmpfunc_never Nigdy
cmpfunc_less Mniej
cmpfunc_equal Equal
cmpfunc_lessequal Mniej lub więcej
cmpfunc_greater Większa
cmpfunc_notequal Nie równa się
cmpfunc_greaterequal Większy lub równy
cmpfunc_always Zawsze

Składnia:

gpu_set_zfunc(cmp_func);

Argument Typ Opis
cmp_func Funkcja porównania Stała The comparison mode to use (see list above)

Zwroty:

N/A

Przykład:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Powyższy kod włącza testowanie bufora z i ustawia jego tryb porównywania przed narysowaniem tła sprite, a następnie wyłącza go ponownie, aby kontynuować rysowanie.

Powrót: Sterowanie procesorem graficznym
