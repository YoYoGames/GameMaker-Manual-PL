gpu_get_texfilter

Za pomocą tej funkcji można sprawdzić, czy filtrowanie texture (interpolacja liniowa) jest włączone (zwraca true), czy nie (zwraca false). Więcej informacji na temat filtrowania texture można znaleźć w funkcji gpu_set_texfilter().

UWAGA: W przypadku celu HTML5 ta funkcja będzie działać tylko z włączoną obsługą WebGL.

Składnia:

gpu_get_texfilter();

Zwroty:

Boolean

Przykład:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

Powyższy kod sprawdza, czy filtrowanie texture jest włączone czy wyłączone, i odpowiednio je przełącza.
