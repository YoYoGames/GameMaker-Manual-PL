gpu_get_zwriteenable

Za pomocą tej funkcji można określić, czy pisanie z-writing jest włączone (funkcja zwraca wartość true), czy nie (funkcja zwraca wartość false). Domyślną wartością jest to, że funkcja z-writing jest włączona, więc funkcja zwraca true.

Składnia:

gpu_get_zwriteenable();

Zwroty:

Przykład:

if gpu_get_zwriteenable() == false
{
    gpu_set_zwriteenable(true);
}

Powyższy kod sprawdza, czy funkcja zapisu z jest włączona, a jeśli jest wyłączona, to jest ponownie włączana.
