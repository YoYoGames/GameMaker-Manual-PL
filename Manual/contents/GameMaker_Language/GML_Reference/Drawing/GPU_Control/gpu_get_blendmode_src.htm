gpu_get_blendmode_src

Za pomocą tej funkcji można uzyskać bieżący źródłowy współczynnik rozszerzonego trybu mieszania używany do rysowania. Zwrócona wartość będzie jedną z poniższych stałych ("s" oznacza wartość pobraną ze źródła, a "d" - z miejsca przeznaczenia):

Stała Współczynnik mieszania (czerwony, zielony, niebieski, alfa)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Ad, Ad, Ad, Ad, Ad)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) gdzie f = min(As, 1-Ad)

Składnia:

gpu_get_blendmode_src();

Zwroty:

Blend Mode Factor Constant (see above table)

Przykład:

var bm;
bm[0] = gpu_get_blendmode_src();
bm[1] = gpu_get_blendmode_dest();
gpu_set_blendmode_ext_sepalpha(bm[0], bm[1], bm_inv_src_alpha, bm_inv_dest_colour);

Powyższy kod tworzy lokalną tablicę i pobiera bieżące źródłowe i docelowe współczynniki mieszania. Tablica ta jest następnie używana do manipulowania składową alfa współczynników mieszania.
