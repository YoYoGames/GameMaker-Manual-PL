surface_get_target_ext

Ta funkcja pobiera identyfikator powierzchni przypisanej do jednego z 4 celów renderowania dostępnych dla powierzchni. Podajesz indeks celu renderowania, który chcesz sprawdzić, a funkcja zwróci -1, jeśli nie ma przypisanej powierzchni, lub wartość całkowitą >= 0, reprezentującą ID przypisanej powierzchni (zwróconą przez funkcję surface_create()).

Składnia:

surface_get_target_ext(index);

ArgumentTyp Opis
index The render target index to check (from 0 to 3).

Zwroty:

Surface ID or -1 (if no target surface is set)

Przykład:

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

Powyższy kod najpierw sprawdzi, czy na stronie shader render target 0 została ustawiona powierzchnia, a jeśli nie, to zostanie ona przypisana.
