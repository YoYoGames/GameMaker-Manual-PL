surface_get_target

Za pomocą tej funkcji możesz uzyskać identyfikator bieżącej powierzchni, do której chcesz rysować. Jeśli utworzyłeś tę powierzchnię używając surface_create()to będzie to dodatnia liczba całkowita, która jest unikalnym identyfikatorem tej powierzchni.

Jeśli nie ustawiono żadnej niestandardowej powierzchni, ale domyślna application_surface jest nadal włączona, to funkcja zwykle zwróci 0. Jednakże, jeśli wcześniej manipulowałeś application_surface, możesz chcieć sprawdzić, czy ID, które otrzymałeś, nie pasuje do bieżącej wartości zmiennej application_surface, na wypadek gdyby nie było to już 0.

Jeśli nie ustawiono żadnej niestandardowej powierzchni i domyślna application_surface również została wyłączona, to funkcja zwróci -1.

Składnia:

surface_get_target();

Zwroty:

Surface ID or -1 (if no target surface is set)

Przykład:

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

Powyższy kod sprawdzi, czy aktualny cel renderowania jest powierzchnią, czy nie, a jeśli jest, resetuje cel, a następnie rysuje powierzchnię.
