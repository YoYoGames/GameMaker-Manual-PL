vertex_get_number

Za pomocą tej funkcji można określić liczbę poszczególnych wierzchołków zdefiniowanych w dowolnym miejscu na stronie vertex buffer.

Składnia:

vertex_get_number(buffer);

Argument Typ Opis
buffer Vertex Buffer ID The vertex buffer to check.

Zwroty:

Real

Przykład:

v_num = vertex_get_number(v_buffer);

Powyższy kod zapisze do zmiennej liczbę wierzchołków przechowywanych w podanym adresie vertex buffer.
