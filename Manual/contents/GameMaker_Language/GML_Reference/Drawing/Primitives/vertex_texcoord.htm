vertex_texcoord

Ta funkcja ustawi współrzędne texture, które mają być użyte dla vertex aktualnie zdefiniowanego dla niestandardowego prymitywu. Podajesz adres buffer, do którego mają być zapisane dane, a także pozycję UV w obrębie texture, która ma być użyta.

Składnia:

vertex_texcoord(buffer, u, v);

Argument Typ Opis
buffer Vertex Buffer ID The buffer to write the information to.
u Real The first texture coordinate to use (0 - 1).
v Real The second texture coordinate to use (0 - 1).

Zwroty:

N/A

Przykład:

vertex_texcoord(buff, 0, 0);

Powyższy kod ustawi wartości UV dla definiowanej strony vertex.

Powrót: Prymitywy i Vertex Formaty
Następny: vertex_float1
