Ta funkcja doda wartość zmiennoprzecinkową do danych vertex. Adres vertex musi być poprawnie sformatowany, aby można było zaakceptować tę funkcję. vertex_format_add_custom() function.

Składnia:

vertex_float1(buffer, float);

Argument Typ Opis
buffer Vertex Buffer ID The buffer to write the information to.
float Real The input value.

Zwroty:

N/A

Przykład:

vertex_float1(buff, 0.05);

Powyższy kod doda wartość zmiennoprzecinkową do definiowanych danych vertex.

Powrót: Prymitywy i Vertex Formaty
Następny: vertex_float2
