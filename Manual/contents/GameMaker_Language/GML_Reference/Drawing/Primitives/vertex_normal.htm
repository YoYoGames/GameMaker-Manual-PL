wierzchołek_normalny

Ta funkcja doda dane normalnej powierzchni do strony vertex aktualnie zdefiniowanej dla niestandardowego obiektu pierwotnego. Użytkownik podaje adres buffer, do którego mają być zapisane dane, a także składowe x, y i z normalnej.

Składnia:

vertex_normal(buffer, nx, ny, nz);

Argument Typ Opis
buffer Vertex Buffer ID The buffer to write the information to.
nx Real The x component of the surface normal.
ny Real The y component of the surface normal.
nz Real The z component of the surface normal.

Zwroty:

N/A

Przykład:

vertex_normal(buff, 0, 1, 1);

Powyższy kod ustawi normalną powierzchniową dla definiowanej strony vertex.
