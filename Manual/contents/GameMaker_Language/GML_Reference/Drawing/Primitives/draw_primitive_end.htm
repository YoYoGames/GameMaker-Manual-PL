draw_primitive_end

Ta funkcja musi zostać wywołana, gdy skończysz definiować punkty swojej bryły. Jeśli nie wywołasz tej funkcji, to nic nie zostanie narysowane, ponieważ funkcja ta mówi GameMakerowi, że skończyłeś i że może teraz narysować zdefiniowany prymityw.

Składnia:

draw_primitive_end()

Zwroty:

Przykład:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Powyższy kod narysuje pasek trójkątów o rozmiarze 4 vertex z teksturą texture zapisaną w zmiennej "tex".
