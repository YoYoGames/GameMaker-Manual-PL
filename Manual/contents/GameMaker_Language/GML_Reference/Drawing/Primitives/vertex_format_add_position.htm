vertex_format_add_position

Powiadom program GameMaker, aby akceptował dwuwymiarowe dane pozycyjne (x i y) jako część nowego formatu vertex, który jest tworzony.

Składnia:

vertex_format_add_position();

Zwroty:

Przykład:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Powyższy kod utworzy nowy format vertex zawierający tylko wartości koloru i położenia, a następnie zapisze identyfikator formatu w zmiennej "my_format".
