wierzchołek_argb

Ta funkcja ustawia wartości ARGB dla strony vertex aktualnie zdefiniowanej dla niestandardowego elementu pierwotnego. Podajesz adres buffer, do którego mają być zapisane dane, jak również wartości czerwona, zielona, niebieska i alfa, których chcesz użyć jako pojedynczej 32-bitowej liczby całkowitej bez znaku - próbka alfa w najwyższych 8 bitach, następnie próbka czerwona, zielona i wreszcie niebieska w najniższych 8 bitach. Do określenia wartości koloru można użyć notacji heksadecymalnej ($AARRGGBB), liczby rzeczywistej lub dowolnej funkcji make_colour_*().

Składnia:

vertex_argb(buffer, argb);

Argument Typ Opis
buffer Vertex Buffer ID The buffer to write the information to.
argb Kolor The ARGB value to set.

Zwroty:

N/A

Przykład:

vertex_argb(buff, $FFFFFFFF);

Powyższy kod ustawi wartości ARGB zdefiniowanej strony vertex na kolor biały.
