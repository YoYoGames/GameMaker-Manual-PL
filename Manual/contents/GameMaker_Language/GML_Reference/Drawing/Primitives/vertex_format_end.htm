vertex_format_end

Funkcja ta musi być wywołana po zdefiniowaniu nowego formatu vertex. Zwraca ona "uchwyt" (indeks) nowego formatu, który musi być używany we wszystkich kolejnych funkcjach vertex, które odwołują się do tego nowego formatu.

Składnia:

vertex_format_end();

Zwroty:

Vertex Format ID

Przykład:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Powyższy kod utworzy nowy format vertex zawierający tylko wartości koloru i położenia, a następnie zapisze identyfikator formatu w zmiennej "my_format".
