video_get_format

Ta funkcja zwraca format kolorów aktualnie załadowanego filmu. Może to być dowolna z poniższych stałych:

Format wideo Stały
Stała Opis
video_format_rgba Powierzchnia wideo wykorzystuje model kolorów RGBA
video_format_yuv Powierzchnia wideo wykorzystuje model kolorów YUV

Windows, macOS, Opera GX, Android, iOS, i HTML5 używają modelu RGBA dla filmów. Wszystkie inne platformy używają modelu YUV.

Ponieważ metody rysowania dla wideo RGBA i YUV są różne, wartość zwracana przez tę funkcję może być użyta do uruchomienia innego kodu w zależności od formatu odtwarzanego wideo. Zobacz przykład Draw Video.

Składnia:

video_get_format();
