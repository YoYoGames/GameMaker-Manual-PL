video_close

Ta funkcja zamyka aktualnie załadowany plik wideo. Upewnij się, że jest ona wywoływana tylko po wywołaniu video_open(), w przeciwnym razie nic nie zrobi.

Składnia:

video_close();

Zwroty:

N/A

Przykład:

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

Powyższy kod powoduje zamknięcie filmu po naciśnięciu klawisza Escape.
