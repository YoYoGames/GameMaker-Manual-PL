wideo_otwarte

Ta funkcja otwiera plik wideo przy użyciu podanego adresu path. Adres path może odnosić się do pliku w folderze Included Files projektu, pliku znajdującego się lokalnie na urządzeniu użytkownika lub adresu URL służącego do załadowania pliku z Internetu (co może nie działać na wszystkich platformach).

Ponieważ w danym momencie można załadować tylko jeden plik wideo, funkcja ta ładuje określony plik wideo do jedynego dostępnego "gniazda wideo", którym następnie można manipulować za pomocą innych funkcji wideo.

Z odtwarzaniem wideo związane są asynchroniczne wywołania zwrotne. Więcej informacji można znaleźć w temacie Wywołania asynchroniczne.

Składnia:

video_open(path);

Argument Typ Opis
path String Path to the video file

Zwroty:

N/A

Przykład:

my_video = video_open("splash.mp4");

Powyższy kod wczytuje plik splash.mp4 z katalogu Included Files gry.
