narysuj_linię

Za pomocą tej funkcji można narysować linię o szerokości 1 piksela między dwoma dowolnymi punktami w grze room. Należy pamiętać, że rysowana linia może wymagać innych wartości (+/-1 na x, y), aby została narysowana w pożądanych wymiarach, ze względu na różnice między różnymi obsługiwanych platform.

UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, musisz wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, Kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem funkcja shader może nie narysować niczego (lub narysować coś, ale niepoprawnie). Jeśli potrzebujesz rysować kształty w ten sposób, to shader powinien być dostosowany do tego celu.

Składnia:

draw_line(x1, y1, x2, y2);

ArgumentTyp Opis
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.

Zwroty:

Przykład:

draw_set_colour(c_lime);
draw_line(50,50,150,50);

Spowoduje to narysowanie jasnozielonej poziomej linii od punktu (50,50) do punktu (150,50).
