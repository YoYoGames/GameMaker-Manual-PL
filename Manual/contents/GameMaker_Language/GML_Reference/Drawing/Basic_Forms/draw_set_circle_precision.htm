draw_set_circle_precision

Aby zoptymalizować wydajność podczas rysowania okręgów, GameMaker rysuje wielokąt o bokach wystarczających do tego, by wydawał się okrągły. Jednak w zależności od tego, jak duże lub małe mają być okręgi, może się okazać, że zmiana może się okazać, że zmiana tej wartości pomoże zwiększyć wydajność gry lub sprawi, że okręgi będą wyglądać lepiej. Wartość precyzji, którą można wprowadzić do tej funkcji, to liczba boków wielokąta kołowego, która może wynosić od 4 do 64, ale musi być liczbą podzielną przez 4. Domyślną wartością jest 24.

UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, powinieneś wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, Kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem funkcja shader może nie narysować niczego (lub narysować coś, ale niepoprawnie). Jeśli potrzebujesz rysować kształty w ten sposób, to shader powinien być dostosowany do tego celu.

Składnia:

draw_set_circle_precision(precision);

ArgumentTyp Opis
precision The precision of all circles drawn after this function is called.

Zwroty:

Przykład:

draw_set_circle_precision(12);

Powyższy kod narysuje wszystkie okręgi przy użyciu wielokąta o 12 bokach.
