draw_enable_drawevent

Za pomocą tej funkcji możesz włączyć (true) lub wyłączyć (false) zdarzenie draw dla wszystkich instancji w grze, dzięki czemu masz kontrolę nad tym, jak i kiedy rzeczy są rysowane, co jest przydatne, jeśli chcesz zastosować technikę "pomijania klatek". Zwróć uwagę, że nie tylko uniemożliwia to rysowanie instancji na ekranie, ale całkowicie wyłącza zdarzenie draw, co oznacza, że należy zachować ostrożność, ponieważ logika gry, która jest obecna w tym zdarzeniu, również nie zostanie uruchomiona.

Należy pamiętać, że jeśli wywołasz tę funkcję na samym początku gry, zanim zostanie wyrenderowana pierwsza klatka (np. zdarzenie Create Event pierwszej strony object w pierwszej stronie room gry), okno gry nie zostanie wyrenderowane. Może to być przydatne w projektach dla wielu graczy, które wymagają dedykowanej strony server, która nie musi niczego renderować, ale należy pamiętać, że nie czyni to gry bezgłową, ponieważ strona GameMaker tego nie obsługuje - nie będzie więc można jej uruchomić na stronie server, która nie ma interfejsu graficznego.

Składnia:

draw_enable_drawevent(enable);

ArgumentTyp Opis
Enable Set to true lub false.

Zwroty:

Przykład:

frame_skip ++;
if (frame_skip mod 5) == 0
{
    draw_enable_drawevent(true);
}
else
{
    draw_enable_drawevent(false);
}

Powyższy kod sprawdza zmienną i jeśli zwraca true, włącza zdarzenie losowania, w przeciwnym razie zdarzenie losowania jest wyłączone.
