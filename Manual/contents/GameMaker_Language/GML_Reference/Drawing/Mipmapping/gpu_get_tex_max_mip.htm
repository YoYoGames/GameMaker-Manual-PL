gpu_get_tex_max_mip

Za pomocą tej funkcji można uzyskać aktualnie ustawiony maksymalny poziom mipmapy, który ma być użyty, gdzie wartość 0 oznacza najwyższą rozdzielczość, 1 oznacza użycie pierwszej mipmapy, 2 - drugiej itd. Zauważ, że może to być całkiem przydatne do uniknięcia artefaktów krwawienia podczas renderowania strony textures, na przykład ustawienie granicy strony texture na 8px, a następnie ustawienie maksymalnego poziomu mipmapy na 3 zapewni, że nie wystąpią żadne problemy z krwawieniem przy większych odległościach renderowania.

Składnia:

gpu_get_tex_max_mip();

Zwroty:

(default: 16)

Przykład:

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

Powyższy kod sprawdzi bieżący maksymalny poziom mapy mip i jeśli nie wynosi on 4, zostanie ustawiony na 4.
