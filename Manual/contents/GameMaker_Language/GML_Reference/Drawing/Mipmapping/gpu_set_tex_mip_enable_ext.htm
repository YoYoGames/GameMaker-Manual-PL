gpu_set_tex_mip_enable_ext

Za pomocą tej funkcji można ustawić, czy mapowanie mipmap jest wyłączone, włączone dla wszystkich lub włączone tylko dla grup texture wybranych w Menedżerze grup tekstur na próbniku shader. Podajesz wartość indeksu dla próbnika shader (zwróconą przez funkcję shader_get_sampler_index(), a następnie jedną z poniższych stałych.

Stała Opis
mip_off Mapowanie Mipmap jest wyłączone.
mip_on Włączone jest mapowanie Mipmap dla wszystkich stron textures.
mip_markedonly Funkcja Mipmapping jest włączona w witrynie textures, w której włączono ją w opcjach grupy Texture (domyślnie).

Składnia:

gpu_set_tex_mip_enable_ext(sampler_index, setting);

Argument Typ Opis
sampler_index Uchwyt do próbnika cieni The index of the shader sampler to get
setting Stała mapowania Mipmappingu The mipmap setting (a constant, default: mip_markedonly)

Zwroty:

N/A

Przykład:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Powyższy kod włącza mipmapping dla podanego samplera shader texture, jeśli nie został on jeszcze włączony.
