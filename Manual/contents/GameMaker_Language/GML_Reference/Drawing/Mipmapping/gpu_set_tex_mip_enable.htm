gpu_set_tex_mip_enable

Za pomocą tej funkcji można zmienić, czy mipmapowanie jest wyłączone, włączone dla wszystkich, czy włączone tylko dla grup texture wybranych w Menedżerze grup tekstur. Do działania funkcji wymagana jest jedna z poniższych stałych.

Stała Opis
mip_off Mapowanie Mipmap jest wyłączone.
mip_on Włączone jest mapowanie Mipmap dla wszystkich stron textures.
mip_markedonly Funkcja Mipmapping jest włączona w witrynie textures, w której włączono ją w opcjach grupy Texture (domyślnie).

Składnia:

gpu_set_tex_mip_enable(setting);

Argument Typ Opis
setting Stała mapowania Mipmappingu The mipmap setting (a constant, default: mip_markedonly)

Zwroty:

N/A

Przykład:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

Powyższy kod sprawdzi, czy funkcja mipmapping jest włączona, a jeśli nie, włączy ją.
