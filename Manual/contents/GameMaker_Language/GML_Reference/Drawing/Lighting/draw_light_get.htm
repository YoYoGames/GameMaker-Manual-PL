draw_light_get

Ta funkcja pobierze określone parametry światła w postaci tablicy zawierającej 6 elementów -

  • [0] = włączony / wyłączony (true / false)
  • [1] = Typ światła Stała (patrz tabela poniżej)
  • [2] = pozycja x
  • [3] = pozycja y
  • [4] = pozycja z
  • [5] = promień światła (tylko dla świateł punktowych, dla kierunkowych będzie to 0)
  • [6] = kolor jasny (prawdziwy)

Typ światła (element [1]) może być jedną z dwóch poniższych stałych:

Typ światła Stałe
Stała Opis
lighttype_dir Światło jest światłem kierunkowym
lighttype_point Światło jest światłem punktowym

Składnia:

draw_light_get(ind);

Argument Typ Opis
ind Real The index number of the light (from 0 to 7)

Zwroty:

Array

Przykład:

light_a = draw_light_get(1);
if light_a[5] < 200
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

Powyższy kod pobierze wartości użyte do zdefiniowania kontrolki indeksowanej jako "1", a następnie sprawdzi promień i jeśli będzie on mniejszy niż 200, zostanie zwiększony, a promień kontrolki ustawiony na nową wartość.
