game_save_buffer

Jest to odmiana funkcji game_save(), dlatego należy najpierw przeczytać stronę poświęconą tej funkcji, ponieważ zawiera ona ważne informacje dotyczące jej użycia, a także tej funkcji.

Za pomocą tej funkcji można zapisać aktualny stan gry w utworzonym wcześniej buforze"grow" buffer (patrz Bufory), który następnie można w dowolnym momencie ponownie załadować za pomocą game_load_buffer() funkcja. Ta funkcja jest przeznaczona do użycia w jednym room w danym czasie (np. nie zaleca się zapisywania buffer w jednym room, a następnie próbowania załadowania buffer z innego ) i powinna być używana tylko do punktów kontrolnych lub restartów poziomów.

UWAGA: Ta funkcja jest bardzo ograniczona i została zaprojektowana dla początkujących graczy, aby szybko uruchomić system punktów kontrolnych. Bardziej zaawansowani użytkownicy mogą jednak preferować kodowanie własnego systemu za pomocą funkcji Plik, ponieważ gra nie będzie zapisywać żadnych zasobów dynamicznych, takich jak struktury danych, powierzchnie, dodane strony sprites itp.

Składnia:

game_save_buffer(buffer);

ArgumentTyp Opis
buffer The buffer id to save to.

Zwroty:

Przykład:

if keyboard_check_pressed(ord("S"))
{
    global.Checkpoint = true;
    game_save_buffer(save_buff);
}

Powyższy kod ustawi zmienną globalną na true, a następnie po naciśnięciu klawisza "S" zapisze aktualny stan gry na stronie buffer zaindeksowanej w zmiennej "save_buff".
