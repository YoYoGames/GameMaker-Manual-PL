game_load

Za pomocą tej funkcji można wczytać grę, która została wcześniej zapisana przy użyciu game_save(). Należy pamiętać, że przywrócona zostanie wersja gry, która została użyta do utworzenia zapisu, więc wszelkie aktualizacje dokonane po tym zapisie nie będą widoczne. Aby uzyskać więcej informacji, przeczytaj stronę na temat game_save().

Składnia:

game_load(filename);

Argument Typ Opis
filename String The name of the file to load from.

Zwroty:

N/A

Przykład:

if keyboard_check_pressed(ord("L"))
{
    if (global.Save) game_load("Save.dat");
}

Powyższy kod wczyta zapisaną wcześniej grę, jeśli zmienna globalna ma adres true po naciśnięciu przez gracza klawisza "L".

Powrót: Ogólne sterowanie grą
Następny: game_load_buffer
