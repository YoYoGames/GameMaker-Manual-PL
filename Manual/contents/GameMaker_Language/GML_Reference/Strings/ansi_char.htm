ansi_char

Ta funkcja zwraca adres string zawierający znak z ustawioną wartością raw BYTE. Znak ten nie będzie i nie powinien być wyświetlany, ale zostanie poprawnie zapisany na dysku w celu użycia podczas kodowania.

Składnia:

ansi_char(val);

ArgumentTyp Opis
val The raw byte value.

Zwroty:

(Single character)

Przykład:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

Powyższy kod tworzy stronę string z surowych danych bajtowych i zapisuje ją do (wcześniej otwartego) pliku.

Powrót: Struny
Następny: chr
