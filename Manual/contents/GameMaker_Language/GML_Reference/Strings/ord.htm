ord

Ta funkcja przyjmuje na wejściu pojedynczy znak string i zwraca wartość Unicode (UTF8) dla tego znaku. Należy pamiętać, że w przypadku użycia funkcji keyboard_check* wejście string może mieć tylko jeden znak i może być tylko liczbą od 0 do 9 lub wielkim znakiem rzymskim od A do Z.

 

Składnia:

ord(string);

Argument Typ Opis
string String The string with which to find the Unicode value.

 

Zwroty:

Real

 

Przykład:

if keyboard_check(ord("W"))
{
    y -= 4;
}

Przytrzymanie klawisza W spowoduje przesunięcie wywołanej instancji o cztery piksele w górę.
