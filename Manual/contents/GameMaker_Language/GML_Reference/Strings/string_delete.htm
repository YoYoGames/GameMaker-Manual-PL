string_delete

Ta funkcja służy do usuwania części podanego adresu string. Podajesz dane wejściowe string, pozycję początkową w obrębie tej strony string, z której mają zostać usunięte znaki (indeks zaczyna się od 1) oraz liczbę znaków do usunięcia. Funkcja zwróci nowy adres string bez tego fragmentu.

Składnia:

string_delete(str, index, count);

ArgumentTyp Opis
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

Zwroty:

String

Przykład:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Spowoduje to ustawienie adresu str2 na "Hello World", ponieważ usunie dodatkowe "o" z "Hello" (które znajduje się na pozycji 5).
