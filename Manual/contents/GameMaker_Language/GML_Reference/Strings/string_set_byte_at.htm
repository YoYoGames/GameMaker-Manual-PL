string_set_byte_at

Ta funkcja ustawia bajt bezpośrednio w pliku string (w formacie UTF8) i zwraca kopię pliku string z wprowadzonymi zmianami.

UWAGA Ta funkcja jest niezwykle powolna, dlatego należy dokładnie rozważyć, czy jest ona konieczna i gdzie ma być używana.

Składnia:

string_set_byte_at(str, pos, byte);

ArgumentTyp Opis
str The string to change the byte of.
pos The position within the string (starting at 1) to change the byte of.
byte The new byte value.

Zwroty:

Przykład:

str = string_set_byte_at("hello", 2, 97);

Powyższy kod zmieniłby wartość bajtu drugiej litery w string, a więc ustawiłby zmienną str na "hallo".

Powrót: Struny
