string_ord_at

Za pomocą tej funkcji można zwrócić kod określonego znaku w określonej pozycji w pliku string, przy czym indeks rozpoczyna się od 1 dla pierwszego znaku. Jeżeli nie zostanie znaleziony żaden znak lub adres string jest krótszy niż wartość podana w indeksie, zwracane jest -1.

Składnia:

string_ord_at(str, index);

ArgumentTyp Opis
str The string to check.
index The position to get the character code from.

Zwroty:

Przykład:

str = "Hello World";
char_code = string_ord_at(str, 7);

Spowoduje to uzyskanie kodu znaku dla siódmego znaku (gdzie "H" liczy się jako pierwszy) w string "str" i zapisanie go w zmiennej "char_code".
