clipboard_set_text

Ta funkcja ustawi schowek na przechowywanie zdefiniowanego string. Możesz ustawić go na pusty string "", aby skutecznie wyczyścić schowek z tekstu.

Składnia:

clipboard_set_text(string);

Argument Typ Opis
string String The text (as a string) to set the clipboard to hold.

Zwroty:

N/A

Przykład:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Powyższy kod sprawdza, czy w schowku znajduje się tekst i jeśli go zawiera, to jest on odczytywany jako string do zmiennej "str". Na koniec schowek jest czyszczony poprzez ustawienie go na pustą string.
