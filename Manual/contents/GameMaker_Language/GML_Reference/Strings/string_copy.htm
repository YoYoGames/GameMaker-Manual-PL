string_copy

Za pomocą tej funkcji można łatwo wybrać pewną liczbę znaków z adresu string, które mają zostać skopiowane do innego adresu. Pierwszy znak w adresie string jest zawsze indeksowany jako 1, a nie 0, jak można by się spodziewać, więc aby skopiować (na przykład) pierwsze pięć znaków z adresu string, należałoby użyć adresu string_copy(str, 1, 5). Kolejny przykład znajduje się poniżej.

Składnia:

string_copy(str, index, count);

ArgumentTyp Opis
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

Zwroty:

Przykład:

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

Powyższy kod umożliwia graczowi wprowadzenie za pomocą klawiatury ciągu string, który jest następnie zapisywany w zmiennej name. Ta zmienna jest następnie sprawdzana, czy nie przekracza maksymalnej długości 15 znaków, i jeśli tak jest, ciąg klawiatury jest zastępowany kopią name string o długości 15 znaków. W ten sposób wprowadzane przez gracza dane są ograniczone do 15 znaków.
