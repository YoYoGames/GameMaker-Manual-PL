string_byte_length

Funkcja zwraca liczbę bajtów w pliku string. Ze względu na to, że są one przechowywane jako UTF8, nie będzie to równe liczbie znaków w pliku string.

Składnia:

string_byte_length(string);

Argument Typ Opis
string String The string to measure the number of bytes of.

Zwroty:

Real

Przykład:

bytesize = string_byte_length("Hello World");

To ustawiłoby bytesize na liczbę bajtów w "Hello World".

Powrót: Struny
Następny: string_set_byte_at
