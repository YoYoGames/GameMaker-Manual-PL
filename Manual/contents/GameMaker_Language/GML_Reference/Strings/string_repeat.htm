string_repeat

Funkcja ta zwraca po prostu tę samą stronę string powtórzoną określoną liczbę razy nad sobą.

Składnia:

string_repeat(str, count);

ArgumentTyp Opis
str The string to repeat.
count The number of times to repeat the string.

Zwroty:

Przykład:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Powyższy kod ustawi adres str2 na "Hello WorldHello WorldHello World".

Powrót: Struny
Następny: string_replace
