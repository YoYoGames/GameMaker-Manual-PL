string_pos

Ta funkcja zwraca pozycję znaku instancji podłańcucha w string, szukając od początku string (aby szukać od końca, należy użyć funkcji string_last_pos()). Funkcja zwróci 0, jeśli w ogóle nie zostanie znaleziona, lub pozycję pierwszego znaku podłańcucha, jeśli zostanie znaleziony. Należy pamiętać, że w starszych wersjach strings są indeksowane od 1, więc 1 jest pierwszą pozycją w string, a nie 0, jak można by się spodziewać. Jednym z zastosowań tej funkcji jest filtrowanie słów, które mogą być uznane za obraźliwe, lub znajdowanie właściwego miejsca do wstawienia tekstu do innej strony string.

Składnia:

string_pos(substr, str);

ArgumentTyp Opis
substr The substring to look for in the string.
str The string.

Zwroty:

Przykład:

if (string_pos(",", text) != 0)
{
    string_insert(name, text, string_pos(",", text));
}

Powyższy kod przeszukuje stronę string zapisaną w zmiennej "text" w poszukiwaniu przecinka, a jeśli go znajdzie, wstawia w tym miejscu podłańcuch "name".
