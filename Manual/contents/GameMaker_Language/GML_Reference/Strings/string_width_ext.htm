string_width_ext

Ta funkcja zwraca maksymalną szerokość (w pikselach) strony wejściowej string, biorąc pod uwagę odstęp między wierszami i szerokość łamania wiersza (zdefiniowaną jako liczba pikseli, które może zająć strona string przed wstawieniem łamania wiersza). Jest to bardzo przydatna do obliczania odległości między elementami tekstu na podstawie maksymalnej szerokości strony string, która jest podzielona na kilka wierszy, tak jak zostałaby narysowana za pomocą draw_text_ext() przy użyciu aktualnie zdefiniowanego font. Odstępy i szerokość można ustawić na -1, aby uzyskać domyślne odstępy.

Składnia:

string_width_ext(string, sep, w);

ArgumentTyp Opis
string The string to measure the width of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Zwroty:

Przykład:

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

Powyższy kod pobierze szerokość podanej strony string, biorąc pod uwagę szerokość separacji wierszy i szerokość łamania wierszy, a następnie narysuje dwa wiersze tekstu, używając zwróconej całkowitej szerokości strony string jako separatora.
