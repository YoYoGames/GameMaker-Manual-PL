string_byte_at

Zwraca surową wartość bajtu jako wartość rzeczywistą w podanej pozycji w podanym adresie string.

Składnia:

string_byte_at(str, index);

ArgumentTyp Opis
str The string to check.
index The position to get the byte from.

Zwroty:

Przykład:

newbyte = string_byte_at("Hello World", 5);

Spowoduje to ustawienie adresu newbyte na surową wartość bajtową szóstej litery napisu "Hello World".

Powrót: Struny
Następny: string_byte_length
