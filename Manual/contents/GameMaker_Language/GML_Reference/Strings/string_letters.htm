string_letters

Za pomocą tej funkcji można usunąć wszystkie znaki, które nie są sklasyfikowane jako litery. Jest to przydatne, aby uniemożliwić wprowadzanie niepożądanych znaków do tekstu (np. wprowadzanie "#" w celu wymuszenia nowej linii).

UWAGA Ta funkcja wykrywa i zwraca tylko 26 liter alfabetu angielskiego od A do Z (duże lub małe litery).

Składnia:

string_letters(string);

ArgumentTyp Opis
string The string to convert to letters.

Zwroty:

Przykład:

username = string_letters(username);

Powyższy kod ustawi zmienną "username" tak, aby zawierała tylko znaki literowe z oryginalnej strony string (jeśli oryginalna strona string to "MacSw33ny", zwrócona strona string będzie miała postać "MacSwny").
