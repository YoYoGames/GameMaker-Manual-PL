długość ciągu

Ta funkcja zwraca liczbę znaków składających się na dany adres string. Może być przydatna do takich rzeczy jak sprawdzenie, kiedy ograniczyć długość znaków we wpisie tekstowym (np.: ograniczenie nazwy gracza do 10 znaków). Pamiętaj, że różni się to od string_width() tym, że mierzy liczbę znaków w string, a nie szerokość w pikselach na ekranie.

Składnia:

string_length(string);

Argument Typ Opis
string String The string to measure the number of characters of.

Zwroty:

Real

Przykład:

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

To sprawdzi, czy string nazwy jest większy niż dziesięć znaków, a jeśli tak, to skopiuje i użyje tylko pierwszych dziesięciu znaków.
