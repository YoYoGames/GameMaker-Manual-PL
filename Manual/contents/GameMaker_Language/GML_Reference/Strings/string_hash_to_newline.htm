string_hash_to_newline

Ta funkcja zwraca adres string, w którym symbol "#" został zamieniony na nowy wiersz. Tak więc strona string sformatowana jako:

"Hello#World"

Na ekranie zostanie wydrukowany jako:

Hello
World

WAŻNE! Funkcja ta została udostępniona w celu zapewnienia kompatybilności importowej między programem GameMaker a poprzednimi wersjami oprogramowania i jako taka nie powinna być używana, chyba że jest to konieczne do odtworzenia przestarzałych funkcji. Zamiast powinieneś używać znaków escape do zaznaczania nowej linii. Więcej informacji na temat kompatybilności importu można znaleźć tutaj.

Składnia:

string_hash_to_newline(string);

ArgumentTyp Opis
string The string to convert over multiple lines

Zwroty:

Przykład:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

Powyższy kod zamienia stronę string z symbolem hash na stronę string podzieloną na dwa wiersze, a następnie wypisuje ją na ekran.
