string_upper

Za pomocą tej funkcji można wymusić, aby strona string zawierała tylko wielkie litery.

UWAGA Ta funkcja wykrywa tylko 26 liter alfabetu angielskiego od A do Z.

Składnia:

string_upper(string);

ArgumentTyp Opis
string The string to convert to uppercase.

Zwroty:

Przykład:

str1 = "Hello World";
str2 = string_upper(str1);

Powyższy kod ustawi zmienną str2 na "HELLO WORLD".

Powrót: Struny
