schowek_get_text

Funkcja zwróci string tekstu zawartego w schowku. jeśli nie ma żadnego tekstu zwróci pusty string "".

Składnia:

clipboard_get_text();

Zwroty:

String

Przykład:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Powyższy kod sprawdza, czy w schowku znajduje się tekst i jeśli go zawiera, to jest on odczytywany jako string do zmiennej "str". Na koniec schowek jest czyszczony poprzez ustawienie go na pustą string.
