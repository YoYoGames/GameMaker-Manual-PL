string_char_at

Za pomocą tej funkcji można zwrócić określony znak w określonej pozycji w pliku string, przy czym indeks rozpoczyna się od 1 dla pierwszego znaku. Jeśli nie zostanie znaleziony żaden znak lub adres string jest krótszy niż podana wartość indeksu, zwracany jest pusty adres string "", jednak jeśli podany indeks jest równy lub mniejszy od 0, zwracany jest pierwszy znak adresu string.

Składnia:

string_char_at(str, index);

ArgumentTyp Opis
str The string to check.
index The position to get the character from.

Zwroty:

Przykład:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Spowoduje to ustawienie adresu str2 na siódmy znak ("H" liczony jako pierwszy) w adresie string str1, w tym przypadku "W".
