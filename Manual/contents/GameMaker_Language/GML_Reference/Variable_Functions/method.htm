metoda

Za pomocą tej funkcji można powiązać istniejącą funkcję (lub metodę) z podaną instancją lub adresem struct, tworząc nową zmienną metody, która może być później użyta.

Podajesz identyfikator instancji, której chcesz użyć (instancji, która jest aktywna w room, a nie w indeksie object ) lub referencję struct oraz identyfikator funkcji (lub referencję metody), którą chcesz powiązać. Funkcja zwróci nową metodę, którą będzie można wywołać ze zmiennej, do której jest przypisana (patrz przykład kodu poniżej).

Zwrócona metoda będzie "związana" z podaną instancją lub struct, co oznacza, że zawsze będzie wykonywana w zakresie tej instancji/struktury.

Można wiązać zarówno funkcje wbudowane, jak i funkcje/metody zdefiniowane przez użytkownika, a także można podać undefined jako argument instancji/struktury, co oznacza, że do wiązania zostanie użyty bieżący self zakres zostanie użyty do powiązania.

Składnia:

method(struct_ref_or_instance_id, func);

Argument Typ Opis
struct_ref_or_instance_id Identyfikatorstruktury lub instancji The unique reference or ID value of the struct or instance to use (can be self lub undefined)
func Funkcja lub metodaskryptu The ID of the function (or method reference) to use

Zwroty:

Method

Przykład:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
    enemy_func = method(_inst, enemy_ai);
}

Powyższy kod sprawdzi, czy w miejscu, w którym znajduje się mysz, istnieje instancja wroga. Jeśli tak, to metoda enemy_ai zostanie powiązana z instancją wroga i zwrócona jako nowa zmienna metody "enemy_func".
