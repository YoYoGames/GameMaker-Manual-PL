array_equals

Za pomocą tej funkcji można sprawdzić, czy dwie tablice są równe (równoważne lub takie same). Podajesz dwie tablice do sprawdzenia, a funkcja zwróci true, jeśli są one równe (równoważne lub takie same) lub false, jeśli nie są. Zauważ, że nie jest to to samo, co sprawdzenie, czy dwie tablice są takie same, przy użyciu ==, które nie sprawdza, czy dwie tablice przechowują równoważne wartości, a jedynie czy tablice odwołują się do tej samej tablicy początkowej. Na przykład:

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

Należy pamiętać, że niektóre stałe mogą nie być sobie równe, co może spowodować niepowodzenie tej funkcji. Oto przykład:

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

Więcej informacji można znaleźć w Tabeli równości.

Składnia:

array_equals(var1, var2);

Argument Typ Opis
var1 Tablica The first array.
var2 Tablica The second array.

Zwroty:

Boolean

Przykład:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

Powyższy kod sprawdzi dwie tablice, czy przechowują one równoważne wartości, a jeśli nie, to skopiuje całą zawartość tablicy "inventory_array" do tablicy "item_array".
