typeof

Funkcja zwraca typ danych dowolnej zmiennej jako string. Możliwe wartości zwrotne są wymienione w poniższej tabeli:

String Opis
"number" Zmienna przechowuje liczbę rzeczywistą (zmiennoprzecinkową) - może to być NaN oraz infinity
"string" Zmienna przechowuje ciąg znaków
"array" Zmienna odwołuje się do tablicy
"bool" Zmienna przechowuje boolean (true / false)
"int32" Zmienna przechowuje 32-bitową liczbę całkowitą
"int64" Zmienna przechowuje 64-bitową liczbę całkowitą
"ptr" Zmienna przechowuje wskaźnik
"undefined" Zmienna jest niezdefiniowana
"null" Zmienna posiada wartość null (nie powinno to być normalnie widoczne)
"vec3" Zmienna przechowuje wektor 3 wartości
"vec4" Zmienna przechowuje wektor 4 wartości
"method" Zmienna przechowuje referencję do funkcji
"struct" Zmienna przechowuje referencję struct
"ref" Zmienna przechowuje referencję do instancji
"unknown" Wartość jest nieznana. To nie powinno być nigdy widoczne i oznacza, że coś poszło nie tak na najbardziej podstawowym poziomie, jak nadpisanie pamięci.

Należy pamiętać, że istnieją przypadki, w których ta funkcja może nie zwrócić prawidłowej wartości dla metody. Rozważmy następujące dwie definicje funkcji:

a = function()
{
    // something
}

function b()
{
    // Something
}

Technicznie, oba są uważane za metody, ponieważ wiążą funkcję ze zmienną zakresu instancji, jednak wywołanie typeof() na funkcji b zwróci "number", a nie "method", podczas gdy wywołanie na a zwróci "method". Wynika to z faktu, że metody utworzone tak jak w przypadku b mają przypisane indeksy script (które są wartościami całkowitymi), ponieważ w ten sposób kompilator rozpoznaje funkcje skryptowe, nawet jeśli funkcja nie została zdefiniowana w script.

Składnia:

typeof(variable);

Argument Typ Opis
variable Każdy The variable to get the data type of.

Zwroty:

String (see table above)

Przykład:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

Powyższy kod pobiera typ danych przechowywany przez podaną zmienną globalną i zwraca string do zmiennej lokalnej, która następnie jest używana do wyprowadzenia komunikatu na konsolę.

Powrót: Funkcje zmienne
Następny: bool
