Funkcje zmienne

Funkcje na tej stronie służą do obsługi różnych zmiennych i typów zmiennych podczas korzystania z witryny GameMaker Language w grach. Więcej informacji na temat zmiennych można znaleźć na stronie Zmienne i zakres zmiennych.

Dostępne są następujące funkcje zmiennych:

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variable_struct_remove
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

Z tablicami związanych jest też kilka funkcji. Zostały one zaprojektowane w celu zapewnienia elastyczności kodu oraz umożliwienia tworzenia bardziej funkcjonalnych i dynamicznych tablic podczas tworzenia gier. Funkcje te to:

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(przestarzałe)
  • array_length_2d(przestarzałe)
  • array_height_2d(przestarzałe)

Wszystkie te funkcje są związane z używaniem zmiennych metody:

  • method
  • method_get_self
  • method_get_index

Do określania typu zmiennej służą również następujące funkcje (więcej informacji o typach danych można znaleźć tutaj, a o tablicach typów - tutaj):

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
