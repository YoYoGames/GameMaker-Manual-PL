tablica_pop

Ta funkcja usuwa ostatni element z podanej tablicy i zwraca jego wartość. Jeśli tablica jest pusta, zwracany jest adres undefined.

Składnia:

array_pop(array);

ArgumentTyp Opis
array The array to remove the last value from.

Zwroty:

Variable or undefined

Przykład:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

Powyższy kod wyciąga ostatnią wartość z podanej tablicy, a następnie rysuje ją wraz z tekstem na ekranie. Pobrany wpis zostanie również usunięty z tablicy wyników.
