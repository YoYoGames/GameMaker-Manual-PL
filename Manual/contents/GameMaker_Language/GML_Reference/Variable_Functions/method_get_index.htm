metoda_get_index

Za pomocą tej funkcji można uzyskać indeks skryptu dla strony script, na której zdefiniowano daną metodę. Jeżeli metoda nie została zdefiniowana na stronie script, funkcja zwróci -1, w przeciwnym razie zwróci wartość indeksu dla strony script.

Składnia:

method_get_index(method);

Argument Typ Opis
method Metoda The method variable to check

Zwroty:

Script Asset or Real

Przykład:

var _index = method_get_index(light_setup);
if _index != -1
{
    show_debug_message(script_get_name(_index));
}

Powyższy kod sprawdzi indeks script dla metody "light_setup", a następnie, jeśli nie będzie on wynosił -1, wypisze na konsolę nazwę script.

Powrót: Funkcje zmienne
Następny: is_string
