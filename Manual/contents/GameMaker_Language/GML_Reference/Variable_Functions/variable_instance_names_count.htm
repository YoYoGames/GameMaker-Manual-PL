zmienna_instancja_nazwy_liczba

Za pomocą tej funkcji możesz znaleźć całkowitą liczbę zmiennych zdefiniowanych dla danej instancji. Podajesz identyfikator instancji, którą chcesz sprawdzić, a funkcja zwróci liczbę całkowitą określającą liczbę napotkanych zmiennych lub (jeśli nie istnieje żadna instancja o podanym identyfikatorze) -1.

Składnia:

variable_instance_names_count(instance_id);

Argument Typ Opis
instance_id Identyfikator instancji The unique ID value of the instance to check.

Zwroty:

Real

Przykład:

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

Powyższy kod pobierze liczbę zmiennych w danej instancji i wyświetli komunikat debugowania w konsoli z tą wartością.

Powrót: Funkcje zmienne
Następny: variable_instance_get
