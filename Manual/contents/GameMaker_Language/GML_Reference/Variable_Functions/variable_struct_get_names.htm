zmienna_struktura_get_names

Za pomocą tej funkcji można pobrać tablicę wypełnioną nazwami zmiennych z adresu struct. Do sprawdzenia przekazywana jest referencja do adresu struct, a każdy wpis w tablicy będzie ciągiem nazw zmiennych zawartych w adresie struct.

Składnia:

variable_struct_get_names(struct);

ArgumentTyp Opis
struct The struct reference to check.

Zwroty:

(each entry is a string)

Przykład:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
    show_debug_message(str);
}

Powyższy kod pobierze tablicę z nazwami zmiennych dla podanej strony struct, a następnie wyświetli je wraz z wartościami na wyjściu debugowania.
