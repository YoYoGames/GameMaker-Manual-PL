struct_set

Za pomocą tej funkcji można ustawić wartość danej zmiennej w witrynie struct. Należy podać odnośnik do witryny struct oraz nazwę zmiennej, której wartość ma zostać ustawiona, w postaci ciągu znaków (patrz przykładowy kod poniżej), a następnie wartość do ustawienia (może to być dowolny poprawny typ danych). Jeśli zmienna nie istnieje jeszcze na stronie struct, zostanie utworzona, a następnie zostanie jej przypisana wartość.

 

Składnia:

struct_set(struct, name, val);

Argument Typ Opis
struct Struct The struct reference to set
name String The name of the variable to set (as a string)
val Any The value to set the variable to

 

Zwroty:

N/A

 

Przykład:

if !struct_exists(mystruct, "shields")
{
    struct_set(mystruct, "shields", 0);
}

Powyższy kod sprawdzi, czy podana zmienna istnieje w podanym adresie struct, a jeśli nie, to zostanie utworzona i ustawiona na 0.
