int64

Funkcja próbuje przekonwertować podaną wartość na podpisaną 64-bitową liczbę całkowitą, gdzie wartość musi być albo realem, albo string, albo int64, albo int32, albo ptr. Każda inna wartość spowoduje awarię gry z komunikatem o błędzie. Możesz sprawdzić, czy zmienna posiada wartość int64 używając funkcji is_int64().

Składnia:

int64(val);

Argument Typ Opis
val Real, String, lub Pointer The value to convert.

Zwroty:

int64 (signed 64-bit integer)

Przykład:

steam_handle = int64(global.fileReadString);

Powyższy kod konwertuje wartość przechowywaną w zmiennej globalnej na 64-bitową liczbę całkowitą.

Powrót: Funkcje zmienne
Następny: variable_instance_exists
