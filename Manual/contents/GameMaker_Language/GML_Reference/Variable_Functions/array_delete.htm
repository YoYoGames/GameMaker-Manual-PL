tablica_usuń

Za pomocą tej funkcji można usunąć wartość (lub wartości) z tablicy na dowolnej pozycji. Funkcja wymaga podania zmiennej zawierającej tablicę, indeks (pozycję) w tablicy, z której ma zostać usunięta, oraz liczbę wartości do usunięcia.

Składnia:

array_delete(variable, index, number);

ArgumentTyp Opis
variable The variable that holds the array.
index The index (position) in the array to delete the value(s) from.
number The number of values to delete.

Zwroty:

Przykład:

array_delete(score_array, 0, 10);

Powyższy kod usunie 10 pierwszych wartości z podanej tablicy, zaczynając od indeksu 0 (zatem indeksy od 0 do 9 zostaną usunięte z tablicy).
