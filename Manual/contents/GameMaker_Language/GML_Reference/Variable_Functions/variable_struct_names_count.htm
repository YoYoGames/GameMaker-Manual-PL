zmienna_struktura_nazwy_liczba

Za pomocą tej funkcji można znaleźć całkowitą liczbę zmiennych zdefiniowanych dla strony struct. Podajesz identyfikator strony struct, którą chcesz sprawdzić, a funkcja zwróci wartość całkowitą oznaczającą liczbę napotkanych zmiennych lub (jeśli nie istnieje strona struct o podanym identyfikatorze) -1.

Składnia:

variable_struct_names_count(struct_id);

ArgumentTyp Opis
struct_id The unique ID value of the struct to check.

Zwroty:

Przykład:

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

Powyższy kod pobierze liczbę zmiennych w podanym adresie struct i wyświetli komunikat debugowania w konsoli z tą wartością.

Powrót: Funkcje zmienne
Następny: instanceof
