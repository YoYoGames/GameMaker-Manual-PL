metoda_get_self

Za pomocą tej funkcji można pobrać identyfikator instancji lub referencję struct, która jest self kontekstu używanego podczas wywoływania metody. Jeśli zmienna nie jest metodą, to funkcja zwróci undefined. Należy pamiętać, że funkcja może również zwracać stałą pointer_null, w którym to przypadku w momencie wywołania metody używana jest bieżąca jaźń.

Składnia:

method_get_self(method);

ArgumentTyp Opis
method The method variable to check.

Zwroty:

Instance ID, Struct, undefined, or pointer_null

Przykład:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

Powyższy kod pobiera kontekst własny dla podanej zmiennej metody i wypisuje go na konsolę.
