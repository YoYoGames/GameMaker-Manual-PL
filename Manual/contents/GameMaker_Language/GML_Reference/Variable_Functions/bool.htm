bool

Ta funkcja próbuje przekonwertować podaną wartość na typ danych boolean, przy czym wartość zostanie zwrócona jako true, jeśli jest większa lub równa 0,5, oraz false w przeciwnym razie.

 

Składnia:

bool(n)

Argument Typ Opis
n Real The value to convert.

 

Zwroty:

Boolean

 

Przykład:

if !is_bool(val)
{
    val = bool(val);
}

Powyższy kod sprawdza, czy zmienna "val" jest typu boolean, a jeśli nie, to jest na nią konwertowany.
