struct_remove

Za pomocą tej funkcji można usunąć zmienną z witryny struct. Należy podać identyfikator struct, z którego ma zostać usunięta zmienna, oraz nazwę string zmiennej, która ma zostać usunięta.

 

Składnia:

struct_remove(struct, name)

Argument Typ Opis
struct Struct The struct reference to remove the variable from
name String The name of the variable to remove (as a string)

 

Zwroty:

N/A

 

Przykład:

if struct_exists(mystruct, "shields")
{
    struct_remove(mystruct, "shields");
}

Powyższy kod sprawdzi, czy dana zmienna istnieje w podanym adresie struct, a jeśli tak, to zostanie usunięta.
