bufor_peek

Za pomocą funkcji buffer_read() można odczytywać dane z podanej strony buffer w bieżącej pozycji wyszukiwania, przy czym każda odczytana część danych jest przesuwana do przodu o kolejne bajty odczytywanych lub zapisywanych danych. Może się jednak zdarzyć, że będziesz musiał odczytać daną część danych bez zmiany aktualnej pozycji wyszukiwania i wtedy właśnie użyjesz tej funkcji. Wystarczy podać tej funkcji identyfikator buffer, a następnie przesunięcie (od początku adresu buffer ) w obrębie adresu buffer, z którego ma nastąpić odczyt, oraz typ danych, które mają być odczytane.

UWAGA: Użycie nieprawidłowego typu danych dla odczytywanych danych spowoduje otrzymanie błędnych wartości!

Jeśli funkcja powiedzie się, to zwrócona zostanie wartość (Real, string, boolean itd.) zależna od typu danych, natomiast niepowodzenie zwróci po prostu 0.

Składnia:

buffer_peek(buffer, offset, type);

Argument Typ Opis
buffer Identyfikator bufora The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to read the given data from.
type Typ danych bufora Stała The type of data that is to be read from the buffer (see the list of constants here).

Zwroty:

Real, Boolean, String or 0 (if it fails)

Przykład:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

Powyższy kod pobierze trzy wartości z trzech różnych miejsc w witrynie buffer indeksowanych w zmiennej "buff", a następnie użyje tych wartości do ustawienia mieszania obrazu instancji.
