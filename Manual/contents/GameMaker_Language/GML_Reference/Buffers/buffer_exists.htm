bufor_istnieje

Ta funkcja służy do sprawdzania, czy zmienna zawiera prawidłową wartość identyfikatora buffer. Jeśli tak, funkcja zwróci true, w przeciwnym razie zwróci false.

Składnia:

buffer_exists(buffer)

ArgumentTyp Opis
buffer The index of the buffer to check.

Zwroty:

Przykład:

if buffer_exists(buff)
{
    buffer_delete(buff);
}

Powyższy kod sprawdza, czy w zmiennej "buff" znajduje się identyfikator buffer, a jeśli tak, to buffer jest usuwany.

Powrót: Bufory
Następny: buffer_create
