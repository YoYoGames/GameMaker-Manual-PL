bufor_set_used_size

Funkcja ta jest przeznaczona przede wszystkim do użytku w witrynie extensions i umożliwia ustawienie "używanego" rozmiaru danej witryny buffer, czyli liczby bajtów, które zostały do niej zapisane.

Podczas zapisywania danych do buffer z extension, GameMaker nie wie, ile miejsca w buffer zostało wypełnione przez kod extension i nie jest w stanie odczytać tych danych. Ta funkcja może być wywołana przez extension, aby poinformować silnik, ile bajtów danych zapisano na stronie buffer, aby silnik mógł odczytać te dane.

Składnia:

buffer_set_used_size(buffer, size);

ArgumentTyp Opis
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

Zwroty:

Przykład:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

Powyższe funkcje byłyby wywoływane w witrynie extension w celu zapisania pewnych danych na stronie GameMaker buffer (poprzez jej adres pamięciowy), a następnie ustawienia liczby bajtów, które zostały do niej zapisane, aby silnik mógł odczytać te dane.
