bufor_wypełnienie

Za pomocą tej funkcji można wypełnić utworzony wcześniej adres buffer danymi określonego typu i o określonej wartości. Dane, którymi wypełnisz stronę buffer, muszą zgadzać się z argumentem "type" tej funkcji, co oznacza, że nie możesz próbować wypełnić strony za pomocą argumentu "a string a użyć typu unsigned 16bit integer, na przykład. Stałe typu są takie same jak te używane przez funkcje buffer_read() oraz buffer_write() funkcji. Wartość "size" to rozmiar strony buffer (w bajtach), którą chcesz wypełnić, natomiast "offset" to wartość przesunięcia (również w bajtach) od początku strony buffer, od której chcesz rozpocząć wypełnianie.

Składnia:

buffer_fill(buffer, offset, type, value, size);

ArgumentTyp Opis
buffer The index of the buffer to fill.
offset The data offset value (in bytes).
type The type of data that is to be written to the buffer (see the list of constants here).
value The data to write.
size The size of the buffer (in bytes) that you wish to fill.

Zwroty:

Przykład:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

Powyższy kod znajduje początek strony buffer o identyfikatorze zapisanym w zmiennej "buff" i zapisuje do niej serię podpisanych 16-bitowych wartości całkowitych.
