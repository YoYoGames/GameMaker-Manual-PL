bufor_base64_encode

Ta funkcja konwertuje dane z podanego adresu buffer na zakodowany w formacie base64 adres string. Jest to powszechnie stosowany schemat kodowania, często wykorzystywany w przypadku wszelkich mediów, które muszą być przechowywane lub przesyłane przez Internet jako tekst. i sprawia, że dane wyjściowe są nieczytelne dla ludzkiego oka. Aby użyć tego schematu, należy określić już utworzony adres buffer, wartość offsetu (czyli punkt w obrębie adresu buffer, od którego ma się rozpocząć kodowanie) oraz rozmiar w bajtach pamięci buffer, która ma zostać zakodowana.

Składnia:

buffer_base64_encode(buffer, offset, size);

ArgumentTyp Opis
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Zwroty:

Przykład:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

Powyższy kod utworzy kodowanie pełnych danych przechowywanych w pliku buffer indeksowanym zmienną "buff" i zapisze zwrócony plik string w zmiennej lokalnej "b_str".
