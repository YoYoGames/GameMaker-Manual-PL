ds_stack_size

Funkcja ta zwraca "rozmiar" stosu, czyli liczbę elementów, które zostały na nim umieszczone.

Składnia:

ds_stack_size(id);

ArgumentTyp Opis
id The id of the data structure to check.

Zwroty:

Przykład:

if !ds_stack_empty(control_stack)
{
    num = ds_stack_size(control_stack);
}

Powyższy kod sprawdza, czy stos DS jest pusty, a jeśli nie jest, pobiera liczbę elementów, które zawiera, i zapisuje tę wartość w zmiennej.

Powrót: ds_stack_create
Następny: ds_stack_copy
