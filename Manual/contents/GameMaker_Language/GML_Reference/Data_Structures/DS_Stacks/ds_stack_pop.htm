ds_stack_pop

Ta funkcja usuwa najwyższą wartość ze stosu DS, usuwając ją ze stosu i zwracając wartość, która ma być przechowywana w zmiennej. Jeśli stos jest pusty, funkcja zwróci stałą undefined, w przeciwnym razie zwróci wartość rzeczywistą lub string zawartą na stosie.

Składnia:

ds_stack_pop(id);

ArgumentTyp Opis
id The id of the data structure to pop from.

Zwroty:

Variable (Data type value that is stored in the stack) or undefined

Przykład:

if !ds_stack_empty(move_stack)
{
    var xx = ds_stack_pop(move_stack);
    var yy = ds_stack_pop(move_stack);
    move_towards_point(xx, yy, 4);
}

Powyższy kod sprawdza, czy stos DS zaindeksowany w zmiennej "move_stack" jest pusty, a jeśli nie jest, to wyciąga ze stosu dwie najwyższe wartości i używa ich do wyznaczenia kierunku ruchu.
