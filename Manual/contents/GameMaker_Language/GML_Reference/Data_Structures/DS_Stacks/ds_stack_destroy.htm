ds_stack_destroy

Ta funkcja usuwa podaną strukturę danych stosu z pamięci, zwalniając zasoby, których używała, i usuwając wszystkie wartości, które zawierała. Funkcja ta powinna być zawsze używana po zakończeniu korzystania ze stosu DS, aby zapobiec wyciekom pamięci, które mogą spowolnić i zawiesić grę.

WAŻNE! Gdy tworzysz strukturę danych, wartość indeksu identyfikująca ją jest liczbą całkowitą zaczynającą się od 0. Oznacza to, że struktury danych różnych typów mogą mieć tę samą wartość indeksu, więc jeśli masz wątpliwości, powinieneś użyć funkcji ds_exists() przed uzyskaniem do nich dostępu. Należy również pamiętać, że indeksy są używane ponownie, więc wartość indeksu zniszczonej struktury danych może zostać użyta przez nowo utworzoną strukturę. Dlatego zalecamy, aby po zniszczeniu zawsze ustawiać zmienną, która przechowywała indeks DS, na -1.

Składnia:

ds_stack_destroy(id);

ArgumentTyp Opis
id The id of the data structure to remove.

Zwroty:

Przykład:

if (lives == 0)
{
    ds_stack_destroy(AI_stack);
    AI_stack = -1;
    room_goto(rm_Menu);
}

Powyższy kod sprawdzi wartość wbudowanej zmiennej globalnej "lives" i jeśli będzie ona równa 0, zniszczy stos DS zaindeksowany w zmiennej "AI_stack", a następnie zmieni adres rooms.

Powrót: ds_stack_create
Następny: ds_stack_clear
