ds_grid_add

Za pomocą tej funkcji można dodać podaną wartość (rzeczywistą lub string) do wartości podanej komórki w siatce. Wartość, która ma zostać dodana, musi być tego samego typu, co wartość przechowywana w komórce siatki, tzn. nie można dodać string do real i odwrotnie, a w przypadku strings odpowiada to konkatenacji.

Składnia:

ds_grid_add(index, x, y, val);

ArgumentTyp Opis
index The index of the grid.
x The x position of the cell in the grid.
y The y position of the cell in the grid.
val The value to add to the cell.

Zwroty:

Przykład:

ds_grid_add(grid, 5, 5, 6)

Spowoduje to dodanie liczby 6 do danej komórki w siatce DS, do której odwołuje się zmienna "siatka".
