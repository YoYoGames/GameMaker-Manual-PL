ds_grid_create

Za pomocą tej funkcji można utworzyć nową strukturę danych DS grid o podanej szerokości i wysokości komórek. Funkcja zwraca identyfikator, który musi być używany we wszystkich innych funkcjach mających do czynienia z tą siatką DS.

WAŻNE! Gdy tworzysz strukturę danych, wartość indeksu identyfikująca ją jest liczbą całkowitą zaczynającą się od 0. Oznacza to, że struktury danych różnych typów mogą mieć tę samą wartość indeksu, więc jeśli masz wątpliwości, powinieneś użyć funkcji ds_exists() przed uzyskaniem do nich dostępu. Należy również pamiętać, że indeksy są używane ponownie, więc wartość indeksu zniszczonej struktury danych może zostać użyta przez nowo utworzoną strukturę.

Składnia:

ds_grid_create(w, h);

ArgumentTyp Opis
w The width of the grid to be created.
h The height of the grid to be created.

Zwroty:

DS Grid ID

Przykład:

mygrid = ds_grid_create(10, 10)

W ten sposób zostanie utworzona siatka o wysokości 10 komórek i szerokości 10 komórek.

Powrót: Siatki DS
Następny: ds_grid_destroy
