ds_grid_copy

Za pomocą tej funkcji można skopiować zawartość jednej siatki do drugiej. Obie siatki muszą być wcześniej utworzone za pomocą funkcji ds_grid_create() funkcja.

Składnia:

ds_grid_copy(destination, source);

Argument Typ Opis
destination Identyfikator siatki DS This index of the grid to copy to.
source Identyfikator siatki DS This index of the grid to copy from.

Zwroty:

N/A

Przykład:

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

Powyższy kod tworzy nową siatkę DS na podstawie szerokości i wysokości poprzednio utworzonej siatki, a następnie kopiuje informacje z poprzedniej siatki do nowej. Na koniec czyści starą siatkę, tak aby wszystkie komórki miały wartość -1.
