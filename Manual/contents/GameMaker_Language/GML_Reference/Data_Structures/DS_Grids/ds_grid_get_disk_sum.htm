ds_grid_get_disk_sum

Tej funkcji można użyć do dodania wszystkich wartości wszystkich komórek znajdujących się w zdefiniowanym obszarze siatki, jak pokazano na poniższym rysunku:

Składnia:

ds_grid_get_disk_sum(index, xm, ym, r);

ArgumentTyp Opis
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.

Zwroty:

Real or String

Przykład:

val = ds_grid_get_disk_sum(grid, 5, 5, 2)

Powyższy kod ustawi zmienną "val" na sumę wszystkich wartości zawartych w danym dysku siatki DS indeksowanym w zmiennej "grid".
