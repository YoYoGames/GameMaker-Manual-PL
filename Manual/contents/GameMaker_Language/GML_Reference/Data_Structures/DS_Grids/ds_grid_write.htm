ds_grid_write

Funkcji tej można użyć do przekształcenia podanej ds_grid na string, którą można następnie zapisać w zewnętrznym pliku (na przykład). Zwróconą z tej funkcji stronę string można wczytać z powrotem do ds_grid za pomocą funkcji ds_grid_read().

UWAGA Zwracany adres string nie jest adresem czytelnym dla człowieka string, ale raczej zrzutem zawartości struktury danych.

Składnia:

ds_grid_write(index);

ArgumentTyp Opis
index The index of the grid to write.

Zwroty:

Przykład:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

Powyższy kod otworzy plik ini (tworząc go, jeśli jeszcze nie istnieje), a następnie zapisze podaną ds_grid jako string do tego pliku.
