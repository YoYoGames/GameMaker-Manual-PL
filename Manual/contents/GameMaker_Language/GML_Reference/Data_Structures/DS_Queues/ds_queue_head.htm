ds_queue_head

Ta funkcja odczyta tylko pierwszą wartość z adresu queue (tę, która znajduje się "na początku"). Nie spowoduje dequeue tej wartości, co oznacza, że może ona zostać odczytana w przyszłości przez tę funkcję lub funkcję ds_queue_dequeue(). Jeśli adres queue jest pusty, funkcja zwróci stałą undefined, w przeciwnym razie zwróci wartość rzeczywistą lub string zawartą w adresie queue.

Składnia:

ds_queue_head(id);

ArgumentTyp Opis
id The id of the data structure to read from.

Zwroty:

(Data type value stored in the queue)

Przykład:

num = ds_queue_head(control_queue);

Powyższy kod odczyta wartość z adresu queue zaindeksowanego w zmiennej "control_queue" i zapisze wartość zwracaną w zmiennej "num".
