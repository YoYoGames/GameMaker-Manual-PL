ds_map_istnieje

Ta funkcja zwróci true, jeśli podany klucz istnieje w (utworzonej wcześniej) mapie DS , i false, jeśli nie istnieje.

Składnia:

ds_map_exists(id, key);

Argument Typ Opis
id Identyfikator mapy DS The id of the data structure to check
key String The key to check for

Zwroty:

Boolean

Przykład:

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

Powyższy kod sprawdzi mapę DS zaindeksowaną w zmiennej "inventory" pod kątem klucza "potions" i jeśli go nie ma, doda go do mapy.
