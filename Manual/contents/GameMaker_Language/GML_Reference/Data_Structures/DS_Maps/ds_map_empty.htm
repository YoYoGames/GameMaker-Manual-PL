ds_map_empty

Ta funkcja zwraca po prostu false, jeśli podana (utworzona wcześniej) mapa DS zawiera jakieś pary klucz-wartość, lub true, jeśli nie zawiera.

Składnia:

ds_map_empty(id);

ArgumentTyp Opis
id The id of the data structure to check.

Zwroty:

Przykład:

if ds_map_empty(inventory)
{
    weight = 0;
}

Powyższy kod sprawdza, czy na mapie DS indeksowanej zmienną "inventory" znajdują się pary klucz-wartość, a jeśli nie, ustawia zmienną "weight" na 0.
