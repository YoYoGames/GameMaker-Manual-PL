ds_map_read

Funkcja ta pobiera adres string, który został wcześniej utworzony przez funkcję ds_map_write() a następnie wczyta go do utworzonej wcześniej mapy DS. Jeśli mapa, do której wczytuje się string, zawiera jakieś pary klucz-wartość, zostaną one najpierw wyczyszczone, zanim zapisana mapa zostanie ponownie utworzona.

Zwróć uwagę, że jeśli określony plik DS string został napisany w programie GameMaker: Studio 1.2.x runtime (lub starszym), należy podać opcjonalny argument "legacy", ustawiając go na true, ponieważ format string zmienił się po tej wersji.

Składnia:

ds_map_read(id, str, [legacy]);

Argument Typ Opis
id Identyfikator mapy DS The id of the data structure to read the string into
str String The string to read
legacy Boolean OPTIONAL Można podać adres true lub false albo całkowicie go pominąć.

Zwroty:

N/A

Przykład:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

Powyższy kod tworzy nową mapę DS i zapisuje jej indeks id w zmiennej "inventory". Następnie otwiera plik ini i wczytuje adres string z tego pliku do zmiennej tymczasowej "t_string". Na koniec sprawdza, czy adres string jest poprawny (nie jest to domyślna wartość ini ""), a jeśli tak, to wczytuje adres string do nowo utworzonej mapy DS, po czym ponownie zamyka plik ini.
