ds_map_replace

Za pomocą tej funkcji można zmienić wartość dla podanego klucza w mapie DS. Podajesz indeks mapy (zwrócony przez funkcję ds_map_create()) a następnie klucz do zastąpienia - albo string, albo liczbę całkowitą - oraz wartość, na którą ma zostać zastąpiona wartość klucza. Jeśli podany klucz nie istnieje, to zostanie utworzony, a jeśli istnieje, to bieżąca wartość zostanie zastąpiona nową wartością. Funkcja zwróci true, jeśli klucz istnieje i wartość została zastąpiona, oraz false, jeśli klucz nie istnieje i został utworzony nowy klucz z podaną wartością.

Składnia:

ds_map_replace( id, key, val );

ArgumentTyp Opis
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

Zwroty:

Przykład:

ds_map_replace(inventory, "torso", 55);

Powyższy kod wyszukuje w mapie DS klucz "tors", a gdy go znajdzie (lub utworzy, jeśli nie istnieje), aktualna wartość jest zastępowana podaną.
