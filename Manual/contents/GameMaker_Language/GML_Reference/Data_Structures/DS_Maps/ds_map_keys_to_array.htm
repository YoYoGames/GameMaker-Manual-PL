ds_map_keys_to_array

Za pomocą tej funkcji można pobrać wszystkie klucze, które zawiera mapa DS. Podajesz identyfikator mapy DS, z której chcesz pobrać klucze (zwrócony przez ds_map_create()), a funkcja zwróci tablicę, w której każdy wpis to klucz z mapy DS. Funkcja posiada opcjonalny drugi argument, w którym można podać utworzoną przez siebie tablicę, w którym to przypadku dane kluczy mapy zostaną dołączone do istniejących danych w tablicy. Należy pamiętać, że funkcja zmodyfikuje bezpośrednio podaną tablicę, ale zwróci również odniesienie do niej (lub odniesienie do nowej tablicy, jeśli nie została podana).

Składnia:

ds_map_keys_to_array(id, [array])

Argument Typ Opis
id Identyfikator mapy DS The id of the map to use.
[array] Tablica OPTIONAL Tablica, do której mają zostać dołączone klucze mapy DS.

Zwroty:

Array

Przykład:

map_keys = ds_map_keys_to_array(inventory);

Powyższy kod pobiera klucze dla mapy DS, a następnie zapisuje je jako tablicę do późniejszego wykorzystania.

Powrót: Mapy DS
