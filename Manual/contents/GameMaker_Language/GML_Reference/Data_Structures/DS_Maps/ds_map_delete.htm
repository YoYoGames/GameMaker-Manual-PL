ds_map_delete

Za pomocą tej funkcji można usunąć dowolny podany klucz (i odpowiadającą mu wartość) z podanej, utworzonej wcześniej mapy DS.

Składnia:

ds_map_delete(id, key);

Argument Typ Opis
id Identyfikator mapy DS The id of the map to change.
key String The key to delete (along with its associated value).

Zwroty:

N/A

Przykład:

ds_map_delete(inventory, "shield");

Powyższy kod usunie klucz "tarcza" (oraz wartość, z którą jest powiązany) z mapy DS (inwentarza).

Powrót: Mapy DS
Następny: ds_map_empty
