ds_map_find_previous

Ta funkcja zwraca poprzedni klucz zapisany w mapie DS przed kluczem podanym w funkcji. Może to być przydatne, gdy trzeba wykonać iterację po mapie DS w poszukiwaniu czegoś, ale w miarę możliwości należy tego unikać, gdyż może to być powolne. Jeśli taki klucz nie istnieje, funkcja zwróci undefined. Należy to zawsze sprawdzać za pomocą funkcji is_undefined() function.

Składnia:

ds_map_find_previous(id, key);

ArgumentTyp Opis
id The id of the map to use.
key The key to find the previous one to.

Zwroty:

Variable or undefined

Przykład:

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

Powyższy kod tworzy kilka zmiennych tymczasowych, a następnie pobiera rozmiar mapy ds_map i znajduje w niej ostatni klucz zapisany przez komputer. Następnie używa adresu for loop do iteracji wstecznej po mapie ds_map w poszukiwaniu wartości klucza "gold". Jeśli ją znajdzie, wyłamuje stronę loop.
