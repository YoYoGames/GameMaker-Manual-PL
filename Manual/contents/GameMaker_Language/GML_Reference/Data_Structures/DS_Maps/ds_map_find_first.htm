ds_map_find_first

Ta funkcja zwraca pierwszy klucz zapisany w mapie DS. Nie jest to pierwszy klucz w kolejności, w jakiej zostały dodane! Mapy DS nie są przechowywane w postaci liniowej, do tego celu należy użyć listy DS, więc funkcja ta znajduje pierwszy klucz zapisany przez komputer. Może to być przydatne, jeśli musisz iterować po mapie DS w poszukiwaniu czegoś, ale w miarę możliwości należy tego unikać, ponieważ może to być powolne.

Należy pamiętać, że ta funkcja zwróci undefined, jeśli podana mapa DS jest pusta.

Składnia:

ds_map_find_first(id);

ArgumentTyp Opis
id The id of the map to use.

Zwroty:

Variable or undefined

Przykład:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if key != "gold"
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

Powyższy kod tworzy kilka zmiennych tymczasowych, a następnie pobiera rozmiar mapy DS i znajduje w niej pierwszy klucz zapisany przez komputer. Następnie używa adresu for loop do iteracji po mapie DS w poszukiwaniu klucza o wartości "złoto". Jeśli ją znajdzie, wyłamuje stronę loop.
