ds_map_add_list

Za pomocą tej funkcji można przypisać (utworzoną wcześniej) listę DS do klucza w podanej mapie DS. Ta funkcja jest przeznaczona do tworzenia map zgodnych z JSON, które następnie zostaną zakodowane przy użyciu json_encode() i powinna być używana tylko w połączeniu z tą funkcją. Jeśli do mapy DS dodano w ten sposób listę, zniszczenie mapy nadrzędnej spowoduje również zniszczenie zawartych w niej list i zwolnienie ich pamięci, a wywołanie funkcji ds_map_clear() na mapie nadrzędnej spowoduje również zniszczenie i wyczyszczenie wszystkich list oznaczonych flagami.

Składnia:

ds_map_add_list(id, key, value)

Argument Typ Opis
id Identyfikator mapy DS The id of the map to use.
key String The key for the added list.
value Identyfikator listy DS The id of the list to add.

Zwroty:

N/A

Przykład:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Powyższy kod utworzy listę i zapełni ją różnymi wartościami zmiennych globalnych. Lista ta jest następnie "zagnieżdżana" w mapie DS, a mapa jest następnie kodowana do pliku JSON string, po czym mapa jest niszczona, co powoduje usunięcie jej i wszystkich list, które zawiera, z pamięci.
