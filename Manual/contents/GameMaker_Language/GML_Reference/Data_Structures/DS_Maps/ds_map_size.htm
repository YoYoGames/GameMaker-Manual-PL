ds_map_size

Za pomocą tej funkcji można sprawdzić, ile par klucz/wartość zawiera (utworzona wcześniej) mapa DS.

Składnia:

ds_map_size(id);

ArgumentTyp Opis
id The id of the data structure to check.

Zwroty:

Przykład:

if (ds_map_size(inventory) > 49)
{
    full = true;
}

Powyższy kod sprawdzi rozmiar mapy DS (tzn. liczbę par klucz/wartość) i jeśli jest ona większa niż 49, ustawi zmienną "full" na true.
