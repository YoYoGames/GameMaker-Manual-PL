ds_map_copy

Za pomocą tej funkcji można skopiować zawartość jednej mapy do innej, utworzonej wcześniej za pomocą strony ds_map_create(). Jeśli mapa DS, do której ma zostać skopiowana, nie jest pusta, to przed kopiowaniem funkcja ta najpierw ją wyczyści. Oryginalna mapa DS pozostaje w tym procesie niezmieniona.

Składnia:

ds_map_copy(id, source);

ArgumentTyp Opis
id The id of the map you are copying to
source The id of the map you are copying from

Zwroty:

Przykład:

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

Powyższy kod utworzy nową mapę i przypisze ją do zmiennej "inventory_2". Następnie skopiuje zawartość mapy DS zindeksowanej w zmiennej "inventory_1" do tej nowej mapy.

Powrót: Mapy DS
Następny: ds_map_replace
