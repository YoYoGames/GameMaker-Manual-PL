ds_map_write

Funkcja ta przekształca dane mapy DS określonej mapy w format string, który można następnie zapisać w pliku *.ini lub *.txt w celu ułatwienia przechowywania. Plik string można później wczytać z powrotem do nowej ds_map za pomocą ds_map_read().

UWAGA: Zwracany adres string nie jest adresem czytelnym dla człowieka string, ale raczej zrzutem zawartości struktury danych.

Składnia:

ds_map_write(id);

ArgumentTyp Opis
id The id of the data structure to use

Zwroty:

Przykład:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Powyższy kod otwiera plik ini gotowy do zapisu. Następnie używa adresu ds_map_write() do wygenerowania adresu string, który jest przechowywany w zmiennej tymczasowej "t_string". Na koniec zapisuje adres string do pliku ini i zamyka go.

Powrót: Mapy DS
