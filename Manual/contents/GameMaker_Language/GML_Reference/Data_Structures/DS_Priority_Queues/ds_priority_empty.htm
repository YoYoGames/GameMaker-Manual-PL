ds_priority_empty

Za pomocą tej funkcji można sprawdzić, czy podany priorytet DS queue jest pusty (zwraca true), czy nie (zwraca false).

Składnia:

ds_priority_empty(id);

ArgumentTyp Opis
id The id of the data structure to check.

Zwroty:

Przykład:

if (count == 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Powyższy kod sprawdza zmienną, czy osiągnęła określoną wartość, a jeśli tak, to czyści priorytet DS queue zaindeksowany w zmiennej "command_queue", ustawia alarm i resetuje zmienną do 0.
