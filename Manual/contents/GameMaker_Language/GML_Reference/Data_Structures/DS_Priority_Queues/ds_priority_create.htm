ds_priority_create

Funkcja ta utworzy nową strukturę danych priorytetu queue i zwróci wartość indeksu. Wartość ta powinna być przechowywana w zmiennej i wykorzystywana we wszystkich dalszych wywołaniach funkcji związanych z priorytetem queue.

WAŻNE! Gdy tworzysz strukturę danych, wartość indeksu identyfikująca ją jest liczbą całkowitą zaczynającą się od 0. Oznacza to, że struktury danych różnych typów mogą mieć tę samą wartość indeksu, więc jeśli masz wątpliwości, powinieneś użyć funkcji ds_exists() przed uzyskaniem do nich dostępu. Należy również pamiętać, że indeksy są używane ponownie, więc wartość indeksu zniszczonej struktury danych może zostać użyta przez nowo utworzoną strukturę.

Składnia:

ds_priority_create();

Zwroty:

DS Priority ID

Przykład:

p_queue = ds_priority_create();

Spowoduje to utworzenie nowego priorytetu queue i przypisanie jego indeksu id do zmiennej instancji "p_queue".

Powrót: Kolejki priorytetowe DS
