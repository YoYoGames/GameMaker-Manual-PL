ds_list_insert

Ta funkcja doda podaną wartość do listy na podanej pozycji. Jeśli lista zawiera więcej wartości po podanej pozycji, ich pozycja zostanie przesunięta o jeden w górę, aby room powiększyć listę o jeden.

Składnia:

ds_list_insert(id, pos, val);

Argument Typ Opis
id Identyfikator listy DS The id of the list to add to.
pos Real The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Zmienna The value to add to the list.

Zwroty:

N/A

Przykład:

ds_list_insert(list, 9, score);

Powyższy kod doda wartość przechowywaną w zmiennej "score" na 10. pozycji listy indeksowanej zmienną "list" (listy są liczone od 0, więc lista dziesięciowartościowa jest numerowana od 0 do 9).

Powrót: Listy DS
Następny: ds_list_replace
