ds_list_is_map

Za pomocą tej funkcji można sprawdzić, czy na podanej pozycji na liście DS znajduje się mapa DS. Jeśli podana pozycja zawiera identyfikator mapy DS, to funkcja zwróci true, w przeciwnym razie zwróci false.

Należy pamiętać, że w ten sposób zostaną wykryte tylko mapy, które zostały ręcznie oznaczone za pomocą funkcji ds_list_mark_as_map() .

Składnia:

ds_list_is_map(id, pos);

ArgumentTyp Opis
id The id of the list to check.
pos The position within the list to check.

Zwroty:

Przykład:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if ds_list_is_map(ships, i)
    {
        ds_map_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Powyższy kod loops przegląda listę DS i sprawdza, czy któryś z wpisów zawiera identyfikatory map. Jeśli tak, to mapy te są niszczone, a następnie po zakończeniu loop niszczona jest główna lista.
