ds_list_add

Za pomocą tej funkcji można dodać do listy nową wartość (rzeczywistą lub string), która zostanie dodana na końcu. Funkcja może przyjmować dalsze opcjonalne argumenty (dowolną ich liczbę), co pozwala na dodawanie wielu wartości kolejno do listy podczas jednego wywołania.

Składnia:

ds_list_add(id, val1 [, val2, ... max_val]);

Argument Typ Opis
id Identyfikator listy DS The id of the list to add to.
val Zmienna The value to add to the list.
[val2, ... max_val] Zmienna Optional values to be added to the list.

Zwroty:

N/A

Przykład:

ds_list_add(sc_list, score);

Powyższy kod doda wartość zapisaną w zmiennej "score" do listy zaindeksowanej w zmiennej "sc_list".

Powrót: Listy DS
Następny: ds_list_set
