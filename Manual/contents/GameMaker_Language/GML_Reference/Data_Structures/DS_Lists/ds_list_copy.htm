ds_list_copy

Za pomocą tej funkcji można skopiować zawartość jednej listy do drugiej. Obie listy muszą być utworzone wcześniej, a jeśli lista, na którą kopiujemy, zawiera już informacje, zostanie ona najpierw wyczyszczona. W rezultacie powstają dwie niezależne listy, które zawierają te same informacje.

Składnia:

ds_list_copy( id, source );

ArgumentTyp Opis
id The id of the list being copied to.
source The id of the list to be copied from.

Zwroty:

Przykład:

if !ds_list_empty(main_list)
{
    old_list = ds_list_create();
    ds_list_copy(old_list, main_list);
    ds_list_clear(main_list);
}

Powyższy kod sprawdzi, czy lista DS jest pusta. Jeśli nie jest pusta, zostaje skopiowana do innej listy DS (utworzonej wcześniej), a następnie oryginalna lista zostaje wyczyszczona.

Powrót: Listy DS
Następny: ds_list_read
