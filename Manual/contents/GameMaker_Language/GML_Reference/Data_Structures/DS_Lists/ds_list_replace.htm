ds_list_replace

Ta funkcja zamienia wartość na podanej pozycji na inną.

Składnia:

ds_list_replace(id, pos, val);

Argument Typ Opis
id Identyfikator listy DS The id of the list to change.
pos Real The position to replace the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Zmienna The new value to replace the given value with.

Zwroty:

N/A

Przykład:

ds_list_replace(n_list, 3, name);

Powyższy kod zastąpi wartość zapisaną na pozycji 3 na liście wartością zapisaną w zmiennej "name".
