ds_list_size

Ta funkcja zwraca "rozmiar" listy, czyli liczbę elementów, które zostały do niej dodane.

Składnia:

ds_list_size(id);

ArgumentTyp Opis
id The id of the data structure to check.

Zwroty:

Przykład:

if !ds_list_empty(control_list)
{
    num = ds_list_size(control_list);
}

Powyższy kod sprawdza, czy lista DS jest pusta, a jeśli nie jest, pobiera liczbę elementów, które zawiera, i zapisuje tę wartość w zmiennej.
