os_powersave_enable

Za pomocą tej funkcji można włączać lub wyłączać funkcje oszczędzania energii w urządzeniu. Jest to ważne, ponieważ niektóre gry (na przykład te, które wykorzystują funkcje pochylania) mogą nie generować zdarzeń, które system operacyjny może zinterpretować jako wprowadzane przez użytkownika i w ten sposób wyłączyć ekran lub zakończyć grę. Ustawiając tę funkcję na false, można wyłączyć funkcje oszczędzania energii i sprawić, że ekran (i gra) będą zawsze działać.

UWAGA: Dotyczy to tylko stron iOS i Android.

Składnia:

os_powersave_enable(flag)

ArgumentTyp Opis
flag Set to true lub false, aby włączyć lub wyłączyć funkcje oszczędzania energii.

Zwroty:

Przykład:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

Powyższy kod sprawdza typ systemu operacyjnego i jeśli jest to Android lub iOS, funkcje oszczędzania energii są wyłączane.

Powrót: System operacyjny i kompilator
Następny: os_lock_orientation
