external_define

Funkcji tej można użyć do zdefiniowania zewnętrznego wywołania funkcji do określonego pliku dll (w przypadku Windows) lub dylib (w przypadku komputerów Mac). Ten plik może być zarówno plikiem dołączonym, jak i częścią extension.

Podajemy nazwę (oraz path) pliku, a następnie nazwę funkcji, którą chcemy zdefiniować. Następnie należy określić konwencję wywoływania (patrz lista stałych poniżej) oraz typ oczekiwanego wyniku (również stały, wymienione poniżej).

Na końcu należy podać liczbę argumentów, jakie może przyjmować funkcja (od 0 do 15), a dla każdego z nich należy określić jego typ. Należy pamiętać, że w przypadku funkcji z 4 lub więcej argumentami, wszystkie muszą być typu ty_real.

UWAGA: Dotyczy to tylko plików dll lub dylib, które zostały dodane jako "Included Files" do pakietu GameMaker IDE. Nie będzie działać z plikami dodanymi jako extensions, ponieważ wymagają one zdefiniowania funkcji w samym pakiecie extension.

Stałe typu wywołania zewnętrznego
Stała Opis
dll_cdecl Jest to domyślne wywołanie w językach C i C++
dll_stdcall Jest to standardowe wywołanie WinAPI (tylkoWindows dll )
Stałe zewnętrznego typu danych
Stała Opis
ty_real Argument liczby rzeczywistej
ty_string Argument string o końcówce zerowej

Składnia:

external_define(dll, name, calltype, restype, argnumb, argtype[0], argtype[1], ...argtype[10]);

Argument Typ Opis
dll String The name of the DLL file (string)
name String The name of the function (string)
calltype Stała typu wywołania zewnętrznego (dll_*) The calling convention used
restype Stała zewnętrznego typu danych (ty_*) The type of the result to expect
argnumb Real The number of arguments (0 - 10)
argtype[0 ... 10] Stała zewnętrznego typu danych (ty_*) The types of the arguments being used

Zwroty:

External Function

Przykład:

my_funcion = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);

W powyższym przykładzie zostanie zdefiniowana zewnętrzna funkcja o nazwie "MojaMin" z dwoma argumentami.

Powrót: System operacyjny i kompilator
Następny: external_call
