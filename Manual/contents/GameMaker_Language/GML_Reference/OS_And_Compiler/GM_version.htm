GM_version

Ta stała przechowuje numer wersji zdefiniowany w opcjach gry dla każdej platformy docelowej. Wartość jest przechowywana jako string.

Składnia:

GM_version;

Zwroty:

Przykład:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

Powyższy kod pobiera stałe GM date oraz version i rysuje je na ekranie.

Powrót: System operacyjny i kompilator
Następny: GM_runtime_version
