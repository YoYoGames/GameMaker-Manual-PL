GM_build_type

Ta stała string przechowuje typ kompilacji: "exe" dla utworzonego pliku wykonywalnego lub "run" dla uruchomienia testowego z IDE.

Składnia:

GM_build_type;

Zwroty:

String

Przykład:

draw_text(32, 32, GM_build_type);

Powyższy kod wyciągnie typ kompilacji aktualnie uruchomionego pliku wykonywalnego.

Powrót: OS i kompilator
Następny: GM_version
