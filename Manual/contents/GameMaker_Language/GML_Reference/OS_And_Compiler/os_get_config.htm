os_get_config

Ta funkcja zwraca nazwę (w postaci string) aktualnie wybranej konfiguracji gry. Więcej informacji o konfiguracjach można znaleźć w rozdziale Konfiguracje.

Składnia:

os_get_config()

Zwroty:

Przykład:

if os_get_config() = "Free_Version"
{
    global.Ads = true;
}
else global.Ads = false;

Powyższy kod sprawdzi, jaka konfiguracja jest używana i jeśli jest to konfiguracja o nazwie "Free_Version", reklamy zostaną włączone w grze.

Powrót: System operacyjny i kompilator
