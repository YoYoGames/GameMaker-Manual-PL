GM_runtime_version

Ta stała przechowuje numer wersji runtime zdefiniowany w Preferencjach jako runtime używany do budowy projektu. Wartość jest przechowywana jako string.

Składnia:

GM_runtime_version;

Zwroty:

String

Przykład:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

Powyższy kod pobiera stałe GM date, version oraz runtime i rysuje je na ekranie.

Powrót: OS i kompilator
Następny: GM_project_filename
