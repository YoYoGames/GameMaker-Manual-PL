scheduler_resolution_get

Ta funkcja służy do pobierania rozdzielczości harmonogramu wątków Windows w milisekundach. Jeżeli rozdzielczość harmonogramu jest ustawiona na wartość domyślną (ustawioną przez Windows), funkcja zwróci -1.

Informacje na temat harmonogramu wątków można znaleźć na stronie scheduler_resolution_set().

UWAGA: Ta funkcja jest przeznaczona tylko dla systemu Windows .

Składnia:

scheduler_resolution_get();

Zwroty:

(or -1 for default)

Przykład:

scheduler_resolution = scheduler_resolution_get();

Ten przykład pobiera rozdzielczość harmonogramu wątków Windows i zapisuje ją w zmiennej "scheduler_resolution".
