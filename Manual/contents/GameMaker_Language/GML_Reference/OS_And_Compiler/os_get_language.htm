os_get_language

Ta funkcja zwraca adres string z dwuliterowym kodem języka systemu operacyjnego, w którym uruchomiona jest gra, zgodnie z normą ISO639. Jeśli informacje te nie są dostępne, funkcja zwraca pusty adres string "" lub "en" dla języka angielskiego. Należy pamiętać, że niektóre języki mają także odpowiedni kod regionalny, więc aby rozróżnić różne regiony w tym samym kraju, należy użyć funkcji os_get_region().

W poniższej tabeli przedstawiono przykłady niektórych głównych dwuliterowych kodów językowych określonych w normie ISO 639:

UWAGA: Zwracany jest nie kod kraju lokalizacji, ale kod języka systemu operacyjnego.

Składnia:

os_get_language()

Zwroty:

Przykład:

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

Powyższy kod sprawdza język systemu operacyjnego i w zależności od zwróconej wartości otwiera inny plik *.ini.
