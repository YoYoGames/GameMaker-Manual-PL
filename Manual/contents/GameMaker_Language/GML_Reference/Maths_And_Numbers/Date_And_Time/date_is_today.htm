data_jest_dzisiejsza

Ta funkcja zwróci true, jeśli podana wartość datetime jest dniem, którego dotyczy sprawdzanie (np. dzisiaj), lub false w przeciwnym wypadku. Funkcja ta może być przydatna w takich sytuacjach jak Wielkanocne Jaja w twoich grach lub do odblokowywania zawartości sezonowej. Należy pamiętać, że na tę funkcję ma wpływ ustawiona strefa czasowa (domyślnie jest to czas lokalny), którą można zmienić za pomocą funkcji date_set_timezone() funkcja.

Składnia:

date_is_today(date);

ArgumentTyp Opis
date The datetime to use.

Zwroty:

Przykład:

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

Powyższy kod sprawdzi, czy data przechowywana w zmiennej globalnej "Halloween" to dzień dzisiejszy, a jeśli tak, to ustawi inną zmienną globalną na nową wartość.
