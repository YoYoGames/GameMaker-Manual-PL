aktualny_czas

Ta zmienna tylko do odczytu zwraca liczbę milisekund, które upłynęły od rozpoczęcia gry.

Składnia:

current_time;

Zwroty:

Przykład:

if (current_time > 600000)
{
    msg = show_question_async("Would you like to rate?");
}

Powyższy kod sprawdza, czy minęło więcej niż 10 minut, zanim użytkownik otrzyma pytanie.

Powrót: Data i czas
Następny: current_second
