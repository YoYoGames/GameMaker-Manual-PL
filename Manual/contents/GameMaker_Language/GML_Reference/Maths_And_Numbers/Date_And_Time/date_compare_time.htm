data_porównania_czasu

Za pomocą tej funkcji można sprawdzić dwie daty i godziny, aby zobaczyć, która z nich ma składnik czasu wcześniejszy lub późniejszy od drugiej. Funkcja zwraca -1, jeśli datetime1 jest wcześniejsze, 1, jeśli datetime1 jest późniejsze i 0, jeśli są takie same, oraz ignoruje datę, ignoruje datę, więc dosłownie tylko to, który z czasów jest późniejszy w stosunku do danego dnia.

Składnia:

date_compare_time(datetime1, datetime2);

ArgumentTyp Opis
datetime1 The first datetime.
datetime2 The datetime to compare the first one to.

Zwroty:

Przykład:

d = date_compare_time(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Spowoduje to ustawienie wartości "d" na odpowiednią wartość w zależności od tego, który z podanych czasów jest najwcześniejszy. Zasadniczo, jeśli aktualny czas jest późniejszy niż 11:04, zwróciłoby 1. Jeśli wcześniej - -1. Jeśli czas jest równy 11:04, zwracane jest 0.
