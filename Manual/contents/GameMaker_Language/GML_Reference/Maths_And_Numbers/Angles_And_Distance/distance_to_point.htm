odległość_do_punktu

Ta funkcja oblicza odległość od krawędzi ramki ograniczającej instancji wywołującej do podanej pozycji x/y na stronie room, przy czym wartość zwracana jest w pikselach. Zwróć uwagę, że jeśli wywołująca funkcja object nie ma zdefiniowanej strony sprite lub maski, wyniki będą nieprawidłowe.

Składnia:

distance_to_point(x, y);

ArgumentTyp Opis
x The x position to check.
y The y position to check.

Zwroty:

Przykład:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    canshoot = true;
}

Powyższy kod sprawdzi odległość do pozycji gracza object x/y i jeśli jest ona mniejsza niż wartość zapisana w zmiennej "range", zmienna "canshoot" zostanie ustawiona na true.
