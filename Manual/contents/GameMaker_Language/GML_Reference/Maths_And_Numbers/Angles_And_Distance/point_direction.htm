punkt_kierunek

Ta funkcja zwraca kierunek wektora utworzonego z określonych składowych [x1,y1] i [x2,y2] w odniesieniu do ustalonych współrzędnych x/y na stronie room. Na przykład na poniższym obrazku, jeśli chcemy uzyskać kierunek od pozycji statku gracza do pozycji wroga, aby móc wystrzelić w niego pocisk, użyjemy tej funkcji (dokładny kod znajduje się w przykładzie poniżej obrazka):

Składnia:

point_direction(x1, y1, x2, y2)

ArgumentTyp Opis
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector

Zwroty:

Przykład:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile))
{
    direction = point_direction(x, y, ex, ey);
}

Powyższy kod pobierze współrzędne x i y najbliższego wroga, a następnie przekaże je do pocisku object w celu użycia w funkcji point_direction, aby prawidłowo ustawić jego kierunek poruszania się.

Powrót: Kąty i odległości
Następny: point_distance
