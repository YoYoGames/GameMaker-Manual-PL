ceil

Funkcja ta przyjmuje dowolną liczbę rzeczywistą i zaokrągla ją w górę do najbliższej liczby całkowitej. Należy zachować ostrożność przy korzystaniu z tej funkcji, ponieważ często popełnianym błędem jest zaokrąglanie losowej wartości i oczekiwanie, że zawsze będzie ona większa od 1, np:

int = ceil(random(5));

Można by się spodziewać, że ten kod zawsze da liczbę całkowitą z przedziału od 1 do 5, ale nie zawsze tak jest, ponieważ istnieje bardzo mała możliwość, że funkcja losowa zwróci 0, a zaokrąglenie 0 w górę nadal da 0. Jest to bardzo odległa możliwość, ale należy ją wziąć pod uwagę przy korzystaniu z tej funkcji.

Składnia:

ceil(val);

ArgumentTyp Opis
x The number to change.

Zwroty:

Przykład:

val = ceil( 3.4 );

Spowoduje to ustawienie wartości val na 4.

Powrót: Funkcje numeryczne
