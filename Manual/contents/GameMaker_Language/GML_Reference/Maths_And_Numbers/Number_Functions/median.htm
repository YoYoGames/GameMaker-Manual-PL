mediana

Ta funkcja zwraca medianę wartości wejściowych, czyli wartość środkową. Gdy liczba argumentów jest parzysta, zwracana jest mniejsza z dwóch wartości środkowych, a funkcja może mieć tyle argumentów, ile jest wymagane (zauważ, że większa liczba argumentów (należy pamiętać, że większa liczba argumentów oznacza, że funkcja będzie wolniej parsowana), z których wszystkie muszą być wartościami rzeczywistymi. Oznacza to, że na przykład funkcja median(43, 12, 25, 19, 6) zwróci wartość 19, ponieważ jest to wartość środkowa pomiędzy wszystkimi pozostałymi.

Składnia:

median(val1, val2, ... max_val);

ArgumentTyp Opis
val0 ... max_val The values to compare.

Zwroty:

Przykład:

x = median( 0, x, room_width );
y = median( 0, y, room_height );

Dzięki temu gracz nie będzie mógł opuścić żadnej strony room, używając mediany jako klamry. Jeśli na przykład gracz przesunie się na lewo od granicy room, jego x będzie mniejsze od 0. Oznacza to, że środkowa liczba pierwszej z median będzie równa 0, więc gracz zostanie przeniesiony do punktu (0,y).
