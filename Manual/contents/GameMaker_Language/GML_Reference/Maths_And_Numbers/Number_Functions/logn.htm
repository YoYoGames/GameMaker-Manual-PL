logn

Ta funkcja jest podobna do funkcji log2(n) i log10(n), z tą różnicą, że określamy wartość podstawy logu. Na przykład funkcja logn(5,25) zwróci nam liczbę 5, które musimy pomnożyć aby otrzymać 25 (czyli 2).

Składnia:

logn(n, val);

ArgumentTyp Opis
n The log base.
val The input value.

Zwroty:

Przykład:

logval = logn(5, num);

Powyższy kod pobiera adres log5 z wartości zapisanej w polu "num".

Powrót: Funkcje numeryczne
