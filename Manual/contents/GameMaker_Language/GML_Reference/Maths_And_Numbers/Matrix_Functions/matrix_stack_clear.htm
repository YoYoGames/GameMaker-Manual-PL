matrix_stack_clear

Ta funkcja powoduje wyczyszczenie stosu macierzy.

Składnia:

matrix_stack_clear();

Zwroty:

Przykład:

if !matrix_stack_is_empty()
{
    matrix_stack_clear();
}

Powyższy kod sprawdza, czy stos macierzy jest pusty, a jeśli nie jest, to go czyści.
