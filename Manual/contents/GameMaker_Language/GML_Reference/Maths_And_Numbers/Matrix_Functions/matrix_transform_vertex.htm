matrix_transform_vertex

Ta funkcja przekształca wektor przez macierz. Podajesz identyfikator macierzy przekształcenia (zwrócony przez funkcję matrix build()) oraz wartości x, y i z dla wektora, który ma zostać przekształcony. Funkcja zwróci tablicę składającą się z 3 elementów, gdzie:

  • tablica[0] = x
  • tablica[1] = y
  • array[2] = z.

Składnia:

matrix_transform_vertex(matrix, x, y, z);

ArgumentTyp Opis
matrix The matrix to use
x The x component of the transform vector
y The y component of the transform vector
z The z component of the transform vector

Zwroty:

(3 elements)

Przykład:

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

Powyższy kod przekształca podane wartości za pomocą macierzy przechowywanej w zmiennej "t_matrix" i zapisuje je w tablicy "verts".

Powrót: Funkcje matrycy
Następny: matrix_stack_is_empty
