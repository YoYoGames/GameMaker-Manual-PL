time_bpm_to_seconds

Ta funkcja przyjmuje wartość uderzeń na minutę i zwraca długość każdego uderzenia w sekundach.

Można tego użyć podczas tworzenia źródła czasu, aby użyć wartości BPM dla źródła czasu. Ważne jest, aby w źródle czasu jako jednostkę stosowano sekundy.

Składnia:

time_bpm_to_seconds(bpm);

Argument Typ Opis
bpm Real The beats-per-minute value to convert into seconds-per-beat

Zwroty:

Real

Przykład:

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

Ten kod zamienia wartość 90 BPM na sekundy i wykorzystuje ją do utworzenia źródła czasu, które będzie działać bez końca.

Przy każdym pobudzeniu wypisuje komunikat "BEAT!" do dziennika wyjściowego.
