time_source_get_period

Ta funkcja zwraca wartość okresu ustawioną dla podanego źródła czasu.

Ta wartość okresu będzie wyrażona w jednostkach ustawionych dla Źródła czasu (sekundy lub ramki).

Składnia:

time_source_get_period(id);

Argument Typ Opis
id Identyfikator źródła czasu The Time Source aby uzyskać wartość okresu dla

Zwroty:

Real
