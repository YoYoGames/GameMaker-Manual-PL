time_source_create

Ta funkcja tworzy nowe Źródło czasu. Więcej informacji na ten temat można znaleźć w części Źródła czasu.

Utworzenie nowego źródła czasu nie powoduje jego automatycznego uruchomienia; należy zadzwonić pod numer time_source_start() aby je uaktywnić.

Nowe źródło czasu jest uruchamiane tylko raz, chyba że podano argument powtórzenia (przeczytaj poniżej).

Musisz zniszczyć Źródło Czasu używając time_source_destroy() kiedy już go nie potrzebujesz.

Argumenty

Poniżej znajduje się zestawienie argumentów funkcji:

Źródło czasu rodzica

Jest to nadrzędne Źródło Czas u, które kontroluje nowe Źródło Czasu. Może to być time_source_global, time_source_game lub niestandardowe źródło czasu, które już istnieje.

Zobacz: Wbudowane źródła czasu

Okres

Jest to okres czasu, który może być wyrażony w sekundach lub klatkach, w zależności od jednostki określonej w kolejnym argumencie.

W przypadku używania ramek jako jednostki okres musi być liczbą całkowitą. Wartości niebędące liczbami całkowitymi są zaokrąglane w dół (floored), z wyjątkiem wartości mniejszych od 1, które są zaokrąglane w górę do 1.

Jednostka

Jest to jednostka, w której wyrażony jest okres; może to być czas_źródła_jednostek_sekund lub czas_źródła_jednostek_ramek.

Zobacz: Jednostki źródła czasu

Wartości uderzeń na minutę można użyć, korzystając z adresu time_bpm_to_seconds().

Wywołanie zwrotne i argumenty

Należy określić "wywołanie zwrotne" method które jest wywoływane po wygaśnięciu źródła czasu.

Opcjonalnie można określić tablicę zawierającą argumenty, które powinny zostać przekazane do metody podczas jej wywoływania.

Do metody nie jest przekazywana sama tablica, lecz każdy element tablicy jako osobny argument.

Na przykład, jeśli Twoja funkcja oczekuje argumentów (x, y, object), to tablica args może wyglądać tak: [30, 600, obj_player].

Powtórzenia

Opcjonalnie można określić, ile razy Źródło czasu powinno powtarzać. Wartość 1 oznacza, że będzie ono uruchamiane tylko raz, co jest wartością domyślną.

Można określić całkowitą liczbę powtórzeń lub -1, aby powtarzać je w nieskończoność.

Na przykład, jeśli ustawisz tę wartość na 3, a okres źródła czasu to "4 sekundy", wówczas źródło czasu będzie aktywne przez 12 sekund i będzie wywoływać metodę wywołania zwrotnego co 4 sekundy.

Typ wygaśnięcia

Może to być time_source_expire_nearest lub time_source_expire_after.

Zobacz: Typy wygaśnięcia źródła czasu

Składnia:

time_source_create(parent, period, units, callback, [args, repetitions, expiry_type]);

Argument Typ Opis
parent Identyfikator źródła czasu The parent Time Source który steruje nowym źródłem czasu
period Real The period that the Time Source dla, w podanych jednostkach
units Źródło czasu Jednostka Stała The units that the given period is in
callback Metoda lub . Funkcja skryptu The method or script function to call when the Time Source wygasa
args Tablica OPTIONAL Tablica zawierająca argumenty, które mają być przekazane do metody
repetitions Real OPTIONAL Liczba powtórzeń źródła czasu, lub -1 dla nieokreślonego powtarzania.
expiry_type Źródło czasu Stała wygaśnięcia OPTIONAL Czy źródło czasu wygasa w najbliższej ramce, czy w następnej ramce

Zwroty:

Time Source ID

Przykład 1:

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

W tym przykładzie chcemy, aby instancja zniszczyła się 300 klatek później.

Kod najpierw tworzy metodę, która po prostu wywołuje instance_destroy().

Następnie tworzy źródło czasu, dziedziczące po źródle czasu gry. Ustawia jego okres na 300 klatek.

Na koniec uruchamia Źródło czasu.

Przykład 2:

var _my_method = function()
{
    show_debug_message("A second has passed!");
}

global.time_per_second = time_source_create(time_source_game, 1, time_source_units_seconds, _my_method, [], -1, time_source_expire_after);

time_source_start(global.time_per_second);

W tym przykładzie tworzymy globalne źródło czasu, które wygasa raz na sekundę. Ten kod zostałby umieszczony w korzeniu pliku script.

Kod najpierw tworzy metodę, która wypisuje komunikat do dziennika wyjściowego o treści "Minęła sekunda!".

Następnie tworzy nowe źródło czasu, dziedziczące po źródle czasu gry. Ustawia jego okres na 1 sekundę.

Metoda jest przekazywana do źródła czasu, aby można ją było wywołać za każdym razem, gdy upływa jego czas. Jako argumenty podawana jest pusta tablica.

Licznik powtórzeń jest ustawiony na -1, więc źródło czasu nigdy się nie zatrzymuje i powtarza w nieskończoność. Typ wygaśnięcia jest ustawiony tak, aby wywołanie zwrotne było uruchamiane w pierwszej klatce po wygaśnięciu.

Na koniec uruchamiane jest źródło czasu.
