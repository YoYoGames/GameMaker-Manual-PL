time_source_get_parent

Ta funkcja zwraca identyfikator Identyfikator źródła czasu rodzica podanego źródła czasu.

Zwracana wartość może być identyfikatorem niestandardowego źródła czasu lub jednego z wbudowanych źródeł czasu, w zależności od tego, w jaki sposób zostało utworzone dane źródło czasu.

Składnia:

time_source_get_parent(id);

Argument Typ Opis
id Identyfikator źródła czasu The Time Source aby uzyskać dane rodzica

Zwroty:

Time Source ID

Przykład:

var _parent = time_source_get_parent(time_source);

Ten kod pobiera informacje o rodzicu źródła czasu i zapisuje je w zmiennej lokalnej.
