call_cancel

Funkcja ta anuluje funkcję callback, która została uruchomiona wcześniej przez call_later.

Składnia:

call_cancel(handle);

Argument Typ Opis
handle Identyfikator źródła czasu The handle to a Time Source zwrócone przez call_later.

Zwroty:

N/A

Przykład:

call_cancel(handle);

Powrót: call_later
Następny: time_source_create
