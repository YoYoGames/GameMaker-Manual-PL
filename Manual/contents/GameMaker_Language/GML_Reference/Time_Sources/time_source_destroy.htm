time_source_destroy

Ta funkcja spowoduje zniszczenie podanego źródła czasu.

Opcjonalnie można określić, czy wszystkie młodsze Źródła czasu również powinny zostać usunięte.

Składnia:

time_source_destroy(id, [destroy_tree]);

Argument Typ Opis
id Identyfikator źródła czasu The Time Source niszczyć
destroy_tree Boolean OPTIONAL Czy niszczyć również podrzędne Źródła Czasu, domyślnie jest to wartość false

Zwroty:

N/A

Przykład:

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

Kod zdarzenia Utwórz tworzy nowe źródło czasu, które wygasa raz na sekundę i powtarza się w nieskończoność.

Kod zdarzenia Clean Up niszczy to źródło czasu, ponieważ nie będzie ono potrzebne po zniszczeniu instancji.
