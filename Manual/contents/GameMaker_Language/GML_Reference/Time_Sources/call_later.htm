zadzwonić później

Funkcja ta wyzwala funkcję callback po określonym czasie lub po określonym interwale. Robi to poprzez utworzenie źródła czasu (patrz Źródła czasu).

Aby anulować funkcję callback, zobacz call_cancel.

WAŻNE Chociaż funkcja ta zwraca rzeczywisty identyfikator źródła czasu, identyfikator ten nie może być użyty z funkcjami źródeł czasu, ponieważ źródła czasu używane przez call_later i call_cancel są "niewidzialne" - nie pojawią się jako dzieci żadnych wbudowanych źródeł czasu i nie mogą być używane z innymi funkcjami źródeł czasu. Są one jednak uwzględniane w liczeniu źródeł czasu zwracanych przez debug_event("ResourceCounts").

Argumenty

Poniżej przedstawiono podział argumentów tej funkcji:

Okres

Jest to okres czasu, który może być wyrażony w sekundach lub klatkach, w zależności od jednostki określonej w kolejnym argumencie.

W przypadku stosowania ramek jako jednostki, okres musi być liczbą całkowitą. Wartości niecałkowite są zaokrąglane w dół (floored), z wyjątkiem wartości mniejszych niż 1, które są zaokrąglane do 1.

Jednostka

Jest to jednostka, w której wyrażony jest okres i może to być time_source_units_seconds lub time_source_units_frames.

Zobacz: Jednostki źródła czasu

Możesz użyć wartości beats-per-minute, używając time_bpm_to_seconds().

Callback

Musisz określić "callback" method , który jest wywoływany po wygaśnięciu źródła czasu.

Pętla

Czy źródło czasu powinno loop.

Składnia:

call_later(period, unit, callback, [loop=false]);

Argument Typ Opis
period Real The period that the Time Source prowadzi dla, w danych jednostkach
units Źródło czasu Jednostka Stała The units that the given period is in
callback Metoda lub Funkcja skryptu The method or script function to call after the specified time
loop Boolean OPTIONAL Czy zapętlić źródło czasu (domyślnie false)

Zwroty:

Time Source ID

Przykład 1:

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

W tym przykładzie tworzymy funkcję callback, która pokazuje komunikat debugowania po upływie pięciu sekund. Po tym czasie bazowe źródło czasu zostaje zniszczone.

Przykład 2:

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

W tym przykładzie tworzymy funkcję callback, która tworzy nowego wroga w losowej pozycji na stronie room. Funkcja ta jest następnie wykonywana co 10 sekund poprzez przekazanie jej do call_later. Ostatni argument jest ustawiony na true więc funkcja będzie wywoływana co 10 sekund.

Użyj call_cancel z uchwytem zwróconym przez call_later, aby zatrzymać Źródło czasu i wywołania zwrotne.
