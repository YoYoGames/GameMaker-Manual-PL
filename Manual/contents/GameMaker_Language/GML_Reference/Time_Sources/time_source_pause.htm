czas_źródła_pauzy

Ta funkcja wstrzymuje działanie podanego źródła czasu. Źródło czasu musi być aktywne, gdy wywoływana jest ta funkcja.

Aby go wznowić, należy zadzwonić pod numer time_source_resume().

Składnia:

time_source_pause(id);

Argument Typ Opis
id Identyfikator źródła czasu The Time Source do przerwy

Zwroty:

N/A

Przykład:

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_resume(time_source);
    }
}

Gdy zostanie naciśnięty klawisz spacji, ten kod pobierze stan źródła czasu.

Gdy stan jest aktywny, wstrzymuje działanie Źródła czasu, a gdy jest wstrzymany, wznawia je.
