czas_sekund_do_bpm

Funkcja pobiera długość taktu w sekundach i zwraca wartość uderzeń na minutę.

Składnia:

time_seconds_to_bpm(seconds);

Argument Typ Opis
seconds Real The seconds-per-beat value to convert into beats-per-minute

Zwroty:

Real

Przykład:

var _period = time_source_get_period(time_source);
var _bpm = time_seconds_to_bpm(_period);

show_debug_message(_bpm);

Funkcja ta pobiera okres ze źródła czasu, konwertuje go na BPM i drukuje tę wartość do dziennika wyjściowego.

Ten kod jest zgodny z przykładem dla time_bpm_to_seconds().
