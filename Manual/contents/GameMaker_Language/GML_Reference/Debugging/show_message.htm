pokaż_komunikat

Ta funkcja tworzy wyskakujące okienko komunikatu, w którym wyświetlany jest podany adres string oraz przycisk "Ok" do jego zamknięcia.

UWAGA Ta funkcja jest przeznaczona do użytku w debugowaniu tylko na obiekcie docelowym Windows, ale jest przestarzała na wszystkich innych obiektach.

Składnia:

show_message(str);

Argument Typ Opis
str String The string to show in the pop-up message.

Zwroty:

N/A

Przykład:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

Powyższy kod loop przegląda wartości przechowywane w tablicy "inv" i dodaje je do zmiennej "tot", a następnie wyświetla komunikat z sumą.
