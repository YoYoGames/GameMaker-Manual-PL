debug_mode

Ta zmienna tylko do odczytu zwraca wartość true, gdy gra jest odtwarzana w trybie debugowania, i false, gdy jest odtwarzana normalnie.

Składnia:

debug_mode

Zwroty:

Przykład:

if debug_mode
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

Powyższy kod otwiera inny plik ini w zależności od wartości zmiennej tylko do odczytu debug_mode.
