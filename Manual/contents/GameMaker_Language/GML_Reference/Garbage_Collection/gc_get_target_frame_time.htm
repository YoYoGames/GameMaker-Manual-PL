gc_get_target_frame_time

Za pomocą tej funkcji można uzyskać aktualną wartość ramki docelowej dla odśmiecacza. Zwracana wartość jest podawana w mikrosekundach (gdzie 1 000 000 mikrosekund równa się jednej sekundzie), a domyślny czas ramki docelowej wynosi 100 mikrosekund. Jeśli chcesz zmienić tę wartość, powinieneś użyć funkcji gc_target_frame_time().

Składnia:

gc_get_target_frame_time(time);

Zwroty:

Przykład:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

Powyższy kod sprawdza bieżącą wartość docelową czasu ramki dla garbage collectora i jeśli nie wynosi ona 50 mikrosekund, jest ustawiana na tę wartość.
