gc_enable

Za pomocą tej funkcji można włączyć lub wyłączyć garbage collector. Wywołanie funkcji z argumentem true włącza go, a użycie false wyłącza (nie jest to zalecane). Domyślnie jest on włączony.

Składnia:

gc_enable(enable);

ArgumentTyp Opis
enable enable (true) lub wyłączyć (false) garbage collector.

Zwroty:

Przykład:

if (global.debug == true)
{
    gc_enable(false);
}

Powyższy kod wyłącza zbieranie śmieci, jeśli podana zmienna globalna ma wartość true.
