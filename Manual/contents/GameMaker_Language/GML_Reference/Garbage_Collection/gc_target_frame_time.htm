gc_target_frame_time

Za pomocą tej funkcji można wskazać odśmiecaczowi, że powinien on dążyć do tego, aby każda ramka była uruchamiana nie dłużej niż przez określony czas. Funkcja przyjmuje wartość czasu podaną w mikrosekundach (gdzie 1 000 000 mikrosekund równa się jednej sekundzie), a domyślny docelowy czas ramki wynosi 100 mikrosekund.

Należy pamiętać, że jest to tylko wartośćdocelowa, ponieważ niektóre operacje nadal muszą być wykonywane w całości w jednej ramce i czasami mogą trwać dłużej niż czas docelowy. Należy również pamiętać, żezwiększenie czasu docelowego spowoduje, że garbage collector będzie lepiej reagował na gwałtowne zmiany w wykorzystaniu pamięci i spowoduje szybsze zwalnianie pamięci, choć w praktyce w większości przypadków nie będzie to raczej wymagane. Należy również pamiętać, że ustawienie docelowego czasu ramki na 0 nie spowoduje całkowitego zatrzymania pracy kolektora - nadal będzie on wykonywał minimalną ilość pracy w każdej ramce. Aby całkowicie wyłączyć kolektor, należy użyć funkcji gc_enable() .

Składnia:

gc_target_frame_time(time);

ArgumentTyp Opis
time The target time - in microseconds -  that the garbage collector should work each frame

Zwroty:

N/A

Przykład:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

Powyższy kod sprawdza bieżącą wartość docelową czasu ramki dla garbage collectora i jeśli nie wynosi ona 50 mikrosekund, jest ustawiana na tę wartość.
