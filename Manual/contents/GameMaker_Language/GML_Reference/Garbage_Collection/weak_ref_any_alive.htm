weak_ref_any_alive

Za pomocą tej funkcji można sprawdzić słabe refer encje do różnych struktur, aby sprawdzić, czy są one nadal "żywe", czy nie. Podajesz tablicę słabych referencji do sprawdzenia (każda słaba referencja powinna zostać utworzona za pomocą funkcji weak_ref_create()), a funkcja zwróci true, jeśli any do któregoś z tych struktów nadal gdzieś się odwołujemy lub false jeśli nie i został on już wyrzucony na śmietnik. Zwróć uwagę, że jeśli podasz tablicę, w której którakolwiek z wartości nie jest słabą referencją, funkcja zwróci undefined.

Funkcja przyjmuje również dwa opcjonalne argumenty, z których pierwszy umożliwia podanie początkowego indeksu tablicy, od którego ma być sprawdzana, oraz argumentu określającego liczbę pozycji (długość) od tego indeksu do sprawdzenia. Jeśli zostaną one podane, sprawdzane będą tylko indeksy tablicy mieszczące się w tych parametrach.

Składnia:

weak_ref_any_alive(array, [index], [length]);

Argument Typ Opis
array Tablica słabych odwołań do struktur Array containing weak references to the structs that you want to check.
[index] Real OPTIONAL Indeks w tablicy, od którego ma się rozpocząć sprawdzanie
[length] Real OPTIONAL Liczba pozycji, począwszy od podanego indeksu, które należy sprawdzić pod kątem

Zwroty:

Boolean (or undefined)

Przykład:

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

Powyższy kod sprawdza tablicę słabych referencji i jeśli jakieś są jeszcze żywe, to instancja jest niszczona.
