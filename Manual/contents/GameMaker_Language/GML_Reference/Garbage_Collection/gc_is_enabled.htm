gc_is_enabled

Za pomocą tej funkcji można sprawdzić, czy garbage collector jest włączony, czy nie. Funkcja zwróci true, jeśli jest włączony, lub false w przeciwnym wypadku.

Składnia:

gc_is_enabled();

Zwroty:

Przykład:

if (!gc_is_enabled())
{
    gc_enable(true);
}

Powyższy kod sprawdza, czy garbage collector jest włączony, a jeśli nie jest, włącza go.
