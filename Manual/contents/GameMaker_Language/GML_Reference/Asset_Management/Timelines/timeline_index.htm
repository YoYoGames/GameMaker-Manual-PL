indeks osi czasu

Ta zmienna przechowuje indeks strony timeline aktualnie powiązanej z instancją. Możesz ją ustawić na konkretną linię czasową, aby jej użyć, lub ustawić na -1, aby przestać używać linii czasowej dla instancji (jeśli dla instancji nie zdefiniowano linii czasowej, zwracane jest również -1). Zauważ, że nie uruchamia to linii czasowej - do tego celu użyj zmiennej timeline_running.

Składnia:

timeline_index;

Zwroty:

Timeline Asset

Przykład:

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

Powyższy kod przypisze instancji wykonującej kod linię czasu indeksowaną w zmiennej "global.tl", jeśli taka linia czasu istnieje.
