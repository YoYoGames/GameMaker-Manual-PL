oś czasu_chwila_dodaj_skrypt

Za pomocą tej funkcji można dynamicznie dodać funkcję skryptu do strony Timelines w dowolnym "momencie" na tej linii czasu, gdzie "moment" odpowiada jednemu tick'owi (lub krokowi) gry. W ten sposób można utworzyć nową linię czasową przy użyciu funkcji timeline_add() i dodawać różne zachowania w dowolnym momencie, albo po prostu zmodyfikować wcześniej utworzony zasób linii czasu, dodając nowe zachowania. Zwróć uwagę, że funkcja nie może wymagać żadnych dodatkowych argumentów podczas korzystania z niej, a jeśli użyjesz jej do modyfikacji strony Timeline asset znajdującej się w przeglądarce Asset, to będzie to dotyczyło wszystkich instancji korzystających z tej strony timeline, a zmiana będzie trwała do końca gry (wywołanie game_restart() również nie zresetuje zmiany).

Składnia:

timeline_moment_add_script(ind, step, script);

Argument Typ Opis
ind Oś czasu Aktywa The index of the timeline to add a moment to.
step Real The moment (step) to add to.
script Funkcja skryptu The index of the script function to add into the moment.

Zwroty:

N/A

Przykład:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

Powyższy kod utworzy nową linię czasu i zapisze jej indeks w zmiennej "global.tl". Następnie doda do linii czasu trzy strony scripts, wybrane losowo, w odstępach jednominutowych.

Powrót: Timelines
Następny: timeline_moment_clear
