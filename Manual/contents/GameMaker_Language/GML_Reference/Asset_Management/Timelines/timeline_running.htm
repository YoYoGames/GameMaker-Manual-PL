linia czasu_bieżąca

Ta zmienna przechowuje bieżący stan przypisanej linii czasu i zwraca true, jeśli linia działa, oraz false, jeśli nie. Można również ustawić tę zmienną na true lub false, aby uruchamiać i zatrzymywać linię czasową w dowolnym momencie. Należy zauważyć, że zatrzymana linia czasowa nie jest resetowana, a zatem ponowne uruchomienie jej w późniejszym czasie spowoduje rozpoczęcie jej od momentu, w którym została zatrzymana.

Składnia:

timeline_running;

Zwroty:

Przykład:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

Powyższy kod sprawdzi, czy dana instancja ma uruchomioną linię czasową, a jeśli nie, to zresetuje przypisaną linię czasową, tak aby rozpoczęła się w pierwszym momencie, a następnie ją uruchomi.
