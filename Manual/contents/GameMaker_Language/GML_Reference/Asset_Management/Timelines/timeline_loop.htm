pętla czasowa

Ta zmienna zwraca, czy linia czasowa jest zapętlona (true) czy nie (false). Możesz zmienić tę zmienną na switch, aby włączyć lub wyłączyć zapętlanie i działa ona z ujemną prędkością linii czasowej (jeśli pozycja linii czasowej spadnie poniżej 0, rozpocznie się ona ponownie w ostatnim zdefiniowanym momencie).

Składnia:

timeline_loop;

Zwroty:

Przykład:

if !timeline_loop
{
    timeline_loop = true;
}

Powyższy kod sprawdza, czy linia czasowa aktualnie przypisana do instancji będzie loop czy nie, a jeśli nie, to ustawia wartość true.
