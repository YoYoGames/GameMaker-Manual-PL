linia_czasowa_istnieje

Za pomocą tej funkcji można sprawdzić, czy linia czasu istnieje (zwraca true), czy nie (zwraca false). Jest to szczególnie przydatne podczas dynamicznego tworzenia strony Timelines za pomocą funkcji timeline_add() Należy jednak pamiętać, że w przypadku szukania niezainicjowanej zmiennej (która w przeciwnym razie zostałaby przypisana do indeksu linii czasu) zostanie wyświetlony błąd.

Składnia:

timeline_exists(ind);

ArgumentTyp Opis
ind The index of the time line to check for.

Zwroty:

Przykład:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

Powyższy kod sprawdza, czy linia czasu istnieje i jest indeksowana w zmiennej globalnej "tl", a jeśli tak, to usuwa tę linię czasu.
