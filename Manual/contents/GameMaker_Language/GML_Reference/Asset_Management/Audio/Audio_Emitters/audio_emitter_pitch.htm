audio_emitter_pitch

Funkcja służy do zmiany wysokości wszystkich dźwięków emitowanych przez dany emiter. Jest to mnożnik wysokości dźwięku, w tym sensie, że wartość wejściowa mnoży aktualną wysokość dźwięku przez tę wartość, więc domyślna wartość 1 oznacza brak zmiany wysokości dźwięku, podczas gdy wartość mniejsza niż 1 obniży wysokość dźwięku, a większa niż 1 podniesie wysokość dźwięku. Najlepiej jest używać małych przyrostów dla tej funkcji, ponieważ każda wartość poniżej 0 lub powyżej 5 może nie być słyszalna.

Dźwięki już odtwarzane przez emiter audio nie muszą być ponownie uruchamiane, aby zmiana wysokości dźwięku była słyszalna. Zmiana wysokości dźwięku jest stosowana natychmiast.

Składnia:

audio_emitter_pitch(emitter, pitch);

Argument Typ Opis
emitter Identyfikator nadajnika audio The index of the emitter to change.
pitch Real The pitch multiplier (default 1).

Zwroty:

N/A

Przykład:

switch (gear)
{
    case 1: audio_emitter_pitch(s_emit, 0.8); break;
    case 2: audio_emitter_pitch(s_emit, 0.9); break;
    case 3: audio_emitter_pitch(s_emit, 0.95); break;
    case 4: audio_emitter_pitch(s_emit, 1); break;
    case 5: audio_emitter_pitch(s_emit, 1.2); break;
}

Powyższy kod zmieni wysokość dźwięku odtwarzanego z emitera indeksowanego w zmiennej "s_emit" na podstawie wartości zmiennej "gear".
