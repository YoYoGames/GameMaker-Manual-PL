audio_pause_sync_group

Funkcja ta wstrzymuje podaną grupę synchronizacji, jeśli jest ona odtwarzana, przy czym indeks grupy jest wartością zwróconą podczas tworzenia grupy za pomocą funkcji audio_create_sync_group(). Nie powoduje to zatrzymania dźwięku, a wywołanie funkcji audio_resume_sync_group()rozpocznie odtwarzanie od tego samego miejsca, w którym zostało wstrzymane.

UWAGA: Ta funkcja nie jest dostępna dla platformy docelowej HTML5.

Składnia:

audio_pause_sync_group(group_index);

Argument Typ Opis
group_index ID grupy synchronizacji audio The group index to pause.

Zwroty:

N/A

Przykład:

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if global.Pause
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

Powyższy kod sprawdza, czy nie został naciśnięty klawisz "P", a jeśli zostanie wykryty, przełącza zmienną "global.Pause", a następnie sprawdza ją, aby wstrzymać lub wznowić działanie grupy synchronizacji indeksowanej w zmiennej "sg".
