audio_create_sync_group

Tworzy grupę synchronizacji i zwraca dla niej unikalną wartość ID, która powinna być następnie używana we wszystkich dalszych wywołaniach funkcji audio dla tej grupy. Jeśli grupa ma mieć adres loop, należy podać adres true, w przeciwnym razie należy podać adres false, ale należy pamiętać, że jeśli grupa ma mieć adres loop, wszystkie ścieżki dodane później muszą mieć tę samą długość. Należy pamiętać, że po utworzeniu grupy synchronizacji należy zwolnić pamięć i dźwięki z nią związane, gdy nie są używane, za pomocą funkcji audio_destroy_sync_group() na przykład w przypadku zdarzeń Room End lub Destroy.

UWAGA: Ta funkcja nie jest dostępna dla platformy docelowej HTML5.

Składnia:

audio_create_sync_group(loop);

Argument Typ Opis
loop Boolean Whether the tracks in the group should loop (true) lub nie (false).

Zwroty:

Audio Sync Group ID

Przykład:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Powyższe polecenie tworzy nową grupę synchronizacyjną i przypisuje jej indeks do zmiennej "sg". Następnie do grupy dodawane są cztery dźwięki, przy czym wzmocnienie dla trzech z nich jest ustawione na 0. Na koniec grupa synchronizacyjna jest odtwarzana.
