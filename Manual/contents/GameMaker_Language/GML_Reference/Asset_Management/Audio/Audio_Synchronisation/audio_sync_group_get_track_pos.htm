audio_sync_group_get_track_pos

Ta funkcja zwraca bieżącą pozycję odtwarzania podanej grupy synchronizacji. Indeks grupy jest wartością zwracaną podczas tworzenia grupy za pomocą funkcji audio_create_sync_group()a wartość zwracana to czas w sekundach, przez jaki utwory były odtwarzane.

UWAGA: Ta funkcja nie jest dostępna dla platformy docelowej HTML5.

Składnia:

audio_sync_group_get_track_pos(group_index);

Argument Typ Opis
group_index ID grupy synchronizacji audio The group index to get the position of.

Zwroty:

Real

Przykład:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

Powyższy kod pobiera pozycję czasową dla grupy synchronizacji zindeksowaną w zmiennej "sg", a następnie używa tej wartości do narysowania na ekranie odtwarzanego czasu.
