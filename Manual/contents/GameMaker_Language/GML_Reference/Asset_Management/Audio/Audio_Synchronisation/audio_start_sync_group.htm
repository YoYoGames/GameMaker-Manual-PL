audio_start_sync_group

Za pomocą tej funkcji można rozpocząć odtwarzanie wcześniej utworzonej grupy synchronizacji. Należy podać indeks grupy, który jest wartością zwróconą podczas tworzenia grupy za pomocą funkcji audio_create_sync_group().

UWAGA: Ta funkcja nie jest dostępna dla platformy docelowej HTML5.

Składnia:

audio_start_sync_group(group_index);

Argument Typ Opis
group_index ID grupy synchronizacji audio The group index to play.

Zwroty:

N/A

Przykład:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Powyższe polecenie tworzy nową grupę synchronizacyjną i przypisuje indeks tej grupy do zmiennej "sg". Następnie do grupy dodawane są cztery dźwięki, przy czym wzmocnienie dla trzech z nich jest ustawione na 0. Na koniec grupa synchronizacyjna jest odtwarzana.

Powrót: Synchronizacja dźwięku
