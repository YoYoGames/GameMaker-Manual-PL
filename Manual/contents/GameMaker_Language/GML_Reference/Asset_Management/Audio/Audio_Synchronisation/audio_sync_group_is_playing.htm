audio_sync_group_is_playing

Funkcja ta może być użyta do sprawdzenia czy jakikolwiek dźwięk w zsynchronizowanej grupie jest odtwarzany. Wymagane jest podanie identyfikatora grupy synchronizacyjnej zwróconego przez funkcję audio_create_sync_group().

Ta funkcjonalność nie jest dostępna dla platformy docelowej HTML5.

Składnia:

audio_sync_group_is_playing(group_index);

Argument Typ Opis
group_index ID grupy Audio Sync The group index to check.

Zwroty:

Boolean

Przykład:

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

Powyższy kod sprawdza, czy grupa synchronizacyjna group_one jest aktualnie odtwarzana, a jeśli tak, to grupa jest zatrzymywana.

Powrót: Synchronizacja dźwięku
Następny: audio_create_sync_group
