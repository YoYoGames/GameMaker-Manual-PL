audio_sound_is_playable

Funkcja ta może być użyta do sprawdzenia, czy podany indeks dźwięku może być aktualnie odtwarzany. Jest to potrzebne ze względu na różne sposoby odtwarzania dźwięku strumieniowego i niestrumieniowego na platformie docelowej HTML5, i zwróci true jeśli dźwięk może być odtwarzany i false jeśli nie może. Zauważ, że na wszystkich innych platformach niż HTML5, funkcja zawsze zwróci true.

Składnia:

audio_sound_is_playable(index);

Argument Typ Opis
index Sound Asset The index of the sound to check.

Zwroty:

Boolean

Przykład:

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

Powyższy kod odtworzy dźwięk "snd_Shot", jeśli jest on odtwarzalny.
