dźwięk_jest_pauzowany

Ta funkcja sprawdza, czy podany dźwięk jest aktualnie wstrzymany. Dźwięk może być pojedynczą instancją dźwięku (indeks dla poszczególnych odtwarzanych dźwięków można przechowywać w zmiennej przy użyciu funkcji audio_play_sound() lub audio_play_sound_at() ) lub dźwiękiem asset, w którym to przypadku zostaną sprawdzone wszystkie instancje danego dźwięku i jeśli którykolwiek z nich jest wstrzymany, funkcja zwróci true, w przeciwnym razie zwróci false.

Składnia:

audio_is_paused(index);

Argument Typ Opis
index Sound Asset lub Sound Instance ID The index of the sound to check.

Zwroty:

Boolean

Przykład:

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

Powyższy kod sprawdza, czy dźwięk zaindeksowany w zmiennej "snd_Waterfall" jest aktualnie wstrzymany, i jeśli zwróci true, odtwarzanie dźwięku zostanie wznowione.
