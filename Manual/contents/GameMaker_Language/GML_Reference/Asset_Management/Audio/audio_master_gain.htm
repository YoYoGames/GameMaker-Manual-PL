audio_master_gain

Za pomocą tej funkcji można ustawić wartość bezwzględną globalnej głośności wszystkich dźwięków i muzyki. Jest ona oparta na skali liniowej od 0 (cisza) do wartości większej od 0, choć normalnie za maksymalną głośność uważa się 1. Można użyć wartości powyżej 1, ale w zależności od użytego dźwięku i platformy, na którą jest kompilowany, może dojść do zniekształceń lub przycięć podczas odtwarzania dźwięku. Ta funkcja wpływa na względną głośność wszystkich dźwięków i muzyki odtwarzanych w grze.

Składnia:

audio_master_gain(gain);

Argument Typ Opis
gain Real Value for the global volume (0 to 1).

Zwroty:

N/A

Przykład:

if keyboard_check(vk_up)
{
    if vol < 1 vol += 0.05;
    audio_master_gain(vol);
}
if keyboard_check(vk_down)
{
    if vol > 0 vol -= 0.05;
    audio_master_gain(vol);
}

Powyższy kod sprawdza, czy naciśnięcie klawiszy strzałek w górę i w dół powoduje zwiększenie lub zmniejszenie zmiennej "vol". Zmienna ta jest następnie używana do ustawiania globalnego wzmocnienia wszystkich dźwięków i muzyki w grze.
