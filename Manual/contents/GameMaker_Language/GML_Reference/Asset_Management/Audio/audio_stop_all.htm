audio_stop_all

Ta funkcja powoduje zatrzymanie wszystkich aktualnie odtwarzanych dźwięków.

Składnia:

audio_stop_all();

Zwroty:

Przykład:

if !global.SFX
{
    audio_stop_all();
}

Powyższy kod sprawdza zmienną globalną "SFX" i jeśli zwróci ona wartość false, zatrzyma wszystkie aktualnie odtwarzane dźwięki.

Powrót: Audio
Następny: audio_is_playing
