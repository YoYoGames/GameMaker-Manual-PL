nazwa_grupy_dźwiękowej

Ta funkcja zwraca adres string zawierający nazwę podanej grupy audio, którą można wyświetlić lub sprawdzić. Kiedy definiujesz grupę audio w Opcjach gry, nadajesz jej unikalną "nazwę", która jest tak naprawdę stałą używaną jako wartośćID dla grupy. Funkcja ta pobiera tylko identyfikator i zwraca string o podanej przez użytkownika nazwie identyfikatora.

Składnia:

audio_group_name(groupID);

Argument Typ Opis
groupID Identyfikator grupy audio The index value constant of the audio group to check (as defined in the Audio Groups Window)

Zwroty:

String

Przykład:

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

Powyższy kod pobiera nazwę podanej stałej grupy audio i wyświetla ją na ekranie.

Powrót: Grupy audio
Następny: audio_group_stop_all
