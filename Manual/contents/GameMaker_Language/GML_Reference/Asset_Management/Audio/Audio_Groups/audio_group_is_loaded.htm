audio_group_is_loaded

Ta funkcja sprawdza, czy określona grupa audio została załadowana do pamięci i jest gotowa do użycia.

Składnia:

audio_group_is_loaded(groupID);

Argument Typ Opis
groupID Identyfikator grupy audio The index of the audio group to check (as defined in the Audio Groups Window)

Zwroty:

Boolean

Przykład:

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

Powyższy kod sprawdza, czy grupa audio została już załadowana, a jeśli tak, to ją rozładowuje.
