audio_group_set_gain

Funkcja ta umożliwia wygaszanie grupy dźwięków przez określony czas lub natychmiastowe ustawianie wzmocnienia grupy.

Czas jest mierzony w milisekundach, a funkcja wymaga wprowadzenia końcowego poziomu wzmocnienia, który grupa ma osiągnąć przed upływem tego czasu. Wzmocnienie to może mieć wartość od 0 (cisza) do 1 (pełna głośność), a skala jest liniowa, czyli wartość 0,5 oznacza połowę głośności. Aby natychmiast zmienić wzmocnienie, wystarczy ustawić argument time na 0. Należy pamiętać, że na niektórych platformach można uzyskać wzmocnienie większe niż 1, chociaż wartość 1 jest uważana za "pełną głośność", a większa może spowodować przycięcie dźwięku.

Składnia:

audio_group_set_gain(groupID, volume, time);

Argument Typ Opis
groupID Identyfikator grupy audio The index of the audio group to stop (as defined in the Audio Groups Window)
volume Real The final value for the group volume.
time Real The length of the change in gain in milliseconds.

Zwroty:

N/A

Przykład:

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

Powyższy kod sprawdza, czy nie został naciśnięty klawisz "spacja", a następnie w ciągu 5 sekund zmniejsza do 0 wszystkie dźwięki dla "audiogrupy1".
