audio_sound_set_track_position

Ta funkcja umożliwia ustawienie pozycji (w sekundach) dla podanego identyfikatora dźwięku lub adresu asset.

Podany "indeks" może być albo pojedynczą instancją dźwięku (zwróconą przez audio_play_sound()) lub dźwiękiem asset (np. dodanym przez przeglądarkę Asset ). Zachowanie tej funkcji będzie się różnić w zależności od rodzaju podanego indeksu:

  • Jeśli jest to używany unikatowy identyfikator dźwięku, który jest już odtwarzany, jego pozycja zostanie natychmiast zmieniona na podany czas.
  • Jeśli jest to dźwięk asset, to wszystkie kolejne odtworzenia danego dźwięku asset rozpoczną się o nowej godzinie.

Ta funkcja zmieni pozycję ścieżki tylko dla aktualnie odtwarzanego dźwięku lub jego następnego odtworzenia (w przypadku dostarczenia dźwięku asset ). Jeśli dźwięk jest odtwarzany z włączoną funkcją zapętlenia, kolejne odtworzenia będą zawsze rozpoczynać się od początku (0,0 sekundy), a nie od pozycji na ścieżce określonej za pomocą tej funkcji. Na przykład rozpoczęcie odtwarzania dźwięku loop od 5,7 sekundy spowoduje odtworzenie pierwszego dźwięku od tego momentu, jednak później będzie on powtarzał całą ścieżkę, od początku do końca.

Składnia:

audio_sound_set_track_position(index, time);

Argument Typ Opis
index Sound Asset lub Sound Instance ID The index of the sound to change.
time Real The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

Zwroty:

N/A

Przykład:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

Powyższy kod odtwarza dźwięk, a następnie wykorzystuje zwróconą wartość ID do ustawienia pozycji początkowej dźwięku na 32 sekundy.
