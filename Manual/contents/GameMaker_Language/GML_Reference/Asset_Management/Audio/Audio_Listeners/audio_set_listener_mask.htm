audio_set_listener_mask

W przypadku korzystania z wielu słuchaczy w systemie można ustawić maskę bitową dla dźwięku, aby był on słyszalny tylko dla słuchaczy oznaczonych przez tę maskę. Można jednak również ustawić maskę globalną za pomocą tej funkcji, a wszystkie dźwięki odtwarzane normalnie będą słyszane przez słuchaczy oznaczonych tą maską, bez konieczności ustawiania maski dla każdego dźwięku z osobna.

Składnia:

audio_set_listener_mask(mask);

Argument Typ Opis
mask Real The bit-mask data to set for the listeners

Zwroty:

N/A

Przykład:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Powyższy kod sprawdza liczbę dostępnych listenerów, a następnie loops pobiera z nich bity masek, które są następnie łączone w jedną maskę bitową, która jest stosowana do globalnego listenera.
