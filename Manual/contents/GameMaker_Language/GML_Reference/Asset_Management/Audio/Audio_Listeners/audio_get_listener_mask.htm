audio_get_listener_mask

Ta funkcja zwraca dane maski bitowej, która określa aktualną domyślną (globalną) maskę dla słuchaczy audio.

Składnia:

audio_get_listener_mask();

Zwroty:

Przykład:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

Powyższy kod pobiera dane aktualnej maski słuchacza i porównuje je z danymi przechowywanymi w zmiennej globalnej. Jeśli dane te nie są takie same, kod sprawdza liczbę dostępnych słuchaczy, a następnie przechodzi przez stronę loops i pobiera bity ich masek, które są następnie łączone w jedną maskę, która jest stosowana w systemie audio w celu zdefiniowania globalnych słuchaczy.
