audio_listener_get_data

Ta funkcja tworzy mapę DS i wypełnia ją wartościami pozycji, prędkości i orientacji dla podanego słuchacza. Domyślnym indeksem słuchacza jest 0, ale można użyć funkcji audio_get_listener_info() aby uzyskać różne indeksy dostępne dla platformy docelowej. Jeśli podasz nieprawidłowy indeks słuchacza, funkcja zwróci -1.

UWAGA Użytkownik jest odpowiedzialny za zniszczenie zwróconej mapy DS za pomocą odpowiedniej funkcji.

Mapa DS będzie zawierać następujące klucze:

  • "x" - pozycja x słuchacza
  • "y" - pozycja y słuchacza
  • "z" - pozycja z słuchacza
  • "vx" - prędkość słuchacza w osi x
  • "vy" - prędkość słuchacza w osi y
  • "vz" - prędkość słuchacza w osi z
  • "lookat_x" - składowa x wektora patrzenia słuchacza
  • "lookat_y" - składowa y wektora spojrzenia słuchacza
  • "lookat_z" - składowa z wektora spojrzenia słuchacza
  • "up_x" - składowa x wektora uniesienia słuchacza
  • "up_y" - składowa y wektora uniesienia słuchacza
  • "up_z" - składowa z wektora uniesienia słuchacza

Składnia:

audio_listener_get_data(index);

Argument Typ Opis
index Identyfikator słuchaczarzeczywistego lub dźwiękowego The listener to get the data for (default is 0).

Zwroty:

DS Map ID

Przykład:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "x"] != 0
    {
        audio_listener_set_position(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

Powyższy kod sprawdza liczbę dostępnych słuchaczy, a następnie loops przez nich przechodzi i jeśli ich pozycja x jest różna od 0, ustawia ich pozycję na 0, 0, 0.
