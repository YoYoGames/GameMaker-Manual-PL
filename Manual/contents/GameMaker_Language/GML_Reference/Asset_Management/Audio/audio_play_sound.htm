audio_play_sound

Dzięki tej funkcji możesz odtwarzać dowolne zasoby dźwiękowe w swojej grze. Podajesz indeks dźwięku i przypisujesz mu priorytet, który następnie jest używany do określenia sposobu postępowania z dźwiękami, gdy liczba odtwarzanych dźwięków przekroczy limit ustalony przez funkcję audio_channel_num(). Dźwięki o niższym priorytecie zostaną zatrzymane na rzecz dźwięków o wyższym priorytecie, a wartość priorytetu może być dowolną liczbą rzeczywistą (rzeczywista wartość jest dowolna i może wynosić od 0 do 1 lub od 0 do 100, ponieważ GameMaker nada im taki sam priorytet). Zauważ, że w przypadku priorytetu, im wyższa liczba tym wyższy priorytet, tak więc dźwięk o priorytecie 100 będzie faworyzowany w stosunku do dźwięku o priorytecie 1. Trzeci argument służy do wykonania dźwięku loop, a ustawienie go na true spowoduje wykonanie dźwięku loop aż do zatrzymania, a ustawienie go na false spowoduje odtworzenie dźwięku tylko raz.

Funkcja ta zwraca również unikalny numer indeksu odtwarzanego dźwięku, który może być przechowywany w zmiennej, aby można było go wstrzymać lub zatrzymać za pomocą odpowiednich funkcji. Oznacza to, że jeśli masz wiele instancji tego samego dźwięku odtwarzanego w tym samym czasie, możesz skierować tylko jedną instancję tego dźwięku do obsługi za pomocą funkcji audio.

Składnia:

audio_play_sound(index, priority, loop, [gain], [offset], [pitch], [listener_mask]);

Argument Typ Opis
index Sound Asset The index of the sound to play.
priority Real Set the channel priority for the sound.
loop Boolean Sets the sound to loop or not.
gain Real OPTIONAL Wartość za zysk.
offset Real OPTIONAL Czas (w sekundach), na który należy ustawić punkt startowy. Wartości dłuższe niż długość podanego dźwięku są ignorowane.
pitch Real OPTIONAL Mnożnik skoku (domyślnie 1).
listener_mask Real OPTIONAL Dane bitmask do ustawienia dla dźwięku. W przypadku celu HTML5 nie będzie to miało żadnego efektu, ponieważ cel nie obsługuje więcej niż jednego słuchacza.

Zwroty:

Sound Instance ID

Przykład 1:

if health <= 0
{
    lives -= 1;
    audio_play_sound(snd_PlayerDead, 10, false);
}

Powyższy kod sprawdza zmienną globalną "health" i jeśli jest ona mniejsza lub równa 0, usunie 1 ze zmiennej globalnej "lives" i odtworzy dźwięk.

Przykład 2:

if bbox_left > room_width
{
    audio_play_sound(snd_Goodbye, 10, false, 1.1, 0, 2);
}

Powyższy kod sprawdza, czy instancja minęła prawą granicę strony room i odtwarza dźwięk o wzmocnieniu 1,1 i wysokości 2 (dwa razy więcej niż normalna wysokość).
