script_execute_ext

Funkcja ta działa podobnie do funkcji script_execute(), z tą różnicą, że można podać tablicę zawierającą argumenty wymagane przez wywoływaną funkcję/skrypt. Można również podać dwa opcjonalne argumenty określające offset w tablicy, z której mają zostać pobrane argumenty, oraz liczbę argumentów, które mają zostać użyte z tablicy (musi to być maksymalnie array_length - offset).

Składnia:

script_execute_ext(scr, array_args, [offset], [num_args]);

Argument Typ Opis
scr Zasób skryptu lub funkcja skryptu The function/script that you want to call
array_args Tablica The array containing the arguments for the function/script
[offset] Real OPTIONAL Przesunięcie w tablicy argumentów
[num_args] Real OPTIONAL Liczba argumentów do użycia (począwszy od offsetu)

Zwroty:

Variable (Will depend on the return value from the function/script being called)

Przykład:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

Powyższy przykładowy kod użyje script_execute_ext do wywołania podanej funkcji, podając tablicę czterech argumentów, ale używając tylko jednego z nich w sposób losowy.

UWAGA Tej funkcji nie można używać z funkcjami wbudowanymi.

Powrót: Skrypty
Następny: script_exists
