obiekt_get_physics

Ta funkcja informuje, czy sprawdzana strona object została oznaczona jako "fizyka włączona" - w takim przypadku zwraca true, - czy też nie - w takim przypadku zwraca false.

Składnia:

object_get_physics(obj);

ArgumentTyp Opis
obj The index of the object to check.

Zwroty:

Przykład:

if object_get_physics(object_index)
{
    phy_active = true;
}

Powyższy kod sprawdzi, czy instancja, z której jest tworzony, object ma włączoną fizykę, a jeśli tak, to uruchomi symulację fizyki dla tej instancji.
