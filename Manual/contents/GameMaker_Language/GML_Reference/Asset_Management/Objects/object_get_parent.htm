obiekt_get_parent

Funkcja zwraca indeks object każdego rodzica, który został przypisany do podanej strony object, w przeciwnym razie zwraca -100, aby pokazać, że strona object nie ma przypisanego rodzica, lub -1, jeśli sprawdzana strona object nie istnieje. Więcej informacji na temat rodziców można znaleźć w rozdziale Edytor obiektów.

Składnia:

object_get_parent(obj);

ArgumentTyp Opis
obj The index of the object to check

Zwroty:

Object Asset

Przykład:

par = object_get_parent(object_index);

Powyższy kod pobierze indeks rodzica object dla instancji uruchamiającej kod i zapisze zwróconą wartość w zmiennej "par".
