obiekt_set_maska

Za pomocą tej funkcji można ustawić indeks maski (mask_index) określonej strony object (więcej informacji o maskach można znaleźć w części Edytor Object ). Oznacza to, że wszystkie instancje tego object, które zostaną utworzone po zmianie maski, będą tworzone z tą nową maską, podczas gdy instancje, które już znajdują się w room, mogą pozostać bez zmian. Można ustawić wartość -1, aby usunąć maskę sprite i domyślnie ustawić maskę zdefiniowaną dla sprite z object (lub bez masek, jeśli nie wybrano sprite ).

Należy pamiętać, że nie jest to funkcja instancji! Możesz ustawić indeks maski poszczególnych instancji za pomocą zmiennej mask_index i w ten sposób mieć dziesięć instancji, każda z inną maską niż object, z którego zostały utworzone, i nawet jeśli zmienisz indeks maski object za pomocą tej funkcji, wszystkie instancje znajdujące się obecnie w room pozostaną takie, jakie były, a tylko instancje utworzone po wywołaniu funkcji będą zaczynały się z nową maską.

Składnia:

object_set_mask(index, spr);

Argument Typ Opis
index Obiekt Aktywa The index of the object to change.
spr Aktywa Sprite The new sprite to use as the object's mask.

Zwroty:

N/A

Przykład:

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

Powyższy kod sprawdza wartość zmiennej globalnej, a następnie zmienia indeks maski object "obj_Platform", jeśli jest on równy dziesięć.
