zdarzenie_obiekt

Ta zmienna tylko do odczytu zwraca indeks object instancji, która uruchamia sprawdzane zdarzenie.

Składnia:

event_object;

Zwroty:

Object Asset

Przykład:

global.obj = event_object;

W powyższym kodzie indeks object instancji wykonującej zdarzenie jest przechowywany w zmiennej globalnej.

Powrót: Generowanie zdarzeń Object
Następny: event_type
