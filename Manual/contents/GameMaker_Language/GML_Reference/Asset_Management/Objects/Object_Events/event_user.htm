zdarzenie_użytkownika

Za pomocą tej funkcji nakazuje się instancji wykonanie akcji lub kodu, który został umieszczony w jednym z 16 zdarzeń zdefiniowanych przez użytkownika. Zdarzenia te mogą być wywoływane tylko w ten sposób lub za pomocą funkcji event_perform funkcji.

Składnia:

event_user(numb);

Argument Typ Opis
numb Real The number of User Event to call, between 0 and 15.

Zwroty:

N/A

Przykład:

event_user(4);

Spowoduje to wywołanie zdarzenia zdefiniowanego przez użytkownika 4.

Powrót: Generowanie zdarzeń Object
Następny: event_action
