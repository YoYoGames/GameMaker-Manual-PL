zdarzenie_typu

Ta zmienna tylko do odczytu zwraca typ aktualnie realizowanego zdarzenia, którym może być jedna z poniższych stałych:

  • ev_create
  • zniszczyć
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_mouse
  • ew_kolizja
  • ew_inne
  • ew_gestykulacja
  • ev_draw


Pełna lista stałych związanych z różnymi zdarzeniami znajduje się na stronie event_perform()a jeśli potrzebujesz numeru zdarzenia (tzn. zdarzenia podrzędnego), powinieneś sprawdzić event_number.

Składnia:

event_type;

Zwroty:

Event Type Constant

Przykład:

show_debug_message("Current Event = " + string(event_type));

Powyższy kod wyświetli komunikat debugowania z typem zdarzenia, które jest aktualnie wykonywane.

Powrót: Generowanie zdarzeń Object
Następny: event_inherited
