obiekt_get_sprite

Ta funkcja powie Ci, czy sprawdzany adres object ma adres sprite, czy nie, a jeśli tak, to zwróci indeks tego adresu sprite lub -1, jeśli nie ma. Należy pamiętać, że nie jest to funkcja instancji! Możesz mieć instancję object bez sprite, podczas gdy instancja tej samej object może ją mieć i odwrotnie, a nawet mogą one mieć różne sprites. Pojedynczą instancję sprite można ustawić za pomocą sprite_index zmiennej instance.

Składnia:

object_get_sprite(obj);

ArgumentTyp Opis
obj The index of the object to check

Zwroty:

Sprite Asset

Przykład:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

Powyższy przykład sprawdzi, czy sprite_index instancji jest zgodny z sprite indeksu object_index instancji. Jeśli nie są one takie same, to przypisuje instancji indeks sprite taki sam jak indeks object.
