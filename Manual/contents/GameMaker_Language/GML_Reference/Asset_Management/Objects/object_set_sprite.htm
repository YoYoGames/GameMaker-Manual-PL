obiekt_set_sprite

Za pomocą tej funkcji możesz ustawić indeks sprite określonej strony object. Oznacza to, że wszystkie instancje tego object, które zostaną utworzone po zmianie sprite_index, zostaną utworzone z tym nowym sprite_index, podczas gdy instancje, które już znajdują się w room, mogą pozostać bez zmian.

Należy pamiętać, że nie jest to funkcja instancji! Możesz ustawić indeks sprite poszczególnych instancji za pomocą zmiennej sprite_index i w ten sposób mieć dziesięć instancji, każda z innym indeksem sprite niż object, z którego zostały utworzone, i nawet jeśli zmienisz indeks sprite w object za pomocą tej funkcji, wszystkie instancje, które są obecnie w room, pozostaną takie, jakie były, a tylko instancje utworzone po wywołaniu funkcji będą zaczynać się od nowego sprite.

Składnia:

object_set_sprite( index, spr );

Argument Typ Opis
index Obiekt Aktywa The index of the object to change.
spr Aktywa Sprite The sprite to assign to the object.

Zwroty:

N/A

Przykład:

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

Powyższy kod ustawi indeks object "obj_Player" sprite na różne wartości, w zależności od tego, w którym room znajduje się aktualnie instancja wykonująca kod. Następnie tworzy instancję "obj_Player".
