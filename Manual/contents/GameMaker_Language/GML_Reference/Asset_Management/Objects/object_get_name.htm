obiekt_get_name

Ta funkcja zwraca nazwę w postaci łańcucha znaków dla podanej strony object. Jest to nazwa, która została podana dla strony object w przeglądarce Asset w głównym oknie GameMakera. Należy pamiętać, że jest to tylko ciąg znaków i nie może być użyty do bezpośredniego odwołania się do object - do tego potrzebny jest indeks obiektu. Możesz jednak użyć tej strony string, aby uzyskać indeks obiektu, używając zwróconej strony string wraz z funkcją asset_get_index().

Składnia:

object_get_name(obj);

ArgumentTyp Opis
obj The index of the object to check.

Zwroty:

Przykład:

str = object_get_name(object_index);

Powyższy kod pobierze nazwę indeksu object dla instancji uruchamiającej kod i zapisze zwróconą wartość w zmiennej "str".
