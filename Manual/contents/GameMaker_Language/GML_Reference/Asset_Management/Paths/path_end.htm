Wywołanie tej funkcji spowoduje zakończenie aktualnej strony path, którą podąża instancja, zgodnie z ustawieniami w momencie wywołania funkcji path_start() została wywołana...

Składnia:

path_end();

Zwroty:

Przykład:

if place_meeting(x, y, obj_Blocker)
{
    path_end();
}

Powyższy kod zakończy działanie bieżącej strony path, jeśli jej instancja wykryje kolizję z jakąkolwiek instancją podanego obiektu.
