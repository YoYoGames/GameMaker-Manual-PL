ścieżka_get_precision

Ta funkcja zwraca, z jaką dokładnością podany adres path został "wygładzony", i będzie to wartość całkowita z zakresu od 1 do 8. Chociaż możesz uzyskać (i ustawić) tę wartość dla prostej linii path, nie będzie ona miała wpływu na sposób, w jaki instancja używa path, ponieważ jest ona istotna tylko wtedy, gdy rodzaj path jest ustawiony na "smooth".

Składnia:

path_get_precision(index);

ArgumentTyp Opis
index The index of the path to check.

Zwroty:

Przykład:

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

Powyższy kod sprawdza stronę path zindeksowaną w "pth_Patrol" pod kątem rodzaju i jeśli jest ona gładka, sprawdza następnie, jak dokładne jest wygładzanie. Jeśli jest ona mniejsza niż 8, ustawiana jest na wartość 8.
