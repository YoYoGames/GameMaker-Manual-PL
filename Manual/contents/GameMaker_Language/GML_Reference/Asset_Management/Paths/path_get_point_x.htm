ścieżka_get_point_x

Ta funkcja zwróci pozycję x (we współrzędnych room ) punktu, który został wprowadzony dla indeksu path. Jeśli punkt znajduje się poza zakresem adresu path (np. adres path ma 8 punktów, a Ty pytasz o pozycję x punktu 10), zwrócona zostanie wartość 0.

Składnia:

path_get_point_x(index, n);

Argument Typ Opis
index Aktywa na ścieżce The index of the path to check.
n Real The point number to check.

Zwroty:

Real

Przykład:

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

Powyższy kod sprawdzi, czy instancja znajduje się na końcu strony path. Jeśli tak, to wybierze losowy punkt na stronie path i przeniesie instancję do tego punktu.
