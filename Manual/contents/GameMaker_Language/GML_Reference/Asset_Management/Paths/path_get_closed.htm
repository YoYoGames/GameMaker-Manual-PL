ścieżka_get_closed

Ta funkcja może być użyta do zwrócenia informacji, czy strona path jest oznaczona jako zamknięta (true) czy otwarta (false), tzn. czy strona path loops ma określony początek i koniec.

Składnia:

path_get_closed(index);

Argument Typ Opis
index Aktywa na ścieżce The index of the path to check.

Zwroty:

Boolean

Przykład:

state = path_get_closed(pth_Patrol);

Spowoduje to ustawienie "state" na true lub false w zależności od stanu zamknięcia strony path indeksowanej w "pth_Patrol".
