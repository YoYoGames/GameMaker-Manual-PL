ścieżka_get_y

Za pomocą tej funkcji można uzyskać współrzędną y pozycji na dowolnej stronie path. 0 oznacza początek strony path, 1 - koniec strony path, a wszystko pomiędzy odpowiada znormalizowanej odległości wzdłuż długości strony path. Wartość ta nie musi być punktem definiującym stronę path, może znajdować się w dowolnym jej miejscu.

Składnia:

path_get_y(ind, pos);

Argument Typ Opis
index Aktywa na ścieżce The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end).

Zwroty:

Real

Przykład:

y = path_get_y(pth_WallWalk, 0.5);

Spowoduje to ustawienie współrzędnej y wywołanej instancji na współrzędną y punktu znajdującego się dokładnie w połowie drogi podanej na stronie path.
