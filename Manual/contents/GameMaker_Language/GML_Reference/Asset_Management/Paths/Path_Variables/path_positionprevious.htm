ścieżka_pozycja_poprzednia

Zmienna ta może być użyta do uzyskania lub ustawienia pozycji instancji wzdłuż jej aktualnej strony path w poprzednim kroku. Jest to znormalizowana wartość z przedziału od 0 do 1, tzn. 0 oznacza pozycję początkową strony path, a 1 pozycję końcową. Jest to podobne do funkcji xprevious i yprevious w sposobie działania, ale jest specyficzna dla paths. Może być przydatna np. do tymczasowego zatrzymania followera path, jeśli coś stoi mu na drodze (patrz przykładowy kod poniżej).

Składnia:

path_positionprevious;

Zwroty:

Przykład:

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if collision_circle(xx, yy, 16, obj_Player, false,true)
{
    path_position = path_positionprevious;
}

Powyższy kod sprawdza obszar przed instancją pod kątem kolizji z object "obj_Player" i jeśli taka kolizja wystąpi, ustawia instancję z powrotem na poprzednią pozycję, którą zajmowała na bieżącej stronie path, przypisaną przez funkcję path_start().
