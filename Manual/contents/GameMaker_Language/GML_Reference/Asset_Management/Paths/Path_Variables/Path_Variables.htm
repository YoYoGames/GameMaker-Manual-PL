Zmienne ścieżki

Poniższe zmienne są wbudowane we wszystkie instancje podczas ich tworzenia i można ich używać do uzyskiwania informacji o instancjach path lub do zmiany pewnych właściwości:

  • path_index
  • path_position
  • path_positionprevious
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

Więcej informacji na temat zmiennych instancji można znaleźć tutaj: Zmienne instancji.
