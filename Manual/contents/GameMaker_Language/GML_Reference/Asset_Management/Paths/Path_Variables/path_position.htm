ścieżka_położenia

Funkcji tej można użyć do uzyskania lub ustawienia pozycji instancji wzdłuż strony path. Wartość jest normalizowana w zakresie od 0 do 1, więc jeśli ustawimy ją na przykład na 0,5, to instancja zostanie przesunięta dokładnie na środek strony path.

Składnia:

path_position;

Zwroty:

Przykład:

path_position = random(1);

Powyższy kod ustawia instancję w losowej pozycji w dowolnym miejscu na bieżącej stronie path.
