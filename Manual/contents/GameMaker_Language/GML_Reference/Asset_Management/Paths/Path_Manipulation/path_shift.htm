przesunięcie_ścieżki

Za pomocą tej funkcji możesz przesunąć stronę path wzdłuż osi poziomej i pionowej. Zwróć uwagę, że nie będzie to miało widocznego efektu w grze, jeśli instancja podążająca za przesuniętą path podąża za nią względnie (a nie bezwzględnie), ponieważ pozycja ścieżki jest wtedy pomijana. Ta funkcja zmienia rzeczywisty zasób path , a więc będzie trwale wpływać na to, jak path jest używany przez wszystkie instancje w grze od momentu użycia funkcji do końca gry. Jeśli to to nie jest to, czego potrzebujesz, to powinieneś użyć funkcji takiej jak path_duplicate(), aby najpierw utworzyć kopię path, a następnie wywołać tę funkcję na powielonej stronie asset (nie zapomnij wywołać ścieżka_usuń() na asset, gdy nie będzie już potrzebna).

Składnia:

path_shift(index, xshift, yshift);

ArgumentTyp Opis
index The index of the path to change.
xshift The number of horizontal pixels to shift the path. Negative=left, positive=right.
yshift The number of vertical pixels to shift the path. Negative=up, positive=down.

Zwroty:

Przykład:

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

Powyższy kod duplikuje stronę path asset i zapisuje jej wartość ID w zmiennej. Zmienna ta jest następnie używana do przesunięcia zduplikowanej strony path na inną pozycję, a następnie instancja uruchamia stronę path.

Powrót: Manipulacja ścieżką
