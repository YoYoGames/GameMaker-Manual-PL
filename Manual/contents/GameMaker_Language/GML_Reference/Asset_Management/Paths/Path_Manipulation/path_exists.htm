ścieżka_istnieje

Ta funkcja zwraca, czy istnieje adres path o podanym indeksie, czy nie. Zwróć uwagę, że jeśli sprawdzisz istnienie path za pomocą zmiennej, która nie została jeszcze zadeklarowana, spowoduje to wyświetlenie błędu.

Składnia:

path_exists(index);

Argument Typ Opis
index Aktywa na ścieżce The index of the path to check for.

Zwroty:

Boolean

Przykład:

if (path_exists(my_path))
{
    path_delete(my_path);
}

Ten kod sprawdza, czy podana zmienna przechowuje indeks path, a jeśli tak, to path jest usuwany z pamięci.
