ścieżka_set_zamknięta

flag Za pomocą tej funkcji można określić, czy dana strona path jest otwarta (false) czy zamknięta (true). Zamknięta strona path ma punkt początkowy połączony z punktem końcowym, tworząc stronę loop, natomiast otwarta strona path ma definitywny, niepołączony początek i koniec. Ta funkcja zmienia rzeczywisty zasób path , a więc będzie trwale wpływać na sposób, w jaki path jest używany przez wszystkie instancje w grze od momentu użycia tej funkcji do końca gry.Jeśli nie tego potrzebujesz, powinieneś użyć funkcji takiej jak path_duplicate(), aby najpierw utworzyć kopię path, a następnie wywołać tę funkcję na zduplikowanej asset (nie zapomnij wywołać path_delete() na asset, gdy nie będzie już potrzebna).

Składnia:

path_set_closed(index, closed);

Argument Typ Opis
index Aktywa na ścieżce The index of the path to change.
closed Boolean Whether the path is closed (true) or not (false).

Zwroty:

N/A

Przykład:

path_set_closed(pth_Patrol, true);

Spowoduje to ustawienie podanej strony path jako zamkniętej strony path.

Powrót: Manipulacja ścieżką
Następny: path_set_kind
