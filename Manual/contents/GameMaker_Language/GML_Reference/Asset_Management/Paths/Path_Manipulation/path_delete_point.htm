ścieżka_usuń_punkt

Za pomocą tej funkcji można usunąć punkt z określonej strony path. Punkty Path są numerowane od 0, więc pierwszy punkt jest zawsze punktem 0, a ostatni punkt path będzie punktem 1. path_get_number() - Jeśli wywołasz tę funkcję na stronie path asset, to usunięcie punktu będzie trwałe i we wszystkich instancjach przypisanych do path w przyszłości będzie brakowało punktu, który został usunięty. Jeśli nie tego potrzebujesz, powinieneś użyć funkcji takiej jak path_duplicate(), aby najpierw utworzyć kopię path, a następnie wywołać tę funkcję na powielonej stronie asset (nie zapomnij wywołać path_delete() na stronie asset, gdy nie będzie już potrzebna).

Składnia:

path_delete_point(ind, n);

Argument Typ Opis
index Aktywa na ścieżce The index of the path to delete a point from.
n Real The defining point to delete.

Zwroty:

N/A

Przykład:

var _col = true;
while (_col)
{
    var _num = path_get_number(mypath) - 1;
    if (_num > 2)
    {
        var _x1 = path_get_point_x(my_path, _num);
        var _y1 = path_get_point_y(my_path, _num);
        var _x2 = path_get_point_x(my_path, _num -1);
        var _y2 = path_get_point_y(my_path, _num -1);
        if collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false)
        {
            path_delete_point(mypath, _num);
            _num -= 1;
        }
        else _col = false;
    }
    else _col = false;
}

Powyższy kod generuje stronę loop, która sprawdza, czy punkty na odcinku od końca do tyłu nie kolidują z podaną stroną object. Jeśli zostanie znaleziona kolizja, punkt path jest usuwany, aż do momentu, gdy nie będzie kolizji lub path będzie miał tylko 2 punkty długości.
