ścieżka_get_number

Tej funkcji można użyć do zwrócenia liczby punktów na stronie path.

Składnia:

path_get_number(index);

ArgumentTyp Opis
index The index of the path to check.

Zwroty:

Przykład:

if (path_get_number(pth_AI) > 1)
{
    path_start( pth_AI, 4, 3, 0);
}

Powyższy kod sprawdza, czy na stronie path znajduje się więcej niż jeden punkt, a jeśli tak, to uruchamia instancję poruszającą się wzdłuż tej strony path.
