sekwencja_get_obiektów

Za pomocą tej funkcji można pobrać tablicę wszystkich indeksów object, w których tworzone są instancje w obrębie podanej strony sequence. Podajesz albo adres sequence object struct (zwrócony przez funkcję sequence_create() lub sequence_get()) lub identyfikator sequence (zwracany przez funkcję layer_sequence_get_sequence() lub z instancji sequence struct właściwość sequence), a funkcja zwróci tablicę, w której każdy element tablicy jest identyfikatorem object_index dla różnych objects używanych przez sequence do tworzenia instancji.

Składnia:

sequence_get_objects(sequence_struct_or_id);

Argument Typ Opis
sequence_struct_or_id Aktywa sekwencji lub sekwencja Object Struktura The sequence object struct or ID to get the objects from.

Zwroty:

Array

Przykład:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

Powyższy kod pobiera adres struct z adresu sequence object, a następnie pobiera adres objects, którego używa, i zapisuje tablicę w zmiennej.

Powrót: Sekwencje
