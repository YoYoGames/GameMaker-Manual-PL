Struktura danych klatek kluczowych

Podczas definiowania struktury klatek kluczowych dla ścieżki należy również zdefiniować dane, które będą z nimi powiązane. Składa się to z różnych kanałów, gdzie każdy kanał to po prostu Keyframe Data Struct. Kanałowi można przypisać określony typ danych, zależnie od rodzaju ścieżki, dla której ustawia się dane keyframe.

Strona Keyframe Data Struct zawiera następujące zmienne:

Sekwencja Keyframe Struktura danych
Zmienna Typ Opis
channel Real Jest to kanał, do którego mają być zastosowane dane keyframe. Jest to dodatnia liczba całkowita, zaczynająca się od 0. Warto zauważyć, że podczas tworzenia ścieżek parametrów dla "pozycji" lub "skali" keyframes należy używać bardzo konkretnych wartości kanałów. Są to: kanał 0 to pozycja X lub skala X, kanał 1 to pozycja Y lub skala Y.
spriteIndex Aktywa Sprite Indeks strony sprite asset, który ma być użyty dla ścieżki. Właściwość ta jest dostępna tylko dla ścieżek typu seqtracktype_graphic i można ją uzyskać lub ustawić.
soundIndex Aktywa dźwiękowe Indeks audio asset, który ma być użyty dla ścieżki. Właściwość ta jest dostępna tylko dla ścieżek typu seqtracktype_audio i można ją uzyskać lub ustawić.
playbackMode Stała sekwencji klucza audio Tryb odtwarzania dźwięku. Ta właściwość jest dostępna tylko dla ścieżek typu seqtracktype_audio i można ją uzyskać lub ustawić. Właściwość powinna być jedną ze stałych podanych w tabeli poniżej.
curve Struktura krzywej animacji Ta właściwość wymaga adresu animation curve struct (więcej informacji można znaleźć tutaj ) i jest dostępna tylko dla ścieżek typu seqtracktype_real. Jeśli nie jest używana krzywa struct, wartość tej właściwości będzie wynosić -1.
value Real Ta właściwość jest po prostu wartością związaną z kanałem danych keyframe i jest dostępna tylko dla ścieżek typu seqtracktype_real, jeśli nie podano adresu curve struct. Może to być na przykład pozycja X lub Y ścieżki, jeśli jest ona umieszczona wewnątrz ścieżki z parametrem "position".
colour Array z Prawdziwes Właściwość ta zwraca (lub wymaga, jeśli jest ustawiona) tablicę wartości kolorów strony keyframe w formacie [A, R, G, B]. Jest ona dostępna tylko dla ścieżek typu seqtracktype_colour. Należy pamiętać, że wartości dla każdego elementu powinny być wyrażone w zakresie od 0 do 1, gdzie 0 odpowiada wartości HEX #00, a 1 odpowiada wartości HEX #FF (0 - 255, jak pokazano w oknie wyboru koloru dla ścieżek kolorowych w edytorze Sequences ).
sequence Sekwencja Object Struct Właściwość ta zwraca (lub wymaga, jeśli jest ustawiona) sekwencję object struct i jest dostępna tylko dla ścieżek typu seqtracktype_sequence.
objectIndex Obiekt Aktywa Właściwość ta zwraca (lub wymaga, jeśli jest ustawiona) indeks object i jest dostępna tylko dla ścieżek typu seqtracktype_instance.
events Array z Strings Ta właściwość umożliwia dostęp do zdarzeń i komunikatów rozgłoszeniowych związanych z danymi keyframe struct . Właściwość tę można uzyskać lub ustawić, przy czym przy uzyskiwaniu zwracana jest tablica strings, a przy ustawianiu należy określić tablicę strings. Więcej informacji o zdarzeniach można znaleźć w sekcji Zdarzenia i momenty sekwencji. Ta właściwość jest dostępna tylko dla ścieżek typu seqtracktype_message.
event Metoda Ta właściwość zwróci (lub może być ustawiona na) wartość method związaną z danymi keyframe struct . Jeśli nie określono żadnej metody lub chcesz usunąć metodę, właściwość ta powinna mieć wartość -1. Właściwość ta jest dostępna tylko dla ścieżek typu seqtracktype_moment.
Wszelkie dodatkowe zmienne wymienione w poniższej tabeli (tabelach)

Zmienna playbackMode może być jedną z następujących stałych:

Stała sekwencji klucza audio
Stała Opis Wartość
seqaudiokey_loop Podczas odtwarzania dźwięk będzie dostępny na stronie loop. 0
seqaudiokey_oneshot Dźwięk zostanie odtworzony tylko raz, a następnie zatrzymany. 1

Dane ścieżki tekstowej

Strona struct będzie zawierać następujące dodatkowe zmienne, jeśli zostanie przypisana do ścieżki tekstowej (seqtracktype_text):

Sekwencja Keyframe Struktura danych
Zmienna Typ Opis
text String Jest to tekst string, który jest rysowany na ścieżce.
wrap Boolean Jest to symbol graficzny określający, czy tekst powinien być zawinięty (true), czy nie (false).
alignmentH Stała wyrównania sekwencji tekstu Jest to wyrównanie tekstu w poziomie i będzie to jedna z poniższych wartości Stała wyrównania tekstu sekwencjipodanych poniżej.
alignmentV Stała wyrównania sekwencji tekstu Jest to wyrównanie tekstu w pionie i będzie to jedna z poniższych wartości Stała wyrównania tekstu sekwencjipodanych poniżej.
fontIndex Czcionka Aktywa Jest to czcionka Zasób czcionek używana przez ścieżkę tekstową.

alignmentH może być dowolną z poniższych stałych:

Stałe wyrównywania sekwencji tekstu (alignmentH)
Stała Opis
seqtextkey_left Tekst zostanie wyrównany do lewej strony ramki
seqtextkey_right Tekst zostanie wyrównany do prawej strony ramki
seqtextkey_center Tekst zostanie wyśrodkowany poziomo w ramce
seqtextkey_justify Tekst zostanie wyjustowany w ramce

alignmentV może być dowolną z poniższych stałych:

Stałe wyrównywania sekwencji tekstu (alignmentV)
Stała Opis
seqtextkey_top Tekst zostanie wyrównany do górnej części ramki
seqtextkey_bottom Tekst zostanie wyrównany do dolnej części ramki
seqtextkey_middle Tekst zostanie wyśrodkowany pionowo w ramce
