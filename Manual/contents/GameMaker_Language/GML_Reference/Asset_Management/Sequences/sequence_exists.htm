sekwencja_istnieje

Za pomocą tej funkcji można sprawdzić, czy strona sequence object istnieje, czy nie. Podaje się albo adres sequence object struct (zwrócony przez funkcję sequence_create() lub sequence_get()) lub identyfikator sequence (zwracany przez funkcję layer_sequence_get_sequence() lub z instancji sequence struct właściwość sequence, lub indeks z przeglądarki asset ), a funkcja zwróci true, jeśli sequence object istnieje, lub false, jeśli nie.

Składnia:

sequence_exists(sequence_struct_or_id);

Argument Typ Opis
sequence_struct_or_id Aktywa sekwencji lub sekwencja Object Struktura The sequence to check for, can be the asset reference itself or its Sequence object struct.

Zwroty:

Boolean

Przykład:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Powyższy kod sprawdza, czy podany adres sequence object istnieje, a jeśli tak, to zostaje zniszczony.

Powrót: Sekwencje
Następny: sequence_create
