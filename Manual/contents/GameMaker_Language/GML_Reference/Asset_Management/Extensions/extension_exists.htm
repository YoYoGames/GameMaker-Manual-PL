rozszerzenie_istnieje

Funkcja mówi czy extension o podanej nazwie istnieje(true) czy nie(false).

Składnia:

extension_exists(extension_name);

Argument Typ Opis
extension_name String The name of your extension asset as a string

Zwroty:

Boolean

Przykład:

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

Ten kod sprawdza, czy MyExtension extension istnieje, a jeśli tak, to pobiera jego opcje jako struct, i odczytuje z niego wartość.

Powrót: Rozszerzenia
Następny: extension_get_options
