extension_get_options

Ta funkcja zwraca stronę struct zawierającą wszystkie opcje i ich wartości dla strony extension o podanej nazwie.

Funkcja zwróci wartość undefined, jeśli podana nazwa extension jest nieprawidłowa.

Składnia:

extension_get_options(extension_name);

Argument Typ Opis
extension_name String The name of your extension asset as a string

Zwroty:

Struct

Przykład:

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

Ten kod pobiera opcje struct dla strony extension i odczytuje z niej wartość.
