extension_get_option_value

Ta funkcja zwraca wartość podanej opcji w witrynie extension o podanej nazwie.

Funkcja zwróci wartość undefined, jeśli podana nazwa extension jest nieprawidłowa.

Składnia:

extension_get_option_value(extension_name, option_name);

Argument Typ Opis
extension_name String The name of your extension asset as a string
option_name String The name of the option to read

Zwroty:

Any

Przykład:

var _enabled = extension_get_option_value("MyExtension", "enabled");

Ten kod pobiera wartość opcji enabled w oknie MyExtension extension.
