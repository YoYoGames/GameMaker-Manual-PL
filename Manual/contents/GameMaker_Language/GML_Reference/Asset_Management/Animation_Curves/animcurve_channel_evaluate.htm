animcurve_channel_evaluate

Funkcja ta może być użyta do uzyskania wartości w określonym punkcie czasu ze struktury kanału. Należy podać wskaźnik struct na kanał (zwrócony przez funkcję animcurve_get_channel()lub zwrócony w animation curve struct przez funkcję animcurve_get()) oraz pozycję "x" (czas) wzdłuż kanału krzywej do oceny. Pozycja ta powinna zawierać się w przedziale od 0 do 1, a funkcja zwróci wartość krzywej w tej pozycji lub zwróci 0, jeśli podany kanał struct jest nieprawidłowy.

Składnia:

animcurve_channel_evaluate(channel_struct, posx);

Argument Typ Opis
channel_struct Struktura kanału krzywej animacji The struct pointer for the channel to evaluate.
posx Real The position in time to check (from 0 to 1).

Zwroty:

Real

Przykład:

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

Powyższy kod pobiera kanał struct dla instancji animation curve asset "ac_AlphaCurve". Następnie używa zwróconej wartości oceny do ustawienia alfa obrazu dla instancji.
