animcurve_destroy

Za pomocą tej funkcji można zniszczyć stronę animation curves, która została wcześniej utworzona za pomocą funkcji animcurve_create(). Wywołanie tej funkcji spowoduje usunięcie strony animation curve z pamięci i wyczyszczenie wszystkich kanałów i punktów, które zawiera (nigdy nie trzeba ich czyścić ręcznie). Należy pamiętać, że nie można zniszczyć żadnych animation curves utworzonych w przeglądarce asset , a jedynie te utworzone w runtime.

Składnia:

animcurve_destroy(curve_struct);

Argument Typ Opis
curve_struct Struktura krzywej animacji The pointer to the curve struct to destroy.

Zwroty:

N/A

Przykład:

animcurve_destroy(my_curve);

Powyższy kod zniszczy (utworzoną wcześniej) stronę animation curve struct zapisaną w zmiennej "my_curve".

Powrót: Krzywe animacji
Następny: animcurve_get
