animcurve_point_new

Za pomocą tej funkcji można utworzyć nową strukturę punktów, która zostanie dodana do kanału animation curve. Po utworzeniu struktura punktów struct jest pusta i należy ustawić następujące zmienne, aby wygenerować dane punktów:

Animacja Punkt krzywej Struktura
Nazwa zmiennej Typ danych Opis
posx prawdziwy Położenie w czasie (znormalizowane od 0 do 1) punktu.
value prawdziwy Wartość punktu.

Każdy punkt struct, który zostanie utworzony, należy dodać do tablicy i to właśnie ta tablica zostanie dodana do zmiennej "channels" kanału animation curve struct (jak pokazano w poniższym przykładzie).

Składnia:

animcurve_point_new();

Zwroty:

Animation Curve Point Struct

Przykład:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Powyższy kod tworzy nową stronę animation curve struct i zapisuje ją w zmiennej "my_curve". Następnie na stronie struct umieszczana jest nazwa i tablica kanałów. Tablica kanałów zawiera pojedynczy kanał z trzema punktami.

Powrót: Krzywe animacji
