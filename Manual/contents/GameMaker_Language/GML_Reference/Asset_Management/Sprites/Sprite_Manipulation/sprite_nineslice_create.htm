sprite_nineslice_create

Dziewięć plasterków to technika stosowana do skalowania prostokątnych obrazów w celu zachowania szczegółów, dzięki czemu po przeskalowaniu zachowują one swój oryginalny kształt. Informacje na temat funkcji Dziewięć plasterków można znaleźć na tej stronie.

Ta funkcja służy do tworzenia struktury Nine Slice, którą można zmodyfikować, a następnie zastosować do strony sprite. Zawartość tej strony struct jest szczegółowo opisana na tej stronie.

Możesz zapisać tę stronę struct w zmiennej, zmodyfikować jej właściwości i zastosować ją do dowolnej strony sprites, używając adresu sprite_set_nineslice().

Składnia:

sprite_nineslice_create();

Zwroty:

Nine Slice Struct

Przykład:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Powyższy kod tworzy nową stronę struct z dziewięcioma plasterkami, włącza ją i ustawia przesunięcia dla prowadnic. Następnie strona struct jest stosowana do strony sprite, zmieniając jej właściwości Nine Slice.
