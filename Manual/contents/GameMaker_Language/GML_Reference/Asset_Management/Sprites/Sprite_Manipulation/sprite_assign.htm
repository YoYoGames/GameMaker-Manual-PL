sprite_assign

Funkcja ta pobiera dwa wcześniej utworzone (lub dołączone) indeksy sprite i kopiuje obraz z jednego do drugiego. W ten sposób można skopiować (lub "sklonować") jeden sprite do innego indeksu. Uwaga: nie można kopiować do zasobów gry. Musisz wcześniej utworzyć stronę sprite, do której ma zostać skopiowany, używając polecenia sprite_add() lub sprite_duplicate() funkcji.

Składnia:

sprite_assign(index, sprite);

ArgumentTyp Opis
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

Zwraca

Przykład:

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

Powyższy kod sprawdza, czy zmienna globalna "Player_Sprite" zawiera sprite, a jeśli tak, to używa sprite_assign, aby zmienić ją na taką, która została załadowana z zewnętrznego pliku. Jeśli nie zawiera sprite, to zostaje załadowany jeden z nich, a jego indeks jest zapisywany w tej zmiennej.
