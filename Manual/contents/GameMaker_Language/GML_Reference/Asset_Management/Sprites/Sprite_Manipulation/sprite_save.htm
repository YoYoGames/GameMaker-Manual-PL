sprite_save

Za pomocą tej funkcji można zapisać na dysku dowolny podobraz strony sprite, podając mu określoną nazwę pliku. Adres sprite musi być dodany na stronie runtime (nie można zapisać adresu sprites dodanego przez stronę IDE), a plik musi być zapisany z adresem *.png extension.

UWAGA: W zależności od wybranej platformy docelowej istnieją ograniczenia co do miejsca, z którego można zapisywać i wczytywać pliki. Więcej informacji można znaleźć w części Obsługa plików.

Składnia:

sprite_save(ind, subimg, fname);

Argument Typ Opis
ind Aktywa Sprite The index of the sprite to save.
subimg Real The sub-image of the sprite to save.
fname String The filename for the saved sprite.

Zwraca

N/A

Przykład:

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

Powyższy kod tworzy powierzchnię i przechowuje jej indeks w zmiennej lokalnej "surf". Następnie celuje w tę powierzchnię, oczyszcza ją i rysuje kilka sprites jeden na drugim. Na podstawie złożonego obrazu narysowanego na powierzchni tworzy nową stronę sprite i przypisuje jej indeks do zmiennej lokalnej "spr_Custom", a następnie zwalnia pamięć używaną przez powierzchnię. Na koniec wynikowa strona sprite jest zapisywana do pliku, a następnie usuwana z pamięci.
