sprite_set_bbox_mode

Ta funkcja służy do ustawiania trybu ramki ograniczającej dla strony sprite. Podaje się indeks strony sprite oraz tryb, który ma być użyty, a który powinien być jedną z poniższych stałych:

Tryb ramki ograniczającej Stała
Stała Opis
bboxmode_automatic Automatycznie - ramka ograniczająca zostanie obliczona automatycznie na podstawie tolerancji ustawionej dla sprite'a.
bboxmode_fullimage Pełny obraz - ramka ograniczająca zostanie ustawiona na pełną szerokość i wysokość strony sprite, niezależnie od tolerancji i "pustych" pikseli.
bboxmode_manual Ręcznie - Ramka ograniczająca jest ustawiana ręcznie na wartości zdefiniowane przez użytkownika, co wymaga użycia funkcji sprite_set_bbox()

UWAGA Ta funkcja wpływa na zasób sprite'a, tak że wszystkie kolejne instancje z tym adresem sprite będą miały ten sam tryb bounding box.

Składnia:

sprite_set_bbox_mode(ind, mode);

Argument Typ Opis
ind Aktywa Sprite The index of the sprite to change the mode of.
mode Tryb ramki ograniczającej Stała The mode to set (a constant).

Zwraca

N/A

Przykład:

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Powyższy kod sprawdza tryb bbox dla bieżącej strony sprite i jeśli nie jest to automatic, to jest ustawiany na tę wartość.
