sprite_index

Ta zmienna zwraca indeks bieżącego adresu sprite dla instancji lub -1, jeśli instancja nie ma powiązanego z nią adresu sprite. Możesz ją zmienić, aby nadać instancji inny adres sprite, podając jej nazwę adresu sprite z drzewa zasobów lub używając zmiennej, w której indeksowany jest załadowany z zewnątrz adres sprite. Zmiana adresu sprite nie powoduje zmiany indeksu aktualnie widocznej ramki, więc jeśli zmienisz adres sprite na ramce numer 3, nowy adres sprite zostanie narysowany z widoczną tą ramką (zakładając, że ma ona taką samą liczbę ramek).

Składnia:

sprite_index;

Zwroty:

Sprite Asset

Przykład:

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

loop Powyższy kod przejdzie przez wszystkie instancje "obj_Check", sprawdzając, czy istnieje linia kolizyjna między nimi a instancją uruchamiającą kod. Adres sprite tych instancji zostanie zmieniony w zależności od zwróconej wartości (true lub false) dla linii kolizyjnej.
