sprite_yoffset

Ta zmienna tylko do odczytu zwraca lokalny yoffset (składową y punktu początkowego zdefiniowaną w edytorze sprite ) strony sprite, która została przypisana do instancji. To yoffset jest zwracane w pikselach i zależy od wartości image_yscaleTak więc (na przykład) jeśli dla kwadratowej strony sprite o wymiarach 32x32 początek y został zdefiniowany na 16, to po przeskalowaniu tej strony sprite o 2 zmienna ta zwróci 32. Jeśli potrzebujesz nieskalowanego yoffset, powinieneś użyć sprite_get_yoffset().

Składnia:

sprite_yoffset;

Zwroty:

Real

Przykład:

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

Powyższy kod sprawdza, czy górna krawędź sprite instancji znajduje się poza room, a jeśli tak, to przesuwa instancję tak, aby widoczna była cała sprite.
