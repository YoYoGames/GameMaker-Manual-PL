sprite_width

Ta zmienna tylko do odczytu zwraca szerokość strony sprite, która została przypisana do instancji. Szerokość ta jest zwracana w pikselach i będzie zależała od wartości image_xscale. Jeśli potrzebujesz nieskalowanej szerokości, powinieneś użyć sprite_get_width().

Składnia:

sprite_width;

Zwroty:

Real

Przykład:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

Powyższy kod sprawdza szerokość zasobu sprite przypisanego do instancji uruchamiającej kod z szerokością zasobu sprite i jeśli nie jest ona taka sama, resetuje zasób image_xscale do wartości 1,
