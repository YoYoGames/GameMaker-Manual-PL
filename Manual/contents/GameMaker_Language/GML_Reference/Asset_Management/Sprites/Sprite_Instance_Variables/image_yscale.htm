image_yscale

Ta wartość określa pionowe skalowanie (wzdłuż osi y) zastosowane do strony sprite, która została przypisana do bieżącej instancji. Skala równa 1 oznacza brak skalowania (1:1), mniejsze wartości będą skalowane w dół (na przykład 0,5 zmniejszy wysokość sprite o połowę), większe wartości będą skalowane w górę, a wartości ujemne będą odbiciem lustrzanym sprite i przeskalują go, chyba że użyta wartość wynosi dokładnie -1 (w tym przypadku sprite jest po prostu odbiciem lustrzanym wzdłuż osi y bez skalowania).

Składnia:

image_yscale;

Zwroty:

(single precision floating point value)

Przykład:

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Powyższy kod skaluje stronę sprite, a następnie po przeskalowaniu jej do pięciokrotności pierwotnego rozmiaru tworzy nową instancję innej strony object i niszczy tę instancję.
