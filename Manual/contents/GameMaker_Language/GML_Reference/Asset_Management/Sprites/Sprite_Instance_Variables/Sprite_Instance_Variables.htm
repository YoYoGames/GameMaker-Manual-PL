Zmienne instancji sprite'a

Zmienne te są domyślnie dołączone do GameMaker objects i wszystkie są związane z sprite przypisanym do instancji w room. Do zmiennych tych można w dowolnym momencie uzyskać dostęp w kodzie lub akcjach instancji i użyć ich do zmiany tego, co i jak jest rysowane.

  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale
  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
