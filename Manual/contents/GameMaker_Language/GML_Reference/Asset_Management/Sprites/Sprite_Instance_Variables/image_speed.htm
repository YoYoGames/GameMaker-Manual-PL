image_speed

Ta zmienna określa szybkość, z jaką GameMaker będzie przechodził przez pod-obrazy dla bieżącej instancji sprite. Podana wartość prędkości jest mnożnikiem, przy czym 1 jest wartością domyślną, a ustawienie jej na 0,5 spowoduje zmniejszenie prędkości animacji o połowę - zgodnie z ustawieniami w Sprite Editor lub Image Editor - podczas gdy ustawienie jej na 2 spowoduje jej podwojenie. Jeśli używana strona sprite nie ma żadnych podobrazów, ta zmienna nie będzie miała żadnego wpływu.

Składnia:

image_speed;

Zwroty:

(single precision floating point value)

Przykład:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

Powyższy kod tworzy instancję obiektu object "obj_Explosion" i ustawia jego image_speed na 0,5 (co oznacza zmniejszenie o połowę prędkości ruchu).
