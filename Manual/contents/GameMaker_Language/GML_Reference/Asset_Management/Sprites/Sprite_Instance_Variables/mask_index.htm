maska_indeks

Po zdefiniowaniu adresu object w programie GameMaker można przypisać mu maskę, która będzie używana do kolizji, a nie maskę odpowiadającą zdefiniowanemu adresowi sprite. Ta zmienna może być użyta do znalezienia sprite_index tej maski (lub zwróci -1, jeśli nie przypisano sprite ) lub ustawić maskę dla instancji na wybraną sprite. Ustawienie indeksu maski oznacza, że możesz mieć na przykład sprite dla instancji o nieregularnym kształcie, a mimo to nadać jej okrągłą maskę kolizyjną, która jest pobierana z innej strony sprite.

Składnia:

mask_index;

Zwroty:

Sprite Asset

Przykład:

mask_index = spr_Round;

Powyższy kod ustawia maskę instancji na maskę strony sprite "spr_Round".

Powrót: Zmienne instancji sprite'a
