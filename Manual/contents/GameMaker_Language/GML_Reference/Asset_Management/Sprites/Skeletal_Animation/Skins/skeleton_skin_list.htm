lista_szkieletów_skóry

Za pomocą tej funkcji można wypełnić (utworzoną wcześniej) listę DS wszystkimi nazwami skórek wchodzących w skład animacji szkieletowej sprite. Nazwy te zostaną umieszczone na stronie strings i będą mogły być następnie używane w innych funkcjach skórek animacji szkieletowej dla tych typów sprite.

Składnia:

skeleton_skin_list(sprite, list);

Argument Typ Opis
sprite Aktywa Sprite The sprite index of the Spine skeletal animation to get the list from.
list Identyfikator listy DS The ID of the DS list to populate with the animation names.

Zwroty:

N/A

Przykład:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Powyższy kod tworzy listę DS, a następnie wypełnia ją nazwami skórek. Następnie wybierana jest losowa skórka, która zostanie zastosowana do instancji, zanim lista zostanie zniszczona.
