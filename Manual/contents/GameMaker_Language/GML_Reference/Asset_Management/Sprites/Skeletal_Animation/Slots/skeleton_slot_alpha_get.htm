szkielet_slot_alpha_get

Za pomocą tej funkcji można uzyskać wartość alfa gniazda dołączania. Podajesz nazwę gniazda ( string), a funkcja zwróci wartość rzeczywistą alfa z zakresu od 0 do 1.

Składnia:

skeleton_slot_alpha_get(slot);

Argument Typ Opis
slot String The slot name to check, a string

Zwroty:

Real

Przykład:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_alpha_get(slot_list[| i]) != 1
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Powyższy kod tworzy listę DS, a następnie wypełnia ją nazwami szczelin znalezionych w miejscu, w którym znajduje się mysz na stronie room. Następnie loops przegląda listę szczelin i ustawia kolor znalezionych szczelin na biały z alfą równą 1, jeśli alfa nie jest już równa 1.
