skeleton_get_minmax

Ta funkcja zwraca tablicę zawierającą całkowitą wartość pól ograniczających dla wszystkich pojedynczych pól ograniczających przypisanych do animacji szkieletowej sprite. Zwrócona tablica będzie miała 4 elementy:

  • [0] - minimalna pozycja x dla wszystkich pól ograniczających
  • [1] - minimalna pozycja y dla wszystkich pól ograniczających
  • [2] - maksymalna pozycja x dla wszystkich ramek ograniczających
  • [3] - maksymalna pozycja y dla wszystkich pól ograniczających

Składnia:

skeleton_get_minmax();

Zwroty:

Array (4 elements: xMin, yMin, xMax, yMax)

Przykład:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
{
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

Powyższy kod sprawdzi bieżącą klatkę animacji przypisaną do ścieżki 0, a następnie wybije kilka cząstek z punktu znajdującego się w środku całkowitego pola ograniczającego dla sprite.
