narysuj_szkielet

Ta funkcja jest przeznaczona tylko do użytku z witrynami sprites, które zostały utworzone za pomocą programu do animacji szkieletowej, takiego jak Spine. Chociaż można rysować takie sprites za pomocą zwykłych funkcji rysowania (np. draw_self()), ale czasami może się zdarzyć, że będziesz chciał narysować pojedynczą klatkę lub pokazać zmianę skóry texture bez faktycznej zmiany właściwości sprite. W takich przypadkach można narysować stronę sprite za pomocą tej funkcji, w której podaje się indeks sprite, a następnie nazwę animacji, z której ma zostać pobrana klatka ( string, zgodnie z ustawieniami programu używanego do tworzenia sprite). Następnie podajesz nazwę skórki, która ma zostać użyta (ponownie ustawioną podczas tworzenia sprite ), oraz wartość ramki. Wartość ramki informuje program GameMaker, jaką część animacji ma pokazać, i jest wartością z przedziału od 0 do image_number. Pozostałe argumenty są takie same jak te używane do normalnego rysowania na stronie sprite i zmieniają skalę, kąt, kolor mieszania i alfa końcowej klatki animacji.

OSTRZEŻENIE! Ze względu na sposób, w jaki animacje szkieletowe są interpolowane między różnymi klatkami "kluczowymi", z tym wywołaniem wiąże się dodatkowy narzut, dlatego zaleca się unikanie rysowania animacji opartych na szkieletach sprites za pomocą tej metody, chyba że jest to absolutnie konieczne.

Składnia:

draw_skeleton(sprite, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Argument Typ Opis
sprite Aktywa Sprite The index of the sprite to draw.
animname String The name of the animation to get the frame from (a string).
skinname String The name of the skin to use (a string).
frame Real The animation frame to draw (from 0 to image_number - 1).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Kolor The colour with which to blend the sprite.
alpha Real The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Zwroty:

N/A

Przykład:

draw_skeleton(spr_Jump, "jump", "skin1", 0, x, y, image_xscale, image_yscale, image_angle, c_white, 0.5);

Powyższy kod narysuje pierwszą klatkę podanej animacji szkieletowej sprite, używając zestawu animacji "skok", skórki zdefiniowanej jako "skin1" oraz w takiej samej skali i położeniu, jakie zdefiniowano we właściwościach instancji, ale z alfą 0,5.
