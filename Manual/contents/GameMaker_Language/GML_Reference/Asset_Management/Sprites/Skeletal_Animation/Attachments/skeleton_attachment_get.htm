szkielet_załącznika_get

Animacja szkieletowa sprite może mieć dodane inne sprites jako załączniki, przy czym te sprites są dodawane do nazwanego slotu (nazwa jest podawana podczas tworzenia slotu załącznika w programie do animacji) i będą rysowane wraz z animacją bieżącego sprite. Za pomocą tej funkcji można uzyskać nazwę (w postaci string) załącznika dla danego slotu aktualnie przypisanego sprite. Zauważ, że do dołączonych sprites odwołujemy się poprzez łańcuch ich nazw przypisany w Spine, lub po wywołaniu funkcji skeleton_attachment_create().

Składnia:

skeleton_attachment_get(slot);

Argument Typ Opis
slot String The slot name (a string) to get the attachment of.

Zwroty:

String

Przykład:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Powyższy kod sprawdzi aktualnie przypisaną nazwę załącznika dla slotu o nazwie "slot_leftHand" i jeśli zwrócona zostanie pusta string, zostanie dołączony nowy sprite.
