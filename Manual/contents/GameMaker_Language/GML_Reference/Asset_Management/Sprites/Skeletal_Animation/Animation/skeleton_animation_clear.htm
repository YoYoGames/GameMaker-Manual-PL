szkielet_animacji_wyczyść

Ta funkcja usuwa z określonej ścieżki animacji wszystkie animacje i umożliwia ich ponowne przypisanie.

Składnia:

skeleton_animation_clear(track);

Argument Typ Opis
track Real The animation track to clear.

Zwroty:

N/A

Przykład:

if mouse_check_button(mb_right)
{
    skeleton_animation_clear(1);
}

Powyższy kod spowoduje wyczyszczenie ścieżki animacji nr 1, jeśli zostanie naciśnięty prawy przycisk myszy.
