szkielet_animacji_jest_zakończony

Za pomocą tej funkcji możesz sprawdzić, czy bieżąca nie zapętlona animacja szkieletowa zakończyła odtwarzanie na podanej ścieżce.

Składnia:

skeleton_animation_is_finished(track);

Argument Typ danych Opis
ścieżka Real Indeks ścieżki animacji

Zwroty:

Boolean

Przykład:

var _finished = skeleton_animation_is_finished(1);
if _finished
{
    show_debug_message("The animation has finished playing");
}

Powyższy kod najpierw sprawdza, czy animacja na ścieżce 1 została zakończona i zapisuje wynik w zmiennej lokalnej _finished. Następnie wyświetla komunikat debugowy w przypadku, gdy animacja zakończyła odtwarzanie.
