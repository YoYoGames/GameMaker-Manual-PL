szkielet_animacji_get_event_frames

Za pomocą tej funkcji można pobrać wszystkie klatki dla danego zdarzenia, w danej animacji. Podajemy nazwę szkieletu animacji (jako string, zdefiniowaną w programie, w którym wykonano animację, lub zwróconą przez funkcję skeleton_animation_get()) oraz nazwę zdarzenia. Funkcja zwraca tablicę z klatkami dla tego zdarzenia.

Jeśli podana nazwa zdarzenia nie istnieje, funkcja zwróci tablicę z -1 jako jej pierwszym (i jedynym) elementem.

Składnia:

skeleton_animation_get_event_frames(anim_name, event_name);

Argument Typ Opis
anim_name String The animation name to get the frames of.
event_name String The event name to get the frames for.

Zwroty:

Array of Reals

Przykład:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

Powyższy kod pobiera ramki dla zdarzenia "Footstep". Jeśli zwrócił tablicę z poprawnymi klatkami, to loops przez nią i sprawdza, czy bieżąca klatka jest równa którejś z klatek w tablicy. W takim przypadku odtwarza efekt dźwiękowy kroku i przerywa (zatrzymuje) loop.
