szkielet_animacji_set_frame

Ta funkcja ustawi animację przypisaną do danej ścieżki na podaną przez Ciebie klatkę. Indeks klatki powinien być wartością rzeczywistą z przedziału od 0 do liczby klatek animacji (którą można uzyskać za pomocą funkcji skeleton_animation_get_frames). Jeśli podasz wartość spoza tego zakresu, animacja zostanie zapętlona, aby nadrobić różnicę, na przykład jeśli animacja ma 5 klatek i ustawisz klatkę na 8, rzeczywista wyświetlona klatka będzie miała wartość 3.

Składnia:

skeleton_animation_set_frame(track, index);

Argument Typ Opis
track Real The animation track to set the frame of.
index Real The new index for the animation frame.

Zwroty:

N/A

Przykład:

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

Powyższy kod ustawi indeks klatki dla animacji w ścieżce 0 na ujemną prędkość obrazu, gdy globalna zmienna "pauza" jest true (zasadniczo wstrzymując animację).
