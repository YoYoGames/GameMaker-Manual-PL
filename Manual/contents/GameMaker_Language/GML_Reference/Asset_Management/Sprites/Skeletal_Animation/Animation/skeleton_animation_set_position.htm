szkielet_animacji_set_pozycja

Funkcja ustawia animację przypisaną do danej ścieżki na podaną przez użytkownika pozycję. Pozycja powinna być wartością rzeczywistą pomiędzy 0 a 1. Jeśli podasz wartość spoza tego zakresu, to pozycja ścieżki zostanie ustawiona na ułamkową część tej wartości. Załóżmy, że użyjesz wartości 0,4, wartości 1,4, wartości 2,4, ... Wszystkie one ustawią pozycję na 0,4.

Składnia:

skeleton_animation_set_position(track, position);

Argument Typ Opis
track Real The animation track to set the frame of.
position Real The new position for the animation frame.

Zwroty:

N/A

Przykład:

skeleton_animation_set_position(1, 0.25);

Powyższy kod ustawi pozycję dla ścieżki 1 animacji szkieletu na 0,25 lub, mówiąc inaczej, na 25% od początku animacji.
