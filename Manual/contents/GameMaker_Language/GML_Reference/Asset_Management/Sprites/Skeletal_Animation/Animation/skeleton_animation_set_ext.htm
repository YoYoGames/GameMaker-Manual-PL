szkielet_animacji_set_ext

Gdy ustawisz zestaw animacji dla animacji szkieletowej sprite, jest on wewnętrznie przypisany do ścieżki 0. Możesz jednak przypisać kolejne zestawy do dalszych ścieżek (od 1 wzwyż) i w ten sposób sprawić, by Twoja sprite odtwarzała dwie (lub więcej) animacje jednocześnie. Zauważ, że sprite powinien być zaprojektowany w twoim programie do animacji z tą myślą, że "bazowa" animacja w ścieżce 0 ma keyframes tylko dla niektórych kości, a następne ścieżki mają keyframes dla pozostałych. Na przykład, jeśli masz animację "chodzenia", a jej keyframes wpływa tylko na nogi sprite, możesz wtedy mieć zestaw animacji głowy i animacji ramion, które możesz ustawić odpowiednio w ścieżkach 1 i 2, pozwalając na posiadanie ruchu bazowego i dostosowywanie się do niego w czasie rzeczywistym podczas gry.
Animacja jest odtwarzana w pętli, co jest domyślne.

Składnia:

skeleton_animation_set_ext(animname, track, [loop]);

Argument Typ Opis
animname String The name (a string) of the animation set to use.
track Real The track for the animation set to use.
loop Boolean Czy animacja powinna loop czy nie (domyślnie jest true)

Zwroty:

N/A

Przykład:

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

Powyższy kod zmieni zestaw animacji używany przez ścieżkę 1 na zestaw "bodyfight" lub zestaw "bodywalk" po wciśnięciu lewego przycisku myszy.
