szkielet_animacji_get_position

Funkcja zwróci pozycję animacji na określonej ścieżce animacji jako wartość z zakresu od 0 do 1. W ten sposób można uzyskać pozycję ścieżki w procentach, gdzie wartość 0 odpowiada 0%, a wartość 1 100%.

Zwróci -1, jeśli do podanej ścieżki nie jest przypisana żadna animacja lub jeśli instancja nie ma ustawionej sprite.

Składnia:

skeleton_animation_get_position(track);

Argument Typ Opis
track Real The animation track to get the position of.

Zwroty:

Real

Przykład:

var _position = skeleton_animation_get_position(0);
var _position_percent = _position * 100;
show_debug_message("The skeleton animation position on track 0 is at " + string(_position_percent) + "%.");

Powyższy kod sprawdza aktualną pozycję animacji szkieletowej przypisanej do ścieżki 0 i zapisuje ją w zmiennej tymczasowej _position. Następnie konwertuje tę wartość na procent mnożąc ją przez 100 i zapisuje wynik tego działania w kolejnej zmiennej tymczasowej _position_percent.
Na koniec pokazuje komunikat debugowy, który wyświetla wartość procentową.
