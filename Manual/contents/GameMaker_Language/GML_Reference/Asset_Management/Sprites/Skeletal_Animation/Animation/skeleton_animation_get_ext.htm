szkielet_animacji_get_ext

Pojedyncza animacja szkieletowa sprite może mieć różne zestawy animacji, a zestawy te mogą być przypisane do różnych ścieżek, dzięki czemu można "mieszać i dopasowywać" zestawy animacji. Ta funkcja zwróci nazwę zestawu animacji aktualnie używanego przez podany numer ścieżki (ustalony za pomocą funkcji skeleton_animation_set_ext).

Składnia:

skeleton_animation_get_ext(track);

Argument Typ Opis
track Real The track number to get the animation set name of.

Zwroty:

String

Przykład:

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

Powyższy kod zmieni zestaw animacji używany przez ścieżkę 1 na "bodyfight", jeśli jeszcze nie jest.
