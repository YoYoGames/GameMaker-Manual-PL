szkielet_animacji_jest_looping

Za pomocą tej funkcji można sprawdzić, czy aktualna animacja na danej ścieżce jest ustawiona na loop.

Składnia:

skeleton_animation_is_looping(track);

Argument Typ Opis
ścieżka Real Indeks ścieżki animacji

Zwroty:

Boolean

Przykład:

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

Powyższy kod sprawdzi, czy animacja na ścieżce 0 jest ustawiona na loop. Jeśli tak jest, pokazuje komunikat debugowania.
