szkielet_animacji_set

Gdy już przypiszesz animację szkieletową sprite do swojej instancji, możesz za pomocą tej funkcji określić, którego zestawu anim acji ma ona używać w danym momencie. Kiedy tworzyłeś swoją sprite (w Spine), zdefiniowałeś wiele zestawów animacji dla różnych stanów i nazwałeś każdy z nich odpowiednio. Te nazwy są tym, czego użyjesz, aby powiedzieć GameMakerowi, który zestaw animacji ma być używany w danym momencie w Twojej grze. Na przykład, jeśli twój sprite jest dla gry platformowej, to możesz mieć zestaw "skok", zestaw "bieg" i zestaw "bezczynność" wszystkie zawarte w jednym sprite i możesz switch między nimi w swojej grze za pomocą tej funkcji.
Animacja jest odtwarzana w pętli, co jest domyślne.

Zauważ, że ustawienie nowej animacji w ten sposób spowoduje wyzerowanie wartości image_index na 0, co spowoduje rozpoczęcie nowej animacji od początku jej animacji loop. Jeśli nie jest to pożądane zachowanie, to możesz ustawić wartość "mix" za pomocą skeleton_animation_mix() i kiedy ustawisz nową animację, będzie ona płynnie interpolowana z poprzednim zestawem. Powinieneś również upewnić się, że masz keyframe na początku każdej animacji, aby zapobiec zatrzymaniu danych z poprzednich animacji, chyba że planujesz używać oddzielnych ścieżek animacji.

Składnia:

skeleton_animation_set(animname, [loop]);

Argument Typ Opis
animname String The name (a string) of the animation set to use.
loop Boolean Czy animacja powinna loop czy nie (domyślnie jest true)

Zwroty:

N/A

Przykład:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

Powyższy kod zmieni używany zestaw animacji na zestaw "jump" po naciśnięciu klawisza spacji, ale tylko wtedy, gdy aktualnie używany zestaw nie jest już "jump".
