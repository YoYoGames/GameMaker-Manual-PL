szkielet_animacji_get_frames

Za pomocą tej funkcji można uzyskać liczbę klatek, jakie ma dana animacja szkieletowa. Podajesz nazwę animacji szkieletowej (w postaci string, zdefiniowanej w programie użytym do stworzenia animacji lub zwróconej za pomocą funkcji skeleton_animation_get()), a funkcja zwraca liczbę klatek, które animacja posiada, w postaci liczby całkowitej. Jeśli podana animacja nie istnieje, funkcja zwróci 0.

Składnia:

skeleton_animation_get_frames(anim_name);

Argument Typ Opis
anim_name String The animation name to get the frames of.

Zwroty:

Real

Przykład:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

Powyższy kod pobierze liczbę klatek animacji, a następnie ustawi adres sprite na ostatnią klatkę i zatrzyma animację.
