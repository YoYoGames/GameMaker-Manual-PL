szkielet_animacji_get_duration

Funkcja zwraca czas potrzebny na uruchomienie podanego zestawu animacji przed powrotem do początku pętli. Wartość zwracana jest w sekundach.

Składnia:

skeleton_animation_get_duration(animname);

ArgumentTyp Opis
animname The name (a string) of the animation set to use.

Zwroty:

Przykład1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

Powyższy kod ustawi image_index na właściwą wartość dla aktualnie przypisanej animacji szkieletowej sprite.

Przykład2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Powyższy kod narysuje podaną animację szkieletową sprite, używając delta-time do ustawienia rysowanej klatki.
