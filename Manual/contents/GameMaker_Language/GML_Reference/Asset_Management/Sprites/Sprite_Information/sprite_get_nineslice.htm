sprite_get_nineslice

Funkcja zwraca strukturę Nine Slice dla strony sprite, zawierającą wszystkie właściwości Nine Slice ustawione w edytorze sprajtów lub za pomocą sprite_set_nineslice(). Zawartość tej strony struct jest szczegółowo opisana na tej stronie.

Jeśli do dostarczonej strony sprite nie jest przypisana strona Nine Slice struct, dla strony struct zostanie utworzona i zwrócona nowa strona sprite z domyślnymi właściwościami Nine Slice.

Zmiana jakichkolwiek wartości w tej witrynie struct spowoduje modyfikację właściwości plastra dziewięciokrotnego oryginalnej witryny sprite, wpływając na wszystkie przyszłe wywołania losowania wykonywane za pomocą tej witryny sprite.

Składnia:

sprite_get_nineslice(ind);

ArgumentTyp Opis
ind The index of the sprite from which the Nine Slice struct will be retrieved

Zwroty:

Nine Slice Struct (or -1 if the sprite doesn't exist)

Przykład:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

Powyższy kod pobiera stronę struct z witryny sprite, włącza dla niej funkcję plasterków dziewięciokrotnych i ustawia przesunięcia prowadnicy.
