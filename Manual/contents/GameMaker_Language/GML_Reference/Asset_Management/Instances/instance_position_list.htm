lista_położenia_instancji

Za pomocą tej funkcji można sprawdzić, czy pozycja nie koliduje ze wszystkimi instancjami innej strony object. Używając tej funkcji, sprawdzasz pojedynczy punkt w witrynie room pod kątem wszystkich instancji witryny object, a sprawdzenie zostanie wykonane względem ramki ograniczającej instancji lub względem maski instancji, jeśli witryna object ma zaznaczone dokładne kolizje.

Podajesz także listę DS, więc wartości id wszystkich instancji, które kolidują z instancją wywołującą, zostaną dodane na koniec podanej listy. Możesz wyczyścić listę przed wywołaniem tej funkcji, aby zawierała ona tylko wyniki z tego wywołania. Możesz także uporządkować listę według odległości sprawdzanej pozycji od początku instancji, które znalazły się w kolizji.

Zwróć uwagę, że funkcja akceptuje również specjalne słowo kluczowe allw którym to przypadku zostaną wypisane wszystkie instancje uznane za kolizyjne. Funkcja zwraca liczbę znalezionych instancji lub 0, jeśli nie znaleziono żadnej.

Składnia:

instance_position_list(x, y, obj, list, ordered);

Argument Typ Opis
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Identyfikatorzasobu lub instancjiobiektu The object to check for instances of.
list Identyfikator listy DS The DS list to use to store the IDs of colliding instances.
ordered Boolean Whether the list should be ordered by distance (true) lub nie (false).

Zwroty:

Real (The number of instances found to be in collision)

Przykład:

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

Powyższy kod sprawdzi kolizję ze wszystkimi instancjami znalezionymi w miejscu, w którym znajduje się mysz. Zostaną one dodane do listy DS, która następnie zostanie zapętlona w celu zniszczenia każdej z kolidujących instancji.
