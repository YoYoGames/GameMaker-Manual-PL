instancja_aktywuj_wszystkie

Za pomocą tej funkcji możesz powiedzieć GameMakerowi, by aktywował wszystkie instancje, które zostały wcześniej wyłączone w room. Zauważ, że aktywacja nie jest natychmiastowa, a instancja, która została aktywowana w ten sposób, nie będzie uważana za aktywną do końca zdarzenia, w którym wywołano funkcję. za aktywną do końca zdarzenia, w którym wywołano tę funkcję.

Składnia:

instance_activate_all();

Zwroty:

Przykład:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

Powyższy kod aktywuje wszystkie instancje w obrębie strony room, a następnie dezaktywuje te, które znajdują się 64px poza granicami bieżącego widoku z kamery.
