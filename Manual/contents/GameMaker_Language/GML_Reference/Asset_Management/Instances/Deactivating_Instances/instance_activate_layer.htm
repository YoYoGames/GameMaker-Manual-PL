aktywuj_instancję_warstwę

Za pomocą tej funkcji można aktywować warstwę, która została wcześniej wyłączona. Należy podać identyfikator warstwy, który może być albo nazwą warstwy zapisaną w edytorze kodu (jako string), albo rzeczywistą wartością identyfikatora warstwy zwróconą przez funkcję layer_create() a wszystkie dezaktywowane instancje na tej warstwie zostaną ponownie aktywowane. Zauważ, że jeśli dezaktywowałeś warstwę, która ma trwałe instancje, musisz ją ponownie aktywować za pomocą tej funkcji przed zmianą adresu room, w przeciwnym razie trwałe instancje nie zostaną przeniesione i zostaną odrzucone. Należy również pamiętać, że aktywacja nie jest natychmiastowa, a instancja, która została aktywowana w ten sposób, nie będzie uważana za aktywną do końca zdarzenia, w którym wywołano tę funkcję.

Składnia:

instance_activate_layer(layer_id);

Argument Typ Opis
layer_id Identyfikator warstwy lub ciągznaków The layer name string (or ID value) to be used

Zwroty:

N/A

Przykład:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

Powyższy kod aktywuje wszystkie instancje w obrębie strony room, a następnie dezaktywuje te, które znajdują się poza granicami bieżącego widoku z kamery, z wyjątkiem instancji na warstwie "Player_Layer", które są ponownie aktywowane na końcu.
