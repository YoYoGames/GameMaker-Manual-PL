instancja_liczba

Za pomocą tej zmiennej tylko do odczytu możesz uzyskać liczbę wszystkich aktywnych instancji znajdujących się na stronie room. Obejmuje ona instancję wykonującą kod, ale nie obejmuje tych instancji, które zostały wyłączone za pomocą funkcji dezaktywacji instancji. Zwróć uwagę, że zmienna ta podaje tylko liczbę instancji w momencie rozpoczęcia kroku, więc wszelkie zmiany w instancjach na stronie room dokonane po rozpoczęciu kroku nie będą brane pod uwagę.

Składnia:

instance_count;

Zwroty:

Przykład:

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

Powyższy kod utworzy wiele instancji object "obj_Star", aż całkowita liczba instancji osiągnie 100.
