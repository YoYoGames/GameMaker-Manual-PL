instancja_najbliższa

Ta funkcja sprawdzi wszystkie instancje podanej strony object, aby zobaczyć, która z nich znajduje się najbliżej podanego punktu x/y. Wszystkie sprawdzenia będą wykonywane od podanej pozycji x/y do początku (pozycji x/y) instancji podanej strony object. Jeśli nie istnieją instancje object, funkcja zwróci słowo kluczowe nooneale jeśli istnieją instancje, to zwróci wartość id znalezionej instancji. Zwróć uwagę, że jeśli instancja wykonująca kod ma ten sam indeks object co sprawdzana object, to zostanie ona uwzględniona w sprawdzeniu (dotyczy to także sprawdzenia instancji nadrzędnej objects, jeśli instancja wywołująca jest także dzieckiem instancji nadrzędnej).

Składnia:

instance_nearest(x, y, obj);

ArgumentTyp Opis
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

Zwroty:

Instance ID or noone

Przykład:

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

Powyższy kod przesuwa bieżącą instancję o 10000 pikseli, a następnie sprawdza jej poprzednią pozycję, aby znaleźć najbliższą instancję tego samego typu object. Jeśli jest to ta sama instancja, to nie zrobi nic więcej niż cofnie się do pierwotnej pozycji, ale jeśli znaleziona instancja jest inna, to narysuje linię między nimi.
