instancja_place

Za pomocą tej funkcji możesz sprawdzić, czy pozycja nie koliduje z inną instancją lub wszystkimi instancjami obiektu object, używając maski kolizji instancji, która uruchamia kod sprawdzający. Używając tej funkcji, prosisz program GameMaker, aby przesunął instancję na nową pozycję, sprawdził, czy nie doszło do kolizji, cofnął się i poinformował, czy kolizja została znaleziona, czy nie.

Działa to w przypadku kolizji precyzyjnych, ale tylko wtedy, gdy zarówno instancja, jak i sprawdzana witryna object mają zaznaczone maski kolizji precyzyjnych, w przeciwnym razie stosowane są tylko kolizje typu bounding box.

Ta funkcja zwróci unikalną instancję id kolidującej strony object, ale jeśli nie jest to potrzebne, nieco szybciej jest użyć funkcji place_meeting(). Funkcja ta akceptuje również specjalne słowa kluczowe all i other i zwraca słowo kluczowe noone, jeśli nie wystąpi kolizja, lub unikatową wartość identyfikatora znalezionej instancji, jeśli kolizja wystąpi.

Składnia:

instance_place(x, y, obj);

Argument Typ Opis
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Identyfikatorzasobu lub instancjiobiektu The object to check for instances of.

Zwroty:

Instance ID or noone

Przykład:

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
{
    hp -= _inst.dmg;
    instance_destroy(_inst);
}

Powyższy kod sprawdzi, czy nie doszło do kolizji z instancjami "obj_Enemy" i jeśli tak się stanie, zmniejszy zmienną "hp" o wartość zapisaną w zmiennej "dmg" kolidującej instancji, a następnie zniszczy kolidującą instancję.
