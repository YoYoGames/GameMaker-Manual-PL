poprzednia strona

Ta wbudowana zmienna zwraca poprzednią pozycję x dla instancji. Zmienna ta zostanie ustawiona tuż przed rozpoczęciem zdarzenia begin step, ale można ją również ustawić w dowolnym momencie za pomocą kodu, co oznacza, że można nadać jej własną wartość, jeśli zajdzie taka potrzeba.

Składnia:

xprevious;

Zwroty:

(single precision floating point value)

Przykład:

if x != xprevious || y != yprevious
{
    moved = true;
}

Powyższy kod sprawdza, czy istnieje różnica między wartościami xprevious i yprevious a bieżącymi wartościami x i y. Jeśli tak, ustawia zmienną na wartość true. Jeśli tak, ustawia zmienną na wartość true.
