hspeed to jedna z "wbudowanych" właściwości, które mają wszystkie instancje. Określa ona prędkość poziomego ruchu (wzdłuż osi x) instancji w pikselach na krok. Tak więc hspeed równe 3 oznacza ruch o 3 piksele w prawo (+x) co krok, a hspeed o wartości -3 oznacza ruch o 3 piksele w lewo (-x) co krok.

Należy pamiętać, że po ustawieniu zmiennych speed i/lub direction wartość hspeed zostanie automatycznie zaktualizowana, aby Podobnie zmiana wartości hspeed będzie miała wpływ na wartości speed i direction. Należy również pamiętać, że gravity, gravity_direction oraz friction mogą modyfikować wartość tej zmiennej, gdy są używane w grze.

Składnia:

hspeed;

Zwroty:

(single precision floating point value)

Przykład:

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

Powyższy kod będzie zmieniał prędkość poziomą w zależności od tego, które klawisze zostaną naciśnięte.
