kierunek_grawitacji

gravity_direction jest jedną z "wbudowanych" właściwości, które posiadają wszystkie instancje i może być użyta do ustawienia kierunku ruchu, gdy instancja gravity jest większa od 0. Zauważ, że kierunki w GameMakerze są zwykle obliczane jako 0° w prawo, 90° w górę, 180° w lewo i 270° w dół.

Składnia:

gravity_direction;

Zwroty:

(single precision floating point value)

Przykład:

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

Powyższy kod zmieni kierunek grawitacji, jeśli gracz object napotka instancję "obj_switch".
