ystart

Ta zmienna przechowuje początkową pozycję y instancji podczas jej pierwszego utworzenia w witrynie room. Nie jest to zmienna tylko do odczytu i można ją ustawiać, jak również odczytywać.

Składnia:

ystart;

Zwroty:

(single precision floating point value)

Przykład:

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

Powyższy kod sprawdzi, czy doszło do kolizji z instancją "obj_spike", a jeśli tak, odejmie 100 od wyniku i przesunie instancję z powrotem na jej pozycję startową.

Powrót: Zmienne instancji
Następny: x
