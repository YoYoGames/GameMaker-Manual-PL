pokój

Ta zmienna przechowuje indeks room dla bieżącej strony room uruchomionej w grze. Nie jest to zmienna tylko do odczytu, jednak jej zmiana nie spowoduje zmiany wartości indeksu dla bieżącej strony room, ale raczej zmianę adresu room, aby odpowiadał indeksowi, który ustawiłeś w zmiennej. Należy przy tym zachować ostrożność, ponieważ jeśli indeks, na który zmieniono zmienną, nie jest prawidłowy, gra wyrzuci błąd i zostanie zamknięta. Ogólnie rzecz biorąc, znacznie lepszym rozwiązaniem jest użycie room_goto() do zmiany adresu rooms.

Uwaga: Identyfikatory Room nie są oparte na ich kolejności w Przeglądarce Asset lub Menedżerze Room, dlatego należy unikać bezpośredniego podawania wartości liczbowej. Zamiast tego należy użyć stałej pokojowej dla strony asset, do której chcemy się odwołać (jest to po prostu jej nazwa) lub pobrać ją za pomocą funkcji.

Składnia:

room;

Zwroty:

Room Asset

Przykład:

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

Powyższy kod sprawdzi, czy bieżący adres room jest pierwszym poziomem, i w takim przypadku odtworzy muzykę z tego poziomu.
