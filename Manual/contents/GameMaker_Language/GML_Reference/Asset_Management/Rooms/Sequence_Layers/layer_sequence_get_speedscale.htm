sekwencja_warstw_get_speedscale

Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementu warstwy - a funkcja zwróci bieżącą skalę prędkości odtwarzania. Jest to wartość mnożnika używanego do zmniejszania lub zwiększania szybkości odtwarzania. Wartość 1 jest wartością domyślną, a wartości mniejsze od 1 oznaczają, że odtwarzanie jest spowolnione, a większe od 1 - że jest przyspieszone.

Składnia:

layer_sequence_get_speedscale(sequence_element_id)

ArgumentTyp Opis
sequence_element_id The unique ID value of the sequence element to target

Zwroty:

Przykład:

if layer_sequence_get_speedscale(title_sequence) != 1
{
    layer_sequence_speedscale(title_sequence, 1);
}

Powyższy kod sprawdza bieżącą skalę prędkości odtwarzania identyfikatora elementu sequence przechowywanego w zmiennej "title_sequence" i jeśli nie jest ona ustawiona na 1, ustawia ją na tę wartość.
