sekwencja_warstw_zniszcz

Za pomocą tej funkcji można zniszczyć (usunąć z room) element sequence. Należy podać identyfikator elementu sequence zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementów warstwy, a element sequence zostanie zniszczony.

Składnia:

layer_sequence_destroy(sequence_element_id)

ArgumentTyp Opis
sequence_element_id The unique ID value of the sequence element to target

Zwroty:

Przykład:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_destroy(a[i])
    }
}

Powyższy kod pobiera identyfikatory wszystkich elementów przypisanych do warstwy instancji uruchamiającej kod. Następnie kod sprawdza, czy wśród zwróconych elementów znajduje się sequence assets i jeśli tak, to zostają one zniszczone.
