layer_sequence_get_headdir

Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementu warstwy - i zwróci ona bieżący kierunek głowicy odtwarzania, który będzie jedną z poniższych stałych.

Składnia:

layer_sequence_get_headdir(sequence_element_id)

ArgumentTyp Opis
sequence_element_id The unique ID value of the sequence element to target

Zwroty:

Stała Opis Wartość
seq_dir_right Strona sequence odtwarza klatki w kolejności od lewej do prawej 1
seq_dir_left Strona sequence odtwarza ramki w kolejności malejącej od prawej do lewej -1

Przykład:

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

Powyższy kod sprawdza bieżący kierunek odtwarzania identyfikatora elementu sequence zapisanego w zmiennej "title_sequence" i jeśli nie jest on ustawiony na seq_dir_left, ustawia go na tę wartość.
