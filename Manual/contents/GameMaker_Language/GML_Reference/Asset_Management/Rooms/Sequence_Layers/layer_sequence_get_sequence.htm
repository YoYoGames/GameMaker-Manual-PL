sekwencja_warstw_get_sequence

Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementów warstwy - a funkcja zwróci strukturę obiektu sequence. Funkcja ta omija konieczność uprzedniego uzyskania instancji sequence struct i umożliwia bezpośredni dostęp do danych sequence. Więcej informacji na temat formatu pliku sequence object struct można znaleźć na tej stronie.

Składnia:

layer_sequence_get_sequence(sequence_element_id)

ArgumentTyp Opis
sequence_element_id The unique ID value of the sequence element to target

Zwroty:

Sequence Object Struct

Przykład:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

Powyższy kod tworzy nowy element sequence, a następnie pobiera jego dane sequence struct . Te dane są następnie wykorzystywane do zmiany szybkości odtwarzania i typu elementu sequence.
