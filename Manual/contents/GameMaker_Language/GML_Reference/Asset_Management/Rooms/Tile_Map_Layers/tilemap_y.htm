tilemap_y

Ta funkcja steruje położeniem wzdłuż osi y elementu mapy room elementu mapy asset tile na warstwie. Podajesz identyfikator elementu mapy tile (który otrzymujesz podczas tworzenia elementu mapy tile za pomocą funkcji layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a następnie ustawiasz wartość y, której chcesz użyć (na podstawie współrzędnych room ).

Składnia:

tilemap_y(tilemap_element_id, y);

Argument Typ Opis
tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to change
y Real The y position for the tile map

Zwroty:

N/A

Przykład:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

Powyższy kod pobiera wartość ID mapy tile asset przypisanej do warstwy "Asset_sky", a następnie ustawia jej pozycję y na losową wartość z przedziału od 0 do wysokości mapy room.

Wstecz: Warstwy mapy kafelkowej
Następny: tilemap_set
