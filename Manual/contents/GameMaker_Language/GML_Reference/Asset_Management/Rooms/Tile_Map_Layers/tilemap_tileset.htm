tilemap_tileset

Za pomocą tej funkcji można zmienić adres tile set asset przypisany do danego elementu mapy tile na warstwie. Podajesz identyfikator elementu mapy tile (który otrzymujesz podczas tworzenia elementu mapy tile za pomocą funkcji layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a następnie podajesz indeks tile set i mapa tile otrzyma nowy adres sprite.

Składnia:

tilemap_tileset(tilemap_element_id, tileset_index)

Argument Typ Opis
tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to change
tileset_index Zestaw płytek Aktywa The new tile set index to use

Zwroty:

N/A

Przykład:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(tile_id) != ts_Nighttime
{
    tilemap_tileset(tile_id, ts_Nighttime);
}

Powyższy kod sprawdza aktualny adres tile set przypisany do mapy tile na warstwie "Tiles_trees" i jeśli nie jest to "ts_Nighttime", to ten adres tile set jest przypisywany do mapy tile.

Wstecz: Warstwy mapy kafelkowej
Następny: tilemap_clear
