tilemap_get_global_mask

Funkcja ta służy do pobierania wartości masek bitowych dla wszystkich map tile. Zwraca bieżącą wartość maski lub -1, jeśli wystąpił błąd, lub 0, jeśli nie podano maski. Więcej informacji o maskach bitowych globalnych map tile można znaleźć w funkcji tilemap_set_global_mask().

Składnia:

tilemap_get_global_mask();

Zwroty:

(0 for no mask, -1 for an error, mask value otherwise)

Przykład:

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_global_mask(new_mask);
}

Powyższy kod pobiera globalną wartość maski powiązaną ze wszystkimi mapami tile. Jeśli nie jest ona taka sama jak wartość zdefiniowana w zmiennej "new_mask", to jest ona ustawiana na tę wartość.

Wstecz: Warstwy mapy kafelkowej
