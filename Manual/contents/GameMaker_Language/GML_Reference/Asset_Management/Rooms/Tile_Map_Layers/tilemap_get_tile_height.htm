tilemap_get_tile_height

Za pomocą tej funkcji możesz pobrać wysokość (w pikselach) każdej komórki tile elementu mapy tile. Podajesz identyfikator elementu mapy tile (który otrzymujesz podczas tworzenia elementu mapy tile za pomocą layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a funkcja zwróci wysokość komórki tile set.

Składnia:

tilemap_get_tile_height(tilemap_element_id);

ArgumentTyp Opis
tilemap_element_id The unique ID value of the tile map element to get the tile cell height of

Zwroty:

Przykład:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

Powyższy kod wykorzystuje pobrany identyfikator mapy tile, aby uzyskać szerokość i wysokość komórki tile i użyć ich do ustawienia dwóch zmiennych globalnych.

Wstecz: Warstwy mapy kafelkowej
