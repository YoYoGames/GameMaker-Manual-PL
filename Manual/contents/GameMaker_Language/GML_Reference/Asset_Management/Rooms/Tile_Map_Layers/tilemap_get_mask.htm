tilemap_get_mask

Za pomocą tej funkcji można pobrać wartość maski bitowej dla podanej mapy tile. Podaje się identyfikator elementu mapy tile (który otrzymuje się podczas tworzenia elementu mapy tile przy użyciu layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a funkcja zwróci bieżącą wartość maski lub -1, jeśli wystąpił błąd, lub 0, jeśli nie podano maski. Więcej informacji o maskach bitów mapy tile można znaleźć w funkcji tilemap_set_mask().

Składnia:

tilemap_get_mask(tilemap_element_id);

Argument Typ Opis
tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to get the mask of

Zwroty:

Real (0 for no mask, -1 for an error)

Przykład:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_mask(map_id, new_mask);
}

Powyższy kod pobiera identyfikator mapy tile z podanej warstwy, a następnie sprawdza powiązaną z nią wartość maski. Jeśli nie jest ona taka sama jak wartość zdefiniowana w zmiennej "new_mask", to jest ona ustawiana na tę wartość.

Wstecz: Warstwy mapy kafelkowej
