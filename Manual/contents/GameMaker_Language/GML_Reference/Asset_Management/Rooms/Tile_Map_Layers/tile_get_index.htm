tile_get_index

Ta funkcja służy do uzyskiwania indeksu tile (pozycja strony tile w obrazie tile set ) z zestawu danych kafelków. Użytkownik określa dane kafelków, które można pobrać za pomocą funkcji tilemap_get()a funkcja zwróci wartość całkowitą indeksu.

Podczas używania tej funkcji należy upewnić się, że podane dane kafelków są poprawne, w przeciwnym razie zwrócona wartość indeksu również nie będzie poprawna, co może spowodować nieoczekiwane zachowanie.

Składnia:

tile_get_index(tiledata)

Argument Typ Opis
tiledata Dane o płytkach the tile-data to check

Zwroty:

Real

Przykład:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

Powyższy kod pobiera identyfikator mapy tile z podanej warstwy, a następnie wykorzystuje go do uzyskania danych o kafelkach dla komórki znajdującej się pod pozycją myszy. Dane te są następnie używane do ustawienia indeksu tile na losową liczbę.

Wstecz: Warstwy mapy kafelkowej
