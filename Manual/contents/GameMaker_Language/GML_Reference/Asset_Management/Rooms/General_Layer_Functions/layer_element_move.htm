warstwa_elementu_przesuń

Tej funkcji można użyć do przeniesienia elementu z jednej warstwy na drugą. Podajesz identyfikator elementu, zwrócony przez funkcję użytą do utworzenia elementu lub edytor room lub funkcję layer_get_all_elements()a następnie podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą funkcji layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność).

Składnia:

layer_element_move(element_id, layer_id)

Argument Typ Opis
element_id Identyfikator elementu warstwy The unique ID value of the element to move
layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to move the element to (or the layer name as a string)

Zwroty:

N/A

Przykład:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_element_move(a[i], asset_layer)
    }
}

Powyższy kod pobiera elementy znajdujące się na warstwie, do której przypisana jest instancja wykonująca kod, następnie sprawdza, czy są one sprite assets, a jeśli tak, to są one przenoszone na warstwę o identyfikatorze zapisanym w zmiennej "asset_layer".
