warstwa_get_all

Funkcja ta zwróci tablicę zawierającą unikalne wartości ID każdej warstwy w witrynie room.

Składnia:

layer_get_all()

Zwroty:

Array (populated with Layer IDs)

Przykład:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
    layer_destroy(a[i]);
}

Powyższy kod pobiera wszystkie warstwy ze strony room i dodaje ich wartości ID do tablicy. Tablica ta jest następnie przetwarzana w celu zniszczenia warstw room.
