layer_is_draw_depth_forced

Za pomocą tej funkcji można sprawdzić, czy dla renderowania warstw w projekcie jest włączona wymuszona głębia Z. Zobacz layer_force_draw_depth() aby uzyskać więcej informacji.

Składnia:

layer_is_draw_depth_forced()

Zwroty:

Przykład:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

Powyższy kod sprawdza, czy głębokość warstwy Z jest wymuszona, a jeśli nie jest, ustawia głębokość Z na 0 i włącza ją.

Powrót: Ogólne funkcje warstwy
