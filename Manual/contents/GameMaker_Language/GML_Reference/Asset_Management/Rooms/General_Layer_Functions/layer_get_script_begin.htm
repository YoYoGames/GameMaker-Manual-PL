warstwa_get_script_begin

Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą polecenia layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność), a funkcja zwróci indeks funkcji skryptu przypisanej do uruchamiania na początku renderowania dla tej warstwy lub zwróci -1, jeśli nie przypisano żadnej funkcji. Funkcje script można przypisać do warstwy za pomocą layer_script_begin() i layer_script_end().

Składnia:

layer_get_script_begin(layer_id);

Argument Typ Opis
layer_id String lub Identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string)

Zwroty:

Script Function or -1 (if no function is assigned)

Przykład:

if layer_get_script_begin(layer) == -1
{
    layer_script_begin(layer, scr_SetShaderValues);
}

Powyższy kod sprawdzi, czy warstwa, na której znajduje się instancja uruchamiająca kod, ma przypisaną funkcję script, a jeśli nie ma, to zostanie ona przypisana.
