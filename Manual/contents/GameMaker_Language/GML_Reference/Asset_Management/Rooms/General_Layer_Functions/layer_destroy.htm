warstwa_zniszczenie

Ta funkcja zniszczy podaną warstwę. Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność), a funkcja usunie ją z bieżącej strony room. Jeśli warstwa została zaprojektowana w edytorze room, to przy następnym opuszczeniu strony room i powrocie warstwa zostanie odtworzona z oryginalną zawartością. Jeśli jednak strona room jest trwała, warstwa zostanie usunięta, chyba że ponownie wyłączysz trwałość strony room. Gdy zniszczysz warstwę w ten sposób, cała jej zawartość również zostanie usunięta, więc wszelkie identyfikatory referencyjne dla teł, map tile itp. nie będą już ważne, a wszystkie instancje przypisane do warstwy zostaną zniszczone (wykonując jednocześnie zdarzenie Zniszcz, jeśli je mają, oraz zdarzenie Wyczyść).

Składnia:

layer_destroy(layer_id)

Argument Typ Opis
layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to destroy (or the layer name as a string)

Zwroty:

N/A

Przykład:

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Powyższy kod sprawdzi, czy istnieje jakakolwiek instancja warstwy object "obj_Bullet_Parent", a jeśli nie, to zniszczy warstwę o identyfikatorze zapisanym w zmiennej globalnej.
