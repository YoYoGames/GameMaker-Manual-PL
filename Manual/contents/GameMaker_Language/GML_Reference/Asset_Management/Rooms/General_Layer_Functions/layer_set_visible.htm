warstwa_set_widoczna

Za pomocą tej funkcji można przełączać widoczność warstwy. Podajemy identyfikator warstwy (który otrzymujemy podczas tworzenia warstwy za pomocą polecenia layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz wartość przełączania dla warstwy, gdzie widoczna to true, a niewidoczna to false.

Gdy warstwa jest niewidoczna, nic, co jest do niej przypisane, nie będzie rysowane, a jeśli do warstwy są przypisane jakieś instancje, to nie będą one nawet uruchamiały swoich zdarzeń rysowania (podobnie jak w przypadku ustawienia zmiennej instancji visible na false). Zauważ, że instancje, które są już oznaczone jako niewidzialne, nie będą oznaczone jako widoczne, jeśli warstwa, na której się znajdują, jest ustawiona jako widoczna, ponieważ są to dwa niezależne ustawienia, choć ich zachowanie jest takie samo.

Składnia:

layer_set_visible(layer_id, visible)

Argument Typ Opis
layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string)
visible Boolean Whether the layer should be visible (true) lub nie (false)

Zwroty:

N/A

Przykład:

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

Powyższy kod pobiera wartość ID dla warstwy o nazwie "Instancje" w edytorze room, a następnie używa tego ID do sprawdzenia, czy warstwa jest widoczna, czy nie, zmieniając jej widoczność w zależności od zwróconej wartości.
