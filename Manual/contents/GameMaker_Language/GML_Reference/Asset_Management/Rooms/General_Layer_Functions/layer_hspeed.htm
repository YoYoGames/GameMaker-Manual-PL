layer_hspeed

Możesz użyć tej funkcji, aby ustawić poziomą prędkość (w pikselach na klatkę gry) warstwy w obrębie bieżącego zakresu room. Podajesz ID warstwy (które otrzymujesz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz wartość prędkości do ustawienia, gdzie wartość dodatnia oznacza przesunięcie w prawo, a wartość ujemna w lewo.

Składnia:

layer_hspeed(layer_id, hspd)

Argument Typ Opis
layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to set the horizontal speed of
hspd Real The horizontal speed (in pixels per game frame) to set

Zwroty:

N/A

Przykład:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

Powyższy kod sprawdza prędkość poziomą i pionową podanej warstwy i jeśli obie nie są ustawione na 0, to ustawia je na 0.
