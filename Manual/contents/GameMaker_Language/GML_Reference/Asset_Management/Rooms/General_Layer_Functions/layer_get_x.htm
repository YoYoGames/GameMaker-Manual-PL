warstwa_get_x

Możesz użyć tej funkcji, aby pobrać pozycję x warstwy w obrębie bieżącego zakresu room. Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą polecenia layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność), a funkcja zwraca liczbę rzeczywistą określającą pozycję x warstwy względem pozycji (0,0) na stronie room. Domyślnie jest to 0.

Składnia:

layer_get_x(layer_id)

Argument Typ Opis
layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to get the x position of

Zwroty:

Real

Przykład:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

Powyższy kod sprawdza podaną pozycję warstwy i jeśli nie jest ona ustawiona na (0, 0), to jest ona ustawiana na tę pozycję.
