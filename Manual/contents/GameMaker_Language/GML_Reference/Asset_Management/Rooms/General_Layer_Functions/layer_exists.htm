warstwa_istnieje

Za pomocą tej funkcji można sprawdzić, czy podana warstwa istnieje. Podajemy identyfikator warstwy (który otrzymujemy podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (w postaci string - będzie to miało wpływ na wydajność), a funkcja zwróci wartość logiczną true, jeśli warstwa istnieje, lub false, jeśli nie.

UWAGA: Ta funkcja działa w zakresie bieżącego celu room - domyślnie jest to room, w którym funkcja jest wywoływana - który można ustawić za pomocą funkcji layer_set_target_room().

Składnia:

layer_exists(layer_name)

Argument Typ Opis
layer_name Ciąg lub identyfikator warstwy The name of the layer (a string or ID value)

Zwroty:

Boolean

Przykład:

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

Powyższy kod sprawdzi, czy warstwa przechowywana w zmiennej globalnej rzeczywiście istnieje, a jeśli nie, to zostanie utworzona.
