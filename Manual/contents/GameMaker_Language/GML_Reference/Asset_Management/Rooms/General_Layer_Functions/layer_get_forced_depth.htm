warstwa_uzyskaj_wymuszoną_głębokość

Za pomocą tej funkcji można pobrać głębię Z ustawioną do renderowania warstw w witrynie room. Zobacz layer_force_draw_depth() aby uzyskać więcej informacji.

Składnia:

layer_get_forced_depth()

Zwroty:

Real

Przykład:

if layer_get_forced_depth() != 0
{
    layer_force_draw_depth(true, 0);
}

Powyższy kod sprawdza, czy głębokość warstwy Z jest wymuszona na wartość 0 czy nie, a jeśli nie, ustawia głębokość warstwy Z na 0.

Powrót: Ogólne funkcje warstwy
Następny: layer_script_begin
