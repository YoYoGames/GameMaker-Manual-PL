fx_get_parameters

Ta funkcja służy do pobierania struktury zawierającej wszystkie parametry dla danego FX struct. Użytkownik określa FX struct (zwrócony przez fx_create() lub layer_get_fx()), a ta funkcja zwraca struct zawierający zmienne dla wszystkich parametrów mających zastosowanie do tego filtra/efektu. Parametry, które używają wektorów (tzn. grup wartości), będą przedstawione w postaci tablic.

Nazwy wszystkich parametrów można uzyskać w osobnej tablicy, wywołując polecenie fx_get_parameter_names(), lub znaleźć je na tej stronie: FX Types & Parameters.

Zmiana wartości na stronie struct nie spowoduje zastosowania tych zmian w filtrze/efektcie; aby zmiany zaczęły obowiązywać, należy wywołać stronę fx_set_parameters() w celu zastosowania zmodyfikowanej strony struct z powrotem do filtra/efektu.

Składnia:

fx_get_parameters(filter_or_effect);

Argument Typ Opis
filter_or_effect Struktura walutowa The FX struct for which the parameters will be returned

Zwroty:

Struct (or -1 on failure)

Przykład:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Powyższy kod pobiera adres FX struct dla warstwy, która z założenia ma zastosowany filtr "Zabarwienie", i pobiera jego parametr struct, wywołując adres fx_get_parameters(). Następnie za pomocą stron sin() i current_time tworzy wartość oscylującą, która jest używana do tworzenia wartości RGBA dla efektu zabarwienia, dzięki czemu efekt jest animowany. Tablica RGBA jest przypisywana do zmiennej g_TintCol w parametrze struct, a następnie struct jest z powrotem stosowana do FX struct przez wywołanie fx_set_parameters().
