warstwa_fx_jest_wyłączona

Ta funkcja zwraca, czy filtr/efekt dla warstwy jest włączony.

Funkcja zwraca wartość logiczną wskazującą, czy FX warstwy jest włączony(prawda) czy wyłączony(fałsz).

Składnia:

layer_fx_is_enabled(layer_name_or_id);

Argument Typ Opis
layer_name_or_id String lub Identyfikator warstwy The name or ID of the layer to check

Zwroty:

Boolean

Przykład:

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

Powyższy kod "odradza" instancję, przesuwając ją na pierwotną pozycję i uzupełniając jej HP. Następnie sprawdza, czy warstwa Desaturate FX jest włączona, a jeśli tak, to ją wyłącza.
