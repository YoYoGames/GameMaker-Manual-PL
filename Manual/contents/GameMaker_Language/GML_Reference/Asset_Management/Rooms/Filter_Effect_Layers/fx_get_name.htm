fx_get_name

Ta funkcja służy do pobierania nazwy filtra/efektu z jego strony struct. Użytkownik określa FX struct do odczytania (zwrócony przez fx_create() lub layer_get_fx()), a funkcja zwraca jego nazwę jako string. Nazwę tę można następnie przekazać do fx_create() w celu utworzenia nowego FX struct z użyciem tego samego typu filtra/efektu.

Składnia:

fx_get_name(filter_or_effect);

ArgumentTyp Opis
filter_or_effect The FX struct to get the name of

Zwroty:

Przykład:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Powyższy kod uruchamia pętlę for pętlę przez wszystkie warstwy znajdujące się na stronie room i sprawdza, czy każda z nich ma filtr FX struct. Jeśli warstwa ma FX struct, sprawdza nazwę tego filtra/efektu, wywołując na niej fx_get_name(). Jeśli jest ona równa "_filter_tintfilter", co oznacza, że jest to filtr "Barwa", zmienia kolor barwy na niebieski.
