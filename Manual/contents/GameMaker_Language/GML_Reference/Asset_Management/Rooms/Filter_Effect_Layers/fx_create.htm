fx_create

Ta funkcja służy do tworzenia nowej strony FX Struct dla danego typu efektu, który można następnie zastosować do warstwy.

Na tej stronie podano nazwy filtrów/efektów, które można włączyć do tej funkcji: Wszystkie typy filtrów/efektów.

Zastosowanie

Parametry utworzonego efektu można odczytać, wywołując polecenie fx_get_parameter/s na zwróconym adresie struct i przypisać im nowe wartości, wywołując fx_set_parameter/s.

Należy pamiętać, że strona struct zwrócona przez tę funkcję nie będzie zawierała żadnych widocznych elementów i będzie można ją zmodyfikować tylko za pomocą jednej z funkcji Filtr/Efekt.

Awarie funkcji

Funkcja zwróci -1, jeśli z jakiegoś powodu nie uda się utworzyć strony struct; może się tak zdarzyć, jeśli podana nazwa FX jest nieprawidłowa lub FX nie został załadowany do pakietu gry.

Aby zapewnić prawidłowe działanie tej funkcji, należy upewnić się, że w pakiecie z grą zostały załadowane potrzebne efekty FX, jak wyjaśniono na stronie Funkcje filtrów i efektów:

Jeśli te warunki nie zostaną spełnione, tzn. nie zostanie przekazany dosłowny adres string, a filtr/efekt nie będzie występował w żadnym z adresów rooms, wówczas funkcja nie będzie mogła załadować tego filtra/efektu i zwróci -1.

Składnia:

fx_create(filter_or_effect_name);

Argument Typ Opis
filter_or_effect_name String The name of the filter/effect to create (get them from All Filter/Effect Types).

Zwroty:

FX Struct or -1 (on failure)

Przykład:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Powyższy kod tworzy nowy FX struct z efektu "_filter_tintfilter", który jest filtrem "Colour Tint" znajdującym się w edytorze Room. Przypisuje on parametrowi "g_TintCol" wartość, która jest kolorem tinty, a ponieważ jest to wewnętrzna strona vec4, przyjmuje tablicę zawierającą cztery wartości (odpowiadające wartościom czerwonej, zielonej, niebieskiej i alfa). Następnie do warstwy FX struct zostaje zastosowana warstwa FX dla odcienia.
