layer_sprite_get_yscale

Za pomocą tej funkcji można uzyskać aktualny mnożnik skali wzdłuż osi y dla elementu sprite. Podajesz identyfikator elementu sprite (który otrzymujesz podczas tworzenia elementu sprite przy użyciu layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość rzeczywistą dla skali y, gdzie 1 oznacza brak skalowania.

Składnia:

layer_sprite_get_yscale(sprite_element_id);

ArgumentTyp Opis
sprite_element_id The unique ID value of the sprite element to get the information from

Zwroty:

Przykład:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_yscale(spr_id) != 1
{
    layer_sprite_yscale(spr_id, 1);
}

Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do pobrania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten zostanie następnie użyty do sprawdzenia wartości skali yscale i jeśli będzie ona różna od 1, zostanie ustawiona na 1.
