warstwa_sprite_change

Za pomocą tej funkcji można zmienić zasób sprite przypisany do danego elementu sprite na warstwie. Podajesz identyfikator elementu sprite (który otrzymujesz podczas tworzenia elementu sprite za pomocą funkcji layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a następnie podajesz indeks sprite. Element asset zostanie zmieniony. Należy pamiętać, że jeśli strona sprite ma podobrazy, to one również będą animowane (można to kontrolować za pomocą przycisków layer_sprite_index() i layer_sprite_speed() ). Zauważ, że możesz przypisać wartość -1 jako nowy indeks sprite, a wtedy nie zostanie wyświetlony żaden element sprite, choć element ten będzie nadal istniał i będzie można go później ponownie zmienić.

Składnia:

layer_sprite_change(sprite_element_id, sprite_index)

Argument Typ Opis
sprite_element_id Identyfikator elementu sprite The unique ID value of the sprite element to change
sprite_index Aktywa Sprite The new sprite index to use

Zwroty:

N/A

Przykład:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
    layer_sprite_change(s, spr_nighttime);
}

Powyższy kod sprawdza indeks sprite elementu o identyfikatorze zapisanym w zmiennej globalnej global.Asset_sprite i jeśli nie jest to "spr_Nighttime", to do elementu przypisywany jest adres sprite.
