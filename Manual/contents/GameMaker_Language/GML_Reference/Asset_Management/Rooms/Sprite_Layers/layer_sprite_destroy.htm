warstwa_sprite_zniszczenie

Ta funkcja zniszczy podany element sprite. Podajesz identyfikator sprite (który otrzymujesz podczas tworzenia sprite za pomocą layer_sprite_create() lub gdy użyjesz identyfikatora warstwy wraz z layer_get_sprite_id()), a ta funkcja go usunie. Zauważ, że nie usuwa to warstwy, tylko sprite z niej, a jeśli sprite jest elementem dodanym w edytorze room, to następnym razem, gdy opuścisz room, a następnie wrócisz, sprite zostanie ponownie utworzony. Jeśli jednak warstwa room jest trwała, warstwa sprite zostanie usunięta, chyba że ponownie wyłączysz trwałość warstwy room.

Składnia:

layer_sprite_destroy(sprite_element_id)

ArgumentTyp Opis
sprite_element_id The unique ID value of the sprite to be destroyed

Zwroty:

Przykład:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
    layer_sprite_destroy(lay_id);
}

Powyższy kod sprawdza warstwę "Asset_Trees", czy podany element sprite istnieje, a jeśli tak, to zostaje on zniszczony (ale nie warstwa).
