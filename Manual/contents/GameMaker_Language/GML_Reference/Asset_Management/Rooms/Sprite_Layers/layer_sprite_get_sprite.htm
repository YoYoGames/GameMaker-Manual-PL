layer_sprite_get_sprite

Za pomocą tej funkcji można uzyskać bieżący indeks sprite elementu sprite. Podajesz identyfikator elementu sprite (który otrzymujesz podczas tworzenia elementu sprite przy użyciu layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość rzeczywistą, która reprezentuje wyświetlany indeks sprite. Jeśli element nie ma przypisanego adresu sprite, funkcja zwróci wartość -1.

Składnia:

layer_sprite_get_sprite(sprite_element_id);

ArgumentTyp Opis
sprite_element_id The unique ID value of the sprite element to get the information from

Zwroty:

Sprite Asset or -1

Przykład:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
    layer_sprite_change(spr_id, spr_Clouds);
}

Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do pobrania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten zostanie następnie użyty do sprawdzenia identyfikatora sprite przypisanego do elementu i ustawienia go na sprite "spr_Clouds", jeśli jeszcze nim nie jest.
