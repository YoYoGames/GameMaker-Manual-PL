warstwa_sprite_yscale

Za pomocą tej funkcji można zmienić, czy dany element sprite na warstwie ma być skalowany wzdłuż osi y, czy nie. Podajesz identyfikator elementu sprite (który otrzymujesz podczas tworzenia elementu sprite za pomocą funkcji layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a następnie ustawiasz wartość skali. Skala o wartości 1 oznacza brak skalowania (1:1), mniejsze wartości będą skalowane w dół (na przykład 0,5 spowoduje zmniejszenie o połowę wysokości użytego elementu sprite ), większe wartości będą skalowane w górę, a wartości ujemne spowodują odbicie lustrzane elementu sprite i jego przeskalowanie, chyba że użyta wartość wynosi dokładnie -1 (w takim przypadku użyty element sprite zostanie odbity lustrzanie od góry do dołu wokół swojej pozycji (0, 0) bez skalowania).

Składnia:

layer_sprite_yscale(sprite_element_id, yscale);

ArgumentTyp Opis
sprite_element_id The unique ID value of the sprite element to change
yscale The yscale value (default is 1)

Zwroty:

Przykład:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

Powyższy kod sprawdzi element sprite przypisany do warstwy, na której znajduje się instancja wykonująca kod, i jeśli jest on skalowany w dowolnym kierunku, ustawi zarówno skalę osi x, jak i skalę osi y na 1.
