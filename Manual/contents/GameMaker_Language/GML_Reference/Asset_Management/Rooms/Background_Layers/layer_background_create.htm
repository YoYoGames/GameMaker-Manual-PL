tworzenie_warstwy_tła

Za pomocą tej funkcji można przypisać zasób sprite do warstwy, która ma być użyta jako tło w projekcie. Podajesz ID warstwy (które otrzymasz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz indeks sprite (czyli nazwę strony sprite widoczną w przeglądarce Asset ), a zasób zostanie dodany do warstwy. Funkcja zwraca unikatową wartość ID elementu, którą można następnie wykorzystać w innych funkcjach warstwy dla tła.

Składnia:

layer_background_create(layer_id, sprite)

Argument Typ Opis
layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string)
sprite Aktywa Sprite The sprite index to be used

Zwroty:

Background Element ID

Przykład:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Powyższy kod tworzy nową warstwę, a następnie dodaje do niej nowy element tła, ustawiając stronę sprite jako obraz tła.

Wstecz: Warstwy tła
Następny: layer_background_destroy
