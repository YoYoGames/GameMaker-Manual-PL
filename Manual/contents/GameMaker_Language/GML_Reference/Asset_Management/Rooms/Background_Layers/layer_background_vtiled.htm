warstwa_tła_vtiled

Za pomocą tej funkcji można zmienić, czy dany element tła na warstwie ma być kafelkowany w pionie, czy nie. Podajesz identyfikator elementu tła (który otrzymujesz podczas tworzenia elementu tła za pomocą funkcji layer_background_create() lub po użyciu funkcji layer_background_get_id()), a następnie ustawia się wartość kafelkowania. Jeśli zostanie ustawiona wartość true, element będzie wyświetlany w kafelkach pionowo w dół strony room, a jeśli zostanie ustawiona wartość false, element nie będzie wyświetlany w kafelkach. Ta funkcja jest przeznaczona wyłącznie do projektów 2D i nie będzie działać poprawnie, jeśli używana jest projekcja z kamery 3D.

Składnia:

layer_background_vtiled(background_element_id, tiled);

Argument Typ Opis
background_element_id Identyfikator elementu tła The unique ID value of the background element to change
tiled Boolean The vertical tiling toggle, which can be true lub . false

Zwroty:

N/A

Przykład:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

Powyższy kod sprawdzi element tła przypisany do warstwy, na której znajduje się instancja wykonująca kod, i jeśli nie jest on ułożony w kafelki w żadnym kierunku, ustawi go na tile w pionie i w poziomie.
