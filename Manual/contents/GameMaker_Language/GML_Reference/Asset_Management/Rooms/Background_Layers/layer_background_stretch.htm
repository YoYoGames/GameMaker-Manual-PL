layer_background_stretch

Za pomocą tej funkcji można przełączać element tła sprite, aby rozciągał się w celu dopasowania do strony room lub pozostał w rozdzielczości 1:1. Podajesz identyfikator elementu tła (który otrzymujesz podczas tworzenia elementu tła za pomocą layer_background_create() lub po użyciu funkcji layer_background_get_id()), a następnie ustaw argument stretch na true lub false. Po ustawieniu na true element sprite zostanie rozciągnięty, aby dopasować się do elementu room (albo przeskalowany w górę, albo przeskalowany w dół, w zależności od rozmiarów elementów sprite i room), a po ustawieniu na false element sprite zostanie wyświetlony w swojej rodzimej rozdzielczości.

Składnia:

layer_background_stretch(background_element_id, stretch)

Argument Typ Opis
background_element_id Identyfikator elementu tła The unique ID value of the background element to change
stretch Boolean The stretch toggle, which can be true lub . false

Zwroty:

N/A

Przykład:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if layer_background_get_stretch(back)
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

Powyższy kod sprawdza, czy nie został naciśnięty przycisk myszy, a jeśli zostanie wykryty, przełącza rozciąganie elementu tła sprite przypisanego do warstwy, na której znajduje się instancja wykonująca kod.

Wstecz: Warstwy tła
