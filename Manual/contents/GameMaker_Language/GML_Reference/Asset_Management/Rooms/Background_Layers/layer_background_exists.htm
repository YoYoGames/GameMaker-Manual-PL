warstwa_tła_istnieje

Za pomocą tej funkcji można sprawdzić, czy na danej warstwie istnieje element tła. Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz identyfikator elementu tła (otrzymywany podczas tworzenia elementu tła za pomocą funkcji layer_background_create() lub po użyciu funkcji layer_background_get_id()), a funkcja zwróci albo true, jeśli element istnieje, albo false, jeśli nie.

UWAGA: Ta funkcja działa w zakresie bieżącego celu room - domyślnie jest to room, w którym funkcja jest wywoływana - który można ustawić za pomocą funkcji layer_set_target_room().

Składnia:

layer_background_exists(layer_id, background_element_id)

Argument Typ Opis
layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string)
background_element_id Identyfikator elementu tła The unique ID value of the background element to check

Zwroty:

Boolean

Przykład:

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
    layer_background_destroy(lay_id, global.TreesBackground);
}

Powyższy kod sprawdza warstwę "Tło_drzewa", czy dany element tła istnieje, a jeśli tak, to zostaje on zniszczony (ale nie warstwa).
