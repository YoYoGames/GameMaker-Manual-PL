warstwa_background_get_visible

Za pomocą tej funkcji można uzyskać widoczny stan elementu tła. Podajesz identyfikator elementu tła (który otrzymujesz podczas tworzenia elementu tła za pomocą layer_background_create() lub po użyciu funkcji layer_background_get_id()), a funkcja zwróci true, jeśli element jest aktualnie widoczny, lub false, jeśli nie jest. Należy pamiętać, że na tę wartość zwracaną nie ma wpływu to, czy warstwa, na której znajduje się element, jest widoczna, czy nie.

Składnia:

layer_background_get_visible(background_element_id);

ArgumentTyp Opis
background_element_id The unique ID value of the background element to get the information from

Zwroty:

Przykład:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_visible(back_id)
{
    layer_background_visible(back_id, false);
}

Powyższy kod pobierze identyfikator warstwy o nazwie "Tło_sky", a następnie użyje go do uzyskania identyfikatora elementu tła na tej warstwie. Identyfikator ten jest następnie używany do sprawdzania widoczności elementu i jeśli jest on widoczny, właściwość ta jest ustawiana na false.
