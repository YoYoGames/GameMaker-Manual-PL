czcionka_istnieje

Ta funkcja zwraca, czy istnieje strona font o podanym indeksie, czy nie. Można sprawdzić indeksy font zdefiniowane w przeglądarce Asset lub fonts dodane za pomocą funkcji takich jak font_add().

Składnia:

font_exists(ind);

ArgumentTyp Opis
ind Index of the font to check.

Zwroty:

Przykład:

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Spowoduje to ustawienie aktywnego rysunku font na fnt_Main, jeśli taki istnieje.
