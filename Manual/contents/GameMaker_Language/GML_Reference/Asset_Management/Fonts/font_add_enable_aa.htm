czcionka_add_enable_aa

Za pomocą tej funkcji można włączać lub wyłączać antyaliasing (AA) dla strony fonts dodanej przy użyciu font_add(). Funkcję tę należy wywołać przed dodaniem jakiejkolwiek strony fonts. Ustawienie jej na true włączy AA, a ustawienie na false wyłączy go. Domyślnie funkcja AA jest włączona. Należy pamiętać, że nie będzie to miało wpływu na stronę fonts, która została dodana przed wywołaniem tej funkcji, a funkcja musi być wywołana tylko raz, gdy dodawana jest strona font, a nie przy każdym losowaniu/kroku, w którym używana jest strona font.

Składnia:

font_add_enable_aa(enable);

ArgumentTyp Opis
enable Whether to enable (true) lub wyłączenie (false) AA w celu dodania fonts.

Zwroty:

Przykład:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

Powyższy kod sprawdza stan antyaliasingu dla strony fonts i jeśli nie jest on włączony, włącza go.
