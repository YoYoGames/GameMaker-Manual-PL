czcionka_zastąp_sprite

Za pomocą tej funkcji można użyć "paska sprite'ów" do zastąpienia wcześniej utworzonego sprite font asset , gdzie każdy podobraz byłby pojedynczym symbolem lub literą. Indeks jest numerem mapy UTF8 dla pierwszego znaku do użycia, a zakresy, których należy użyć, są takie same jak w przypadku zwykłego font, zaczynając od "!", ale pamiętaj, że powinieneś uwzględnić tylko to, co jest potrzebne, i że powinieneś użyć funkcji ord() aby uzyskać prawidłową wartość UTF8 dla początkowej litery.

Można także ustawić odstępy między literami font, aby były proporcjonalne (true) lub nie (false), przy czym odstępy proporcjonalne font są zależne od rzeczywistej szerokości liter (na przykład litera "i" zajmuje mniej miejsca room niż litera "w"), a odstępy nieproporcjonalne font są zależne od szerokości podobrazia, podobnie jak odstępy jednoprzestrzenne font. Na koniec można określić odstęp, jaki ma być pozostawiony między każdą literą podczas pisania. Może to być dowolna wartość całkowita, przy czym 0 oznacza brak odstępu (litery będą się "stykać", jeśli są proporcjonalne).

Jeśli funkcja zawiedzie, to zakończy działanie bez zastąpienia podanego adresu sprite font.

Warto również zauważyć, że w mapie znaków nie może być żadnych spacji, więc jeśli na przykład używasz znaków "." i "0", ale nie "/", musisz albo dołączyć "/", albo pozostawić pusty obrazek między dwoma używanymi znakami. Na poniższym rysunku przedstawiono typowy układ strony sprite font:

Składnia:

font_replace_sprite(ind, spr, first, prop, sep);

Argument Typ Opis
ind Czcionka Aktywa The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr Aktywa Sprite The sprite to add a font based on.
first Real The index of the first available character.
prop Boolean Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep Real The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

Zwroty:

N/A

Przykład:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Powyższy kod zastąpi stronę font asset zindeksowaną w zmiennej globalnej "Font" nową stroną font asset z zewnętrznego pliku graficznego.
