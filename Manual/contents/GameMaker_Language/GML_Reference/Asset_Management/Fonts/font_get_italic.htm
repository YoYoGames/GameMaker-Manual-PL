font_get_italic

Za pomocą tej funkcji można sprawdzić, czy dowolny adres font asset ma kursywę flag czy nie. Jeśli tak, funkcja zwróci true, w przeciwnym razie zwróci false.

Składnia:

font_get_italic(ind);

ArgumentTyp Opis
ind Index of the font to check.

Zwroty:

Przykład:

if font_get_italic(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Spowoduje to ustawienie aktywnego rysunku font na fnt_Main, jeśli we właściwościach font jest on ustawiony jako kursywny.
