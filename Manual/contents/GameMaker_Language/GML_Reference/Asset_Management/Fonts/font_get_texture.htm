font_get_texture

Ta funkcja zwraca specjalny wskaźnik na stronę font texture. Wartość tę można następnie wykorzystać w innych funkcjach rysowania, zwłaszcza w rysowaniu ogólnym przy użyciu prymitywów oraz w funkcjach Shadera. Więcej informacji o zwracanej stronie texture można uzyskać, korzystając z różnych funkcji texture_, które można znaleźć tutaj.

Składnia:

font_get_texture(font);

Argument Typ Opis
font Czcionka Aktywa The index of the font to use.

Zwroty:

Texture

Przykład:

tex = font_get_texture(fnt_Main);

Powyższy kod pobierze wskaźnik texture dla adresu font indeksowanego jako "fnt_Main".

Powrót: Czcionki
Następny: font_get_uvs
