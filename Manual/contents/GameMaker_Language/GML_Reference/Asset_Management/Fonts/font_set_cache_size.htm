font_set_cache_size

Podczas mieszania i kolorowania obrazów HTML5 nie może robić tego dynamicznie w taki sam sposób, w jaki robiłby to program wykonywalny. Dlatego GameMaker musi tymczasowo zapisać kopię mieszanych obrazów i wczytać je w razie potrzeby. je w razie potrzeby. Ta funkcja określa, ile mieszanych kopii danego font może zostać zbuforowanych, zanim stare zostaną nadpisane. Domyślną wartością jest 4.

UWAGA dla strony sprite fonts należy użyć funkcji równoważnej dla strony sprites, sprite_set_cache_size().

Składnia:

font_set_cache_size(ind, max);

ArgumentTyp Opis
ind The index of the font to change the cache size of.
max The maximum number of cached copies of the font that can be stored.

Zwroty:

Przykład:

font_set_cache_size(fnt_MainMenu, 2);

Spowoduje to ustawienie adresu font cache strony font indeksowanej w zmiennej "fnt_MainMenu" na 2 kopie.
