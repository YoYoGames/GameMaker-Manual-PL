Rollback Multiplayer

Rollback multiplayer w GameMaker pozwala na szybkie skonfigurowanie i stworzenie sieciowej gry wieloosobowej dla celu Opera GX.

Ta funkcjonalność wykorzystuje serwery GX do łączenia graczy i synchronizacji danych wejściowych, więc deweloper nie wymaga konfiguracji server, aby rozpocząć nową grę online.

Podczas gdy funkcjonalność online może być używana tylko z celem Opera GX, możesz użyć "Sync Test" do testowania gry wieloosobowej lokalnie, z losowym lub wyśmiewanym wejściem stosowanym przez innych graczy. Sync Test działa na platformach Windows i macOS oprócz Opery GX.

Rozpoczęcie pracy

Przeczytaj Create a Multiplayer Game i wykonaj kroki, aby stworzyć swój pierwszy projekt multiplayer.

Po stworzeniu swojej pierwszej gry wieloosobowej przeczytaj kolejne strony, aby lepiej zrozumieć system Rollback i wiedzieć, co możesz z nim zrobić:

  • System Rollback
  • Definiowanie danych wejściowych
  • Ograniczenia cofania
  • Zdarzenia związane z cofaniem
  • Preferencje cofania
  • Strony referencyjne:
    • Funkcje Rollback
    • Zmienne Rollback
