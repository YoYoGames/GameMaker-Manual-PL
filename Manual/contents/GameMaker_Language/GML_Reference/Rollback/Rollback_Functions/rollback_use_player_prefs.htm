cofnąć_użycie_gracza_prefs

Ta funkcja włącza preferencje gracza i pozwala ustawić domyślne preferencje dla wszystkich graczy.

Domyślne preferencje mogą być dowolnego typu. Jeśli nie określisz domyślnych preferencji, to wszyscy gracze muszą mieć ustawione preferencje przed rozpoczęciem gry.

Wywołanie tej funkcji wyłączy auto-start, więc musisz uruchomić grę ręcznie. Zobacz: Manual vs. Auto Start

Jeśli jest używana, funkcja ta musi być uruchomiona przed funkcją join/create.

Proszę przeczytać: Preferencje gracza

Składnia:

rollback_use_player_prefs([default]);

Argument Typ Opis
default Każdy OPTIONAL Domyślne preferencje do ustawienia dla wszystkich graczy.

Zwroty:

N/A

Przykład:

rollback_use_player_prefs({
character: 0,
kolor: c_white
});

Powyższy kod włącza preferencje gracza, oraz ustawia domyślne preferencje w struct.

Powrót: Funkcje cofania
Następny: rollback_set_player_prefs
