cofnij_define_player

Ta funkcja definiuje object, który powinien być używany dla graczy w multiplayerze. Jest to opcjonalne, a gdy zostanie użyte, system automatycznie stworzy instancje dla graczy, którzy się łączą i zniszczy instancje dla graczy, którzy się rozłączają.

Domyślnie tworzy to instancje graczy w warstwie "Instances", która musi istnieć w twojej grze room. Jeśli chcesz użyć innej warstwy, przekaż jej nazwę jako drugi argument.

Aby uzyskać informacje na temat tworzenia gry z lub bez tej funkcji, zobacz Definiowanie obiektu gracza. Jeśli jest używana, ta funkcja musi być uruchomiona przed funkcją join/create.

Każda instancja utworzona za pomocą tej funkcji otrzymuje zmienne instancji wymienione na tej stronie.

Składnia:

rollback_define_player(object, [layer_name]);

Argument Typ Opis
object Obiekt Aktywa The object that will be used for creating player instances.
layer_name String OPTIONAL Nazwa warstwy, na której będą tworzone instancje graczy. Domyślnie "Instancje".

Zwroty:

N/A

Przykład:

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

Powyższy kod definiuje gracza object, a następnie próbuje dołączyć do gry. Jeśli nie został dołączony, zamiast tego rozpoczyna nową grę, z maksymalnie 2 graczami i wyłączonym testem synchronizacji.
