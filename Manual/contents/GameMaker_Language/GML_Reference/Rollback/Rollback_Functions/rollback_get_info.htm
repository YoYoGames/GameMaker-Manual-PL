rollback_get_info

Funkcja zwraca adres struct zawierający informacje o graczu. Powinna być używana tylko podczas ręcznego zarządzania instancjami graczy. Jeśli używasz rollback_define_player()to każda instancja gracza automatycznie otrzyma te informacje jako zmienne.

Funkcja przyjmuje jako argument ID gracza i zwraca struct z następującymi zmiennymi:

Zmienna Typ Opis
player_avatar_url String URL wskazujący na obrazek awatara gracza
player_avatar_sprite Aktywa Sprite Sprite zawierający awatar gracza
player_name String Nazwa zawodnika
player_type String Informuje, czy gracz jest zalogowany w GX.games czy nie.

Dla użytkowników gościnnych będzie to "Gość", a dla zalogowanych "Użytkownik".
player_user_id String ID gracza w GX.games
player_prefs Każdy Preferencje ustawione dla gracza

Składnia:

rollback_get_info(player_id);

Argument Typ Opis
player_id Real The ID of the player for which information will be returned.
