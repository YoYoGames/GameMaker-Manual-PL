cofanie_używanie_ręcznego_startu

Domyślnie gra wieloosobowa rozpoczyna się automatycznie po dołączeniu wszystkich graczy.

Wywołanie tej funkcji wyłącza to zachowanie autostartu, więc gra czeka, aż uruchomisz ją ręcznie.

Jeśli jest używana, funkcja ta musi być uruchomiona przed funkcją join/create.

Składnia:

rollback_use_manual_start();

Zwroty:

N/A

Przykład:

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

Powyższy kod umożliwia ręczne uruchomienie, a następnie dołącza lub tworzy nową grę wieloosobową.
