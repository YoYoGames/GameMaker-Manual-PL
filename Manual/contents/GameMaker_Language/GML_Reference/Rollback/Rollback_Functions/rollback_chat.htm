cofanie_chatu

Funkcja ta służy do wysyłania wiadomości string do innych graczy.

Określasz adres string do wysłania, o maksymalnym rozmiarze 128 bajtów.

Gdy drugi argument nie jest określony, wiadomość jest wysyłana do wszystkich graczy, łącznie z nadawcą.

W drugim argumencie możesz podać ID gracza, który jest graczem, który otrzyma wiadomość.

Jak używać?

Ta funkcja ma być używana "lokalnie", a więc wynikałaby z lokalnych wejść, a nie wejść Rollback.

Zobacz przykład użycia na dole. Słuchaj odebranych wiadomości w zdarzeniu Rollback Event.

Składnia:

rollback_chat(message, [to]);

Argument Typ Opis
message String The message to send, as a string.
to Real OPTIONAL ID gracza, do którego zostanie wysłana wiadomość. Nieokreślenie tego argumentu lub określenie -1 powoduje wysłanie wiadomości do wszystkich graczy.

Zwroty:

N/A

Przykład:

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

Jeśli lokalnie zostanie wciśnięty klawisz enter, wartość keyboard_string jest wysyłana do wszystkich graczy. Ta strona string jest następnie czyszczona, aby można było wpisać nową wiadomość.

Spowoduje to wywołanie zdarzenia Rollback Event, gdy wiadomość zostanie odebrana przez wszystkich klientów (w tym nadawcę).

Zdarzenie cofnięcia

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
