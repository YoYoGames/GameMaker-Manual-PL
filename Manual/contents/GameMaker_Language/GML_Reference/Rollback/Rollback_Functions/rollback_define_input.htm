rollback_define_input

Ta funkcja służy do definiowania własnych wejść dla wszystkich graczy. Pobiera ona adres struct, który powinien zawierać wszystkie wejścia wraz z przypisanymi do nich stałymi wejściowymi.

Aby dowiedzieć się, jak przypisać wejścia niestandardowe, zobacz sekcję Sterowanie niestandardowe.

Jeśli funkcja ta jest używana, musi być uruchamiana przed funkcją join/create.

Składnia:

rollback_define_input(input_struct);

Argument Typ Opis
input_struct Struktura A struct containing input names as variables, and their assigned inputs as values.

Zwroty:

N/A

Przykład:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

Powyższy kod definiuje niestandardowe wejścia dla wszystkich graczy. Dwa ostatnie wejścia wykorzystują wiele kontrolek, umieszczając je w tablicy.

Rozszerzony przykład można znaleźć w temacie Definiowanie danych wejściowych.
