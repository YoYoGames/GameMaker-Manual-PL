rollback_get_input

Ta funkcja zwraca adres struct zawierający wartości wejściowe dla gracza. Przyjmuje opcjonalny argument określający identyfikator gracza, dla którego zostaną zwrócone dane wejściowe.

Gdy jest używana w instancji gracza, która została utworzona przez rollback_define_player()nie musisz podawać identyfikatora gracza, więc możesz wywołać tę funkcję bez żadnych argumentów.

Jeśli używasz go w innych instancjach lub w instancjach graczy utworzonych ręcznie, musisz podać identyfikator gracza.

Informacje o tym, jakie dane wejściowe są zwracane i jak można je zmienić, można znaleźć w temacie Definiowanie danych wejściowych.

Składnia:

rollback_get_input(player_id);

Argument Typ Opis
player_id Real OPTIONAL Identyfikator gracza, dla którego zostaną zwrócone dane wejściowe.

Zwroty:

Struct

Przykład:

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

Powyższy kod pobiera dane wejściowe struct z instancji gracza, która została automatycznie utworzona przez system Rollback. Na podstawie zwróconych wartości przesuwa instancję o 2 piksele w każdym kierunku.

Rozszerzony przykład można znaleźć w sekcji Tworzenie gry wieloosobowej.
