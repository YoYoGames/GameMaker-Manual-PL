rollback_use_random_input

Kiedy grasz w grę używając Sync Test, wszyscy "zdalni" gracze otrzymują losowe wartości dla swoich zdefiniowanych wejść, jako podstawowy sposób testowania. Możesz wyłączyć to zachowanie, wywołując tę funkcję z wartością false jako argumentem, i włączyć je ponownie, podając wartość true.

Składnia:

rollback_use_random_input(enabled);

Argument Typ Opis
enabled Boolean Specifies whether random input should be enabled (true) lub wyłączony(false). Domyślnie jest ona włączona.

Zwroty:

N/A

Przykład:

rollback_use_random_input(false);

Powyższy kod wyłącza losowe wejście podczas testu synchronizacji.

Powrót: Funkcje cofania
Następny: rollback_use_manual_start
