rollback_get_player_prefs

Funkcja zwraca preferencje ustawione dla podanego ID gracza.

Jeśli nie podano ID gracza, funkcja musi zostać wywołana wewnątrz instancji gracza, w takim przypadku zostaną zwrócone preferencje dla tego gracza.

Możesz również użyć zmiennej instancji player_prefs.

Ta funkcja zwraca ostatnie preferencje, które zostały zsynchronizowane dla danego gracza, więc ustawienie preferencji nie spowoduje natychmiastowej aktualizacji preferencji, które zwraca ta funkcja.

Proszę przeczytać: Preferencje gracza

Składnia:

rollback_get_player_prefs([player_id]);

Argument Typ Opis
player_id Real OPTIONAL ID zawodnika, dla którego zwracane są preferencje.

Zwroty:

Any

Przykład:

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

Powyższy kod pobiera preferencje ustawione dla bieżącego gracza i stosuje preferencję do zmiennej instancji.
