rollback_define_input_frame_delay

Funkcja ta określa stałe opóźnienie ramki dla wejść lokalnego odtwarzacza.

Domyślnie twoja gra będzie używać adaptacyjnego opóźnienia wejścia, które jest oparte na opóźnieniu między graczami. Będzie to działać dla prawie każdej gry, jednak niektóre specyficzne typy gier (np. gry walki) mogą preferować używanie stałego opóźnienia wejściowego, zamiast adaptacyjnego opóźnienia, które może się zmienić w dowolnym momencie.

Nie zaleca się zmieniać tego, chyba że wiesz dokładnie, co robisz. Możesz wywołać tę funkcję w dowolnym momencie przed i w trakcie gry wieloosobowej.

Podana wartość opóźnienia musi być wyrażona w liczbie klatek. Jeśli podasz wartość -1, gra powróci do używania opóźnienia adaptacyjnego. Maksymalne opóźnienie jakie możesz ustawić to 10 klatek.

Składnia:

rollback_define_input_frame_delay(frames);

Argument Typ Opis
frames Real The number of frames to set as the fixed input delay, or -1 aby przywrócić je do ustawień domyślnych.

Zwroty:

N/A

Przykład:

// 'Options' script
global.frame_delay = 5;

// obj_game: Create event
rollback_define_input_frame_delay(global.frame_delay);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

Możesz zdefiniować swoją wartość opóźnienia ramki jako zmienną globalną i użyć jej podczas wywoływania rollback_define_input_frame_delay() przed rozpoczęciem gry rollback.
