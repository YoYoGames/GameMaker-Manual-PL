cofnąć się do tyłu_start_gry

Ta funkcja wymusza rozpoczęcie bieżącej gry wieloosobowej, zanim wszyscy gracze do niej dołączą. Musisz najpierw stworzyć grę wieloosobową używając rollback_create_game().

System normalnie czeka, aż wszyscy gracze dołączą, zanim automatycznie rozpocznie grę. Ta funkcja może być użyta, aby rozpocząć grę wcześniej, zanim wszyscy gracze dołączą.

Możesz wywołać rollback_use_manual_start() aby Rollback czekał na tę funkcję nawet po dołączeniu wszystkich graczy.

Wywołanie tej funkcji spowoduje wywołanie zdarzenia Rollback Start.

Składnia:

rollback_start_game();

Zwroty:

N/A

Przykład:

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

Jeśli gra Rollback nie jest uruchomiona, a Enter jest wciśnięty, to rozpocznie to grę.

Dzięki temu gracz może rozpocząć grę ręcznie, zanim dołączą do niej wszyscy gracze.
