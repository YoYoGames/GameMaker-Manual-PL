rollback_display_events

Domyślnie po uruchomieniu gry w lewym górnym rogu wyświetlane są komunikaty debugowania, informujące o zdarzeniach w grze wieloosobowej. Możesz wyłączyć te komunikaty, wywołując tę funkcję z argumentem false, a następnie włączyć je ponownie, podając true.

Składnia:

rollback_display_events(enabled);

Argument Typ Opis
enabled Boolean Specifies whether event display should be enabled (true) lub wyłączona(false). Domyślnie jest ona włączona.

Zwroty:

N/A

Przykład:

rollback_display_events(false);

Powyższy kod wyłącza wyświetlanie zdarzeń.
