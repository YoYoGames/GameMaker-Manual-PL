cofnij_włączenie_gry

Ta funkcja próbuje dołączyć do gry. Musisz być na stronie URL, która zawiera ID wirtualnej strony room, aby dołączyć.

Jeśli room został dołączony, zwraca true, a jeśli nie dołączono żadnej gry, zwraca false.

Kiedy nowy gracz dołącza do gry, w której jesteś już obecny, uruchamiane jest zdarzenie Rollback Event. Więcej informacji na ten temat znajduje się w części Rollback Events.

Argument dry_run jest opcjonalny, a gdy ustawiony na true, pozwala uruchomić funkcję bez faktycznego dołączenia do gry. Jest to używane do sprawdzenia, czy można dołączyć do gry bez faktycznego dołączenia do niej.

Składnia:

rollback_join_game([dry_run]);

Argument Typ Opis
dry_run Boolean Disabled by default (false). Gdy jest włączona(true), funkcja nie dołącza do gry, a jedynie zwraca true, jeśli można do niej dołączyć, lub false w przeciwnym wypadku.

W ten sposób możesz użyć tej funkcji, aby sprawdzić tylko, czy gracz został zaproszony.

Zwroty:

Boolean

Przykład:

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_create_game(2, false);
}

Powyższy kod definiuje gracza object, a następnie próbuje dołączyć do gry. Jeśli nie został dołączony, tworzy zamiast niego nową grę, z maksymalnie 2 graczami i wyłączonym testem synchronizacji.

Rozbudowany przykład znajdziesz w rozdziale Tworzenie gry wieloosobowej.
