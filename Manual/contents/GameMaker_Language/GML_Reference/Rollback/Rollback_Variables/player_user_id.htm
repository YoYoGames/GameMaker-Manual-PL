identyfikator gracza_użytkownika

Ta zmienna przechowuje GX.games ID bieżącej instancji gracza, do użycia przy bezpośrednich połączeniach API z serwerem.

Ta zmienna istnieje tylko w instancjach graczy utworzonych przez rollback_define_player()i jest inna dla każdego gracza.

Gracze gościnni będą mieli przypisaną do tej zmiennej pustą string.

Składnia:

player_user_id;

Zwroty:

String

Powrót: Zmienne cofania
Następny: player_prefs
