typ_gracza

Ta zmienna określa, czy instancja gracza jest zalogowanym użytkownikiem GX.games, czy gościem. Może zwrócić jeden z poniższych ciągów znaków:

String Opis
"Guest" Gracz nie jest zalogowany w GX.games
"User" Gracz jest zalogowany w GX.games

Użytkownicy gościnni otrzymują zastępcze awatary i nazwy.

Ta zmienna istnieje tylko w instancjach graczy utworzonych przez rollback_define_player()i jest inna dla każdego gracza.

Składnia:

player_type;

Zwroty:

String

Powrót: Zmienne cofania
Następny: player_user_id
