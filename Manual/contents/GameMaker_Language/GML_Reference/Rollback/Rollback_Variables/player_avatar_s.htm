gracz_avatar_sprite

Ta zmienna przechowuje adres sprite aktualnej instancji gracza.

Ta zmienna istnieje tylko w instancjach graczy utworzonych przez rollback_define_player()i jest inna dla każdego gracza.

Odtwarzacze gościnne będą wyświetlane w miejscu sprite.

Składnia:

player_avatar_sprite;

Zwroty:

Sprite Asset

Przykład:

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

Wyświetlany jest awatar instancji gracza uruchamiającego kod, w obszarze 64x64, 256px nad instancją.
