nazwa_gracza

Ta zmienna przechowuje nazwę bieżącej instancji gracza.

Ta zmienna istnieje tylko w instancjach graczy utworzonych przez rollback_define_player()i jest inna dla każdego gracza.

Gracze gościnni otrzymają nazwę zastępczą.

Składnia:

player_name;

Zwroty:

String

Przykład:

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

Rysuje nazwę gracza 100px nad jego instancją, z wyrównaniem do środka.
