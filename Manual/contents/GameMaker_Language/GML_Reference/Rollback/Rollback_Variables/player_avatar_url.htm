player_avatar_url

Ta zmienna przechowuje adres URL do awatara gracza, który można załadować za pomocą sprite_add().

Można użyć strony player_avatar_sprite, na której jest już załadowana strona sprite.

Ta zmienna istnieje tylko w instancjach graczy utworzonych przez rollback_define_player()i jest inna dla każdego gracza.

Odtwarzacze gościnne będą wyświetlane w miejscu sprite.

Składnia:

player_avatar_url;

Zwroty:

String
