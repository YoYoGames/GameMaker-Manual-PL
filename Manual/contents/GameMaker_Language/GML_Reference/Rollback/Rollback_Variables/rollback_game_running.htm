cofanie_gry_bieżącej

Ta zmienna globalna określa, czy gra jest aktualnie uruchomiona. Jeśli jest to prawda, oznacza to, że gra została uruchomiona i jest w toku. W pozostałych przypadkach wartość ta będzie równa false.

Składnia:

rollback_game_running

Zwroty:

Boolean

Przykład:

if (!rollback_game_running) return;

x += move_x;
y += move_y;

Powyższy kod zostałby uruchomiony w zdarzeniu Step gracza object. Jeśli gra nie jest uruchomiona, spowoduje wyjście ze zdarzenia, więc nie będzie można sterować graczem, dopóki gra nie zostanie uruchomiona.
