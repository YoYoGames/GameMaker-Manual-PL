cofnij_potwierdzoną_ramkę

Ta zmienna globalna przechowuje numer ostatniej klatki, w której wejścia dla wszystkich graczy zostały zsynchronizowane i potwierdzone.

Można to porównać z adresem rollback_current_frame, aby sprawdzić, czy bieżąca ramka ma potwierdzone wejścia dla wszystkich graczy.

Można także użyć funkcji rollback_sync_on_frame(), aby sprawdzić, czy bieżąca ramka jest zsynchronizowana.

Składnia:

rollback_confirmed_frame

Zwroty:

Real

Przykład:

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

Ten kod sprawdza, czy bieżąca ramka ma potwierdzone dane wejściowe od wszystkich graczy, sprawdzając, czy wartość bieżącej ramki jest równa wartości ramki potwierdzonej.

Jeśli jest to prawda, uruchamia funkcję sprawdzającą, czy został spełniony warunek zakończenia gry, dzięki czemu może zakończyć grę.

Robi się to, aby uniknąć zakończenia gry w stanie opartym na przewidywaniach, i czeka na w pełni zsynchronizowaną ramkę przed próbą sprawdzenia warunków zakończenia gry.
