phy_col_normal_y

Ta zmienna tylko do odczytu zwraca składową y normalnej kolizyjnej odpowiadającej wartości phy_collision_y wartości tablicy. Dla każdego punktu styczności istnieje powiązana normalna styczności (zazwyczaj jest to ta sama normalna dla wszystkich punktów styczności w zderzeniu). Normalna kontaktu to wektor jednostkowy, który wskazuje od jednego punktu kolizji do drugiego. Można jej użyć na przykład do obliczenia prawidłowego kierunku "pchania" w celu rozwiązania kolizji.

UWAGA: Ta zmienna jest dostępna tylko w zdarzeniu kolizji instancji z włączoną fizyką.

Składnia:

phy_col_normal_y;

Zwroty:

(single precision floating point value, or undefined if the instance is not physics enabled)

Przykład:

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris)
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

Powyższy kod wykorzystuje normalną kontaktową do ustawienia kierunku ruchu dla instancji utworzonej w zdarzeniu kolizji między dwiema instancjami z włączoną fizyką.
