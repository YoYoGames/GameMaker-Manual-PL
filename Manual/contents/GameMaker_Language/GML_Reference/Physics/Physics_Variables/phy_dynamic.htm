phy_dynamiczny

Instancja dynamiczna to taka, która jest w pełni symulowana w świecie fizyki. Ta zmienna tylko do odczytu zwraca true, jeśli sprawdzana instancja jest w pełni symulowana, lub false, jeśli nie jest.

Składnia:

phy_dynamic;

Zwroty:

(or undefined if the instance is not physics enabled)

Przykład:

if other.phy_dynamic
{
    with (other)
    {
        var dir;
        dir = point_direction(x, y, other.x, other.y);
        physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
    }
}

Powyższy kod tworzy nową instancję, a następnie definiuje ją jako "pocisk" w świecie fizyki, po czym nadaje jej potężny impuls wzdłuż osi y.
