phy_kinematic

Ta zmienna tylko do odczytu zwraca true, jeśli instancja jest sklasyfikowana jako kinematyczna object, lub false, jeśli nie jest. Instancja kinematyczna to taka, która ma nieskończoną masę (gęstość równą 0), ale może się poruszać. Zatem aby uczynić instancję kinematyczną, należy najpierw utworzyć instancję statyczną, a następnie ustawić jedną lub więcej zmiennych instancji związanych z ruchem (np: phy_speed_x, phy_speed_y, lub phy_angular_velocity)

Składnia:

phy_kinematic;

Zwroty:

(or undefined if the instance is not physics enabled)

Przykład:

if !phy_kinematic
{
    phy_speed_x = 5;
}

Powyższy kod sprawdza, czy instancja jest kinematyczna, a jeśli nie, ustawia prędkość poziomą na 5.
