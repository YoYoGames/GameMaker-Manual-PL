phy_sleeping

Ta zmienna tylko do odczytu zwraca, czy instancja jest obecnie "śpiąca" (true) czy nie (false), Instancja "śpiąca" to taka, która nie jest aktywnie zaangażowana w żadną fizyczną symulację. GameMaker usypia stronę objects, by zaoszczędzić cykle symulacji, gdy instancja jest w stanie spoczynku i nie wchodzi w kolizję z inną instancją.

Składnia:

phy_sleeping;

Zwroty:

(or undefined if the instance is not physics enabled)

Przykład:

if phy_sleeping
{
    instance_destroy();
}

Powyższy kod sprawdza, czy strona object jest aktywnie symulowana, a jeśli nie jest, to jest niszczona.
