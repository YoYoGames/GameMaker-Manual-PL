phy_bullet

Ta zmienna określa, czy instancja porusza się z bardzo dużą prędkością (na przykład pocisk) czy nie. Domyślną wartością jest false, ale ustawienie jej na true mówi GameMakerowi, że instancja będzie poruszać się z tak dużą prędkością, że będzie wymagać bardziej kosztownego wykrywania kolizji, aby upewnić się, że nie przejdzie przez inne instancje niezauważona.

Składnia:

phy_bullet;

Zwroty:

(or undefined if the instance is not physics enabled)

Przykład:

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

Powyższy kod tworzy nową instancję, a następnie definiuje ją jako "pocisk" w świecie fizyki, po czym nadaje jej potężny impuls wzdłuż osi y.
