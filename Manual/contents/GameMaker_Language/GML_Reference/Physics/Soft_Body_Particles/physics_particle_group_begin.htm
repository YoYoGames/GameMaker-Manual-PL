physics_particle_group_begin

Za pomocą tej funkcji można utworzyć grupę cząstek w witrynie room. Grupa cząstek to taka, w której cząstki potrzebne do utworzenia określonej formy są tworzone wszystkie naraz jako grupa, co pozwala tworzyć symulowane ciała miękkie o różnych właściwościach, określanych przez używaną stronę flags. Flagi" są wartościami zwracanymi przez kombinację stałych cząsteczek (szczegóły na ten temat można znaleźć w funkcji physics_particle_create() ), natomiast "groupFlags" to kombinacja następujących stałych grupowych (lub 0 dla braku flags):

Fizyka Grupa cząstek Flag Stała
Stała Opis
phy_particle_group_flag_solid Stała grupa cząstek stałych zapobiega umieszczaniu w niej innych elementów. W przypadku przedostania się do środka, grupa cząstek stałych wypycha uszkodzony element z powrotem na swoją powierzchnię, dzięki czemu grupa cząstek stałych posiada szczególnie silną siłę odpychania.
phy_particle_group_flag_rigid Sztywne grupy cząstek to takie, których kształt nie zmienia się, nawet gdy zderzają się z innymi elementami.

Ta grupa flags wykorzystuje maskowanie bitów do utworzenia końcowej wartości wyjściowej, która jest następnie sprawdzana w celu ustawienia różnych podstawowych właściwości grupy, oprócz tych ustawionych przez cząstkę flags. Można nie używać żadnej z tych stałych, jednej lub obu, stosując maskowanie bitowe lub "|", aby zamaskować odpowiednie bity (jak pokazano w poniższym przykładzie).

Oprócz adresu flags można także ustawić położenie w witrynie room, w którym ma zostać utworzona grupa cząstek (podane położenie będzie środkiem masy gotowej grupy), początkową prędkość poziomą i pionową oraz prędkość kątową grupy, co umożliwia tworzenie grup poruszających się i obracających.

Można także ustawić początkowy kolor i alfa wszystkich cząstek w grupie, a wartości te będą używane podczas rysowania cząstek za pomocą funkcji physics_particle_draw(). Następnie ustawia się siłę spójności cząstek w grupie, czyli inaczej siłę, z jaką cząstki się do siebie "przyklejają". Siła ta może mieć wartość z przedziału od 0 do 1, gdzie 0 oznacza brak spójności, a 1 - maksymalną spójność.

Na koniec musisz podać kategorię dla swojej grupy cząstek. Ta wartość jest dowolną liczbą całkowitą, którą nadajesz każdej grupie cząstek, która ma mieć podobne właściwości, i która może być użyta do późniejszego ustawienia strony flags dla tej konkretnej kategorii, a także do wielu innych rzeczy.

UWAGA: Dla kategorii można użyć dowolnej wartości całkowitej z wyjątkiem 0, która jest zarezerwowana przez symulację i służy do wybierania wszystkich kategorii w innych funkcjach (np. w rysowaniu).

Należy zauważyć, że wywołanie tej funkcji nie powoduje utworzenia grupy w fizyce room. Funkcja ta po prostu rozpoczyna proces i ustawia podstawowe właściwości cząstek dla grupy. Aby faktycznie utworzyć grupę cząstek, należy nadać jej formę (patrz przykład poniżej), a następnie wywołać funkcję physics_particle_group_end(), po czym zostanie ona utworzona.

UWAGA: Typ cząstek flags, który ustawisz, będzie miał także wpływ na spójność grupy cząstek, np. jeśli na stronie flag zaznaczysz, że cząstki są typu wodnego (na przykład), to gdy grupa zderzy się z innym elementem, rozpadnie się, natomiast jeśli na stronie flag zaznaczysz, że cząstki są typu sprężynowego, to przy zderzeniu zachowają spójność.

Składnia:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Argument Typ Opis
flags Fizyka Cząstka Flag Stała(s) The flags to set on the particle.
groupflags Fizyka Grupa cząstek Flag Stała(s) The group flags to set on the particle group (see table at the top).
x Real The x position to create the particle group at.
y Real The y position to create the particle group at.
ang Real The angle (in degrees) to rotate the group to on create.
xv Real The initial horizontal velocity.
yv Real The initial vertical velocity.
ang_velocity Real The initial angular velocity.
col Kolor The base colour to use for the particle group.
alpha Real The base alpha to use for the particle group.
strength Real Set the cohesion strength between particles in the group.
category Real The user defined category to which the group belongs.

Zwroty:

N/A

Przykład:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Powyższy kod przechowuje w zmiennych adres flags dla typu cząstki oraz właściwości grupy cząstek, a następnie wykorzystuje je do utworzenia okrągłej grupy cząstek o promieniu 100px w miejscu, w którym znajduje się mysz.
