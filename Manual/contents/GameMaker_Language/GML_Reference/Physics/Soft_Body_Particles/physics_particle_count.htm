physics_particle_count

Ta funkcja zwraca liczbę cząstek aktywnych w fizyce room.

Składnia:

physics_particle_count()

Zwroty:

Przykład:

if (physics_particle_count() < physics_particle_get_max_count())
{
    physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)
}

Powyższy kod sprawdzi, czy na stronie room znajduje się mniej niż maksymalna dozwolona liczba cząstek, a jeśli tak, to utworzy jeszcze jedną.
