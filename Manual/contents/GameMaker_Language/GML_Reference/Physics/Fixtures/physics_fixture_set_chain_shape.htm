physics_fixture_set_chain_shape

Ta funkcja definiuje kształt uchwytu "łańcuchowego". Uchwyt łańcuchowy składa się z pewnej liczby punktów, które są następnie połączone ze sobą za pomocą kształtów krawędziowych, aby je połączyć. Sama funkcja pobiera indeks (ID) elementu i może określić, czy element ma być loop czy nie. Ustawienie na true połączy ostatni punkt z pierwszym za pomocą krawędzi, natomiast ustawienie na false nie. Zauważ, że ta funkcja przygotowuje uchwyt do przyjęcia punktów wymaganych do utworzenia łańcucha, które powinny być dodane po wywołaniu tej funkcji za pomocą physics_fixture_add_point()podobnie jak podczas budowania umocowania wielokąta.

Podczas tworzenia łańcucha musisz nadać mu co najmniej dwa punkty, ale nie jesteś ograniczony liczbą kolejnych punktów, które możesz dodać, aby utworzyć gotowy element.

Składnia:

physics_fixture_set_chain_shape(fixture, loop)

ArgumentTyp Opis
fixture The index of the fixture
loop Whether to close the chain (true) lub nie (false).

Zwroty:

Przykład:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(fix, xx, yy);
    xx += 50 + random(150);
    yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

Powyższy kod utworzy linię opraw "krawędziowych" o różnych wysokościach na długości strony room.
