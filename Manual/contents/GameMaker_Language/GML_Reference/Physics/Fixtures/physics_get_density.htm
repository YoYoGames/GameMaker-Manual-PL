physics_get_density

Kiedy wiążesz uchwyt z instancją używając physics_fixture_bind() to zwraca "id" dla powiązanego elementu. Możesz użyć tego id, aby uzyskać wartość gęstości powiązanego elementu(nie "bazowego") w dowolnym momencie przy użyciu tej funkcji.

Składnia:

physics_get_density(fixture)

ArgumentTyp Opis
fixture the id of the bound fixture

Zwroty:

Przykład:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

Powyższy kod pobiera wartość gęstości prądu dla powiązanych właściwości fizycznych instancji, a następnie ustawia je na inną wartość.
