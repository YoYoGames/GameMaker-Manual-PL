przerwa

Polecenie break jest używane do przedwczesnego zakończenia a for, repeat, while lub do / until loop jakiegoś rodzaju, lub do nakazania instrukcji switch że instrukcja ma się zakończyć w tym miejscu, lub do przedwczesnego zakończenia with połączenie. Aby uzyskać bardziej dogłębne wyjaśnienie, jak można użyć tej funkcji w poszczególnych okolicznościach, należy zapoznać się z poszczególnymi stronami poświęconymi tym funkcjom. Uwaga: jeśli break zostanie użyty poza którymkolwiek z tych kontekstów, spowoduje to wyświetlenie błędu.

Poniżej przedstawiono kilka podstawowych przykładów użycia strony break, której składnia jest bardzo prosta:

break;

break w pętli for:

for (var i = 0; i < 10; i += 1)
{
    if (array[i] == 234)
    {
        break;
    }
}

num = i;

break w pętli repeat:

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

break w pętli while:

var i = 0;
while (!place_free(x, y))
{
    x = random(room_width);
    y = random(room_height);
    if (i > 50)
    {
        break;
    }
    else
    {
        i+=1;
    }
}

break w pętli do / until:

var _id = noone;
do
{
    _id = list[| 0];
    if instance_exists(_id)
    {
        break;
    }
    ds_list_delete(list, 0);
}
until (ds_list_empty(list));
target = _id;

break w przypadku korzystania z witryny with:

var count = 0;
with (obj_Enemy)
{
    count++;
    if (count > 10)
    {
        break;
    }
    hp = 100;
}

break na stronie switch:

switch (keyboard_key)
{
    case vk_left:
    case ord("A"):
        x -= 4;
    break;

    case vk_right:
    case ord("D"):
        x += 4;
    break;

    case vk_up:
    case ord("W"):
        y -= 4;
    break;

    case vk_down:
    case ord("S"):
        y += 4;
    break;
}
