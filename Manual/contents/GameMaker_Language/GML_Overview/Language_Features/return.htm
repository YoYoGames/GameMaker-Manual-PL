return

Polecenie return ma następującą składnię:

return (<expression>)

Instrukcji return używa się tylko w funkcjach i metodach skryptu. Służy ona do zwracania wartości z funkcji, która ma być użyta w dalszym kodzie lub wywołaniu funkcji. Należy zauważyć, że wykonanie funkcji kończy się na instrukcji return , co oznacza, że kod, który pojawi się po wywołaniu return, nie zostanie uruchomiony. Poniżej znajduje się krótki przykład script o nazwie sqr_calc, który oblicza kwadrat dowolnej przekazanej mu wartości i zawiera funkcję wyłapującą błędy w przypadku, gdy przekazany mu parametr nie jest liczbą rzeczywistą:

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if !is_real(value)
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

Aby wywołać funkcję z wnętrza kodu, wystarczy użyć jej w taki sam sposób, jak w przypadku wywoływania funkcji runtime - to znaczy napisać nazwę funkcji z wartościami parametrów w nawiasach. Tak więc powyższa funkcja zostałaby wywołana w następujący sposób:

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

W tym przypadku zmienna val będzie miała wartość 0 (ponieważ zmienna amount nie jest liczbą rzeczywistą) lub sumę dla value * value.
