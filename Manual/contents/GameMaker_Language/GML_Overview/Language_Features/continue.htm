kontynuuj

Podstawowa składnia instrukcji continue jest następująca:

continue;

Jeśli zostanie użyty wewnątrz instrukcji, która tworzy loop (for, repeat, while lub do / until), spowoduje natychmiastowe zakończenie bieżącej iteracji i przeskoczenie z powrotem na początek loop, rozpoczynając nową iterację i pomijając kod, który pojawił się po continue w obrębie loop. Można go także użyć wewnątrz instrukcji with gdzie spowoduje, że kod zostanie przeskoczony do następnej instancji i uruchomiony ponownie. Zauważ, że użycie continue poza jednym z tych kontekstów spowoduje wystąpienie błędu.

Poniżej przedstawiono przykład użycia w pętli for:

var _val = 0;

for (var i = 0; i < 10; i += 1)
{
    if (val_array[i] <= 0)
    {
        continue;
    }
    _val += val_array[i];
}

draw_text(32, 32, "Positive Values Total = " + string(_val));

Poniżej przedstawiono przykład użycia w pętli while:

var file = file_text_open_read("Game_Data.txt");
var _num = 0;

while (!file_text_eof(file))
{
    var _s = file_text_readln(file);
    if (_s == "")
    {
        continue;
    }
    str[num++] = _s;
}

file_text_close(file);

Poniżej przedstawiono przykład użycia w pętli do / until:

do
{
    var _x = random(room_width);
    var _y = random(room_height);

    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }

    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}
until (instance_count(obj_Enemy) >= 10);

Poniżej zamieszczono przykład zastosowania w pętli repeat:

repeat(10)
{   
    var _x = random(room_width);
    var _y = random(room_height);

    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }

    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}

Na koniec przykład zastosowania w oświadczeniu with:

with (obj_Enemy_Parent)
{
    if (object_index == obj_Enemy_InDestructible)
    {
        continue;
    }

    hp -= 100;

    if (hp <= 0)
    {
        instance_destroy();
    }
}
