async_load

Ta zmienna ma zasięg globalny i jest używana do przechowywania mapy DS, gdy jest używana w zdarzeniach asynchronicznych, oraz -1 we wszystkich innych przypadkach. Rzeczywista zawartość mapy DS zależy od typu wywołania zwrotnego zdarzenia asynchronicznego, a także od funkcji użytej do wywołania zdarzenia, dlatego szczegółowe informacje na temat wszystkich możliwych zawartości mapy DS można znaleźć na stronach poświęconych tym zdarzeniom.

Składnia:

async_load;

Zwroty:

DS Map ID

Przykład rozszerzony:

sprite =  sprite_add("site.com/path/image.png", 16, true, true, 0, 0);

Powyższy kod zostałby wywołany w zdarzeniu w celu załadowania strony sprite z zewnętrznego adresu URL. Następnie wywołałby zdarzenie asynchroniczne Image Loaded, w którym nastąpiłoby przetworzenie mapy async_load:

if ds_map_find_value(async_load, "id") == sprite
{
    if (ds_map_find_value(async_load, "status") >= 0)
    {
        sprite_index = sprite;
    }
}

Powyższy kod najpierw sprawdzi identyfikator mapy async_load, a następnie sprawdzi stan wywołania zwrotnego. Jeśli wartość jest większa lub równa 0 (co sygnalizuje sukces), wynik wywołania zwrotnego zostanie użyty do ustawienia indeksu instancji sprite na nowo załadowany obraz.
