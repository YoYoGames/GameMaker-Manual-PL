życie

Ta zmienna ma zasięg globalny i jest używana do przechowywania wartości liczbowej, która jest zwykle używana do określania życia gracza. Ta zmienna jest przeznaczona tylko do obsługi starszych projektów z poprzednich wersji GameMakera i powinna nie może być używana w nowych projektach ponieważ w przyszłości może zostać wycofana z użycia.

Składnia:

lives;

Zwroty:

Real (single precision floating point value)

Przykład

if (lives < 5)
{
    lives += 1;
}

Powyższy kod sprawdza zmienną lives i jeśli jest ona mniejsza niż 5, dodawana jest do niej 1.
