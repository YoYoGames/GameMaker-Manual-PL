wynik

Ta zmienna ma zasięg globalny i jest używana do przechowywania wartości liczbowej, która jest zwykle używana do określania wyniku gracza. Ta zmienna jest przeznaczona tylko do obsługi starszych projektów z poprzednich wersji GameMakera i powinna nie może być używana w nowych projektach ponieważ w przyszłości może zostać wycofana z użycia.

Składnia:

score;

Zwroty:

Real (single precision floating point value)

Przykład

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

Powyższy kod sprawdza indeks object instancji wykonującej kod za pomocą a switcha następnie dodaje różne wartości do zmiennej score w zależności od tego, jaki jest adres object.

Powrót: Zmienne globalne
Następny: lives
