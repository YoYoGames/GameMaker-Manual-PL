argument

Ta zmienna przechowuje tablicę, która jest używana wraz ze zmienną tylko do odczytu argument_count w funkcjach i metodach skryptu. Każda pozycja tablicy przechowuje wartość wejściową dla funkcji i jest przeznaczona specjalnie do stosowania w funkcjach o zmiennych argumentach.

Należy zauważyć, że istnieje również szereg niezależnych zmiennych globalnych, które mogą być używane w funkcjach definiowanych przez użytkownika do odwoływania się do różnych argumentów wejściowych: argument0, argument1, argument2, itd... aż do argument15.

UWAGA: Nie jest to już ściśle wymagane i służy bardziej do obsługi starszych funkcji niż do czegokolwiek innego. Wszystkie funkcje zdefiniowane przez użytkownika będą przyjmować zmienne argumenty, a każdy argument przekazany do funkcji, który nie jest częścią nazwanych danych wejściowych, będzie domyślnie zainicjowany jako niezdefiniowany, co można sprawdzić za pomocą funkcji is_undefined().

Składnia:

argument[n]
argument1
argument2
...
argument15

Zwroty:

Variable (can be of any data type supplied to the function)

Przykład:

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

Powyższy kod używa zmiennej argument_count wraz z tablicą argument do inicjalizacji zmiennych i jest przykładem na to, jak można pozwolić, by funkcja zdefiniowana przez użytkownika przyjmowała zmienną liczbę argumentów.
