GML Code

GameMaker Language (zwany również po prostu GML) jest zastrzeżonym językiem skryptowym GameMaker. Język ten jest skonstruowany tak, aby umożliwić użytkownikom tworzenie gier w intuicyjny i elastyczny sposób, oferując jednocześnie wszystkie możliwości każdego innego dużego języka programowania. Jest on również podstawą dla GML Visual i może być używany w połączeniu z nim, jeśli jest to wymagane.

Część podręcznika dotycząca GML Code podzielona jest na następujące rozdziały:

  • Przegląd GML Code
  • Odnośnik do GML Code
