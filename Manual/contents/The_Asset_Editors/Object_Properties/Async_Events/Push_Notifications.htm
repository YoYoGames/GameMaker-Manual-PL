Powiadomienia push

Zdarzenie powiadomienia Push jest wyzwalane przez wywołanie zwrotne z powiadomienia push w systemie operacyjnym urządzenia, pochodzącego z lokalnego lub zdalnego źródła. Generuje ono mapę DS Map, która jest przypisana wyłącznie do tego zdarzenia i jest przechowywana w zmiennej specjalnej async_load.

To zdarzenie jest wykorzystywane wyłącznie przez stronę extensions (np. Firebase Cloud Messaging extension), dlatego instrukcje dotyczące korzystania z tego zdarzenia można znaleźć w dokumentacji rozszerzenia.
