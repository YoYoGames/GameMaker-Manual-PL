YoYo Compiler

YoYo Compiler (YYC) jest specjalnym kompilatorem do tworzenia pakietów wykonywalnych, które używają kodu maszynowego zamiast kodu interpretowanego i runnera (co jest tym, czego używa kompilacja GameMaker VM). Skompilowany kod jest szybszy do uruchomienia, ale zajmuje więcej czasu, aby skompilować plik wykonywalny.

Licencja

Twoja licencja GameMaker może, ale nie musi obejmować kompilatora YoYo. Aby się tego dowiedzieć, zobacz stronę Centrum Pomocy Która subskrypcja jest dla mnie odpowiednia?

Konfiguracja zestawu SDK

Aby kompilator YoYo mógł skompilować się dla platformy docelowej, musisz mieć zainstalowane odpowiednie SDK.

Aby uzyskać ogólny przegląd na ten temat, zobacz następującą stronę Help Center: GameMaker Required SDKs.

Informacje na temat pakietów SDK wymaganych dla poszczególnych platform można znaleźć również w Centrum Pomocy:

  • Opera GX
  • Windows
  • macOS
  • Ubuntu
  • Android
  • iOS (w tym iPadOS)
  • tvOS
